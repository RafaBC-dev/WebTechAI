# Nueva placa de desarrollo STM32U575 con conectores de display y microSD

**Fecha:** 2026-04-13
**Categoría:** embebidos
**Tags:** microcontroladores, embebidos, placas-de-desarrollo
**Título original:** $15 STM32U575 development board features FPC display connector, microSD card slot, two 48-pin GPIO headers

---

## Introducción

Maker Go ha lanzado una nueva placa de desarrollo STM32U575, una opción compacta para proyectos de interfaz de usuario (UI) que aprovecha el procesador ultra-bajo consumo STM32U5.

## ¿Qué es?

La placa STM32U575 es una placa de desarrollo que utiliza el procesador STM32U575VGT6, un microcontrolador ultra-bajo consumo con núcleo Cortex-M33 a 160 MHz, 786 KB de SRAM y 1024 KB de flash. También cuenta con un conector de display FPC compatible con pantallas de 1,47 pulgadas y 2,00 pulgadas, un slot de microSD, y dos conectores de 48 pines para GPIO.

## ¿Cómo funciona?

La placa STM32U575 se puede programar mediante una variedad de métodos, incluyendo SWD, USB y puerto serial. El procesador STM32U575VGT6 es compatible con el software STM32Cube IDE y HAL libraries de STMicroelectronics. Además, aunque no hay una versión oficial de MicroPython para esta placa, el núcleo de MicroPython v1.27 ya soporta el STM32U5, lo que significa que se puede utilizar con esta placa.

## ¿Por qué importa?

Esta placa es relevante para proyectos de interfaz de usuario (UI) compactos que requieren una potencia de procesamiento baja y una interfaz de usuario intuitiva. También es una opción atractiva para desarrolladores que buscan una placa de desarrollo fiable y fácil de usar.

## Consejo técnico

Si estás interesado en utilizar la placa STM32U575, te recomendamos descargar el software STM32Cube IDE y HAL libraries de STMicroelectronics para empezar a desarrollar tus proyectos.

```bash
stm32cubeide --help
```
