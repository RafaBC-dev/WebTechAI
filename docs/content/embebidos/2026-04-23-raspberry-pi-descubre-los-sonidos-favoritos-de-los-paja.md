# Raspberry Pi descubre los sonidos favoritos de los pájaros cantores

**Fecha:** 2026-04-23
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** Raspberry Pi reveals songbirds’ favourite sounds

---

## Introducción

Un equipo de investigadores ha desarrollado un sistema para estudiar las preferencias auditivas de los pájaros cantores utilizando Raspberry Pi. Este proyecto revoluciona la forma en que se estudian las habilidades auditivas de los pájaros.

## ¿Qué es?

El proyecto utiliza un Raspberry Pi para crear un sistema de reproducción de sonidos que permite a los pájaros cantores elegir sus sonidos favoritos. Los investigadores han diseñado un sistema que utiliza sensores infrarrojos para detectar el movimiento de los pájaros y reproducir el sonido correspondiente.

## ¿Cómo funciona?

El sistema utiliza un Raspberry Pi Zero 2 W como cerebro principal y se conecta a un HiFiBerry DAC+ Zero y a 8-ohm Dayton Audio full-range speakers. Los sensores infrarrojos Adafruit IR Break Beam detectan el movimiento de los pájaros y reproducen el sonido correspondiente.

## ¿Por qué importa?

Este proyecto es importante porque permite a los investigadores estudiar las preferencias auditivas de los pájaros cantores de manera más accesible y económica. También puede ayudar a entender mejor las habilidades auditivas de los pájaros y cómo pueden ser afectadas por el entorno.

## Consejo técnico

Si deseas crear un sistema similar, puedes utilizar el código proporcionado en el artículo para configurar tu Raspberry Pi y conectar los sensores infrarrojos. Recuerda que debes utilizar un HiFiBerry DAC+ Zero y 8-ohm Dayton Audio full-range speakers para reproducir los sonidos de manera adecuada.

```bash
sudo apt-get install rpi-hifiberry-dac
```
