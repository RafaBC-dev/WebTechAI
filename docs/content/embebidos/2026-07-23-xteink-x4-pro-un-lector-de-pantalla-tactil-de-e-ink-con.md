# Xteink X4 Pro: un lector de pantalla táctil de E Ink con firmware abierto

**Fecha:** 2026-07-23
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y linux

## Introducción

La empresa Xteink ha lanzado el Xteink X4 Pro, un lector de pantalla táctil de E Ink con una pantalla de 4,3 pulgadas, conectividad WiFi y Bluetooth, y una batería de 1100 mAh. Lo que hace que este dispositivo sea especial es que su firmware es abierto y puede ser personalizado.

## ¿Qué es?

El Xteink X4 Pro es un lector de pantalla táctil de E Ink que cuenta con una pantalla de 4,3 pulgadas, una tarjeta de memoria microSD de 16 GB, conectividad WiFi y Bluetooth, y una batería de 1100 mAh. Su firmware es abierto, lo que significa que los usuarios pueden personalizarlo y agregar nuevas funcionalidades.

## ¿Cómo funciona?

El Xteink X4 Pro utiliza el microcontrolador ESP32-S3, que proporciona conectividad WiFi y Bluetooth. La pantalla táctil de E Ink permite a los usuarios interactuar con el dispositivo de manera intuitiva. El firmware abierto permite a los usuarios personalizar el dispositivo y agregar nuevas funcionalidades.

## ¿Por qué importa?

El Xteink X4 Pro es importante porque ofrece una opción de lector de pantalla táctil de E Ink con firmware abierto, lo que significa que los usuarios pueden personalizarlo y agregar nuevas funcionalidades. Esto puede ser especialmente útil para aquellos que buscan una experiencia de lectura personalizada.

## Consejo técnico

Si estás interesado en personalizar el firmware del Xteink X4 Pro, puedes descargar el firmware CrossPoint Reader y seguir las instrucciones en el sitio web de Xteink para instalarlo.

```bash
git clone https://github.com/CrossPoint-Reader/CrossPoint-Reader.git
```
