# Pantalla flexible AMOLED de 6,67 pulgadas compatible con Raspberry Pi

## Introducción

DFRobot ha lanzado una pantalla flexible AMOLED de 6,67 pulgadas con una resolución de 2400x1080, diseñada para ser fácilmente compatible con computadoras de placa única como Raspberry Pi y LattePanda.

## ¿Qué es?

La pantalla flexible AMOLED es un tipo de pantalla que se puede doblar y estira sin romperse, lo que la hace ideal para aplicaciones en las que el espacio y la forma son limitados. Esta pantalla en particular tiene una resolución de 2400x1080 y puede mostrar 16,7 millones de colores.

## ¿Cómo funciona?

La pantalla utiliza una placa de control MIPI-HDMI que convierte el señal de HDMI en un formato que puede ser entendido por la pantalla AMOLED. La placa de control también suministra la energía necesaria para la pantalla. La pantalla misma tiene una conexión de alta densidad que es difícil de trabajar con directamente, por lo que se utiliza una placa de adaptación para facilitar la conexión.

## ¿Por qué importa?

Esta pantalla flexible AMOLED es importante porque puede ser utilizada en una variedad de aplicaciones, incluyendo pantallas de robot, consolas curvas para automoción, dispositivos de vestimenta y sistemas de interfaz de usuario industrial.

## Consejo técnico

Si deseas utilizar esta pantalla con tu Raspberry Pi, asegúrate de utilizar la placa de control MIPI-HDMI y la placa de adaptación para evitar problemas de conexión.
