# Raspberry Pi Connect: Acceso remoto a dispositivos Raspberry Pi

## Introducción

Raspberry Pi Connect es una herramienta que permite acceder a dispositivos Raspberry Pi desde cualquier lugar del mundo, directamente desde un navegador web. Con las últimas actualizaciones, se han agregado nuevas funcionalidades que mejoran la experiencia de usuario, especialmente para equipos que utilizan Connect para administrar flotas de dispositivos.

## ¿Qué es?

Raspberry Pi Connect es una plataforma que permite acceder y administrar dispositivos Raspberry Pi de manera remota. Permite a los administradores acceder a los dispositivos, ver su estado y realizar tareas de mantenimiento desde cualquier lugar.

## ¿Cómo funciona?

Para acceder a un dispositivo Raspberry Pi, solo es necesario tener una cuenta de Raspberry Pi ID y conectarse a la plataforma de Raspberry Pi Connect. Una vez dentro, se puede ver una lista de dispositivos disponibles y seleccionar el que se desee acceder. También se pueden aplicar etiquetas a los dispositivos para facilitar la búsqueda y la organización.

## ¿Por qué importa?

Raspberry Pi Connect es importante porque permite a los administradores acceder a los dispositivos de manera remota, lo que facilita la gestión y el mantenimiento de las flotas de dispositivos. También permite aplicar seguridad adicional mediante la autenticación en dos factores, lo que reduce el riesgo de acceso no autorizado.

## Consejo técnico

Si estás utilizando Raspberry Pi Connect, te recomendamos aplicar etiquetas a tus dispositivos para facilitar la búsqueda y la organización. Puedes hacer esto seleccionando el dispositivo y pulsando en la pestaña 'Configuración' para agregar o eliminar etiquetas.
