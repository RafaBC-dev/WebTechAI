# Makerfabs MaTouch ESP32-P4: una pantalla táctil con 4G LTE y Ethernet

**Fecha:** 2026-06-27
**Categoría:** embebidos
**Tags:** esp32, robotica, embebidos
**Título original:** Makerfabs MaTouch ESP32-P4 10.1-inch HMI display features 4G LTE and Ethernet connectivity, a 2MP camera

---

## Introducción

Makerfabs ha lanzado el MaTouch ESP32-P4, una pantalla táctil de 10.1 pulgadas con conectividad 4G LTE y Ethernet, perfecta para aplicaciones de HMI y edge AI.

## ¿Qué es?

El MaTouch ESP32-P4 es una plataforma de desarrollo para pantallas táctiles basada en el SoC ESP32-P4NRW32, que cuenta con un procesador dual-core RISC-V, una GPU y un VPU. La pantalla tiene una resolución de 1280x800 y soporta hasta 1920x1080 en la cámara de 2MP.

## ¿Cómo funciona?

La pantalla se conecta a Internet mediante el módulo 4G LTE SIMCom SIM7670G y tiene una tarjeta SD para almacenamiento adicional. La plataforma también incluye un módulo Wi-Fi 6 y Bluetooth 5.x LE, así como un módulo de expansión con GPIO y interfaces de comunicación. El sistema operativo es Linux, y se puede programar con herramientas como Python y Arduino.

## ¿Por qué importa?

El MaTouch ESP32-P4 es ideal para aplicaciones de HMI y edge AI, como monitoreo de sistemas, automatización de procesos y análisis de datos en tiempo real. También se puede utilizar para crear dispositivos de IoT que requieren conectividad 4G LTE y Ethernet.

## Consejo técnico

Si estás interesado en desarrollar aplicaciones con el MaTouch ESP32-P4, recomiendo investigar la documentación oficial de Makerfabs y la comunidad de desarrolladores de ESP32 para obtener más información y recursos.
