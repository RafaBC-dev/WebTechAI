# Neuro N6: un desarrollo de AI vision modular y compacto para proyectos de robótica

## Introducción

Ohm Lab ha lanzado el Neuro N6, un desarrollo de AI vision compacto y modular que permite a los desarrolladores crear proyectos de robótica avanzados con facilidad. Este módulo está diseñado para trabajar con cámaras de diferentes tipos y ofrece una gran cantidad de recursos para implementar inteligencia artificial en aplicaciones de robótica.

## ¿Qué es?

El Neuro N6 es un módulo de desarrollo de AI vision que se basa en el microcontrolador STMicro STM32N6, que cuenta con un procesador ARM Cortex-M55 de 32 bits y un acelerador de neuronas de 600 GOPS. Esto permite a los desarrolladores crear aplicaciones de inteligencia artificial avanzadas con una gran eficiencia energética.

## ¿Cómo funciona?

El Neuro N6 se puede programar utilizando el IDE de Arduino, lo que facilita la creación de proyectos de robótica para desarrolladores que ya están familiarizados con este entorno. Además, el módulo cuenta con una gran cantidad de conectores y interfaces para conectar cámaras, sensores y otros dispositivos periféricos.

## ¿Por qué importa?

El Neuro N6 importa porque permite a los desarrolladores crear proyectos de robótica avanzados con facilidad, lo que puede ser útil en una variedad de aplicaciones, desde la automatización industrial hasta la robótica de servicio.

## Consejo técnico

Si estás interesado en probar el Neuro N6, te recomendamos descargar el firmware y las bibliotecas de ejemplo de GitHub y probarlas en tu proyecto de robótica. También es recomendable utilizar la aplicación NeuroStudio para visualizar la salida de la cámara y la inteligencia artificial en tiempo real.

```bash
git clone https://github.com/OhmLab/NeuroN6.git
```
