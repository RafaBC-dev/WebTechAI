## Introducción

La comunidad de desarrolladores de Raspberry Pi ha lanzado un conjunto de ejemplos de código en Rust para la nueva placa Raspberry Pi Pico 2, aprovechando la potencia de la arquitectura RP2350 y la plataforma Embassy async.

## ¿Qué es?

Rust Embassy es un framework de programación en Rust que permite crear aplicaciones concurrentes y asíncronas de manera sencilla. El conjunto de ejemplos para Raspberry Pi Pico 2 incluye proyectos que demuestran la capacidad de la placa para ejecutar código en Rust y utilizar la plataforma Embassy async.

## ¿Cómo funciona?

Los ejemplos de código en Rust Embassy para Raspberry Pi Pico 2 se centran en demostrar la capacidad de la placa para ejecutar aplicaciones concurrentes y asíncronas. Utilizan la plataforma Embassy async para crear programas que puedan manejar múltiples tareas simultáneamente, aprovechando la arquitectura RP2350 de la placa.

## ¿Por qué importa?

Este conjunto de ejemplos es importante porque demuestra la potencia de la placa Raspberry Pi Pico 2 y la plataforma Embassy async para crear aplicaciones concurrentes y asíncronas. Los desarrolladores pueden utilizar estos ejemplos como base para crear sus propias aplicaciones innovadoras y aprovechar al máximo la capacidad de la placa.

## Consejo técnico

Si deseas empezar a trabajar con Rust Embassy y Raspberry Pi Pico 2, crea un proyecto nuevo en Rust y utiliza el comando `cargo new` para crear un nuevo proyecto básico. Luego, instala la biblioteca Embassy async utilizando el comando `cargo add embassy`.

```bash
cargo new, cargo add embassy
```
