# Generador de señal RF abierto y de alta gama a un precio asequible

**Fecha:** 2026-03-09
**Categoría:** embebidos
**Tags:** RF, Generador de señal, Atek Midas, SCPI, Laboratorio
**Título original:** DSG-22.6 GHz is a $1,590 open-source RF signal generator based on Atek Midas’s custom ICs (Crowdfunding)

## Introducción

La empresa turca Atek Midas ha lanzado un generador de señal RF de alta gama y abierto que ofrece una gama de frecuencia de 0.15 GHz a 22.6 GHz y puede ser controlado a través de una interfaz web o mediante comandos SCPI.

## ¿Qué es?

El DSG-22.6 GHz es un generador de señal RF de alta gama y abierto que se diseñó para proporcionar una gama de frecuencia de 0.15 GHz a 22.6 GHz. Ofrece una resolución de ajuste de 1 Hz y puede generar una potencia de salida de -15 dBm a +20 dBm.

## ¿Cómo funciona?

El generador de señal RF utiliza una IC personalizada de Atek Midas y cuenta con una pantalla táctil capacitiva y una conexión USB Type-C para controlar el dispositivo. También puede ser controlado a través de una interfaz web o mediante comandos SCPI. El dispositivo cuenta con una velocidad de ajuste de menos de 100 μs y puede realizar barridos de frecuencia lineales y logarítmicos.

## ¿Por qué importa?

El DSG-22.6 GHz es importante porque ofrece una gama de frecuencia más amplia y una resolución de ajuste más alta que otros generadores de señal RF de alta gama, pero a un precio más asequible. Esto lo hace ideal para uso en laboratorios, pruebas de producción y experimentación con tecnologías de radiofrecuencia.

## Consejo técnico

Si estás interesado en utilizar el DSG-22.6 GHz, te recomiendo investigar sobre la interfaz web y los comandos SCPI para aprender a controlar el dispositivo y aprovechar todas sus funciones.

```bash
scpi
```
