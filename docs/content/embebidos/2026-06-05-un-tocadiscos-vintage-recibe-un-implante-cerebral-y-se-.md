# Un tocadiscos vintage recibe un implante cerebral y se integra con Home Assistant

**Fecha:** 2026-06-05
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** Vintage Turntable Gets Brain Transplant and Home Assistant Integration

---

## Introducción

Un entusiasta de la música en vinilo ha llevado a cabo un proyecto emocionante: convertir un tocadiscos vintage en un dispositivo inteligente con control automático gracias a un microcontrolador ESP32 y la integración con Home Assistant.

## ¿Qué es?

El proyecto consiste en reemplazar la electrónica de un tocadiscos vintage Sansusi P-L45 con un microcontrolador ESP32, lo que le permite controlar el movimiento del cabezal de lectura de manera precisa y automática. El objetivo es crear un dispositivo que pueda seguir la pista del disco de vinilo de manera lineal, sin necesidad de un mecanismo de brazo pivotante.

## ¿Cómo funciona?

El microcontrolador ESP32 se encarga de controlar el movimiento del cabezal de lectura, ajustando la velocidad para seguir la pista del disco de vinilo. Para lograr esto, se utiliza un mecanismo de seguimiento lineal, que requiere una precisión y control electrónicos precisos. El ESP32 se comunica con la placa base del tocadiscos a través de un nivelador de voltaje y un circuito de desbordamiento de hardware.

## ¿Por qué importa?

Este proyecto es importante porque permite a los entusiastas de la música en vinilo disfrutar de un dispositivo de alta calidad y precisión, con características de control automático y seguimiento lineal. Además, es un ejemplo de cómo se pueden utilizar tecnologías modernas para restaurar y mejorar dispositivos antiguos.

## Consejo técnico

Si deseas implementar un seguimiento lineal en tu propio proyecto de tocadiscos, puedes utilizar la documentación y los diseños de la placa base de ESP32 proporcionados en el blog de WeAreAllGeeks como punto de partida.

```bash
git clone https://github.com/WeAreAllGeeks/Sansusi-P-L45-ESP32.git
```
