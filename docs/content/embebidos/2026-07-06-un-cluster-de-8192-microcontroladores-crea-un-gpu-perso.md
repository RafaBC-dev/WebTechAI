# Un cluster de 8192 microcontroladores crea un GPU personalizado

**Fecha:** 2026-07-06
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, diseño-3d
**Título original:** Gluing 8192 MCUs Together to Make a GPU

---

## Introducción

Un ingeniero ha creado un sistema de GPU personalizado utilizando 8192 microcontroladores, cada uno con su propio LED RGB. Este proyecto es un ejemplo innovador de cómo se puede aprovechar la tecnología de los microcontroladores para crear sistemas complejos.

## ¿Qué es?

El proyecto consiste en un cluster de 8192 microcontroladores CH570, cada uno con un núcleo QingKe 32-bit RISC-V, que se comunican entre sí para crear un sistema de GPU personalizado. Cada microcontrolador tiene su propio LED RGB para crear una matriz de LEDs personalizable.

## ¿Cómo funciona?

Los microcontroladores se conectan a una backplane y se comunican entre sí a través de una red de comunicaciones. Cada microcontrolador tiene su propio controlador MCU, que se encarga de gestionar la comunicación y la programación de los demás microcontroladores. El sistema utiliza un PSU de 3 kW para suministrar energía a los microcontroladores.

## ¿Por qué importa?

Este proyecto es importante porque demuestra la posibilidad de crear sistemas complejos utilizando tecnología de microcontroladores. También puede tener aplicaciones en campos como la visualización de datos, la iluminación de espacios y la creación de efectos visuales.

## Consejo técnico

Si deseas crear un sistema similar, considera utilizar una backplane de alta densidad y un PSU de alta capacidad para suministrar energía a los microcontroladores. También es importante programar cuidadosamente el sistema para asegurarte de que todos los microcontroladores se comuniquen correctamente.
