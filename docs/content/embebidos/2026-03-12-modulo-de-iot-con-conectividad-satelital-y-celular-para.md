# Módulo de IoT con conectividad satelital y celular para aplicaciones de seguimiento

## Introducción

La empresa Blues Notecard ha lanzado un módulo de IoT que combina conectividad satelital y celular para aplicaciones de seguimiento en transporte, logística, energía y equipo comercial. Este módulo revoluciona la forma en que se conectan los dispositivos IoT en áreas remotas.

## ¿Qué es?

El Blues Notecard for Skylo es un módulo de IoT que ofrece conectividad satelital, celular, WiFi y GNSS (sistema de navegación por satélite) para aplicaciones de seguimiento. No requiere una suscripción satelital adicional ni hardware adicional. Se conecta por defecto a través de WiFi y se cambia a celular cuando no está dentro del rango. Sólo se recurre a la conectividad satelital de pago de Skylo cuando no hay acceso a WiFi ni celular.

## ¿Cómo funciona?

El módulo utiliza un módulo de comunicación satelital y celular Quectel BG95-S5, que ofrece conectividad LTE-M, NB-IoT, GPRS y Skylo. También cuenta con un módulo de WiFi Silicon Labs WFM200S y un sensor de aceleración LIS2DTW para detectar movimiento y apagar el GNSS cuando el dispositivo está inmóvil. El módulo se puede conectar a través de I2C, UART o USB y soporta MCUs de 1.8V o 3.3V.

## ¿Por qué importa?

Este módulo es importante porque ofrece una solución de conectividad satelital y celular para aplicaciones de seguimiento en áreas remotas, lo que permite a los dispositivos IoT permanecer conectados en la mayoría de las veces. Esto es especialmente relevante para industrias como la logística, la energía y el equipo comercial, donde la conectividad es esencial para la eficiencia y la seguridad.

## Consejo técnico

Si estás desarrollando una aplicación de seguimiento que requiere conectividad satelital y celular, considera utilizar el Blues Notecard for Skylo para aprovechar su conectividad múltiple y su capacidad para permanecer conectado en áreas remotas.
