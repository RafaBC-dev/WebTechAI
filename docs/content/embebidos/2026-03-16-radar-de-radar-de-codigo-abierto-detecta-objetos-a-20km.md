## Introducción

El proyecto AERIS-10 es un radar de código abierto que utiliza tecnología de radar de alta frecuencia para detectar objetos a larga distancia. Este sistema es capaz de rastrear múltiples objetos y ofrece una gran flexibilidad en términos de configuración y personalización.

## ¿Qué es?

El AERIS-10 es un sistema de radar de código abierto que utiliza una frecuencia de 10,5 GHz y una modulación de frecuencia lineal (LFM) para detectar objetos en un rango de hasta 20km. El sistema se basa en un FPGA AMD Artix-7 y utiliza una arquitectura de radar de alta frecuencia para proporcionar una gran precisión y velocidad.

## ¿Cómo funciona?

El AERIS-10 funciona mediante la generación de señales de radar que se envían hacia el objetivo y luego se reciben y procesan para determinar la posición y velocidad del objeto. El sistema utiliza una serie de componentes, incluyendo un DAC, un ADC, un procesador de señales y un sistema de control de potencia, para proporcionar una gran flexibilidad y precisión en la detección de objetos.

## ¿Por qué importa?

El AERIS-10 es importante porque ofrece una solución de radar de código abierto que puede ser utilizada en una variedad de aplicaciones, incluyendo la detección de objetos en el espacio aéreo, la navegación marítima y la seguridad en la industria. El sistema es capaz de proporcionar una gran precisión y velocidad en la detección de objetos, lo que lo hace ideal para aplicaciones que requieren una alta precisión y velocidad.
