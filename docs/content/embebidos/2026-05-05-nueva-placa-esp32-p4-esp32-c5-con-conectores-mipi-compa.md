# Nueva placa ESP32-P4 + ESP32-C5 con conectores MIPI compatibles con Raspberry Pi

## Introducción

La empresa Maker Go ha lanzado una nueva placa de desarrollo llamada ESP32P4C5 Core Board, que se basa en el procesador ESP32-P4 y el SoC ESP32-C5. Esta placa es ideal para proyectos de IoT y robótica, ya que cuenta con conectores MIPI compatibles con Raspberry Pi y una gran cantidad de recursos de hardware.

## ¿Qué es?

La ESP32P4C5 Core Board es una placa de desarrollo que combina el procesador ESP32-P4 con el SoC ESP32-C5. El ESP32-P4 es un procesador dual-core de 32 bits que cuenta con un CPU de alto rendimiento y un MCU de bajo consumo. El ESP32-C5 es un SoC dual-band que cuenta con WiFi 6, Bluetooth 5.0 LE y 802.15.4.

## ¿Cómo funciona?

La ESP32P4C5 Core Board cuenta con cuatro conectores MIPI, dos de los cuales son compatibles con Raspberry Pi. Estos conectores permiten conectar pantallas y cámaras a la placa. Además, la placa cuenta con tres puertos USB, uno para depuración y dos Type-C/A que comparten una conexión de datos USB 2.0. También cuenta con un micrófono MEMS, un conector de altavoz y dos headers de 34 pines para expansión.

## ¿Por qué importa?

La ESP32P4C5 Core Board es importante porque ofrece una gran cantidad de recursos de hardware para proyectos de IoT y robótica. La compatibilidad con Raspberry Pi hace que sea fácil conectar pantallas y cámaras a la placa, lo que la hace ideal para proyectos de visión artificial y reconocimiento de voz.
