# Puente de adquisición de datos industrial con Raspberry Pi CM0

## Introducción

La empresa EDATEC ha lanzado el ED-IPC1200, un puente de adquisición de datos industrial basado en el Raspberry Pi Compute Module Zero (CM0) que ofrece una amplia gama de interfaces I/O y conectividad para aplicaciones de automatización industrial y de borde.

## ¿Qué es?

El ED-IPC1200 es un dispositivo de adquisición de datos industrial que utiliza el procesador quad-core Cortex-A53 del Raspberry Pi CM0 para ofrecer una amplia gama de interfaces I/O, incluyendo RS-485, RS-232, cuatro entradas digitales y cuatro salidas digitales. También cuenta con conectividad Ethernet, Wi-Fi y Bluetooth, y puede ser alimentado con una corriente de 9 a 28V DC.

## ¿Cómo funciona?

El ED-IPC1200 utiliza el procesador del Raspberry Pi CM0 para ejecutar el sistema operativo Raspberry Pi OS Desktop/Lite y ofrece una variedad de interfaces I/O para la adquisición de datos y el control de equipos. También cuenta con un módulo de Wi-Fi y Bluetooth opcional y un módulo de 4G LTE para conectividad a Internet.

## ¿Por qué importa?

El ED-IPC1200 es importante para aplicaciones de automatización industrial y de borde, ya que ofrece una solución flexible y escalable para la adquisición de datos y el control de equipos. También es útil para proyectos de IoT y de automatización de procesos industriales.

## Consejo técnico

Si estás desarrollando un proyecto de automatización industrial, considera utilizar el ED-IPC1200 como puente de adquisición de datos y control de equipos. Puedes utilizar las herramientas de desarrollo de Raspberry Pi, como CODESYS y Node-RED, para crear aplicaciones de automatización personalizadas.

sudo apt-get install raspbian-os
