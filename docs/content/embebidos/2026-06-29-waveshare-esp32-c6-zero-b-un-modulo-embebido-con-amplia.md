## Introducción

Waveshare ha lanzado un nuevo módulo embebido llamado ESP32-C6-Zero-B, que ofrece una amplia gama de voltaje y una serie de características avanzadas. Este módulo es ideal para aplicaciones industriales y de IoT.

## ¿Qué es?

El ESP32-C6-Zero-B es un módulo embebido basado en el SoC ESP32-C6FH8 de Espressif Systems, que cuenta con un procesador RISC-V de 32 bits y una memoria de 8MB. Ofrece conectividad WiFi 6, Bluetooth 5.3 y una radio 802.15.4 para Thread y Zigbee.

## ¿Cómo funciona?

El módulo cuenta con una serie de conectores y pines que permiten la expansión de funciones, como la comunicación serial, I2C, SPI y ADC. También cuenta con un LED RGB y botones de reinicio y arranque.

## ¿Por qué importa?

Este módulo es importante debido a su amplia gama de voltaje, que va desde 5V hasta 36V, lo que lo hace ideal para aplicaciones industriales. También ofrece una serie de características avanzadas, como la conectividad WiFi 6 y Bluetooth 5.3.

## Consejo técnico

Si deseas utilizar el ESP32-C6-Zero-B con Arduino, puedes utilizar la librería oficial de Waveshare, que ofrece una serie de funciones y ejemplos para ayudarte a empezar.

```bash
sudo apt-get install libnewlib-dev
```
