## Introducción

Los entusiastas de la retrocomputación tienen una nueva herramienta para recrear sistemas antiguos: el picoZ80, un reemplazo de microprocesador Z80 basado en Raspberry Pi y ESP32.

## ¿Qué es?

El picoZ80 es un reemplazo de microprocesador Z80 que utiliza un microcontrolador Raspberry Pi RP2350B y un SoC ESP32 para proporcionar conectividad WiFi y Bluetooth. Se puede conectar directamente a sistemas antiguos que utilizaban un Z80.

## ¿Cómo funciona?

El picoZ80 utiliza las máquinas de estado de I/O programable (PIO) del microcontrolador Raspberry Pi RP2350B para reproducir las señales de la Z80 de manera precisa. El ESP32 se encarga de la conectividad WiFi y Bluetooth. El sistema se puede configurar mediante un archivo JSON que se almacena en una tarjeta microSD.

## ¿Por qué importa?

El picoZ80 es importante porque permite a los entusiastas de la retrocomputación recrear sistemas antiguos de manera precisa y fácil de usar. También proporciona conectividad WiFi y Bluetooth, lo que facilita la configuración y el uso de los sistemas.

## Consejo técnico

Si deseas crear tu propio picoZ80, asegúrate de utilizar la documentación oficial y las herramientas recomendadas, como KiCad para el diseño de la placa y la herramienta de configuración de la tarjeta microSD.
