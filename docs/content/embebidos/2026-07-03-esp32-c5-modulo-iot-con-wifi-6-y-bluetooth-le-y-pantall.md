# ESP32-C5: módulo IoT con WiFi 6 y Bluetooth LE y pantalla de 1,47 pulgadas

**Fecha:** 2026-07-03

## Introducción

Waveshare ha lanzado el ESP32-C5-LCD-1.47, un módulo IoT que combina WiFi 6 y Bluetooth LE con una pantalla de 1,47 pulgadas. Este módulo es ideal para proyectos de IoT que requieren una conexión a Internet segura y una interfaz visual

## ¿Qué es?

El ESP32-C5-LCD-1.47 es un módulo IoT basado en el chip ESP32-C5, que ofrece conectividad dual-banda WiFi 6 y Bluetooth LE. También cuenta con una pantalla de 1,47 pulgadas y 262K de colores, y tiene una memoria de 4MB de flash. El módulo también incluye una ranura para tarjeta microSD y dos conectores de 9 pines para expansión de GPIO

## ¿Cómo funciona?

El ESP32-C5-LCD-1.47 utiliza el chip ESP32-C5 como procesador principal, que tiene un núcleo de 32 bits RISC-V que alcanza velocidades de hasta 240 MHz. El módulo también cuenta con una memoria de 384 KB de SRAM y 320 KB de ROM. La pantalla de 1,47 pulgadas se controla mediante el driver ST7789. El módulo también incluye una antena integrada y un conector USB Type-C para alimentación, programación y datos

## ¿Por qué importa?

El ESP32-C5-LCD-1.47 es importante porque ofrece una conexión a Internet segura y una interfaz visual para proyectos de IoT. Esto lo hace ideal para aplicaciones como monitoreo de sensores, control de dispositivos y visualización de datos. El módulo también es compatible con la plataforma Arduino y ofrece una gran cantidad de recursos y ejemplos de código

## Consejo técnico

Si estás trabajando con el ESP32-C5-LCD-1.47, te recomendamos utilizar la biblioteca LVGL para crear interfaces de usuario personalizadas en la pantalla de 1,47 pulgadas. LVGL es una biblioteca de código abierto que ofrece una gran cantidad de características y ejemplos de código para crear interfaces de usuario en tiempo real
