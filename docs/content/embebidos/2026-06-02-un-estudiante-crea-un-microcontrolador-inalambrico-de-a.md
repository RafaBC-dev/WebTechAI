# Un estudiante crea un microcontrolador inalámbrico de alta calidad en su dormitorio

## Introducción

Nick Winans, un estudiante de primer año de universidad, ha creado un microcontrolador inalámbrico llamado nice!nano, que es compatible con Pro Micro y utiliza la tecnología nRF52840. Este proyecto demuestra la capacidad de los estudiantes para crear tecnología innovadora en sus propios entornos.

## ¿Qué es?

El nice!nano es un microcontrolador inalámbrico que utiliza la tecnología nRF52840 de Nordic Semiconductor. Es compatible con Pro Micro y ofrece una alta calidad y una gran cantidad de características. Puede ser utilizado para crear proyectos de robótica, automatización y otros dispositivos electrónicos.

## ¿Cómo funciona?

El nice!nano utiliza la tecnología nRF52840 para conectarse a dispositivos inalámbricos y enviar datos. La placa se diseñó utilizando KiCad y se creó utilizando la tecnología de fabricación en serie (PCB). El microcontrolador utiliza la biblioteca Adafruit para interactuar con el hardware y el software.

## ¿Por qué importa?

El nice!nano importa porque demuestra la capacidad de los estudiantes para crear tecnología innovadora en sus propios entornos. También muestra la importancia de la colaboración entre estudiantes y la comunidad de desarrollo de software para crear proyectos de alta calidad.

## Consejo técnico

Si deseas crear un proyecto de robótica o automatización, considera utilizar la tecnología nRF52840 y la biblioteca Adafruit. También es importante utilizar herramientas de diseño como KiCad para crear placa de circuito impreso (PCB) de alta calidad.
