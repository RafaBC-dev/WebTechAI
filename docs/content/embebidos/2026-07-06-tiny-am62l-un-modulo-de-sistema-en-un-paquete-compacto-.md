# Tiny AM62L: un módulo de sistema en un paquete compacto para aplicaciones industriales

**Fecha:** 2026-07-06
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** Tiny Calixto Systems AM62L STAMP SOM targets industrial automation, HMI applications, EV chargers

---

## Introducción

Calixto Systems ha lanzado el AM62L STAMP SOM, un módulo de sistema en un paquete compacto de 40x40mm que integra un procesador dual-core Arm Cortex-A53 de Texas Instruments. Este módulo está diseñado para aplicaciones industriales, como automatización, HMIs, cargadores de vehículos eléctricos y retail inteligente.

## ¿Qué es?

El AM62L STAMP SOM es un módulo de sistema que integra un procesador dual-core Arm Cortex-A53 de Texas Instruments, con hasta 2GB de memoria LPDDR4x y 8GB o 16GB de almacenamiento eMMC. También cuenta con interfaces de red, USB, SD, display y una variedad de entradas/salidas industriales.

## ¿Cómo funciona?

El módulo utiliza el procesador AM62Lx de Texas Instruments, que se ejecuta a una frecuencia de hasta 1,25 GHz. La memoria LPDDR4x y el almacenamiento eMMC permiten una alta velocidad de datos y una gran capacidad de almacenamiento. Las interfaces de red y USB permiten la comunicación con otros dispositivos, mientras que las entradas/salidas industriales permiten la conexión a sensores y actuadores.

## ¿Por qué importa?

El AM62L STAMP SOM es importante porque ofrece una solución compacta y eficiente para aplicaciones industriales, como automatización, HMIs y cargadores de vehículos eléctricos. También cuenta con características de seguridad, como cifrado AES y aceleración de RSA/ECC, lo que lo hace ideal para aplicaciones que requieren alta seguridad.

## Consejo técnico

Si estás buscando una solución para tu proyecto de automatización industrial, considera utilizar el AM62L STAMP SOM. Puedes conectar sensores y actuadores a través de las entradas/salidas industriales y utilizar la interfaz de red para comunicarte con otros dispositivos.
