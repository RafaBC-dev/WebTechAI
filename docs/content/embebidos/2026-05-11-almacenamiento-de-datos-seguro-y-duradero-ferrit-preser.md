# Almacenamiento de datos seguro y duradero: FERRIT preserva datos durante 200 años

**Fecha:** 2026-05-11
**Categoría:** embebidos
**Tags:** ia-local, robotica, embebidos, linux
**Título original:** Up to 256 MB FERRIT modular F-RAM storage device preserves critical data for up to 200 years

---

## Introducción

Machdyne ha presentado FERRIT, un dispositivo de almacenamiento de datos seguro y duradero que puede preservar datos durante hasta 200 años. Este dispositivo utiliza tecnología F-RAM y es ideal para la conservación de documentos críticos, registros históricos y claves criptográficas.

## ¿Qué es?

FERRIT es un dispositivo de almacenamiento de datos modular que utiliza tecnología F-RAM (memoria ferroeléctrica) para almacenar datos. Puede almacenar hasta 256 MB de datos y es capaz de soportar un número ilimitado de escrituras. Esto lo hace ideal para aplicaciones que requieren alta seguridad y durabilidad.

## ¿Cómo funciona?

FERRIT utiliza un controlador basado en el microcontrolador Raspberry Pi RP2040 y un bus SPI/QSPI para conectar los módulos de memoria. Cada módulo de memoria puede almacenar hasta 16 MB de datos y se puede conectar hasta 256 módulos para alcanzar una capacidad total de 256 MB.

## ¿Por qué importa?

FERRIT es importante porque ofrece una solución de almacenamiento de datos seguro y duradero para aplicaciones críticas. Puede preservar datos durante 200 años, lo que lo hace ideal para la conservación de documentos históricos y claves criptográficas.

## Consejo técnico

Si deseas implementar un sistema de almacenamiento de datos seguro y duradero en tu proyecto, considera utilizar la tecnología F-RAM y el controlador Raspberry Pi RP2040. Puedes encontrar más información sobre la implementación de FERRIT en el repositorio de GitHub de Machdyne.
