# El precio de Raspberry Pi 5 sube debido a la escasez de RAM

**Fecha:** 2026-04-03
**Categoría:** embebidos
**Tags:** microcontroladores, software y linux, diseño y hardware
**Título original:** The Raspberry Pi 4 With 3 GB RAM is No Joke

---

## Introducción

La escasez de RAM LPDDR4 y LPDDR5 ha llevado a Raspberry Pi a aumentar los precios de sus modelos con 4 o más GB de RAM. Esto afecta a los modelos Raspberry Pi 4 y 5, lo que puede hacer que sea difícil para los entusiastas de la electrónica acceder a estas platinas.

## ¿Qué es?

Raspberry Pi es una serie de platinas de desarrollo de código abierto diseñadas para proyectos de electrónica y robótica. Estas platinas están equipadas con procesadores ARM y memoria RAM, y se pueden programar utilizando lenguajes como Python o C++.

## ¿Cómo funciona?

La placa Raspberry Pi 4 utiliza dos chips de memoria LPDDR4 de 1.5 GB cada uno, lo que la hace más asequible que modelos con 4 o más GB de RAM. Esto se debe a que la escasez de RAM LPDDR4 y LPDDR5 ha llevado a una subida de precios en la industria.

## ¿Por qué importa?

La subida de precios de Raspberry Pi puede afectar a los proyectos de electrónica y robótica que dependen de estas platinas. Los entusiastas de la electrónica pueden tener que buscar alternativas más asequibles, como utilizar platinas con memoria LPDDR2 o microcontroladores como el ESP32.

## Consejo técnico

Si necesitas una placa de desarrollo con memoria RAM, considera utilizar un microcontrolador como el ESP32, que es más asequible y no está afectado por la escasez de RAM. Puedes utilizar la biblioteca de desarrollo de ESP32 para programar tu proyecto.

```bash
Para instalar la biblioteca de desarrollo de ESP32 en Linux, puedes utilizar el comando `sudo apt-get install esp32-devkit`.
```
