# Nueva placa de desarrollo ESP32-P4-Pi-VIEWE con Wi-Fi 6 y Bluetooth 5

## Introducción

VIEWE ha lanzado una nueva placa de desarrollo inspirada en la Raspberry Pi, que combina un microcontrolador ESP32-P4 con un chip ESP32-C6 para Wi-Fi 6 y Bluetooth 5. Esta placa ofrece una gran cantidad de conectividad y recursos para proyectos de robótica y IoT.

## ¿Qué es?

La ESP32-P4-Pi-VIEWE es una placa de desarrollo que combina un microcontrolador ESP32-P4 dual-core RISC-V con un chip ESP32-C6 para Wi-Fi 6 y Bluetooth 5. Ofrece 32MB de PSRAM y 16MB de NOR flash, así como una gran cantidad de conectividad y recursos para proyectos de robótica y IoT.

## ¿Cómo funciona?

La placa utiliza el microcontrolador ESP32-P4 para ejecutar aplicaciones y el chip ESP32-C6 para proporcionar Wi-Fi 6 y Bluetooth 5. La placa también cuenta con una gran cantidad de conectividad, incluyendo Ethernet, MIPI DSI y CSI, así como una tarjeta SD y un puerto USB 2.0.

## ¿Por qué importa?

Esta placa es importante para desarrolladores de robótica y IoT, ya que ofrece una gran cantidad de recursos y conectividad para proyectos complejos. También es una excelente opción para aquellos que buscan una placa de desarrollo más potente que la Raspberry Pi.

## Consejo técnico

Si estás trabajando con la ESP32-P4-Pi-VIEWE, asegúrate de utilizar el framework ESP-IDF para desarrollar aplicaciones y aprovechar al máximo los recursos de la placa.

```bash
esp-idf
```
