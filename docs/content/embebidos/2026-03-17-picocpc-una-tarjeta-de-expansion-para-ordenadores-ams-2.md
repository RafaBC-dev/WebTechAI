# PicoCPC: una tarjeta de expansión para ordenadores Amstrad CPC

**Tags:** robotica, diseño-3d, electronica, embebidos

## Introducción

Después de décadas de olvido, los ordenadores Amstrad CPC están experimentando un renacimiento gracias a la creación de hardware y software personalizado. La comunidad de entusiastas ha desarrollado una serie de proyectos para mejorar las capacidades de estos ordenadores, incluyendo la tarjeta de expansión PicoCPC.

## ¿Qué es?

La PicoCPC es una tarjeta de expansión para ordenadores Amstrad CPC que se basa en un microcontrolador Raspberry Pi Pico 2. Ofrece una variedad de características, incluyendo memoria extendida, emulación de controlador de disco flexible, carga de software de cartucho y audio de seis voces.

## ¿Cómo funciona?

La PicoCPC se conecta a la CPC mediante una interfaz de expansión estándar y utiliza el microcontrolador Raspberry Pi Pico 2 para proporcionar las características adicionales. La tarjeta emula un controlador de disco flexible, lo que permite a los usuarios acceder a software de cartucho y almacenar datos en una tarjeta SD.

## ¿Por qué importa?

La PicoCPC es importante porque permite a los entusiastas de la CPC aprovechar al máximo la potencia de sus ordenadores. Con la capacidad de expandir la memoria hasta 1024 kB, emular un controlador de disco flexible y cargar software de cartucho, la PicoCPC es una herramienta esencial para cualquier aficionado a la CPC.

## Consejo técnico

Si deseas crear tu propia tarjeta de expansión para la CPC, considera utilizar el microcontrolador Raspberry Pi Pico 2 y la biblioteca de software de la PicoCPC como punto de partida. Puedes personalizar la tarjeta para agregar características adicionales y mejorar la experiencia de usuario.
