# Ampisu: un laboratorio de poder compacto y aislado para el trabajo de campo

## Introducción

La tecnología de los suministros de poder USB-C se ha vuelto más accesible y asequible, pero la mayoría de los dispositivos disponibles tienen limitaciones. El Ampisu es un laboratorio de poder compacto y aislado diseñado para el trabajo de campo y la prueba de sistemas embebidos.

## ¿Qué es?

El Ampisu es un laboratorio de poder compacto y aislado que se puede llevar en un bolsillo. Está diseñado para el trabajo de campo y la prueba de sistemas embebidos. Tiene tres salidas de poder aisladas y puede adaptarse a diferentes configuraciones de poder.

## ¿Cómo funciona?

El Ampisu utiliza un microcontrolador Raspberry Pi RP2040 para controlar las salidas de poder. Cada salida tiene un conversor de voltaje aislado y puede ser configurada para proporcionar una corriente constante o una tensión constante. El dispositivo también tiene una función de protección contra sobretensión y sobrecorriente.

## ¿Por qué importa?

El Ampisu es importante porque ofrece una solución compacta y asequible para el trabajo de campo y la prueba de sistemas embebidos. Es ideal para desarrolladores de software y hardware que necesitan un laboratorio de poder portátil y flexible.

## Consejo técnico

Si estás desarrollando un proyecto de hardware y necesitas un laboratorio de poder compacto, considera utilizar el Ampisu. Puedes configurarlo para proporcionar una corriente constante o una tensión constante y proteger tus dispositivos contra sobretensión y sobrecorriente.
