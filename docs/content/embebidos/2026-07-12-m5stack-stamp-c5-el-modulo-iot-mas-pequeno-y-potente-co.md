# M5Stack Stamp-C5: el módulo IoT más pequeño y potente con ESP32-C5

## Introducción

M5Stack ha lanzado el Stamp-C5, un módulo IoT de tamaño reducido pero con características avanzadas. Este pequeño pero potente módulo se basa en el microcontrolador ESP32-C5 de Espressif y ofrece conectividad inalámbrica y una gran cantidad de GPIO.

## ¿Qué es?

El M5Stack Stamp-C5 es un módulo IoT de tamaño reducido que se basa en el microcontrolador ESP32-C5 de Espressif. Este módulo ofrece conectividad inalámbrica a través de WiFi 6, Bluetooth LE y 802.15.4, y cuenta con una gran cantidad de GPIO para conectar sensores y actuadores.

## ¿Cómo funciona?

El Stamp-C5 se basa en el microcontrolador ESP32-C5, que cuenta con un procesador RISC-V de 32 bits a 240 MHz y 384 KB de SRAM. El módulo también cuenta con una gran cantidad de GPIO, incluyendo 11 pines en headers de 2,54 mm y 8 pines en un conector FPC. La conectividad inalámbrica se logra a través de un conector IPEX que se puede conectar a un antena externa.

## ¿Por qué importa?

El Stamp-C5 es importante porque ofrece una solución de IoT de tamaño reducido pero con características avanzadas. Esto lo hace ideal para aplicaciones de IoT que requieren una gran cantidad de conectividad y flexibilidad, como sensores de temperatura y humedad, sistemas de automatización doméstica y más.

## Consejo técnico

Si estás interesado en usar el Stamp-C5 con Arduino, puedes comenzar configurando el IDE de Arduino para que se comunique con el módulo. Puedes hacer esto siguiendo las instrucciones en la documentación de M5Stack y utilizando la biblioteca de Arduino para ESP32-C5.

```bash
sudo apt-get install esp32
```
