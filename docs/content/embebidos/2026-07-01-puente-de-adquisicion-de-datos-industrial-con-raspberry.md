## Introducción

La empresa EDATEC ha lanzado un nuevo dispositivo de adquisición de datos industrial basado en el módulo de computación Raspberry Pi CM0. Este dispositivo es ideal para aplicaciones de automatización industrial y de borde con múltiples interfaces de entrada/salida.

## ¿Qué es?

El EDATEC ED-IPC1200 es un puente de adquisición de datos industrial que utiliza el módulo de computación Raspberry Pi CM0. Este dispositivo cuenta con varias interfaces de entrada/salida, incluyendo RS-485, RS-232, cuatro entradas digitales y cuatro salidas digitales. También admite Ethernet y conectividad Wi-Fi/Bluetooth y/o 4G LTE opcional.

## ¿Cómo funciona?

El dispositivo utiliza el procesador quad-core Cortex-A53 de 1,0 GHz del módulo de computación Raspberry Pi CM0, que cuenta con 512 MB de memoria RAM LPDDR2 y 8 GB o 16 GB de almacenamiento eMMC. El dispositivo admite varias interfaces de entrada/salida, incluyendo RS-485, RS-232, cuatro entradas digitales y cuatro salidas digitales. También cuenta con un puerto USB 2.0 y un puerto USB Type-C para eMMC.

## ¿Por qué importa?

Este dispositivo es importante para aplicaciones de automatización industrial y de borde, ya que ofrece una solución compacta y fiable para la adquisición de datos y el control de equipos. También admite conectividad Wi-Fi/Bluetooth y/o 4G LTE, lo que lo hace ideal para aplicaciones que requieren conectividad remota.

## Consejo técnico

Si estás desarrollando una aplicación de automatización industrial, considera utilizar el EDATEC ED-IPC1200 como puente de adquisición de datos. Puedes utilizar la biblioteca Python para interactuar con las interfaces de entrada/salida y enviar datos a un servidor de datos en la nube.

```bash
pip install python-rpi-gpio
```
