# StreamCrate: un servidor de medios portátil y versátil

**Fecha:** 2026-06-07
**Categoría:** embebidos
**Tags:** raspberry-pi, linux, servidor-de-medios
**Título original:** StreamCrate: portable media server

---

## Introducción

Daveed Walzer, un entusiasta de la tecnología, ha creado un servidor de medios portátil llamado StreamCrate, que combina la versatilidad de un Raspberry Pi con la capacidad de almacenamiento de un disco duro externo. Este dispositivo puede reproducir contenido durante 10 horas con una batería de 20.000mAh y un disco duro de 5TB.

## ¿Qué es?

StreamCrate es un servidor de medios portátil que se basa en un Raspberry Pi 4 y un disco duro externo de 5TB. Permite la reproducción de contenido en diferentes dispositivos a través de Wi-Fi o HDMI, y se puede controlar mediante un Amazon Fire TV stick.

## ¿Cómo funciona?

StreamCrate se construyó utilizando un Raspberry Pi 4, un disco duro externo de 5TB, una batería de 20.000mAh y un adaptador de Wi-Fi externo. El sistema se ejecuta con Raspberry Pi OS Lite y utiliza Kodi como reproductor de medios. El dispositivo se puede controlar mediante un Amazon Fire TV stick y se puede conectar a una red Wi-Fi para reproducir contenido en diferentes dispositivos.

## ¿Por qué importa?

StreamCrate es importante porque ofrece una solución portátil y versátil para la reproducción de contenido en diferentes dispositivos. También permite la sincronización de contenido con un servidor de medios local, lo que es útil para aquellos que desean acceder a su biblioteca de medios desde cualquier lugar.

## Consejo técnico

Si deseas crear un servidor de medios portátil similar a StreamCrate, considera utilizar un Raspberry Pi 4 y un disco duro externo de 5TB. También es importante configurar Kodi como reproductor de medios y conectar el dispositivo a una red Wi-Fi para acceder a tu biblioteca de medios desde cualquier lugar.

```bash
sudo apt-get install kodi
```
