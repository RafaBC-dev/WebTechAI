# Waveshare UGV Beast: un robot off-road con inteligencia artificial

**Fecha:** 2026-04-22
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** UGV Beast – An off-road tracked AI robot built for Raspberry Pi 4/5

## Introducción

Waveshare ha presentado el UGV Beast, un robot off-road con ruedas continuas diseñado para funcionar con la plataforma Raspberry Pi 4 o 5. Este robot es capaz de realizar tareas de visión artificial y planificación de estrategias, mientras que un sub-controlador ESP32 se encarga de la control de movimiento y el procesamiento de datos de sensores.

## ¿Qué es?

El UGV Beast es un robot off-road con ruedas continuas diseñado para funcionar con la plataforma Raspberry Pi 4 o 5. Está equipado con una sub-placa de controlador multi-funcional, un SoC ESP32, motores controlados por TB6612FNG y una variedad de interfaces periféricas para sensores y dispositivos.

## ¿Cómo funciona?

El robot utiliza la plataforma Raspberry Pi 4 o 5 como núcleo de procesamiento, que ejecuta Debian Bookworm y ROS2-HUMBLE-LTS. El sub-controlador ESP32 se encarga de la control de movimiento y el procesamiento de datos de sensores, mientras que la sub-placa de controlador multi-funcional gestiona la comunicación con los sensores y dispositivos periféricos.

## ¿Por qué importa?

El UGV Beast es relevante porque ofrece una solución para realizar tareas de visión artificial y planificación de estrategias en entornos off-road. Puede ser utilizado en aplicaciones como la exploración de terrenos, la supervisión de infraestructuras o la búsqueda y rescate en entornos peligrosos.

## Consejo técnico

Para empezar a trabajar con el UGV Beast, es recomendable instalar la plataforma ROS2-HUMBLE-LTS en la Raspberry Pi 4 o 5 y configurar el sub-controlador ESP32 para que se comunique con los sensores y dispositivos periféricos.

```bash
sudo apt-get install ros2-humble-desktop
```
