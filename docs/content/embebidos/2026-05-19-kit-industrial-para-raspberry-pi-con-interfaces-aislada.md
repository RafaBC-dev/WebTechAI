## Introducción

Waveshare ha lanzado un kit que convierte un Raspberry Pi 4 o 5 en un ordenador industrial con interfaces aisladas y una carcasa para DIN rail. Esto permite a los usuarios crear sistemas de control industrial más fáciles y económicos.

## ¿Qué es?

El kit 'Isolated RS232 / RS485 / CAN / CAN FD Expansion Board For Raspberry Pi' es una placa de expansión que se conecta a un Raspberry Pi 4 o 5 y proporciona interfaces aisladas para RS232, RS485 y CAN Bus. También incluye una carcasa para DIN rail y una alimentación de 7-36V DC.

## ¿Cómo funciona?

La placa de expansión utiliza la interfaz SPI + UART del Raspberry Pi para conectarse a las interfaces aisladas. Las interfaces RS232, RS485 y CAN Bus están implementadas utilizando chips específicos como el MCP2518FD + MCP2562FD para CAN FD y el MCP2515 + SN65HVD230 para CAN Bus.

## ¿Por qué importa?

Este kit es importante porque permite a los usuarios crear sistemas de control industrial más fáciles y económicos. También ofrece una solución flexible para conectar dispositivos industriales a una red de control.

## Consejo técnico

Si deseas utilizar este kit con un Raspberry Pi 5, asegúrate de conectar la placa de expansión a la interfaz PCIe para aprovechar las características adicionales de la placa.
