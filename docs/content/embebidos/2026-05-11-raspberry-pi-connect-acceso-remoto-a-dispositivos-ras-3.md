## Introducción

Raspberry Pi Connect es una herramienta que permite acceder a dispositivos Raspberry Pi desde cualquier lugar del mundo, directamente desde un navegador web. La última actualización ha agregado nuevas funcionalidades que mejoran la experiencia de usuario, especialmente para equipos que utilizan Connect para administrar flotas de dispositivos.

## ¿Qué es?

Raspberry Pi Connect es una herramienta que permite acceder a dispositivos Raspberry Pi desde cualquier lugar del mundo, directamente desde un navegador web. Es una herramienta en línea que permite a los administradores acceder a los dispositivos de sus equipos y realizar tareas de administración, como la actualización del software y la configuración de los dispositivos.

## ¿Cómo funciona?

Para utilizar Raspberry Pi Connect, los administradores deben crear una cuenta y agregar los dispositivos Raspberry Pi a su equipo. Una vez que los dispositivos estén agregados, los administradores pueden acceder a ellos desde cualquier lugar del mundo, directamente desde un navegador web. La herramienta utiliza la tecnología de autenticación de dos factores para garantizar la seguridad de los dispositivos y la información de los administradores.
