## Introducción

El proyecto AERIS-10 es un radar de radar abierto que puede rastrear múltiples objetos a una distancia de hasta 20km. Este sistema de radar es una innovación importante en el campo de la robótica y la tecnología de radar.

## ¿Qué es?

AERIS-10 es un sistema de radar de radar abierto que utiliza una técnica de modulación de frecuencia lineal (LFM) para generar ondas de radar. El sistema consta de una placa base que contiene un procesador de campo-programable (FPGA) AMD Artix-7, un microcontrolador STM32 y una variedad de componentes electrónicos. El sistema también incluye una antena de radar que puede ser configurada para rastrear objetos en diferentes direcciones.

## ¿Cómo funciona?

El sistema AERIS-10 funciona generando ondas de radar que se envían hacia el objetivo y luego se reciben y procesan para determinar la distancia y la velocidad del objetivo. El sistema utiliza una técnica de procesamiento de señal llamada Fourier transformada rápida (FFT) para analizar las ondas de radar y determinar la posición y la velocidad del objetivo. El sistema también incluye una interfaz de usuario que permite al usuario configurar y controlar el sistema.

## ¿Por qué importa?

AERIS-10 es importante porque puede ser utilizado en una variedad de aplicaciones, incluyendo la detección de objetos en movimiento, la navegación y la localización. El sistema también puede ser utilizado en la investigación y el desarrollo de tecnologías de radar avanzadas.
