# Sistema de soporte para multímetro 3D impreso

## Introducción

La comunidad de impresión 3D ha creado un sistema de soporte para multímetros compactos, lo que permite organizar instrumentos en la mesa de trabajo de manera eficiente.

## ¿Qué es?

El sistema de soporte es un diseño 3D creado para el multímetro FNIRSI DST-210, que combina funciones de multímetro, osciloscopio y generador de señales. Se puede imprimir en una impresora 3D y se utiliza un sistema de anclaje para fijarlo a una mesa de trabajo.

## ¿Cómo funciona?

El sistema de soporte utiliza un anclaje de tipo Flush Big Thread Multipoint x5 para unirse a una mesa de trabajo de tipo Multiboard. Esto permite una fijación segura y estable del multímetro, evitando movimientos innecesarios y permitiendo una mayor organización en la mesa de trabajo.

## ¿Por qué importa?

Este sistema de soporte es importante porque permite una mayor organización y eficiencia en la mesa de trabajo, especialmente en entornos de trabajo compartidos. También es una buena oportunidad para practicar la impresión 3D y el diseño de sistemas de soporte personalizados.

## Consejo técnico

Si deseas crear un sistema de soporte personalizado para tus instrumentos, puedes utilizar herramientas de diseño 3D como Tinkercad o Fusion 360 para crear un diseño adaptado a tus necesidades. Recuerda utilizar materiales adecuados y seguir las instrucciones de seguridad al imprimir y ensamblar el sistema de soporte.
