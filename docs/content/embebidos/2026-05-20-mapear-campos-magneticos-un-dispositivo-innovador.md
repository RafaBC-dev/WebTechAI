## Introducción

Un ingeniero ha creado un dispositivo para mapear campos magnéticos, utilizando tecnología de última generación. Este dispositivo puede ser útil para investigadores y científicos que buscan detectar anomalías magnéticas en la Tierra.

## ¿Qué es?

El dispositivo utiliza un microcontrolador ESP32, un panel táctil de 4.3 pulgadas y sensores de campo magnético EMS100. El ESP32 se conecta a un Arduino Nano, que se encarga de leer los datos de los sensores y enviarlos al ESP32.

## ¿Cómo funciona?

El dispositivo utiliza un sistema de lectura de datos en tiempo real, que permite visualizar los campos magnéticos en una representación de calor en la pantalla táctil. Los datos también se pueden exportar a una tarjeta SD para análisis posterior.

## ¿Por qué importa?

Este dispositivo puede ser útil para investigadores y científicos que buscan detectar anomalías magnéticas en la Tierra, como los geólogos que buscan recursos naturales. También puede ser utilizado en aplicaciones de detección de minas y búsqueda de artefactos históricos.

## Consejo técnico

Si estás interesado en crear un dispositivo similar, considera utilizar la biblioteca de software de Arduino para leer los datos de los sensores y enviarlos al microcontrolador ESP32. También es importante utilizar un sistema de protección de la batería para evitar daños al dispositivo.
