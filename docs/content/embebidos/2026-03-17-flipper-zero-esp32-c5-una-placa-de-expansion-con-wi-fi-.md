# Flipper Zero ESP32-C5: una placa de expansión con Wi-Fi 6 y GPS

## Introducción

Rabbit-Labs ha lanzado una nueva placa de expansión para el Flipper Zero, una herramienta de seguridad y análisis de red. Esta placa, llamada ESP32-C5, cuenta con Wi-Fi 6, GPS y otros componentes para ofrecer una mayor funcionalidad a la herramienta.

## ¿Qué es?

La placa ESP32-C5 es una expansión para el Flipper Zero que se basa en el microcontrolador ESP32-C5, que cuenta con Wi-Fi 6, Bluetooth 5.0 y otros componentes. Esta placa ofrece una mayor funcionalidad a la herramienta, permitiendo análisis de red y seguridad más avanzados.

## ¿Cómo funciona?

La placa ESP32-C5 se conecta al Flipper Zero mediante un puerto USB-C y utiliza el microcontrolador ESP32-C5 para ofrecer Wi-Fi 6, Bluetooth 5.0 y otros componentes. La placa también cuenta con un módulo GPS y un slot de tarjeta SD para almacenar datos.

## ¿Por qué importa?

La placa ESP32-C5 es importante porque ofrece una mayor funcionalidad a la herramienta Flipper Zero, permitiendo análisis de red y seguridad más avanzados. Esto es especialmente útil para profesionales de la seguridad y la red que necesitan herramientas más potentes para su trabajo.

## Consejo técnico

Si estás interesado en utilizar la placa ESP32-C5 con tu Flipper Zero, asegúrate de instalar el firmware ESP32 Marauder, que ofrece acceso a herramientas de análisis de red y seguridad más avanzadas.

```bash
esp32c5
```
