# Un pequeño dispositivo de notificación se convierte en un hackeable notificador de escritorio

**Fecha:** 2026-07-10
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** A Super Cheap Desk Toy Becomes a Hackable Desktop Notifier

---

## Introducción

El GeekMagic SmallTV es un dispositivo pequeño y versátil que puede mostrar información de todo tipo, desde la hora y el clima hasta el precio actual del Bitcoin. Recientemente, un desarrollador ha creado un firmware personalizado para este dispositivo, lo que lo convierte en un notificador de escritorio hackeable.

## ¿Qué es?

El GeekMagic SmallTV es un dispositivo de notificación de escritorio que se vende como un pequeño televisor con pantalla. Dependiendo del firmware que se le instale, puede mostrar información como la hora, el clima, el precio del Bitcoin o incluso el estado de un tracker de ADS-B. Estos dispositivos están disponibles en AliExpress y tienen varias versiones, incluyendo una que utiliza el chip ESP8266 y otra que utiliza el chip ESP32-C2.

## ¿Cómo funciona?

El dispositivo utiliza un firmware que se puede actualizar a través de Wi-Fi, lo que permite a los desarrolladores personalizar su comportamiento y agregar nuevas funcionalidades. El firmware actualmente soporta varios tipos de información, incluyendo tickers de acciones, un tracker de ADS-B y un indicador de uso de tokens de Claude AI. Los desarrolladores pueden personalizar el firmware para agregar nuevas funcionalidades o conectarlo a fuentes de datos personalizadas.

## ¿Por qué importa?

Este dispositivo es interesante porque ofrece una forma barata y fácil de crear un notificador de escritorio personalizado. Además, el hecho de que se pueda actualizar a través de Wi-Fi y que tenga un firmware personalizable lo convierte en un proyecto atractivo para los desarrolladores de software y hardware.

## Consejo técnico

Si deseas personalizar el firmware del GeekMagic SmallTV, puedes utilizar la herramienta de desarrollo de ESP32 para crear un nuevo firmware y conectarlo a una fuente de datos personalizada.

```bash
git clone https://github.com/Giovi321/smalltv-firmware
```
