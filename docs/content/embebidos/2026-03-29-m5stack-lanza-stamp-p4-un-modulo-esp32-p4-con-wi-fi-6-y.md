# M5Stack lanza Stamp-P4, un módulo ESP32-P4 con Wi-Fi 6 y Bluetooth 5.4

**Fecha:** 2026-03-29
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** M5Stack Stamp-P4 – A tiny ESP32-P4 USB-C board with optional Wi-Fi 6 and Bluetooth 5.4

## Introducción

M5Stack ha presentado Stamp-P4, un módulo de desarrollo de tamaño reducido basado en el ESP32-P4, un procesador de alta performance de 16MB de Flash y 32MB de PSRAM, con soporte opcional para Wi-Fi 6 y Bluetooth 5.4. Este módulo es ideal para proyectos de IoT y robótica.

## ¿Qué es?

Stamp-P4 es un módulo de desarrollo de tamaño reducido que se basa en el ESP32-P4, un procesador de alta performance de 16MB de Flash y 32MB de PSRAM. Ofrece una amplia gama de interfaces, incluyendo una conexión para cámaras MIPI-CSI, una interfaz de pantalla MIPI DSI, Ethernet RMII, USB 2.0 HS y hasta 44 pines GPIO.

## ¿Cómo funciona?

Stamp-P4 se basa en el ESP32-P4, un procesador de alta performance que cuenta con un núcleo dual-core RISC-V de 360 MHz, un núcleo LP de 40 MHz y un acelerador de procesamiento de píxeles 2D. También cuenta con un VPU que soporta codecs H.264 y JPEG. El módulo ofrece una amplia gama de interfaces, incluyendo una conexión para cámaras MIPI-CSI, una interfaz de pantalla MIPI DSI, Ethernet RMII, USB 2.0 HS y hasta 44 pines GPIO.

## ¿Por qué importa?

Stamp-P4 es ideal para proyectos de IoT y robótica, ya que ofrece una amplia gama de interfaces y un procesador de alta performance. También es compatible con la plataforma de programación UiFlow2 y con los frameworks Arduino IDE, ESP-IDF y PlatformIO.
