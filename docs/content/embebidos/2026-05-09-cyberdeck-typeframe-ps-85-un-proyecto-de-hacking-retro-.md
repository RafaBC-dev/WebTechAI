## Introducción

El mundo de los cyberdecks, inspirados en la ciencia ficción, sigue creciendo gracias a proyectos como el Typeframe PS-85, un dispositivo de hacking retro construido con Raspberry Pi y componentes personalizados.

## ¿Qué es?

El Typeframe PS-85 es un proyecto de hacking retro que utiliza un Raspberry Pi Zero 2 W como base, combinado con un teclado mecánico de 40 por ciento y un panel LED personalizado. El dispositivo tiene una forma de tableta y es inspirado en la película Alien y su universo.

## ¿Cómo funciona?

El Typeframe PS-85 se construyó a partir de un proyecto anterior llamado Penkesu, que utilizaba un Raspberry Pi Zero. Jeff Merrick, el creador del proyecto, decidió modificar el diseño para crear un dispositivo más pequeño y sencillo, utilizando un teclado mecánico de 40 por ciento y un panel LED personalizado. El dispositivo utiliza el sistema operativo Linux y se puede programar para realizar tareas de hacking y otros proyectos.

## ¿Por qué importa?

El Typeframe PS-85 es importante porque muestra cómo la tecnología Raspberry Pi puede ser utilizada para crear dispositivos personalizados y innovadores. El proyecto también demuestra cómo la comunidad de hackers y makers puede inspirarse en la ciencia ficción y crear proyectos que son a la vez funcionales y estéticamente agradables.

## Consejo técnico

Si deseas crear un proyecto similar, puedes utilizar el diseño del Typeframe PS-85 como base y personalizarlo con tus propios componentes y programación. Recuerda que la clave para crear un proyecto exitoso es la creatividad y la perseverancia.
