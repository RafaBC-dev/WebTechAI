# PicoCPC: un módulo de expansión para ordenadores Amstrad CPC

## Introducción

Después de décadas de olvido, los ordenadores Amstrad CPC están experimentando un resurgimiento gracias a la creación de hardware y software personalizado. El PicoCPC es un ejemplo de esto, un módulo de expansión que agrega nuevas funcionalidades a estos clásicos ordenadores de 8 bits.

## ¿Qué es?

El PicoCPC es un módulo de expansión creado para ordenadores Amstrad CPC, que se basa en un Raspberry Pi Pico 2. Ofrece una amplia gama de funcionalidades, incluyendo la expansión de memoria hasta 1024 kB, emulación de controlador de disco flexible, soporte para hasta 16 ROMs emulados y carga de software para la consola GX4000.

## ¿Cómo funciona?

El PicoCPC utiliza un Raspberry Pi Pico 2 como base y se conecta a los ordenadores Amstrad CPC mediante un conector de expansión estándar. El módulo emula un controlador de disco flexible, lo que permite a los usuarios cargar software y datos desde un disco flexible emulado. También ofrece soporte para hasta 16 ROMs emulados, lo que permite a los usuarios acceder a software y sistemas operativos alternativos.

## ¿Por qué importa?

El PicoCPC es importante porque permite a los usuarios de ordenadores Amstrad CPC acceder a nuevas funcionalidades y mejorar la experiencia de usuario. También es un ejemplo de cómo la comunidad de desarrollo de hardware y software puede crear soluciones personalizadas para ordenadores antiguos.

## Consejo técnico

Si deseas crear tu propio módulo de expansión para ordenadores Amstrad CPC, considera utilizar un Raspberry Pi Pico 2 como base y aprovecha las bibliotecas de programación disponibles para este microcontrolador.

```bash
python -m pip install rp2
```
