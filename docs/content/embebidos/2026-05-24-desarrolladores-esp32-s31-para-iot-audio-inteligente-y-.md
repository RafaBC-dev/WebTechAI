# Desarrolladores ESP32-S31 para IoT, audio inteligente y HMI

**Fecha:** 2026-05-24
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y linux
**Título original:** ESP32-S31 development boards unveiled for IoT, Smart Audio, and HMI applications

---

## Introducción

Espera a que lleguen los desarrolladores ESP32-S31, dos tableros de desarrollo que ofrecen características impresionantes para aplicaciones de IoT, audio inteligente y HMI. Estos tableros están diseñados para aprovechar al máximo la tecnología ESP32-S31, que cuenta con dos núcleos RISC-V, Gigabit Ethernet, WiFi 6 y Bluetooth.

## ¿Qué es?

Los ESP32-S31-Function-CoreBoard-1 y ESP32-S31-Korvo-1 son dos tableros de desarrollo diseñados para aprovechar las características del ESP32-S31. El primero se centra en aplicaciones de IoT conectadas, mientras que el segundo se enfoca en audio y video conectados, con un enfoque en la voz activa y la interacción humana.

## ¿Cómo funciona?

Ambos tableros utilizan el módulo ESP32-S31-WROOM-3, que cuenta con dos núcleos RISC-V, uno de alta velocidad y otro de bajo consumo. También incluyen interfaces de LCD y cámara, así como conectividad WiFi 6, Bluetooth y 802.15.4. El ESP32-S31-Korvo-1 también incluye un array de micrófonos y conectores de altavoz.

## ¿Por qué importa?

Estos tableros ofrecen una plataforma sólida para desarrollar aplicaciones de IoT, audio inteligente y HMI. Permiten a los desarrolladores aprovechar las características del ESP32-S31 para crear soluciones conectadas y eficientes.

## Consejo técnico

Si estás desarrollando una aplicación de IoT conectada, considera utilizar el ESP32-S31-Function-CoreBoard-1 para aprovechar su conectividad Gigabit Ethernet y su capacidad para manejar grandes cantidades de datos.

```bash
Para conectar el ESP32-S31-Function-CoreBoard-1 a tu computadora, necesitarás un cable USB 2.0 Type-A a Type-C y una computadora con Windows, Linux o macOS.
```
