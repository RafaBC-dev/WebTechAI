## Introducción

El desarrollador de Esparagus ha lanzado un nuevo proyecto en Crowd Supply: Esparagus Audio Brick, un módulo de amplificación de audio Hi-Fi basado en ESP32 que soporta Home Assistant y Squeezelite. Este dispositivo compacto puede ser utilizado para crear sistemas de audio en todo el hogar, mejorar los altavoces vintage con características de streaming modernas o alimentar configuraciones de altavoces personalizadas.

## ¿Qué es?

El Esparagus Audio Brick es un módulo de amplificación de audio Hi-Fi compacto que utiliza un microcontrolador ESP32 o ESP32-S3 para procesar audio y soportar Wi-Fi y Ethernet. Este dispositivo incluye un DAC de alta calidad, un amplificador de clase-D integrado y un DSP para igualar el audio y gestionar fallos.

## ¿Cómo funciona?

El Esparagus Audio Brick utiliza un microcontrolador ESP32 o ESP32-S3 para procesar audio y soportar Wi-Fi y Ethernet. El dispositivo incluye un DAC Texas Instruments TAS5825M para procesar audio y un amplificador de clase-D integrado para amplificar el audio. El DSP se utiliza para igualar el audio y gestionar fallos, mientras que la interfaz de usuario se puede personalizar mediante un display OLED opcional.

## ¿Por qué importa?

El Esparagus Audio Brick puede ser utilizado para crear sistemas de audio en todo el hogar, mejorar los altavoces vintage con características de streaming modernas o alimentar configuraciones de altavoces personalizadas. Este dispositivo es especialmente útil para aquellos que buscan una solución de audio de alta calidad y fácil de configurar.

## Consejo técnico

Si deseas aprovechar al máximo el Esparagus Audio Brick, asegúrate de configurar correctamente el DSP para igualar el audio y gestionar fallos. También es recomendable utilizar un display OLED para personalizar la interfaz de usuario y mejorar la experiencia de usuario.
