# Un reloj de memoria magnética USB que revoluciona la tecnología

**Fecha:** 2026-06-30
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, diseño-3d
**Título original:** Making a Magnetic Core Memory USB Drive

## Introducción

La falta de memoria DRAM y NAND Flash en el mercado de consumo ha generado preocupación entre los entusiastas de la tecnología. Sin embargo, un proyecto reciente ha revolucionado la forma en que se almacenan los datos, utilizando una memoria magnética de 64 bits

## ¿Qué es?

La memoria magnética es un tipo de almacenamiento que utiliza pequeños núcleos de hierro que pueden ser magnetizados para almacenar información. En este proyecto, se utiliza una memoria magnética de 64 bits para crear un reloj de memoria USB

## ¿Cómo funciona?

La memoria magnética se utiliza junto con un pequeño router CNC para crear las placas de circuito impreso (PCB) que contienen los componentes necesarios. El ESP32 se utiliza como el cerebro del proyecto y se conecta a la memoria magnética a través de un hilo de cobre. La memoria magnética se utiliza para almacenar la información, mientras que el ESP32 se encarga de leer y escribir la información en la memoria

## ¿Por qué importa?

Este proyecto es importante porque demuestra que es posible crear un almacenamiento de datos utilizando una tecnología antigua y poco utilizada. La memoria magnética es una forma de almacenamiento que se utiliza en algunos sistemas de alta velocidad, pero no se utiliza comúnmente en dispositivos de almacenamiento portátiles

## Consejo técnico

Si deseas crear tu propio reloj de memoria magnética USB, asegúrate de utilizar un router CNC para crear las PCB con precisión. También es importante experimentar con diferentes voltajes para encontrar el valor óptimo para la memoria magnética
