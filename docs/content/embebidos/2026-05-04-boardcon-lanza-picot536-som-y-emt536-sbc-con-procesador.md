# Boardcon lanza PICOT536 SoM y EMT536 SBC con procesador Allwinner T536

**Fecha:** 2026-05-04
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** Boardcon PICOT536 SoM and EMT536 SBC feature Allwinner T536 Edge AI processor

## Introducción

La empresa Boardcon ha lanzado dos nuevos productos: el SoM (System-on-Module) PICOT536 y el SBC (Single-Board Computer) EMT536. Estos dispositivos están diseñados para aplicaciones industriales y de edge computing, y están basados en el procesador Allwinner T536, que cuenta con una arquitectura de CPU quad-core Arm Cortex-A55 y un acelerador de AI de 2 TOPS.

## ¿Qué es?

El PICOT536 SoM es un módulo de sistema que incluye el procesador Allwinner T536, memoria LPDDR4/LPDDR4X, eMMC flash y una variedad de interfaces de entrada y salida, como MIPI DSI, LVDS, camera inputs y audio inputs y outputs. El EMT536 SBC es una placa de desarrollo que utiliza el PICOT536 SoM como núcleo y ofrece una variedad de interfaces y conectores para permitir la conexión de sensores, actuadores y otros dispositivos.

## ¿Cómo funciona?

El PICOT536 SoM funciona con un procesador quad-core Arm Cortex-A55 que puede alcanzar velocidades de hasta 1.6 GHz, y cuenta con un acelerador de AI de 2 TOPS que puede realizar operaciones de inteligencia artificial en tiempo real. El EMT536 SBC utiliza el PICOT536 SoM como núcleo y ofrece una variedad de interfaces y conectores para permitir la conexión de sensores, actuadores y otros dispositivos.

## ¿Por qué importa?

Estos dispositivos son importantes para aplicaciones industriales y de edge computing, ya que ofrecen una solución compacta y eficiente para realizar tareas de procesamiento y análisis de datos en tiempo real. También pueden ser utilizados en aplicaciones de machine vision, robotics y otros campos que requieren procesamiento de datos en tiempo real.

## Consejo técnico

Si estás desarrollando una aplicación industrial o de edge computing, considera utilizar el PICOT536 SoM o el EMT536 SBC como base para tu proyecto. Puedes utilizar la documentación de Boardcon para aprender a programar y configurar estos dispositivos.
