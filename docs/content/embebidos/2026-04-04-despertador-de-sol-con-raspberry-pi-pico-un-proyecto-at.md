## Introducción

Un proyecto innovador creado por Pawel Skiba, un ingeniero que se inspiró en la necesidad de su esposa de un despertador de sol para mejorar su sueño. Con un presupuesto de 250 euros, Pawel decidió crear su propio despertador de sol utilizando un Raspberry Pi Pico 2 W.

## ¿Qué es?

El despertador de sol es un dispositivo que simula el proceso del amanecer, gradualmente aumentando la iluminación en la habitación para ayudar a la persona a despertar de manera suave y natural. Esto se logra utilizando un LED que emite luz roja, seguida de colores más brillantes, para simular la luz del sol

## ¿Cómo funciona?

Pawel utilizó un Raspberry Pi Pico 2 W para controlar el LED y programar el despertador. El dispositivo utiliza un algoritmo para simular el proceso del amanecer, comenzando con luz roja y gradualmente aumentando la intensidad y el color. El Raspberry Pi Pico 2 W es un microcontrolador pequeño y económico que permite controlar el LED y programar el despertador.

## ¿Por qué importa?

Este proyecto es importante porque demuestra la capacidad de crear dispositivos innovadores y personalizados utilizando tecnología de código abierto. El despertador de sol puede ser una solución efectiva para personas que tienen dificultades para despertar temprano o que necesitan ayuda para regular su ciclo de sueño.

## Consejo técnico

Si deseas crear tu propio despertador de sol, puedes utilizar el código de Pawel como referencia y personalizarlo según tus necesidades. Puedes encontrar el código en GitHub y utilizar herramientas como el IDE de Raspberry Pi para programar y depurar tu dispositivo.

```bash
git clone https://github.com/pawelskiba/sunrise-alarm-clock
```
