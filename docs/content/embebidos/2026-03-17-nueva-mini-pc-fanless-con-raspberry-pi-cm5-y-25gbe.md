## Introducción

Waveshare ha lanzado una nueva mini PC fanless que utiliza el SoC Raspberry Pi CM5 y ofrece una conexión de 2,5GbE, HDMI 2.0 y dos puertos USB.

## ¿Qué es?

La 'Gigabit / 2.5G Dual Ethernet Mini-Computer (B)' es una mini PC fanless que utiliza el SoC Raspberry Pi CM5 y ofrece una conexión de 2,5GbE, HDMI 2.0 y dos puertos USB. También cuenta con un socket M.2 Key-M para conectar una tarjeta de estado sólido (SSD) o un acelerador de inteligencia artificial (AI).

## ¿Cómo funciona?

La mini PC utiliza el SoC Raspberry Pi CM5, que se conecta a un módulo de red 2,5GbE y un módulo de red Gigabit Ethernet. El sistema también cuenta con un socket M.2 Key-M para conectar una tarjeta de estado sólido (SSD) o un acelerador de inteligencia artificial (AI). La conexión HDMI 2.0 permite una resolución de hasta 4K a 60Hz.

## ¿Por qué importa?

Esta mini PC fanless es importante porque ofrece una conexión de 2,5GbE, lo que la hace ideal para aplicaciones que requieren una velocidad de red alta. También cuenta con un socket M.2 Key-M para conectar una tarjeta de estado sólido (SSD) o un acelerador de inteligencia artificial (AI), lo que la hace útil para aplicaciones que requieren una gran cantidad de almacenamiento y procesamiento.

## Consejo técnico

Si deseas utilizar la conexión 2,5GbE de esta mini PC, asegúrate de conectarla a un router o un switch que soporte esta velocidad de red. También es importante configurar la tarjeta de red para que funcione correctamente con el sistema operativo Raspberry Pi OS.

```bash
sudo apt-get update && sudo apt-get install raspbian
```
