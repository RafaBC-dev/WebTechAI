## Introducción

El proyecto Typeframe PS-85 es un ejemplo de cómo la tecnología de retrocomputación puede revivir viejas ideas y crear algo nuevo y emocionante. Inspirado en la película Alien, este cyberdeck utiliza un Raspberry Pi Zero 2 W y un teclado mecánico para crear una experiencia de usuario única.

## ¿Qué es?

El Typeframe PS-85 es un proyecto de retrocomputación que utiliza un Raspberry Pi Zero 2 W y un teclado mecánico para crear un cyberdeck inspirado en la película Alien. El proyecto se basa en la idea de crear una computadora portátil que se pueda usar para tareas básicas como escribir texto y navegar por la web.

## ¿Cómo funciona?

El Typeframe PS-85 utiliza un Raspberry Pi Zero 2 W como procesador y un teclado mecánico de 65% para la entrada de datos. El sistema está basado en Linux y utiliza una pantalla táctil para la interfaz de usuario. El proyecto también utiliza un GPIO expuesto para conectar periféricos adicionales.

## ¿Por qué importa?

El Typeframe PS-85 es importante porque muestra cómo la tecnología de retrocomputación puede ser utilizada para crear algo nuevo y emocionante. El proyecto también demuestra la flexibilidad y la personalización que se pueden lograr con tecnologías como el Raspberry Pi y Linux.

## Consejo técnico

Si deseas crear un proyecto similar, considera utilizar un Raspberry Pi Zero 2 W y un teclado mecánico para crear un sistema de retrocomputación personalizado. Puedes utilizar herramientas como la Adafruit CharliePlex para conectar periféricos adicionales.

```bash
sudo apt-get install raspbian -y
```
