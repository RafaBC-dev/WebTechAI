# Encuadernación 3D para LED Matrix Portal S3 de Adafruit

## Introducción

Adafruit ha lanzado una encuadernación 3D para su panel LED de 62×32 con la placa Matrix Portal S3. Esta encuadernación permite montar la placa y el panel de manera segura y fácil de usar.

## ¿Qué es?

La encuadernación 3D es un proyecto de impresión 3D diseñado para albergar la placa Matrix Portal S3 y el panel LED de 62×32 de Adafruit. Utiliza tornillos M2.5×4 para montar la placa y tornillos M3×8 para atornillar el panel.

## ¿Cómo funciona?

La encuadernación 3D está diseñada para llenar la cama de impresión P1/X1 completa. El panel LED se atornilla a la encuadernación utilizando tornillos M3×8, mientras que la placa Matrix Portal S3 se monta con tornillos M2.5×4.

## ¿Por qué importa?

Esta encuadernación 3D es importante porque permite a los usuarios montar la placa y el panel de manera segura y fácil de usar, lo que facilita la creación de proyectos de LED y otros dispositivos electrónicos.
