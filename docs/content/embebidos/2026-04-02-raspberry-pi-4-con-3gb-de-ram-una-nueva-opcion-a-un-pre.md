# Raspberry Pi 4 con 3GB de RAM: una nueva opción a un precio asequible

**Fecha:** 2026-04-02
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, linux
**Título original:** Raspberry Pi 4 3GB launched for $83.75, further price increases announced across the board for 4GB+ RAM hardware

## Introducción

La Raspberry Pi Foundation ha lanzado una nueva versión del Raspberry Pi 4 con 3GB de RAM, a un precio de $83.75. Además, se han anunciado aumentos de precio para los modelos con 4GB o más de RAM.

## ¿Qué es?

El Raspberry Pi 4 es una placa de desarrollo de código abierto que se utiliza para proyectos de robótica, IA y otros dispositivos embebidos. La nueva versión con 3GB de RAM ofrece una opción más asequible para desarrolladores y usuarios que necesitan una placa con un poco más de memoria que la versión básica.

## ¿Cómo funciona?

La placa Raspberry Pi 4 con 3GB de RAM utiliza dos chips LPDDR4 de 1.5GB cada uno, lo que da un total de 3GB de memoria. Esto se logra mediante una variante de la placa que utiliza dos PCBs para conectar los chips de memoria.

## ¿Por qué importa?

La nueva placa con 3GB de RAM es importante porque ofrece una opción más asequible para desarrolladores y usuarios que necesitan una placa con un poco más de memoria. Además, se espera que esta placa sea utilizada en proyectos de IA, robótica y otros dispositivos embebidos que requieren una cantidad de memoria moderada.

## Consejo técnico

Si estás considerando comprar una placa Raspberry Pi 4 con 3GB de RAM, asegúrate de verificar la compatibilidad de tus proyectos con la placa y de que tengas suficiente memoria para ejecutarlos sin problemas.
