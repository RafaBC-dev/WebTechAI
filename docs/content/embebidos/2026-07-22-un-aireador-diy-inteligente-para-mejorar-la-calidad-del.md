# Un aireador DIY inteligente para mejorar la calidad del aire

**Fecha:** 2026-07-22
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y linux
**Título original:** A Smarter DIY Air Filter

---

## Introducción

Un hacker ha diseñado un sistema de aireado inteligente que monitorea y mejora la calidad del aire en un espacio. Este proyecto utiliza tecnología de microcontroladores y sensores para ofrecer una solución innovadora para mantener un ambiente saludable.

## ¿Qué es?

AirSense es un sistema de aireado DIY diseñado por Kishan Pratap Singh que utiliza un microcontrolador ESP32 para monitorear y mejorar la calidad del aire en un espacio. El sistema incluye sensores para medir la concentración de partículas PM2.5, el índice de calidad del aire (AQI), la temperatura, la humedad y la presión atmosférica.

## ¿Cómo funciona?

El sistema funciona conectando un filtro HEPA a un ventilador de 150 mm impulsado por un microcontrolador ESP32. Los sensores se montan en la vía de escape del filtro, lo que permite monitorear la calidad del aire que se produce. El sistema también incluye un display LCD para mostrar los datos de la calidad del aire.

## ¿Por qué importa?

Este sistema es importante porque ofrece una solución innovadora para mantener un ambiente saludable en un espacio. La calidad del aire es fundamental para la salud humana, y este sistema puede ayudar a reducir la exposición a partículas y gases nocivos.

## Consejo técnico

Si deseas implementar un sistema similar, puedes utilizar la biblioteca de software de ESP32 para programar el microcontrolador y conectar los sensores. También puedes utilizar la plataforma de desarrollo de Arduino para crear un prototipo y probar el sistema.

```bash
Utiliza la biblioteca de software de ESP32 para programar el microcontrolador: `git clone https://github.com/espressif/esp32-arduino.git`
```
