# PicoCPC: una tarjeta de expansión para ordenadores Amstrad CPC

**Fecha:** 2026-03-16
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y linux
**Título original:** PicoCPC custom board

## Introducción

Después de décadas de olvido, los ordenadores Amstrad CPC están experimentando un resurgimiento, gracias a la creación de hardware y software personalizados. La tarjeta de expansión PicoCPC es el último capítulo de esta historia.

## ¿Qué es?

La PicoCPC es una tarjeta de expansión para ordenadores Amstrad CPC que se basa en un microcontrolador Raspberry Pi Pico 2. Ofrece una amplia gama de características, incluyendo memoria extendida, emulación de controlador de disco flexible, soporte para hasta 16 ROMs emulados y carga de cartuchos para software del GX4000.

## ¿Cómo funciona?

La PicoCPC utiliza el microcontrolador Raspberry Pi Pico 2 para proporcionar una amplia gama de características. La memoria se extiende hasta 1024kB, y la emulación de controlador de disco flexible permite utilizar ROMs emulados. La carga de cartuchos se realiza mediante una tarjeta de sonido PlayCity emulada.

## ¿Por qué importa?

La PicoCPC es importante porque ofrece una forma de actualizar y personalizar los ordenadores Amstrad CPC, que en su momento fueron populares pero luego se olvidaron. Ahora, gracias a la PicoCPC, estos ordenadores pueden ser utilizados para juegos y aplicaciones más complejas.

## Consejo técnico

Si deseas utilizar la PicoCPC con tu ordenador Amstrad CPC, asegúrate de instalar el software de emulación adecuado y de configurar la tarjeta de expansión correctamente. Puedes encontrar más información en el sitio web del proyecto PicoCPC.
