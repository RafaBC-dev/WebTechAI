# Caja de exhibición con pantalla LED y compartimiento secreto

## Introducción

Adafruit presenta una caja de exhibición futurista que integra una pantalla LED y un compartimiento secreto. Esta caja es el resultado de la fusión de la impresión 3D y la electrónica.

## ¿Qué es?

La caja de exhibición es un proyecto de impresión 3D que combina una pantalla LED con un compartimiento secreto. El compartimiento está oculto detrás de una tapa magnética y puede ser accedido mediante un botón de encendido. La caja está diseñada para exhibir objetos electrónicos y tiene una ventana de acrílico para mostrar la pantalla LED.

## ¿Cómo funciona?

La caja de exhibición se construye mediante impresión 3D utilizando un modelo disponible en MakerWorld. La pantalla LED se integra mediante el kit de lámpara LED Bambu Lab. El compartimiento secreto se acciona mediante un botón de encendido que activa un mecanismo magnético que abre la tapa.

## ¿Por qué importa?

Esta caja de exhibición es importante porque muestra la posibilidad de combinar la impresión 3D con la electrónica para crear objetos innovadores y útiles. También puede ser utilizada como una base para proyectos de electrónica y robótica.

## Consejo técnico

Si deseas crear una caja de exhibición similar, puedes descargar el modelo de impresión 3D desde MakerWorld y utilizar el kit de lámpara LED Bambu Lab para integrar la pantalla LED.

```bash
Descarga el modelo de impresión 3D desde MakerWorld: https://makerworld.com/en/models/1188705-sci-fi-enclosure
```
