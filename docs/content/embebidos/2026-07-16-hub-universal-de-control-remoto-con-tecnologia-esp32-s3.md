## Introducción

El proyecto OpenInfrared Point busca revolucionar la forma en que controlamos dispositivos electrónicos en casa, eliminando la necesidad de remotos dedicados y ofreciendo una solución universal y flexible.

## ¿Qué es?

OpenInfrared Point es un hub de control remoto universal construido con la tecnología ESP32-S3 que permite a los usuarios controlar dispositivos compatibles desde un smartphone mediante el escaneo de un código QR o el toque de una etiqueta NFC.

## ¿Cómo funciona?

El hub se conecta a través de USB-C y puede aprender códigos IR de remotos existentes. Soporta control de dispositivos a través de infrarrojo, IP (HTTP) y Bluetooth Low Energy (BLE), con soporte de RF planeado como una actualización futura.

## ¿Por qué importa?

Este proyecto importa porque ofrece una solución flexible y universal para controlar dispositivos electrónicos en casa, sin necesidad de remotos dedicados. También permite la integración con sistemas de automatización como Home Assistant y Alexa.

## Consejo técnico

Si deseas aprovechar al máximo el potencial de OpenInfrared Point, considera aprender a utilizar la API HTTP integrada para crear macros y configuraciones personalizadas para tus dispositivos.
