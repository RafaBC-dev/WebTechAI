# Nueva plataforma de inteligencia artificial para Internet de las Cosas

**Fecha:** 2026-04-21
**Categoría:** embebidos
**Tags:** ia-local, robotica, embebidos

## Introducción

Ezurio ha lanzado la plataforma Tungsten 510/700, una plataforma de inteligencia artificial (IA) para Internet de las Cosas (IoT) que combina la potencia de los procesadores MediaTek Genio 510 y 700 con una amplia gama de interfaces y conectividad.

## ¿Qué es?

La plataforma Tungsten 510/700 es un módulo de sistema en un solo chip (SoM) que se basa en la arquitectura SMARC 2.1 y está diseñado para aplicaciones de IoT que requieren procesamiento de IA en tiempo real. El módulo cuenta con dos opciones de procesador: el MediaTek Genio 510, un procesador hexa-core con un núcleo de IA de hasta 3,2 TOPS, y el MediaTek Genio 700, un procesador octa-core con un núcleo de IA de hasta 4,0 TOPS.

## ¿Cómo funciona?

La plataforma Tungsten 510/700 funciona mediante la combinación de los procesadores MediaTek Genio 510 y 700 con una amplia gama de interfaces y conectividad, incluyendo Gigabit Ethernet, WiFi 6 y Bluetooth 5.2. El módulo también cuenta con una memoria de hasta 8 GB de LPDDR4 y almacenamiento de hasta 128 GB de eMMC 5.1. La plataforma es compatible con una variedad de interfaces de video, incluyendo HDMI, DisplayPort y eDP, y cuenta con dos interfaces CSI para cámaras.

## ¿Por qué importa?

La plataforma Tungsten 510/700 es importante porque ofrece una solución de procesamiento de IA en tiempo real para aplicaciones de IoT, lo que permite a los desarrolladores crear sistemas más inteligentes y eficientes. La plataforma también cuenta con una amplia gama de interfaces y conectividad, lo que la hace ideal para una variedad de aplicaciones, desde la automatización industrial hasta la atención al cliente.

## Consejo técnico

Si estás desarrollando una aplicación de IoT que requiere procesamiento de IA en tiempo real, considera utilizar la plataforma Tungsten 510/700. Puedes comenzar configurando el módulo con el comando `sudo apt-get install linux-headers-$(uname -r)` y luego instalar la herramienta de desarrollo de MediaTek `mediatek-developer-toolkit`.

```bash
sudo apt-get install linux-headers-$(uname -r)
```
