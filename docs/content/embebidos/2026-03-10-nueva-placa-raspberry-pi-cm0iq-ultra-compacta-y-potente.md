# Nueva placa Raspberry Pi CM0IQ: ultra compacta y potente

**Fecha:** 2026-03-10
**Categoría:** embebidos
**Tags:** Raspberry Pi, CM0 Lite, placa de desarrollo, robótica, dispositivos IoT
**Título original:** Makerfabs CM0IQ – An ultra-compact (42x36mm) Raspberry Pi CM0 Lite board

## Introducción

Makerfabs ha lanzado la CM0IQ, una placa ultra compacta basada en la Raspberry Pi CM0 Lite. Con un tamaño de apenas 42x36mm, esta placa es ideal para aplicaciones de robótica, dispositivos IoT y integraciones de hardware personalizadas. A pesar de su pequeño tamaño, la CM0IQ incluye una variedad de interfaces y conectores, lo que la convierte en una opción atractiva para proyectos de desarrollo de hardware.

## ¿Qué es?

La CM0IQ es una placa de desarrollo ultra compacta basada en la Raspberry Pi CM0 Lite. Está diseñada para ser lo más pequeña posible mientras sigue incluyendo una variedad de interfaces y conectores, como una conexión USB-A, una salida HDMI micro, conectores CSI y DSI para cámaras y pantallas, y un conector de antena IPEX. Esto la hace ideal para aplicaciones de robótica, dispositivos IoT y integraciones de hardware personalizadas.

## ¿Cómo funciona?

La CM0IQ utiliza el SoC Broadcom BCM2710A1, que incluye un procesador cuádruple núcleo Cortex-A53 a 1,0 GHz, una GPU VideoCore IV y un VPU para decodificar y codificar video. La placa también incluye 512MB de memoria RAM LPDDR2 y una ranura para tarjeta microSD. La CM0IQ se puede conectar a una pantalla o cámara a través de los conectores DSI y CSI, respectivamente, y se puede alimentar a través de la conexión USB-C o de los conectores de alimentación de 5V.

## ¿Por qué importa?

La CM0IQ es importante porque ofrece una opción compacta y potente para proyectos de desarrollo de hardware. Su tamaño pequeño y variedad de interfaces la convierten en una opción ideal para aplicaciones de robótica, dispositivos IoT y integraciones de hardware personalizadas. Además, su compatibilidad con la Raspberry Pi OS y su capacidad para detectar cámaras y pantallas sin necesidad de software adicional la hacen una opción atractiva para desarrolladores de hardware.

## Consejo técnico

Si estás planeando desarrollar un proyecto de hardware que requiera una placa compacta y potente, considera la CM0IQ como una opción. Puedes conectar una pantalla o cámara a la placa a través de los conectores DSI y CSI, respectivamente, y programar la placa utilizando la Raspberry Pi OS.

```bash
sudo apt-get install raspbian
```
