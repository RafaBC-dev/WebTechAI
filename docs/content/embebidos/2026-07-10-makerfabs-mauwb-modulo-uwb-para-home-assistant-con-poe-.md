# Makerfabs MaUWB: módulo UWB para Home Assistant con PoE y carcasa

## Introducción

Makerfabs ha lanzado el MaUWB, un módulo UWB (Ultra-Wideband) basado en ESP32-S3 que se integra con Home Assistant para crear sistemas de ubicación en tiempo real. Este módulo es ideal para aplicaciones de ubicación en interiores, como seguimiento de dispositivos en hogares y oficinas.

## ¿Qué es?

El MaUWB es un módulo UWB que combina un ESP32-S3 con un módulo UWB MaUWB basado en el Qorvo DW3000. Este módulo soporta PoE (Alimentación por Conectividad Ethernet) y viene en una carcasa para instalaciones fijas. El MaUWB se puede programar utilizando el IDE de Arduino o el framework ESP-IDF para aplicaciones personalizadas.

## ¿Cómo funciona?

El MaUWB utiliza el módulo UWB MaUWB para realizar mediciones de distancia y velocidad con una precisión de hasta 500 metros. El módulo UWB se integra con el ESP32-S3, que se encarga de la comunicación Wi-Fi y Bluetooth. El MaUWB también soporta Ethernet y PoE, lo que lo hace ideal para aplicaciones de ubicación en interiores.

## ¿Por qué importa?

El MaUWB es importante porque permite crear sistemas de ubicación en tiempo real en interiores, lo que puede ser útil para aplicaciones como seguimiento de dispositivos en hogares y oficinas. El MaUWB también soporta PoE, lo que lo hace más fácil de instalar y mantener.

## Consejo técnico

Si deseas utilizar el MaUWB con Home Assistant, asegúrate de seguir la guía de configuración de Makerfabs para configurar el módulo UWB y establecer las roles de los dispositivos.

```bash
uwbtools.exe (herramienta de configuración de Makerfabs)
```
