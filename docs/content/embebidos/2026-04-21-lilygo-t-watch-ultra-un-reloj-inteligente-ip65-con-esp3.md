# LILYGO T-Watch Ultra: Un reloj inteligente IP65 con ESP32-S3 y AMOLED

**Fecha:** 2026-04-21
**Categoría:** embebidos
**Tags:** esp32, arduino, robotica, embebidos
**Título original:** LILYGO T-Watch Ultra – An IP65-rated ESP32-S3 smartwatch with 2.01-inch AMOLED, LoRa, and GNSS

---

## Introducción

LILYGO ha lanzado el T-Watch Ultra, un reloj inteligente con pantalla AMOLED de 2.01 pulgadas y un SoC ESP32-S3. Este dispositivo es ideal para aplicaciones de IoT, seguimiento de GPS y desarrollo de firmware personalizado.

## ¿Qué es?

El T-Watch Ultra es un reloj inteligente basado en el SoC ESP32-S3, que cuenta con una pantalla AMOLED de 2.01 pulgadas y un diseño IP65. Esto significa que es resistente a la agua y el polvo. El dispositivo también incluye un módulo GNSS, un transceptor LoRa y un sensor de movimiento inteligente.

## ¿Cómo funciona?

El T-Watch Ultra funciona con el SoC ESP32-S3, que cuenta con un procesador dual-core de 240 MHz y 512 KB de SRAM. La pantalla AMOLED de 2.01 pulgadas ofrece una resolución de 410 x 502 píxeles y soporta 16,7 millones de colores. El dispositivo también cuenta con una batería de 1.100 mAh y un puerto USB-C para carga y programación.

## ¿Por qué importa?

El T-Watch Ultra es importante porque ofrece una solución completa para aplicaciones de IoT, seguimiento de GPS y desarrollo de firmware personalizado. El dispositivo es ideal para proyectos que requieren una pantalla AMOLED y un SoC potente, como el ESP32-S3.

## Consejo técnico

Si deseas programar el T-Watch Ultra con Arduino IDE, asegúrate de instalar la versión 3.3.0-alpha1 o superior del core Arduino-ESP32. Además, configura las opciones del board para el radio correcto.

```bash
Arduino-ESP32 core versión 3.3.0-alpha1 o superior
```
