# Convierte tu LILYGO T-LoRa Pager en un reproductor de MP3 y radio por Internet

**Fecha:** 2026-04-26
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** Melody Machine firmware transforms LILYGO T-LoRa Pager into a portable MP3 player and Internet radio

---

## Introducción

El proyecto Melody Machine permite transformar el dispositivo LILYGO T-LoRa Pager en un reproductor de MP3 y radio por Internet, ofreciendo una experiencia de usuario gráfica y funcionalidades como la reproducción de música desde una tarjeta SD y la conexión a Internet.

## ¿Qué es?

El proyecto Melody Machine es un firmware open-source que se puede instalar en el dispositivo LILYGO T-LoRa Pager, convirtiéndolo en un reproductor de MP3 y radio por Internet. Esto se logra mediante la utilización de la tecnología ESP32-S3 y la biblioteca LVGL para crear una interfaz gráfica de usuario.

## ¿Cómo funciona?

El firmware utiliza la tecnología ESP32-S3 para reproducir música desde una tarjeta SD y conectarse a Internet para reproducir radios en vivo. La interfaz gráfica de usuario se crea utilizando la biblioteca LVGL, lo que permite al usuario navegar por la lista de reproducción, seleccionar canciones y ajustar el volumen.

## ¿Por qué importa?

Este proyecto importa porque ofrece una forma de convertir un dispositivo existente en un reproductor de MP3 y radio por Internet, lo que puede ser útil para aquellos que buscan una solución económica y flexible para reproducir música y escuchar radios en vivo.

## Consejo técnico

Si deseas instalar el firmware Melody Machine en tu LILYGO T-LoRa Pager, asegúrate de copiar la música en el directorio mp3 y la lista de reproducción en el directorio m3u, y configura tus credenciales de Wi-Fi en el archivo settings.json.

```bash
esptool
```
