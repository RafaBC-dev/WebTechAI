# Peppe's Ghost: un escáner LiDAR casero con Raspberry Pi 5

## Introducción

Un ingeniero ha creado un escáner LiDAR casero utilizando un Raspberry Pi 5 y un diseño 3D impresionado. El proyecto, llamado Peppe's Ghost, captura el mundo como un punto nube y ha generado interés en la comunidad de diseño y robótica.

## ¿Qué es?

Peppe's Ghost es un escáner LiDAR casero que utiliza un Raspberry Pi 5 como cerebro y un diseño 3D impresionado como carcasa. El escáner utiliza un sensor LiDAR para capturar datos tridimensionales del entorno y un sistema de cámaras para capturar imágenes en color.

## ¿Cómo funciona?

El Raspberry Pi 5 gestiona el escáner LiDAR y las cámaras, sincronizando los datos en tiempo real. El sensor LiDAR envía datos a través de la conexión Ethernet, mientras que las cámaras capturan imágenes en color a través de los puertos CSI. Un SSD se utiliza para almacenar los datos de punto nube generados por el escáner.

## ¿Por qué importa?

Peppe's Ghost tiene aplicaciones en campos como la arquitectura, el diseño y la robótica. El escáner puede ser utilizado para capturar datos tridimensionales de edificios, monumentos y otros objetos, lo que puede ser útil para la planificación y el diseño de proyectos.

## Consejo técnico

Si deseas crear un escáner LiDAR casero similar, puedes utilizar el software de Raspberry Pi para configurar el escáner y el sistema de cámaras. Recuerda que debes utilizar un sensor LiDAR compatible con el Raspberry Pi y un diseño 3D impresionado adecuado para la carcasa.

```bash
sudo apt-get install libraspberrypi-dev
```
