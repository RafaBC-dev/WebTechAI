# DEBIX T62P-01: un SBC industrial para IoT y Edge Computing

**Fecha:** 2026-06-29
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux

## Introducción

Polyhex Technology ha lanzado el DEBIX T62P-01, un SBC compacto para aplicaciones industriales que integra un SoC Texas Instruments Sitara AM62P y ofrece conectividad de red y Wi-Fi 6. Este dispositivo está diseñado para uso en Internet de las cosas (IoT), Edge Computing y terminales inteligentes.

## ¿Qué es?

El DEBIX T62P-01 es un SBC industrial que se basa en el SoC Texas Instruments Sitara AM62P, un procesador cuádruple Cortex-A53 con frecuencia de reloj de hasta 1,4 GHz y un núcleo real-time Cortex-R5F. Este dispositivo ofrece conectividad de red a través de dos puertos GbE, incluyendo uno con soporte para PoE, y también cuenta con Wi-Fi 6 y Bluetooth 5.4.

## ¿Cómo funciona?

El DEBIX T62P-01 cuenta con 2 GB de memoria LPDDR4 estándar, con opciones de 4 GB y 8 GB. También ofrece almacenamiento mediante eMMC flash hasta 256 GB y una ranura para tarjeta MicroSD. El dispositivo cuenta con una variedad de puertos de expansión, incluyendo dos puertos USB 2.0, un puerto USB Type-C para depuración y un puerto USB Type-C para alimentación.

## ¿Por qué importa?

El DEBIX T62P-01 es relevante para aplicaciones industriales que requieren conectividad de red y procesamiento de datos en tiempo real. Puede ser utilizado en Edge Computing, IoT y terminales inteligentes, lo que lo hace una opción interesante para empresas que buscan mejorar la eficiencia y la seguridad de sus operaciones.

## Consejo técnico

Si estás interesado en utilizar el DEBIX T62P-01 para Edge Computing, considera utilizar la tecnología de seguridad integrada del dispositivo, como el Secure Boot y el TrustZone, para proteger tus datos y aplicaciones.
