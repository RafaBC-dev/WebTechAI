# Nueva placa IoT con ESP32-C5 y pantalla LCD de 1,47 pulgadas

**Fecha:** 2026-07-04
**Categoría:** embebidos
**Tags:** esp32, robotica, embebidos
**Título original:** ESP32-C5 dual-band WiFi 6 and Bluetooth LE IoT board features 1.47-inch color LCD

## Introducción

La placa IoT Waveshare ESP32-C5-LCD-1.47 ofrece conectividad WiFi 6 y Bluetooth LE, junto con una pantalla LCD de alta resolución. Esta placa es ideal para proyectos de IoT que requieren una conexión estable y una interfaz gráfica intuitiva.

## ¿Qué es?

La placa Waveshare ESP32-C5-LCD-1.47 es una placa de desarrollo de IoT basada en el chip ESP32-C5, que ofrece conectividad WiFi 6 y Bluetooth LE. La placa cuenta con una pantalla LCD de 1,47 pulgadas con resolución 320x172, un slot de tarjeta microSD y dos headers de expansión de 9 pines.

## ¿Cómo funciona?

La placa ESP32-C5-LCD-1.47 utiliza el chip ESP32-C5, que cuenta con un procesador RISC-V de 32 bits a 240 MHz y 384 KB de memoria SRAM. La placa también cuenta con una antena cerámica integrada y un conversor de voltaje MP1605GTF para suministrar 5V a la placa.

## ¿Por qué importa?

La placa ESP32-C5-LCD-1.47 es importante porque ofrece una conexión estable y una interfaz gráfica intuitiva para proyectos de IoT. La placa es ideal para aplicaciones como monitoreo de actividad WiFi, medición de proximidad con Bluetooth LE y visualización de datos en una pantalla LCD.

## Consejo técnico

Si deseas utilizar la placa ESP32-C5-LCD-1.47 con el ESP-IDF framework, puedes comenzar con el ejemplo de código proporcionado en la wiki de Waveshare y adaptarlo a tus necesidades. También puedes utilizar la biblioteca LVGL para crear interfaces gráficas personalizadas.

```bash
esp32-idf flash --chip esp32 --port /dev/ttyUSB0 --baud 115200
```
