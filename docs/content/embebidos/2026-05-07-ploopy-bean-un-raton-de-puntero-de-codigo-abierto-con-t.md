# Ploopy Bean: un ratón de puntero de código abierto con tecnología de vanguardia

**Fecha:** 2026-05-07

**Tags:** robotica, microcontroladores, software-y-linux

## Introducción

La empresa canadiense Ploopy ha lanzado un nuevo producto innovador: un ratón de puntero de código abierto llamado Ploopy Bean, que utiliza un microcontrolador Raspberry Pi RP2040 y una tecnología de puntero de alta precisión.

## ¿Qué es?

El Ploopy Bean es un ratón de puntero de código abierto que se puede imprimir en 3D y que utiliza un microcontrolador Raspberry Pi RP2040 para controlar cuatro botones Omron D2LS-21 y un nub de fricción. La tecnología de puntero utiliza un sensor de efecto Hall de alta precisión de Texas Instruments TMAG5273.

## ¿Cómo funciona?

El Ploopy Bean funciona mediante un microcontrolador Raspberry Pi RP2040 que lee los datos del sensor de efecto Hall y controla los cuatro botones Omron D2LS-21. El nub de fricción utiliza una tecnología de alta precisión para detectar movimientos de hasta 20,000 ksps.

## ¿Por qué importa?

El Ploopy Bean es relevante porque ofrece una alternativa de código abierto a los ratones de puntero tradicionales y permite a los usuarios personalizar su experiencia de usuario mediante la configuración de los botones y la tecnología de puntero.

## Consejo técnico

Si deseas personalizar tu Ploopy Bean, puedes utilizar la herramienta VIA web para agregar una configuración de archivo y cambiar la configuración de los botones y la tecnología de puntero.
