# Impresión de circuitos impresos con 3D Printers: una solución rápida

**Fecha:** 2026-05-20
**Categoría:** embebidos
**Tags:** diseño-3d, electronica, herramientas

## Introducción

La impresión de circuitos impresos ha mejorado en la última década, pero la creación de prototipos rápidos sigue siendo un desafío. Ahora, un equipo de investigación ha encontrado una solución innovadora utilizando 3D Printers.

## ¿Qué es?

La impresión de circuitos impresos con 3D Printers consiste en crear circuitos impresos personalizados utilizando un impresor 3D. Se utiliza software de diseño asistido por computadora (CAD) para crear el diseño del circuito, que luego se imprime en un material plástico. Posteriormente, se aplica cinta de cobre para hacerlo conductor y se cortan las trazas a lo largo de sus bordes elevados.

## ¿Cómo funciona?

El proceso comienza con el diseño del circuito en software de CAD, donde se crean trazas gruesas para mantener su fuerza. Luego, se exporta el diseño y se abre en software de CAD 3D, donde se elevan las trazas a 2 mm. Finalmente, se imprime el circuito en un 3D Printer y se aplica cinta de cobre para hacerlo conductor.

## ¿Por qué importa?

Esta solución rápida para la impresión de circuitos impresos es importante porque permite a los diseñadores y desarrolladores crear prototipos rápidos y asequibles, lo que acelera el proceso de desarrollo de productos. Además, es una alternativa más accesible que la milla de circuitos impresos utilizando una máquina de control numérico (CNC).

## Consejo técnico

Si deseas intentar esta técnica, utiliza el software de diseño asistido por computadora (CAD) KiCad para crear el diseño del circuito y el software de CAD 3D Tinkercad para elevar las trazas. Recuerda que las trazas deben ser gruesas para mantener su fuerza.

```bash
kiCad exportar diseño: 'File > Export > PCB > KiCad PCB file (.kicad_pcb)'
```
