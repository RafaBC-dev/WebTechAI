# Boardcon lanza el Tiny1126B, un módulo de sistema en un solo chip más pequeño y ligero

**Fecha:** 2026-04-18
**Categoría:** embebidos
**Tags:** robotica, ia-local, linux
**Título original:** Boardcon Tiny1126B is a smaller and lighter Rockchip RV1126B system-on-module, yet with more I/Os

---

## Introducción

Boardcon ha presentado el Tiny1126B, un módulo de sistema en un solo chip (SoC) más pequeño y ligero que su antecesor, el MINI1126B-P. Este módulo está diseñado para ser utilizado en sistemas de visión artificial para cámaras inteligentes, puertas inteligentes, cámaras de inspección y robots de limpieza.

## ¿Qué es?

El Tiny1126B es un módulo de sistema en un solo chip que utiliza el procesador Rockchip RV1126B, un cuatocore con tecnología ARM Cortex-A53. Este módulo incluye un acelerador de inteligencia artificial (IA) con capacidad para 3 TOPS (INT8), lo que lo hace ideal para aplicaciones de visión artificial y procesamiento de imágenes.

## ¿Cómo funciona?

El Tiny1126B funciona conectando dos conectores de 100 pines de 0,4 mm de diámetro, que ofrecen más entradas/salidas (I/O) que los conectores de 80 pines de 0,5 mm de diámetro utilizados en su antecesor. Esto permite una mayor flexibilidad y capacidad para conectar dispositivos externos. El módulo también incluye un conector USB 3.0 DRD (Dual-Role Device) y un conector de almacenamiento SDMMC.

## ¿Por qué importa?

El Tiny1126B es importante porque ofrece una solución compacta y eficiente para aplicaciones de visión artificial y procesamiento de imágenes. Esto lo hace ideal para sistemas de cámaras inteligentes, puertas inteligentes, cámaras de inspección y robots de limpieza.

## Consejo técnico

Si estás desarrollando un proyecto de visión artificial, considera utilizar el Tiny1126B como módulo de sistema en un solo chip. Puedes utilizar el SDK de Boardcon para desarrollar aplicaciones y conectar dispositivos externos a través de los conectores de 100 pines.

```bash
Para conectar el Tiny1126B a un sistema Linux, puedes utilizar el comando `sudo apt-get install librockchip-ia` para instalar la biblioteca de IA de Rockchip.
```
