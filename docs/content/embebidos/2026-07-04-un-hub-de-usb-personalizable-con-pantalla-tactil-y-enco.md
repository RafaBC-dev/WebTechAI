# Un hub de USB personalizable con pantalla táctil y encoder rotativo

**Fecha:** 2026-07-04
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** HALO TOUCH V2 – A customizable desktop USB hub with rotary encoder fitted with round touchscreen display

---

## Introducción

La empresa Innoelement ha lanzado el HALO TOUCH V2, un hub de USB multifuncional con pantalla táctil y encoder rotativo diseñado para ser un compañero de escritorio interactivo. Este dispositivo integra monitoreo de sistema, reproducción de medios y características de productividad para escritorios, oficinas y configuraciones de escritorio.

## ¿Qué es?

El HALO TOUCH V2 es un hub de USB con una pantalla táctil circular de 360x360 píxeles y un encoder rotativo que permite controlar el dispositivo de manera interactiva. Ofrece una variedad de características, incluyendo un temporizador Pomodoro, un reproductor de fotos y animaciones, un reproductor de MP3 y un visualizador de espectro de audio.

## ¿Cómo funciona?

El dispositivo cuenta con un módulo MCU no especificado (potencialmente ESP32-S3) que gestiona la interacción con el usuario y la conexión a la red. La pantalla táctil permite realizar gestos como deslizar hacia arriba para abrir el menú y deslizar hacia la izquierda o derecha para navegar entre aplicaciones. El encoder rotativo proporciona control similar al de la Microsoft Surface Dial en Windows.

## ¿Por qué importa?

El HALO TOUCH V2 importa porque ofrece una forma innovadora de interactuar con el escritorio y el sistema operativo, permitiendo a los usuarios personalizar su experiencia de trabajo y entretenimiento. También ofrece características de productividad y monitoreo de sistema que pueden ser útiles en entornos de trabajo y estudio.

## Consejo técnico

Si deseas personalizar el HALO TOUCH V2 con temas y animaciones, puedes utilizar archivos en una tarjeta microSD para configurar el dispositivo según tus preferencias.
