# WipperSnapper para Raspberry Pi y más: Adafruit IO y Adafruit Learning System

## Introducción

Adafruit ha lanzado una serie de actualizaciones importantes en su plataforma de IoT y aprendizaje en línea. Entre ellas, WipperSnapper para Raspberry Pi, una herramienta que facilita la conexión de dispositivos a la plataforma, y una actualización de la herramienta de aprendizaje en línea que mejora la visualización de código.

## ¿Qué es?

WipperSnapper es una herramienta que permite conectar dispositivos a la plataforma de IoT de Adafruit, Adafruit IO, de manera sencilla y rápida. También se ha actualizado la herramienta de aprendizaje en línea, Adafruit Learning System, para mejorar la visualización de código y facilitar el aprendizaje de programación.

## ¿Cómo funciona?

WipperSnapper funciona mediante un instalador único que se puede ejecutar en un terminal, lo que facilita la conexión de dispositivos a la plataforma de IoT. La herramienta de aprendizaje en línea ha sido actualizada para que los elementos de código se puedan desplegar automáticamente y se mejoren las características de visualización y descarga de código.

## ¿Por qué importa?

Estas actualizaciones son importantes porque facilitan el acceso a la plataforma de IoT y el aprendizaje de programación para desarrolladores y usuarios no expertos. También permiten a los usuarios crear proyectos de IoT de manera más sencilla y rápida.

## Consejo técnico

Si deseas empezar a trabajar con WipperSnapper, puedes empezar ejecutando el comando `curl -fsSL https://get.wippersnapper.io | sh` en tu terminal, lo que instalará la herramienta y la configurará para que funcione correctamente.

```bash
curl -fsSL https://get.wippersnapper.io | sh
```
