# Nueva familia de microcontroladores Fortior FU75xx para control de motores

## Introducción

Fortior Technology ha lanzado la familia de microcontroladores FU75xx, diseñada para controlar motores de manera eficiente y precisa. Estos microcontroladores son ideales para aplicaciones de alta velocidad y control en tiempo real, como sistemas de movimiento y control de motores en industria y vehículos eléctricos.

## ¿Qué es?

La familia de microcontroladores FU75xx es un conjunto de dispositivos que combinan un núcleo RISC-V de 32 bits con el núcleo Motor Engine (ME2) de segunda generación. El núcleo RISC-V se utiliza para la configuración de parámetros y el procesamiento rutinario, mientras que el núcleo ME2 integra módulos de control de flujo de corriente (FOC) y cálculo de coordenadas de Dirichlet (CORDIC) para realizar cálculos rápidos y precisos.

## ¿Cómo funciona?

Los microcontroladores FU75xx funcionan integrando el núcleo RISC-V y el núcleo ME2 para realizar cálculos y controlar motores de manera eficiente. El núcleo ME2 utiliza el módulo FOC para controlar motores con flujo de corriente y el módulo CORDIC para realizar cálculos de coordenadas de Dirichlet. Esto permite realizar cálculos rápidos y precisos para controlar motores de manera eficiente.

## ¿Por qué importa?

La familia de microcontroladores FU75xx es importante porque ofrece una solución eficiente y precisa para controlar motores en aplicaciones de alta velocidad y control en tiempo real. Estos microcontroladores son ideales para sistemas de movimiento y control de motores en industria y vehículos eléctricos, donde la precisión y la eficiencia son fundamentales.
