# M5Stack Stamp-C5: el módulo IoT más pequeño con ESP32-C5

**Fecha:** 2026-07-11
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, esp32
**Título original:** M5Stack Stamp-C5 – A Tiny ESP32-C5 USB-C board/module with up to 19 GPIO, 3.7V battery support

---

## Introducción

M5Stack ha lanzado el Stamp-C5, un módulo IoT extremadamente pequeño basado en el ESP32-C5, que ofrece conectividad WiFi 6, Bluetooth LE y 802.15.4, y hasta 19 pines GPIO.

## ¿Qué es?

El M5Stack Stamp-C5 es un módulo IoT compacto que utiliza el procesador RISC-V ESP32-C5, con un solo núcleo de 32 bits a 240 MHz, 384 KB de SRAM y 320 KB de ROM, además de 4 MB de SPI flash. Ofrece conectividad dual-banda WiFi 6, Bluetooth LE y 802.15.4, y cuenta con hasta 19 pines GPIO.

## ¿Cómo funciona?

El Stamp-C5 utiliza el procesador ESP32-C5 para proporcionar conectividad WiFi 6, Bluetooth LE y 802.15.4, y cuenta con una arquitectura de 2x 7-pines y 1x 12-pines para expansiones. El módulo también incluye un conector IPEX para una antena externa y un puerto USB-C para programación y alimentación.

## ¿Por qué importa?

El Stamp-C5 es importante porque ofrece una solución compacta y potente para aplicaciones IoT, con una gran cantidad de GPIO y conectividad avanzada. Es ideal para proyectos que requieren una gran flexibilidad y capacidad de expansión.

## Consejo técnico

Si deseas aprovechar al máximo el Stamp-C5, asegúrate de utilizar la biblioteca de Arduino para ESP32-C5, que ofrece una gran cantidad de funciones y ejemplos para ayudarte a empezar.

```bash
sudo apt-get install arduino-esp32
```
