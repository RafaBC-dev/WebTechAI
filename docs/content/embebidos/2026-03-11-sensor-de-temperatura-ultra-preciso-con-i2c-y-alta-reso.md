## Introducción

Adafruit ha lanzado un nuevo sensor de temperatura ultra preciso que ofrece una resolución de 16 bits y una precisión de ±0.03°C en un rango de -55 a 155°C. Este sensor es ideal para aplicaciones que requieren una medición precisa de la temperatura.

## ¿Qué es?

El TMP119 es un sensor de temperatura I2C que permite agregar una medición precisa de la temperatura a cualquier proyecto. Ofrece una resolución de 16 bits y una precisión de ±0.03°C en un rango de 0 a 45°C. El sensor también puede monitorear la temperatura y alertar cuando sea necesario tomar acción correctiva.

## ¿Cómo funciona?

El TMP119 utiliza la tecnología I2C para comunicarse con el microcontrolador o la placa de desarrollo. El sensor tiene una alta resolución de 16 bits, lo que significa que puede medir pequeñas variaciones en la temperatura. El sensor también tiene una función de alerta para notificar cuando la temperatura sea demasiado alta o baja.

## ¿Por qué importa?

Este sensor es ideal para aplicaciones que requieren una medición precisa de la temperatura, como termostatos, seguimiento de activos en cadena fría, medidores de gas y calor. El TMP119 también es compatible con placas de desarrollo como Arduino y Raspberry Pi.

## Consejo técnico

Si deseas utilizar el TMP119 con tu proyecto, asegúrate de leer la documentación de la placa de desarrollo y la librería de código correspondiente. También es importante configurar correctamente el sensor y el microcontrolador para obtener resultados precisos.

```bash
python -m pip install adafruit-circuitpython-tmp119
```
