# ESP32-S31: un microcontrolador dual-core RISC-V con conectividad inalámbrica

**Fecha:** 2026-03-25
**Categoría:** embebidos
**Tags:** microcontroladores, esp32, embebidos

## Introducción

Espressif Systems ha lanzado un nuevo microcontrolador inalámbrico potente, el ESP32-S31, que cuenta con una arquitectura dual-core RISC-V y conectividad Gigabit Ethernet, WiFi, Bluetooth y 802.15.4.

## ¿Qué es?

El ESP32-S31 es un microcontrolador dual-core RISC-V que cuenta con un núcleo de alto rendimiento con FPU y SIMD, y un núcleo de bajo consumo RISC-V. Ofrece 62 pines GPIO, un MAC Gigabit Ethernet, WiFi, Bluetooth y conectividad 802.15.4 para Zigbee, Thread y Matter.

## ¿Cómo funciona?

El ESP32-S31 cuenta con un núcleo de alto rendimiento que ejecuta instrucciones RISC-V con FPU y SIMD, lo que le permite realizar cálculos matemáticos y operaciones de procesamiento de imágenes de manera eficiente. El núcleo de bajo consumo se encarga de realizar tareas de bajo consumo, como la gestión de sensores y la comunicación inalámbrica.

## ¿Por qué importa?

El ESP32-S31 es relevante porque ofrece una solución de microcontrolador inalámbrico potente y flexible, ideal para aplicaciones que requieren conectividad inalámbrica y procesamiento de datos en tiempo real, como sensores de IoT, robots y dispositivos de automatización.

## Consejo técnico

Si estás trabajando con el ESP32-S31, asegúrate de utilizar la biblioteca de desarrollo ESP-IDF para aprovechar al máximo las características del microcontrolador y reducir el tiempo de desarrollo.
