# Microcontrolador de bajo costo para control de motores

## Introducción

Microchip ha lanzado un nuevo microcontrolador de bajo costo para aplicaciones de control de motores, que ofrece un rendimiento determinista de hasta 100 MIPS y una integración de alta resolución de PWM y ADC.

## ¿Qué es?

El dsPIC33CK es una familia de microcontroladores de 16 bits de la línea dsPIC33, diseñados para aplicaciones de control de motores y precisión. Ofrecen una amplia gama de características, incluyendo memoria flash de hasta 256 KB, un ADC de 12 bits y un PWM de alta resolución.

## ¿Cómo funciona?

El dsPIC33CK funciona a una frecuencia de hasta 100 MHz y cuenta con un DSP de 40 bits con acumuladores de 40 bits. También ofrece una variedad de periféricos, incluyendo UART, I2C, SPI, CAN FD y SENT. La familia también incluye una serie de opciones de seguridad, como el código de protección y el monitoreo de reloj.

## ¿Por qué importa?

El dsPIC33CK es importante porque ofrece una solución de bajo costo para aplicaciones de control de motores y precisión. Puede ser utilizado en una variedad de aplicaciones, incluyendo motores de corriente continua, motores de corriente alterna y sistemas de control de velocidad.

## Consejo técnico

Si estás diseñando un sistema de control de motores, considera utilizar el dsPIC33CK por su rendimiento determinista y su integración de alta resolución de PWM y ADC. Puedes utilizar el MPLAB X IDE y el MPLAB XC-DSC compiler para programar y configurar el microcontrolador.

```bash
microchip dspic33ck
```
