## Introducción

El proyecto Typeframe PS-85 es un cyberdeck creado por Jeff Merrick, un maker que se inspiró en la película Alien para diseñar un dispositivo portátil que combine una Raspberry Pi con un teclado mecánico y una pantalla táctil. Este proyecto es relevante ahora mismo porque muestra la creatividad y la innovación que se pueden lograr con tecnología de código abierto.

## ¿Qué es?

El Typeframe PS-85 es un cyberdeck que se basa en una Raspberry Pi Zero 2 W y un teclado mecánico de 65% con keycaps inspirados en la película Alien. El dispositivo tiene una pantalla táctil y es portátil, lo que lo hace ideal para ser utilizado en entornos de trabajo o de entretenimiento.

## ¿Cómo funciona?

El Typeframe PS-85 funciona conectando la Raspberry Pi a un teclado mecánico y una pantalla táctil. El sistema operativo utilizado es Linux, y el dispositivo puede ser programado utilizando herramientas como Python y Adafruit CharliePlex. El proyecto también incluye un sistema de GPIO expuesto, que permite conectar periféricos adicionales.

## ¿Por qué importa?

Este proyecto importa porque muestra la posibilidad de crear dispositivos portátiles y personalizados utilizando tecnología de código abierto. El Typeframe PS-85 puede ser utilizado en una variedad de aplicaciones, desde la escritura y la programación hasta la creación de contenido multimedia.

## Consejo técnico

Si deseas crear un proyecto similar, considera utilizar una Raspberry Pi y un teclado mecánico de 65% con keycaps personalizados. También puedes utilizar herramientas como Adafruit CharliePlex para crear un sistema de GPIO expuesto y conectar periféricos adicionales.

```bash
sudo apt-get install python3-pip
```
