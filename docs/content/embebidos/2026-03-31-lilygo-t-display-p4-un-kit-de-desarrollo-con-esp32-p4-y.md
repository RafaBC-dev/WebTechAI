# LILYGO T-Display-P4: un kit de desarrollo con ESP32-P4 y ESP32-C6

**Fecha:** 2026-03-31
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** T-Display-P4 smartphone-like devkit features ESP32-P4 MCU, ESP32-C6 wireless SoC, and SX1262/LR2021 LoRa transceiver

## Introducción

LILYGO ha lanzado el T-Display-P4, un kit de desarrollo que combina la potencia del ESP32-P4 con la conectividad del ESP32-C6. Este dispositivo ofrece una amplia gama de características, incluyendo GPS, Ethernet y LoRaWAN, lo que lo convierte en una herramienta ideal para proyectos de IoT y robótica.

## ¿Qué es?

El T-Display-P4 es un kit de desarrollo que se basa en el ESP32-P4, un microcontrolador dual-core RISC-V con instrucciones de AI y un procesador gráfico 2D. También incluye el ESP32-C6, un módulo de conectividad que ofrece Wi-Fi 6, Bluetooth 5.x y 802.15.4. El dispositivo cuenta con una pantalla táctil de 4,05 o 4,1 pulgadas, una cámara de 2MP y una variedad de conectores y puertos para expansiones.

## ¿Cómo funciona?

El T-Display-P4 funciona mediante el ESP32-P4, que es responsable de la ejecución de la lógica de programación y la gestión de los sensores y actuadores. El ESP32-C6 se encarga de la conectividad inalámbrica, permitiendo a la placa comunicarse con dispositivos remotos a través de Wi-Fi, Bluetooth o LoRaWAN. La pantalla táctil permite al usuario interactuar con el dispositivo y visualizar datos en tiempo real.

## ¿Por qué importa?

El T-Display-P4 es importante porque ofrece una plataforma de desarrollo completa para proyectos de IoT y robótica. Con su capacidad para comunicarse con dispositivos remotos y su pantalla táctil, este kit es ideal para proyectos que requieren una gran cantidad de datos y una interfaz de usuario intuitiva. Además, la integración de LoRaWAN permite a los desarrolladores aprovechar las redes de IoT para enviar y recibir datos de manera eficiente.

## Consejo técnico

Si estás desarrollando un proyecto de IoT o robótica, considera utilizar el T-Display-P4 como plataforma de desarrollo. Puedes utilizar la biblioteca de drivers de ESP32 para programar el dispositivo y aprovechar sus características de conectividad y pantalla táctil. Recuerda que es importante leer la documentación de la biblioteca y ajustar el código según sea necesario para asegurarte de que funcione correctamente.

```bash
esp32/boards/esp32-p4/
```
