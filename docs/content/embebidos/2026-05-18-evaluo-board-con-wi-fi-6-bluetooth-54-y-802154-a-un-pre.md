# Evalúo board con Wi-Fi 6, Bluetooth 5.4 y 802.15.4 a un precio inmejorable

**Fecha:** 2026-05-18
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** $13 Qualcomm QCC74xM EVK offers Wi-Fi 6, Bluetooth 5.4, and 802.15.4 connectivity (Bouffalo Lab BL618 inside)

## Introducción

La empresa Bouffalo Lab ha lanzado un evalúo board llamado BL618, que es básicamente igual que el Qualcomm QCC74xM EVK. Este board ofrece Wi-Fi 6, Bluetooth 5.4 y 802.15.4 a un precio increíblemente bajo de solo $13.

## ¿Qué es?

El Qualcomm QCC74xM EVK es un evalúo board que ofrece Wi-Fi 6, Bluetooth 5.4 y 802.15.4. Está basado en el procesador RISC-V RV32IMAFCP y cuenta con 484 KB de SRAM y 8 MB de pSRAM. También incluye una interfaz I2S para audio y una interfaz DBI para display.

## ¿Cómo funciona?

El board utiliza el procesador RISC-V RV32IMAFCP para ejecutar el código y cuenta con una memoria de 484 KB de SRAM y 8 MB de pSRAM. También incluye una interfaz I2S para audio y una interfaz DBI para display. La comunicación se realiza a través de Wi-Fi 6, Bluetooth 5.4 y 802.15.4.

## ¿Por qué importa?

Este board es importante porque ofrece una forma económica de desarrollar proyectos que requieren Wi-Fi 6, Bluetooth 5.4 y 802.15.4. También es útil para desarrollar proyectos de IoT que requieren una conexión a Internet.

## Consejo técnico

Si estás desarrollando un proyecto de IoT que requiere Wi-Fi 6, Bluetooth 5.4 y 802.15.4, considera utilizar el Qualcomm QCC74xM EVK como evalúo board. Puedes conectarlo a tu dispositivo y probar la comunicación Wi-Fi y Bluetooth.

```bash
Para conectar el board a tu dispositivo, puedes utilizar el comando `sudo apt-get install libqcc74x-dev` en tu sistema Linux. Luego, puedes utilizar el comando `qcc74x-evk` para configurar el board y establecer la comunicación Wi-Fi y Bluetooth.
```
