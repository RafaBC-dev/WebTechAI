## Introducción

La comunidad de Raspberry Pi sigue creando proyectos innovadores, como el cyberdeck Typeframe PS-85, un dispositivo inspirado en la película Alien que combina una Raspberry Pi con una pantalla táctil y un teclado mecánico.

## ¿Qué es?

El Typeframe PS-85 es un proyecto de retroinformática que utiliza una Raspberry Pi Zero 2 W como base y se enfoca en la creación de un dispositivo portátil que pueda ser utilizado para escribir y navegar por Internet. El proyecto se inspira en la película Alien y utiliza una pantalla táctil y un teclado mecánico para su interfaz.

## ¿Cómo funciona?

El Typeframe PS-85 utiliza una Raspberry Pi Zero 2 W como procesador y se conecta a una pantalla táctil y un teclado mecánico. El dispositivo utiliza el sistema operativo Linux y se puede configurar para utilizar una variedad de aplicaciones y herramientas. El proyecto también utiliza la tecnología de hot-swap para permitir que el usuario cambie fácilmente la batería del dispositivo.

## ¿Por qué importa?

El Typeframe PS-85 es importante porque muestra la creatividad y la innovación de la comunidad de Raspberry Pi. El proyecto también demuestra la posibilidad de crear dispositivos portátiles y sostenibles que puedan ser utilizados para una variedad de tareas.
