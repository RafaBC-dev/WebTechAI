# Fortior introduce la familia de microcontroladores FU75xx para control de motores

**Fecha:** 2026-06-12
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** Fortior FU75xx dual-core motor control MCU family combines 32-bit RISC-V core with 2nd-gen Motor Engine (ME2) core

---

## Introducción

Fortior Technology ha lanzado la familia de microcontroladores FU75xx, diseñada para controlar motores con alta precisión y velocidad. Estos microcontroladores están destinados a aplicaciones de alta velocidad y control en tiempo real en la robótica y los sistemas de movimiento.

## ¿Qué es?

La familia de microcontroladores FU75xx es un conjunto de microcontroladores dual-core que combina un núcleo RISC-V de 32 bits con el núcleo Motor Engine (ME2) de segunda generación. El núcleo RISC-V se utiliza para la configuración de parámetros y el procesamiento rutinario, mientras que el núcleo ME integra módulos de FOC y CORDIC que permiten cálculos rápidos de control de velocidad y control de onda cuadrada para motores BLDC/PMSM sensados o sin sensores.

## ¿Cómo funciona?

El microcontrolador FU75xx cuenta con un núcleo RISC-V que opera a 48 MHz y un núcleo ME2 que también opera a 48 MHz. El núcleo ME2 incluye módulos de FOC y CORDIC que permiten cálculos rápidos de control de velocidad y control de onda cuadrada. El microcontrolador también cuenta con una memoria de 12 kB de SRAM y 4 kB de PRAM para la ejecución del programa, así como una memoria de 64 kB de Flash con ECC y CRC para la protección del código.

## ¿Por qué importa?

La familia de microcontroladores FU75xx es importante porque ofrece una solución de alta precisión y velocidad para controlar motores en aplicaciones de robótica y sistemas de movimiento. Estos microcontroladores pueden ser utilizados en aplicaciones como motores de servo, articulaciones robóticas, electrodomésticos inteligentes y sistemas de vehículos eléctricos.

## Consejo técnico

Si deseas utilizar la familia de microcontroladores FU75xx, es recomendable utilizar la herramienta de desarrollo Fortior Interactive Connectivity (FICE) para configurar y programar el microcontrolador. La FICE ofrece una interfaz gráfica de usuario para configurar los parámetros del microcontrolador y cargar el código de programación.
