# Kit de DIY para un dron miniatura basado en ESP32-S3

**Tags:** robotica, drones, embebidos

## Introducción

La Seeed Studio ha lanzado el ESP-FLY DIY Kit, un kit de DIY para un dron miniatura basado en el ESP32-S3. Este dron es ideal para la educación STEM, proyectos DIY y prácticas de vuelo indoor.

## ¿Qué es?

El ESP-FLY DIY Kit es un kit de DIY para un dron miniatura que utiliza la placa XIAO ESP32-S3 de Seeed Studio. Este dron es muy pequeño, con dimensiones de 67 x 67 x 31mm, lo que lo hace ideal para almacenarlo en un bolsillo o caja pequeña.

## ¿Cómo funciona?

El dron utiliza la placa XIAO ESP32-S3 como controlador principal, que cuenta con un procesador dual-core Tensilica LX7 a 240 MHz, 512 KB de SRAM, 8 MB de PSRAM y conectividad Wi-Fi 4 y Bluetooth 5.0. El dron también cuenta con un sensor IMU MPU6050 y cuatro motores coreless de 70,000 RPM.

## ¿Por qué importa?

El ESP-FLY DIY Kit es importante porque ofrece una forma accesible y divertida de aprender sobre la robótica y la programación de drones. Además, es ideal para prácticas de vuelo indoor y puede ser utilizado en proyectos de educación STEM.

## Consejo técnico

Si deseas empezar a trabajar con el ESP-FLY DIY Kit, asegúrate de descargar el firmware desde GitHub y utilizar la aplicación ESP-Drone Android para controlar el dron desde tu smartphone.

```bash
git clone https://github.com/esp-drone/esp-drone-firmware.git
```
