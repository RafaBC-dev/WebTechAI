# Visualiza señales de radio con QuadRF, un tile de radio definido por software

## Introducción

El proyecto QuadRF permite visualizar señales de radio en el entorno utilizando una aplicación de realidad aumentada. Esta tecnología podría ser útil para identificar dispositivos inalámbricos en un área determinada.

## ¿Qué es?

QuadRF es un tile de radio definido por software que utiliza un Raspberry Pi 5 como procesador principal. Este tile es capaz de capturar señales de radio y renderizar una imagen en vivo de las señales en un dispositivo móvil o computadora.

## ¿Cómo funciona?

QuadRF utiliza cuatro antenas coherentes para medir las diferencias en el tiempo de llegada de las señales. El procesamiento de señales se realiza en el Lattice ECP5 FPGA y el Raspberry Pi 5. La imagen resultante se envía a un dispositivo móvil o computadora para visualización en tiempo real.

## ¿Por qué importa?

QuadRF puede ser útil para identificar dispositivos inalámbricos en un área determinada, como Wi-Fi, cámaras inalámbricas, teléfonos móviles, drones y transmisores de laboratorio.

## Consejo técnico

Si deseas experimentar con QuadRF, puedes descargar el software open-source y utilizarlo con un Raspberry Pi 5 y cuatro antenas coherentes.

```bash
git clone https://github.com/quadrf/quadrf.git
```
