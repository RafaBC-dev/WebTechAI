# ¿La Cadena de Newton es realmente perpetua?

**Fecha:** 2026-07-13
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y Linux
**Título original:** Newton’s Cradle Isn’t Really Perpetual

---

## Introducción

Una creación de hacking ha revelado que la Cadena de Newton, un objeto de oficina popular, no es tan perpetuo como parece. Un pequeño truco electrónico hace que parezca que funciona sin ayuda.

## ¿Qué es?

La Cadena de Newton es un objeto de oficina que consiste en una serie de bolas que se balancean y golpean entre sí, creando un efecto visual atractivo. Sin embargo, un hackero ha demostrado que se puede hacer que parezca perpetuo utilizando un pequeño circuito electrónico.

## ¿Cómo funciona?

El hackero utilizó un ESP32 y un sensor de proximidad para detectar el movimiento de las bolas y enviar un impulso electromagnético en el momento adecuado para mantener el movimiento. El circuito es relativamente simple y utiliza módulos estándar.

## ¿Por qué importa?

Este proyecto es importante porque demuestra cómo se puede utilizar la tecnología para crear objetos que parezcan funcionar de manera mágica. También puede ser utilizado como una herramienta educativa para enseñar a los estudiantes sobre la física y la electrónica.

## Consejo técnico

Si deseas crear tu propia Cadena de Newton perpetua, puedes utilizar el ESP32 y un sensor de proximidad para detectar el movimiento de las bolas. Puedes encontrar más información sobre el proyecto en el artículo original y utilizar la librería de Python para programar el ESP32.

```bash
python -m esptool.py --port /dev/ttyUSB0 --baud 115200
```
