## Introducción

Volos Projects ha presentado un proyecto DIY de rádio internet basado en el desarrollo de la tarjeta de circuito impreso Waveshare ESP32-S3-LCD-1.54 y un sketch de Arduino con una interfaz de usuario estilo Winamp. Este proyecto es una excelente opción para aquellos que buscan crear un dispositivo de entretenimiento casero a un costo razonable.

## ¿Qué es?

El proyecto consiste en un dispositivo de rádio internet que utiliza la tarjeta de circuito impreso Waveshare ESP32-S3-LCD-1.54, que tiene un procesador dual-core ESP32-S3R8 con una velocidad de hasta 240 MHz, 512KB de SRAM, 8MB de PSRAM, WiFi 4 y Bluetooth 5.0 LE + Mesh. El dispositivo también cuenta con una pantalla de 1.54 pulgadas de resolución 240x240, un altavoz y dos micrófonos.

## ¿Cómo funciona?

El dispositivo utiliza el sketch de Arduino para crear una interfaz de usuario estilo Winamp, que permite a los usuarios escuchar estaciones de radio en línea. El sketch utiliza la biblioteca de Arduino para comunicarse con la tarjeta de circuito impreso y controlar la pantalla y el altavoz. Los usuarios pueden agregar sus propias estaciones de radio a la lista de reproducción modificando el código del sketch.

## ¿Por qué importa?

Este proyecto es importante porque ofrece una forma fácil y asequible de crear un dispositivo de entretenimiento casero con una interfaz de usuario personalizable. También permite a los usuarios experimentar con la programación de Arduino y la creación de dispositivos de Internet de las cosas (IoT).
