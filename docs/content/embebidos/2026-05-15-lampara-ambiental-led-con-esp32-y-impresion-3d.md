## Introducción

Adafruit presenta una lámpara ambiental compacta que combina una LED ring WS2812B con un ESP32 D1 Mini y una conexión USB-C. Esta creación es ideal para integrarse con Home Assistant y otras configuraciones de automatización WLED.

## ¿Qué es?

La lámpara ambiental LED es un proyecto de impresión 3D que utiliza un ESP32 D1 Mini como controlador y una LED ring WS2812B para crear un efecto de luz ambiental. Se puede conectar a una fuente de alimentación USB-C y se puede controlar a través de una aplicación o de manera remota.

## ¿Cómo funciona?

El ESP32 D1 Mini se encarga de controlar la LED ring WS2812B, que se conecta a través de una interfaz de comunicación serie. El proyecto utiliza la biblioteca WLED para crear una interfaz de usuario y controlar la lámpara. La conexión USB-C permite alimentar la lámpara y transferir datos.

## ¿Por qué importa?

Esta lámpara ambiental LED es importante porque ofrece una forma sencilla de crear un entorno de iluminación personalizado y automático. Puede ser utilizada en una variedad de entornos, desde hogares hasta oficinas, y se puede integrar con sistemas de automatización como Home Assistant.

## Consejo técnico

Si deseas crear una lámpara similar, puedes descargar los archivos de impresión 3D y el código del proyecto desde MakerWorld y utilizar la biblioteca WLED para crear tu propia interfaz de usuario.
