# nanoBASIC_UNO: Un Basic para Arduino con características de los clásicos

## Introducción

Si te gustaba escribir código en Basic en tu computadora clásica o en un Basic Stamp, ahora puedes volver a disfrutar de esta experiencia con Arduino gracias a nanoBASIC_UNO.

## ¿Qué es?

nanoBASIC_UNO es una implementación de Basic para el microcontrolador Arduino UNO, que permite programar el dispositivo utilizando un lenguaje similar al clásico Basic. Esta versión tiene suficientes extensiones para manejar GPIO, sistemas analógicos y otros componentes del Arduino.

## ¿Cómo funciona?

nanoBASIC_UNO se ejecuta en la RAM del Arduino y puede almacenar programas en la memoria flash. El usuario puede decidir si utilizar números enteros de 16 o 32 bits. El lenguaje es similar al clásico Basic, pero con algunas diferencias, como la necesidad de utilizar el comando PROG para indicar que se está escribiendo un programa.

## ¿Por qué importa?

nanoBASIC_UNO es importante porque permite a los programadores experimentados y principiantes disfrutar de la experiencia de escribir código en Basic en un dispositivo embebido como el Arduino. Esto puede ser especialmente útil para aquellos que desean aprender a programar dispositivos embebidos de manera sencilla y divertida.

## Consejo técnico

Si deseas probar nanoBASIC_UNO, puedes descargar el código y compilarlo en tu computadora utilizando un compilador de C++ o utilizar un IDE como Arduino IDE.

```bash
nanoBASIC_UNO
```
