## Introducción

Raspberry Pi, conocida por sus tarjetas de desarrollo, ha lanzado un podcast para explicar su tecnología de microcontroladores, incluyendo su nuevo chip RP2350.

## ¿Qué es?

Raspberry Pi ha diseñado su propio microcontrolador, RP2350, como sucesor del popular RP2040. Este chip es capaz de realizar tareas complejas y cuenta con características de seguridad robustas.

## ¿Cómo funciona?

El RP2350 utiliza un procesador RISC-V, lo que permite una mayor flexibilidad y personalización. También cuenta con más memoria y I/O que su predecesor, RP2040.

## ¿Por qué importa?

El RP2350 es importante porque ofrece una solución de microcontrolador asequible y potente para proyectos de robótica, IoT y otros aplicativos embebidos.

## Consejo técnico

Si deseas aprovechar al máximo el RP2350, investiga la librería de desarrollo de Raspberry Pi y sus herramientas de programación, como el IDE Thonny.

```bash
sudo apt-get install thonny
```
