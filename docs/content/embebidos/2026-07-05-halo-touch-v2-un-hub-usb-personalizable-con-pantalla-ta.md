# HALO TOUCH V2: un hub USB personalizable con pantalla táctil y encoder rotativo

**Fecha:** 2026-07-05
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y linux
**Título original:** HALO TOUCH V2 – A customizable desktop USB hub with rotary encoder fitted with round touchscreen display

---

## Introducción

El HALO TOUCH V2 es un hub USB multifuncional diseñado para ser un compañero de escritorio interactivo. Combina monitoreo de sistema, reproducción de medios y características de productividad para escritorios, oficinas y configuraciones de escritorio.

## ¿Qué es?

El HALO TOUCH V2 es un dispositivo de escritorio que combina un hub USB con una pantalla táctil circular y un encoder rotativo. Ofrece monitoreo de sistema, reproducción de medios y características de productividad, y se puede personalizar con temas, medios y animaciones a través de archivos en una tarjeta microSD.

## ¿Cómo funciona?

El HALO TOUCH V2 cuenta con un MCU no especificado (potencialmente ESP32-S3), un puerto USB 2.0 Type-C upstream y dos puertos USB 2.0 Type-A downstream. La pantalla táctil circular de 360x360 píxeles soporta gestos táctiles y se puede personalizar con temas y animaciones. El encoder rotativo proporciona controles compatibles con Microsoft Surface Dial en Windows.

## ¿Por qué importa?

El HALO TOUCH V2 importa porque ofrece una forma sencilla de personalizar y monitorear su escritorio, y se puede utilizar en entornos de trabajo, hogares y configuraciones de escritorio. También soporta actualizaciones de firmware OTA y sincronización de reloj.

## Consejo técnico

Para personalizar el HALO TOUCH V2, asegúrese de crear un archivo 'wifi.txt' en la tarjeta microSD para configurar la red Wi-Fi y guarde archivos de música y imágenes en la otra tarjeta microSD para reproducirlos en el dispositivo.
