## Introducción

El mundo de los cyberdecks retro sigue en auge gracias a los entusiastas de la tecnología y la creación de proyectos innovadores. Un ejemplo de esto es el cyberdeck PS-85, creado por Jeff Merrick, que combina la estética de la película Alien con la tecnología de Raspberry Pi.

## ¿Qué es?

El cyberdeck PS-85 es un dispositivo portátil que combina un Raspberry Pi Zero 2 W con un teclado mecánico de 65% y una pantalla táctil. Inspirado en la película Alien, cuenta con un diseño retro y un conjunto de teclas con símbolos futuristas creados por Ron Cobb.

## ¿Cómo funciona?

El cyberdeck PS-85 utiliza un Raspberry Pi Zero 2 W como procesador y un teclado mecánico de 65% para la entrada de datos. La pantalla táctil permite interactuar con el dispositivo de manera intuitiva. El proyecto se basa en la creación de un sistema de escritorio portátil que pueda ser utilizado para tareas de escritura y navegación en Internet.

## ¿Por qué importa?

El cyberdeck PS-85 es importante porque muestra la creatividad y la innovación que se pueden lograr con la tecnología de Raspberry Pi. También demuestra la posibilidad de crear dispositivos portátiles con una estética retro y una funcionalidad moderna.

## Consejo técnico

Si deseas crear un proyecto similar, considera utilizar un Raspberry Pi como procesador y un teclado mecánico de 65% para la entrada de datos. También puedes utilizar una pantalla táctil para interactuar con el dispositivo de manera intuitiva.

```bash
sudo apt-get install raspbian
```
