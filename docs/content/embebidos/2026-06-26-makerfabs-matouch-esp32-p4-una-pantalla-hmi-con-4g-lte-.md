# Makerfabs MaTouch ESP32-P4: una pantalla HMI con 4G LTE y Ethernet

**Fecha:** 2026-06-26
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y linux
**Título original:** Makerfabs MaTouch ESP32-P4 10.1-inch HMI display features 4G LTE and Ethernet connectivity, a 2MP camera

---

## Introducción

Makerfabs ha lanzado el MaTouch ESP32-P4, un desarrollo de plataforma de pantalla HMI con 4G LTE y Ethernet. Esta pantalla es ideal para aplicaciones de HMI, inteligencia artificial en el borde y sistemas embebidos conectados.

## ¿Qué es?

El MaTouch ESP32-P4 es una pantalla de 10.1 pulgadas con tecnología IPS LCD, que cuenta con una resolución de 1280x800 píxeles. Está equipada con un procesador dual-core ESP32-P4NRW32, que incluye un núcleo de alto rendimiento y un núcleo de bajo consumo. La pantalla también cuenta con una cámara de 2MP, un módulo de 4G LTE SIMCom SIM7670G y una tarjeta de sonido ES8311.

## ¿Cómo funciona?

La pantalla MaTouch ESP32-P4 funciona mediante un sistema operativo basado en Linux, que permite la ejecución de aplicaciones de HMI y la conexión a Internet a través de 4G LTE o Ethernet. La pantalla también cuenta con una interfaz de usuario capacitiva de 10 puntos, que permite la interacción con la pantalla de manera intuitiva.

## ¿Por qué importa?

El MaTouch ESP32-P4 es importante porque ofrece una solución completa para aplicaciones de HMI y sistemas embebidos conectados. La pantalla es ideal para proyectos que requieren una interfaz de usuario capacitiva y conexión a Internet, como monitoreo de sistemas, control de procesos y aplicaciones de inteligencia artificial.

## Consejo técnico

Si deseas conectar tu MaTouch ESP32-P4 a Internet a través de 4G LTE, asegúrate de configurar el módulo SIMCom SIM7670G correctamente y de obtener un plan de datos adecuado para tu proyecto.

```bash
sudo apt-get install libesp32-dev
```
