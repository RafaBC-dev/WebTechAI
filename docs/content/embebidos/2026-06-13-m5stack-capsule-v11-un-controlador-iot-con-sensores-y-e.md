# M5Stack Capsule v1.1: un controlador IoT con sensores y expansión

**Fecha:** 2026-06-13
**Categoría:** embebidos
**Tags:** robotica, esp32, arduino
**Título original:** M5Stack Capsule Kit v1.1- A Battery-powered ESP32-S3 IoT controller with IMU sensor, MEMS microphone, and IR transmitter

---

## Introducción

M5Stack ha lanzado el Capsule v1.1, un controlador IoT con sensores y expansión que se enfoca en aplicaciones como el control de actitud y seguimiento de movimiento, grabación de audio y reconocimiento de voz, y recolección de datos con bajo consumo de energía.

## ¿Qué es?

El M5Stack Capsule v1.1 es un controlador IoT basado en el módulo Stamp-S3A, que cuenta con un procesador dual-core ESP32-S3 con instrucciones de vectores de AI, 512KB de SRAM, 8MB de flash y conectividad WiFi 4 y Bluetooth 5.0 LE + Mesh. También incluye sensores como un IMU 6-axial, un micrófono MEMS y un transmisor IR.

## ¿Cómo funciona?

El Capsule v1.1 funciona con un procesador dual-core ESP32-S3 que se enfoca en aplicaciones de IoT. El módulo Stamp-S3A incluye un procesador de bajo consumo de energía (ULP) basado en RISC-V, que ayuda a reducir el consumo de energía del dispositivo. El controlador también cuenta con una ranura para tarjeta microSD y una conexión Grove para expansión.

## ¿Por qué importa?

El M5Stack Capsule v1.1 es importante porque ofrece una plataforma de desarrollo para aplicaciones de IoT con bajo consumo de energía, lo que la hace ideal para aplicaciones como el seguimiento de movimiento y la recolección de datos con bajo consumo de energía.

## Consejo técnico

Si deseas aprovechar al máximo el M5Stack Capsule v1.1, asegúrate de utilizar la biblioteca UIFlow2 para programar el dispositivo y aprovechar las características de AI del procesador ESP32-S3.
