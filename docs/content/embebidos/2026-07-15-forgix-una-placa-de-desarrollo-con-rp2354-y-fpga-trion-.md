## Introducción

Adiuvo Engineering ha lanzado la placa de desarrollo Forgix, que combina un microcontrolador Raspberry Pi RP2354 con un FPGA Efinix Trion T8 en el formato Teensy 4.0. Esta placa es única y ofrece una gran cantidad de interfaces y recursos para desarrolladores.

## ¿Qué es?

La placa Forgix es una placa de desarrollo que integra un microcontrolador Raspberry Pi RP2354 con un FPGA Efinix Trion T8. El RP2354 es un procesador dual-core con tecnología Arm Cortex-M33 y RISC-V, mientras que el FPGA Trion T8 es un procesador de lógica programable con 7.384 elementos lógicos y 8 multiplicadores embutidos.

## ¿Cómo funciona?

La placa Forgix funciona conectando el RP2354 con el FPGA Trion T8 a través de una interfaz SPI. El RP2354 puede ejecutar código en tiempo real y controlar el FPGA, mientras que el FPGA puede realizar cálculos complejos y controlar dispositivos externos. La placa también incluye 16 Mbit de memoria PSRAM y una conexión USB-C para programación y poder.

## ¿Por qué importa?

La placa Forgix es importante porque ofrece una gran cantidad de interfaces y recursos para desarrolladores. Puede ser utilizada para crear proyectos de robótica, automatización y otros aplicativos que requieren cálculos complejos y control de dispositivos externos.
