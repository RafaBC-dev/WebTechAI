# Lámpara de NeoPixel inspirada en EPCOT Spaceship Earth

## Introducción

La empresa Adafruit ha lanzado un proyecto de aprendizaje para crear una lámpara de NeoPixel inspirada en el icónico Spaceship Earth de EPCOT. Esta lámpara puede ser controlada por un microcontrolador ESP32 y mostrar efectos de iluminación impresionantes.

## ¿Qué es?

El proyecto consiste en crear una lámpara de NeoPixel utilizando un microcontrolador QT Py ESP32 PICO y un driver NeoPixel BFF. La lámpara puede ser controlada mediante el software WLED y mostrar efectos de iluminación dinámicos.

## ¿Cómo funciona?

El microcontrolador ESP32 PICO se conecta a la lámpara de NeoPixel y recibe instrucciones del software WLED. El software controla la secuencia de colores y patrones de la lámpara, creando efectos de iluminación impresionantes.

## ¿Por qué importa?

Este proyecto importa porque permite a los usuarios crear sus propias lámparas de NeoPixel con efectos de iluminación personalizables. También es una excelente oportunidad para aprender sobre la programación de microcontroladores y la electrónica.

## Consejo técnico

Si deseas crear una lámpara de NeoPixel similar, asegúrate de utilizar la librería WLED y el driver NeoPixel BFF adecuados para tu microcontrolador. También es importante configurar correctamente el software WLED para obtener los mejores resultados.

```bash
pip install wled
```
