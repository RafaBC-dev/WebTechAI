# Echo Pyramid: dispositivo de voz inteligente para M5Stack Atom

**Fecha:** 2026-03-08
**Categoría:** embebidos
**Dificultad:** Intermedio
**Tags:** ESP32, M5Stack, voz-inteligente
**Título original:** Echo Pyramid enables smart voice interaction applications on M5Stack Atom ESP32 IoT controllers

---

## Resumen

Echo Pyramid es un dispositivo de voz inteligente diseñado para M5Stack Atom, AtomS3 y AtomS3R. Ofrece voz asistente, reconocimiento de voz a distancia y control de voz. Utiliza un micrófono MEMS, un codec de audio HD y un MCU STM32. Se puede conectar a través de USB-C y expandir con módulos I2C.

## Puntos clave

- Dispositivo de voz inteligente para aplicaciones IoT
- Funciona con M5Stack Atom y otros controles IoT
- Permite voz asistente, reconocimiento de voz y control de voz

## Consejo técnico

Asegúrate de conectar el dispositivo a un controlador M5Stack Atom compatible.
