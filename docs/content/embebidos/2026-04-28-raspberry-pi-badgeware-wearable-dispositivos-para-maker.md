## Introducción

Pimoroni ha lanzado una nueva gama de dispositivos wearable basados en Raspberry Pi, diseñados para la comunidad de makers. Estos dispositivos ofrecen una mejora significativa sobre las versiones anteriores.

## ¿Qué es?

Los dispositivos wearable de Pimoroni son pequeños dispositivos electrónicos que se pueden llevar en el cuello o en la mano. Están diseñados para ser personalizados y programados por los usuarios. Cada dispositivo tiene un display, botones y una batería recargable.

## ¿Cómo funciona?

Los dispositivos wearable de Pimoroni están basados en el procesador RP2350 de Raspberry Pi. Cada dispositivo tiene un display diferente: e-ink, LCD color o matriz de LED. Los dispositivos también tienen botones y una batería recargable a través de USB-C. Los usuarios pueden programar los dispositivos utilizando MicroPython.

## ¿Por qué importa?

Estos dispositivos wearable pueden ser utilizados para una variedad de aplicaciones, como crear identificaciones personales, juegos, aplicaciones de productividad o incluso dispositivos de monitoreo. La comunidad de makers puede personalizar y programar los dispositivos para adaptarse a sus necesidades específicas.

## Consejo técnico

Si deseas crear tu propio dispositivo wearable, considera utilizar el kit de desarrollo de Pimoroni, que incluye el procesador RP2350, el display y los componentes necesarios. Puedes programar el dispositivo utilizando MicroPython y personalizarlo según tus necesidades.

```bash
python -m pyboard -p /dev/ttyUSB0
```
