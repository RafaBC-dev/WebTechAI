## Introducción

La empresa Argon ha lanzado un nuevo encuadre industrial para Raspberry Pi 5 y la pantalla táctil de 5 pulgadas del Pi, diseñado para crear sistemas de interfaz humana máquina compactos y robustos.

## ¿Qué es?

El Argon Industria HMI 5C es un encuadre industrial de aluminio diseñado para albergar un Raspberry Pi 5 y la pantalla táctil de 5 pulgadas del Pi, convirtiéndolo en un sistema compacto de interfaz humana máquina (HMI) para paneles de control, interfaces de automatización y aplicaciones de visualización embebida.

## ¿Cómo funciona?

El encuadre soporta montaje VESA y paneles con un goma suave para una instalación limpia, y ofrece acceso a puertos USB, Ethernet, HDMI y GPIO a través de una tarjeta de IO interna que reenvía conexiones (incluyendo la conversión de micro-HDMI a HDMI completo), y incluye agujeros para cables para una instalación limpia.

## ¿Por qué importa?

Este encuadre importa porque ofrece una solución compacta y robusta para crear sistemas de HMI en aplicaciones industriales, permitiendo una fácil integración con Raspberry Pi 5 y la pantalla táctil de 5 pulgadas del Pi.

## Consejo técnico

Si planeas utilizar un HAT, asegúrate de remover el columna de enfriamiento pasiva para evitar conflictos de espacio y utilizar un enfriador activo o un disipador de calor bajo perfil.
