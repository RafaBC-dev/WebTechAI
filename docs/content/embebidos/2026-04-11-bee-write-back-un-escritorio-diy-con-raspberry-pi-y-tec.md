## Introducción

Un ingeniero llamado Simon ha creado un escritorio DIY llamado Bee Write Back, basado en Raspberry Pi Zero 2 W, que busca ofrecer una experiencia de escritura sin distracciones.

## ¿Qué es?

Bee Write Back es un escritorio DIY que utiliza un Raspberry Pi Zero 2 W como base, un display OLED de 5,5 pulgadas y un teclado mecánico para ofrecer una experiencia de escritura sin distracciones.

## ¿Cómo funciona?

El escritorio utiliza un Raspberry Pi Zero 2 W con un procesador quad-core Cortex-A53, 512 MB de RAM y conectividad WiFi y Bluetooth. El display OLED de 5,5 pulgadas ofrece una resolución de 1280 x 720 píxeles. El teclado mecánico es compatible con el protocolo YMDK Air40.

## ¿Por qué importa?

Bee Write Back puede ser una herramienta útil para aquellos que buscan una experiencia de escritura sin distracciones, como escritores o estudiantes que necesitan enfocarse en su trabajo.

## Consejo técnico

Si deseas crear un escritorio DIY similar, puedes encontrar las instrucciones completas en GitHub, incluyendo dos scripts de Python: uno para un programa de escritura minimalista y otro para un cliente de chat en un terminal.

```bash
python writerdeck.py
```
