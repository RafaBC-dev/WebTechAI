# Módulo NuMaker-GestureAI-M55M1: control de gestos con IA y visión

## Introducción

Nuvoton ha lanzado el módulo NuMaker-GestureAI-M55M1, un desarrollo basado en el MCU NuMicro M55M1, que combina un núcleo Cortex-M55 con un microNPU Ethos-U55 para control de gestos y visión con IA. Este módulo es ideal para aplicaciones como control de gestos, sistemas de visión básicos y interfaces sin tocar.

## ¿Qué es?

El NuMaker-GestureAI-M55M1 es un módulo de desarrollo que integra el MCU NuMicro M55M1 con un sensor de imagen CMOS GC0308, un micrófono digital y una ranura de tarjeta microSD para almacenar modelos de IA. Está diseñado para aplicaciones que requieren control de gestos, visión y interfaces sin tocar.

## ¿Cómo funciona?

El módulo utiliza el MCU NuMicro M55M1 para procesar la información de la cámara y el micrófono, y el microNPU Ethos-U55 para realizar la detección de gestos y personas. La información se envía a través de la UART5 para ser procesada por un dispositivo externo.

## ¿Por qué importa?

El NuMaker-GestureAI-M55M1 es importante porque permite a los desarrolladores crear aplicaciones de control de gestos y visión con IA de manera sencilla y eficiente. Esto puede ser utilizado en una variedad de aplicaciones, desde sistemas de seguridad hasta interfaces de usuario innovadoras.
