# Escalas inteligentes con ESP32 y HX711: SparkFun OpenScale IoT

## Introducción

SparkFun ha lanzado el OpenScale IoT, un módulo de escala inteligente basado en ESP32 que permite leer datos de peso con precisión sin necesidad de código personalizado. Conectividad Wi-Fi y Bluetooth, este módulo es ideal para proyectos de IoT y monitoreo de peso.

## ¿Qué es?

El SparkFun OpenScale IoT es un módulo de escala inteligente que utiliza el procesador ESP32 para leer datos de peso con precisión. Incluye un amplificador HX711 de 24 bits y un sensor de temperatura TMP102, lo que permite calibrar y compensar la temperatura. También cuenta con una conexión Qwiic I²C para agregar sensores o displays adicionales.

## ¿Cómo funciona?

El módulo OpenScale IoT funciona mediante la lectura de señales de un cargador de peso (load cell) a través del amplificador HX711. El procesador ESP32 procesa estas señales y las envía a través de Wi-Fi o Bluetooth. El módulo también incluye una función de calibración y compensación de temperatura para asegurar lecturas precisas.

## ¿Por qué importa?

El SparkFun OpenScale IoT es importante porque permite a los desarrolladores crear proyectos de IoT y monitoreo de peso sin necesidad de código personalizado. Esto hace que sea ideal para aplicaciones como la medición de peso en tiempo real, el monitoreo de la calidad del aire o la medición de la humedad.

## Consejo técnico

Para conectar el módulo OpenScale IoT a tu proyecto de IoT, asegúrate de utilizar la biblioteca ESP32 en tu código y configura la conexión Wi-Fi y Bluetooth según sea necesario. También es recomendable calibrar el módulo antes de utilizarlo en un proyecto real.

```bash
`esp32`
```
