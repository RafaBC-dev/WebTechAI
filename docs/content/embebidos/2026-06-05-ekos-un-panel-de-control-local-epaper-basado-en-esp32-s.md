## Introducción

El EKOS es un panel de control local ePaper diseñado para ofrecer una experiencia de usuario personalizada y privada. Sin necesidad de conexión a la nube o suscripciones, EKOS permite controlar dispositivos y acceder a información en tiempo real de manera segura y eficiente.

## ¿Qué es?

El EKOS es un panel de control ePaper local que utiliza un SoC ESP32-S3 para ofrecer una experiencia de usuario personalizada y privada. Puede ser actualizado y configurado a través de una API local, permitiendo controlar dispositivos y acceder a información en tiempo real de manera segura y eficiente.

## ¿Cómo funciona?

El EKOS utiliza un SoC ESP32-S3 para procesar información y controlar dispositivos. La interfaz de usuario se gestiona mediante la librería LVGL, mientras que la actualización y configuración se realizan a través de una API local. El panel cuenta con un sensor BME280 para medir temperatura, humedad y presión atmosférica.

## ¿Por qué importa?

El EKOS importa porque ofrece una experiencia de usuario personalizada y privada, sin necesidad de conexión a la nube o suscripciones. También permite controlar dispositivos y acceder a información en tiempo real de manera segura y eficiente, lo que lo hace ideal para aplicaciones de automatización y control de hogares inteligentes.

## Consejo técnico

Si deseas crear un panel de control similar al EKOS, considera utilizar el ESP32-S3 como SoC y la librería LVGL para gestionar la interfaz de usuario. También es importante tener en cuenta la seguridad y la privacidad de la información que se almacena y transfiere.

Para actualizar el firmware del EKOS, utiliza el comando `idf.py flash` y asegúrate de tener instalado el IDE de Espressif.
