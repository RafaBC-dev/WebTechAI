# Simulador de Arduino, Raspberry Pi y ESP32 en línea y self-hosted

**Fecha:** 2026-04-04
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** Velxio is an open-source, self-hosted Arduino, Raspberry Pi, and ESP32 simulator

---

## Introducción

Velxio es un simulador de código abierto que permite simular y programar placa de Arduino, Raspberry Pi y ESP32 directamente en el navegador web. Es una herramienta muy útil para desarrolladores y estudiantes de robótica y electrónica.

## ¿Qué es?

Velxio es un simulador de código abierto que permite simular y programar placa de Arduino, Raspberry Pi y ESP32. Es una herramienta que permite a los desarrolladores y estudiantes de robótica y electrónica simular y probar sus proyectos sin necesidad de tener hardware físico. El simulador es compatible con 19 objetivos de cinco arquitecturas diferentes, incluyendo AVR8, Arm Cortex-M0+, RISC-V RV32IMC/EC y Xtensa LX6/LX7.

## ¿Cómo funciona?

El simulador de Velxio funciona mediante la emulación de hardware y software de las placas de Arduino, Raspberry Pi y ESP32. Permite a los usuarios arrastrar y soltar componentes y módulos, conectarlos a los pines de la placa y escribir y ejecutar código en Arduino o Python. También ofrece una consola de serie para acceder a la salida de datos de los proyectos.

## ¿Por qué importa?

Velxio es importante porque permite a los desarrolladores y estudiantes de robótica y electrónica simular y probar sus proyectos de manera rápida y eficiente. También es una herramienta muy útil para aprender y practicar la programación de placas de Arduino, Raspberry Pi y ESP32 sin necesidad de tener hardware físico.

## Consejo técnico

Si deseas instalar Velxio en tu máquina local, puedes utilizar el comando `sudo docker run -d -p 3080:80 ghcr.io/davidmonterocrespo24/velxio:master` para ejecutar el simulador en tu navegador web.

```bash
sudo docker run -d -p 3080:80 ghcr.io/davidmonterocrespo24/velxio:master
```
