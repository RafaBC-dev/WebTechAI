# M5Stack PaperColor: un kit de desarrollo con pantalla E Ink color

**Tags:** microcontroladores, esp32, robotica, diseño-3d

## Introducción

M5Stack ha lanzado un nuevo kit de desarrollo llamado PaperColor, que combina una pantalla E Ink color con un procesador ESP32-S3. Esto permite crear dispositivos con bajo consumo de energía y alta visibilidad

## ¿Qué es?

El M5Stack PaperColor es un kit de desarrollo que incluye una pantalla E Ink color de 4 pulgadas con resolución de 600x400, un procesador ESP32-S3, memoria RAM de 8MB y conectividad WiFi y Bluetooth. También cuenta con un micrófono, un altavoz, un sensor de temperatura y humedad, y varias conexiones para expansiones

## ¿Cómo funciona?

La pantalla E Ink color se basa en la tecnología ED2208-DOA y ofrece una tasa de refresco de aproximadamente 15-19 segundos. El procesador ESP32-S3 es un microcontrolador dual-core con instrucciones de vectorización para acelerar la IA. El kit también incluye una batería de 1250mAh que se puede recargar a través del puerto USB Type-C

## ¿Por qué importa?

El M5Stack PaperColor es relevante porque permite crear dispositivos con bajo consumo de energía y alta visibilidad, lo que los hace ideales para aplicaciones como marcos digitales de fotos, paneles de datos y señalización electrónica

## Consejo técnico

Si estás interesado en programar el M5Stack PaperColor, puedes utilizar la biblioteca M5Unified y las bibliotecas M5GFX y M5PM1 para acceder a las funciones del kit y crear aplicaciones personalizadas

Para programar el M5Stack PaperColor, puedes utilizar el IDE Arduino y la biblioteca M5Unified. Puedes empezar con el ejemplo de código que se proporciona en la documentación de M5Stack
