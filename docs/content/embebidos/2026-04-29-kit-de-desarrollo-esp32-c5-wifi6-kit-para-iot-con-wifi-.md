## Introducción

La empresa Waveshare ha lanzado un kit de desarrollo para IoT que se basa en el procesador ESP32-C5 y ofrece una amplia gama de opciones, incluyendo diferentes capacidades de PSRAM y flash, y la opción de elegir entre una antena integrada o externa.

## ¿Qué es?

El ESP32-C5-WIFI6-KIT es un kit de desarrollo para IoT que se basa en el procesador ESP32-C5, un procesador de 32 bits de una sola núcleo con una velocidad de hasta 240 MHz. Ofrece una amplia gama de opciones, incluyendo diferentes capacidades de PSRAM y flash, y la opción de elegir entre una antena integrada o externa.

## ¿Cómo funciona?

El ESP32-C5-WIFI6-KIT se basa en el procesador ESP32-C5, que cuenta con un núcleo de 32 bits de una sola núcleo con una velocidad de hasta 240 MHz. También cuenta con 384 KB de SRAM en chip y 320 KB de ROM. La conectividad se basa en WiFi 6 dual-banda (2,4 GHz y 5 GHz) con una velocidad de hasta 9,6 Gbps y Bluetooth 5.0 Low Energy.

## ¿Por qué importa?

Este kit de desarrollo es importante porque ofrece una amplia gama de opciones para desarrollar aplicaciones de IoT, incluyendo la capacidad de elegir entre diferentes capacidades de PSRAM y flash, y la opción de elegir entre una antena integrada o externa. Esto lo hace ideal para desarrollar aplicaciones de IoT que requieren una gran cantidad de memoria y almacenamiento.

## Consejo técnico

Si estás desarrollando una aplicación de IoT con el ESP32-C5-WIFI6-KIT, considera utilizar la biblioteca ESP-IDF para acceder a las funciones de hardware y software del procesador. También puedes utilizar la biblioteca Arduino para desarrollar aplicaciones de IoT de manera más sencilla.

```bash
Es posible utilizar la biblioteca ESP-IDF con el comando `idf.py flash` para programar el ESP32-C5-WIFI6-KIT.
```
