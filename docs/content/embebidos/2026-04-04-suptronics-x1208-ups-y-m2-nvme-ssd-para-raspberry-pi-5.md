# SupTronics X1208: UPS y M.2 NVMe SSD para Raspberry Pi 5

**Fecha:** 2026-04-04
**Categoría:** embebidos
**Tags:** robotica, raspberry-pi, embebidos

## Introducción

La empresa SupTronics ha lanzado el X1208, un dispositivo que combina una batería de UPS con un slot M.2 NVMe SSD para Raspberry Pi 5. Esto permite a los usuarios almacenar datos de manera segura y rápida, incluso en caso de apagones.

## ¿Qué es?

El SupTronics X1208 es un dispositivo de alto rendimiento que se conecta a la Raspberry Pi 5 y proporciona una batería de UPS para proteger el sistema contra pérdidas de datos en caso de apagones. También cuenta con un slot M.2 NVMe SSD para almacenar datos de manera rápida y segura.

## ¿Cómo funciona?

El X1208 utiliza una pila de batería de 21700 y un slot M.2 NVMe SSD para almacenar datos. La batería se carga a través de un puerto USB-C y puede proporcionar hasta 5A de corriente. El dispositivo también cuenta con un sensor de temperatura y un LED para indicar el estado de la batería.

## ¿Por qué importa?

El X1208 es importante porque permite a los usuarios almacenar datos de manera segura y rápida, incluso en caso de apagones. Esto es especialmente útil para aplicaciones de Edge Computing, NAS y data logging, donde la disponibilidad y la confiabilidad son fundamentales.

## Consejo técnico

Si deseas utilizar el X1208 con tu Raspberry Pi 5, asegúrate de configurar el PSU_MAX_CURRENT=5000 en tu archivo de configuración de Raspberry Pi. Esto te permitirá aprovechar al máximo la capacidad de la batería.

```bash
PSU_MAX_CURRENT=5000
```
