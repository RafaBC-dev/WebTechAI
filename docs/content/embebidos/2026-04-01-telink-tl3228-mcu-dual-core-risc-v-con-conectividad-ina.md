# Telink TL3228: MCU dual-core RISC-V con conectividad inalámbrica avanzada

**Fecha:** 2026-04-01
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** Telink TL3228 – Low-power, low-latency dual-core RISC-V wireless MCU supports Bluetooth 6.0, 802.15.4, and 2.4 GHz proprietary

---

## Introducción

Telink Semiconductor acaba de lanzar el TL3228, un microcontrolador dual-core RISC-V con conectividad inalámbrica avanzada, diseñado para aplicaciones de Internet de las cosas (IoT) y dispositivos inteligentes.

## ¿Qué es?

El TL3228 es un microcontrolador dual-core RISC-V que integra una CPU de 192 MHz y una memoria de 384 KB de SRAM. También cuenta con conectividad inalámbrica avanzada, incluyendo Bluetooth 6.0, 802.15.4 y protocolos propietarios de 2,4 GHz.

## ¿Cómo funciona?

El TL3228 utiliza una arquitectura dual-core RISC-V, lo que le permite ejecutar múltiples tareas simultáneamente. La conectividad inalámbrica se realiza a través de una interfaz de radio de 2,4 GHz, que permite transmisiones de datos a velocidades de hasta 6 Mbps.

## ¿Por qué importa?

El TL3228 es relevante para aplicaciones de IoT y dispositivos inteligentes, ya que ofrece una conectividad inalámbrica avanzada y una eficiencia energética mejorada. Esto lo hace ideal para aplicaciones como el seguimiento de activos, la automatización del hogar y los accesorios de juego de alta definición.

## Consejo técnico

Si deseas desarrollar aplicaciones con el TL3228, puedes utilizar el Telink IoT Studio IDE, que ofrece herramientas de compilación, descarga de firmware y depuración. También puedes utilizar los SDKs proporcionados por Telink, que se encuentran disponibles en GitHub.

```bash
Para empezar a desarrollar con el TL3228, puedes descargar el Telink IoT Studio IDE desde la página de Telink y seguir las instrucciones de instalación. Luego, puedes utilizar el IDE para crear y compilar tu aplicación.
```
