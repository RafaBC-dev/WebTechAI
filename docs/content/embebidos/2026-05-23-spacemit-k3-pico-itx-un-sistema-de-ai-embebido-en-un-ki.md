# SpacemiT K3 Pico-ITX: un sistema de AI embebido en un kit de construcción

## Introducción

SpacemiT ha lanzado un kit de construcción para un sistema de AI embebido basado en el procesador RISC-V K3 16-core. Este kit es ideal para desarrolladores y entusiastas de la tecnología que quieren experimentar con la IA en un entorno de construcción

## ¿Qué es?

El SpacemiT K3 Pico-ITX es un kit de construcción que incluye un chasis compatible con la placa base K3 Pico-ITX, un procesador RISC-V K3 16-core y una placa base SpacemiT MUSE_Pico-ITX. El kit también incluye un dock USB-C con varios puertos y una placa base con varios conectores para conectar dispositivos externos

## ¿Cómo funciona?

El kit se puede conectar a un monitor y un teclado mediante los puertos USB-C y HDMI. La placa base SpacemiT MUSE_Pico-ITX tiene varios conectores para conectar dispositivos externos, como M.2 Key-M 2280 para NVMe SSD o M.2 B-Key 2242/3052 para 4G LTE o 5G celular. El kit también incluye un cable de debug serial para depurar el sistema

## ¿Por qué importa?

Este kit es importante porque ofrece una forma fácil de experimentar con la IA en un entorno de construcción. El procesador RISC-V K3 16-core es un ejemplo de cómo la tecnología de procesamiento de RISC-V está evolucionando y se está convirtiendo en una opción viable para aplicaciones de AI embebida

## Consejo técnico

Si estás interesado en experimentar con la IA en un entorno de construcción, considera conectar un dispositivo de almacenamiento externo (como un SSD NVMe) a la placa base SpacemiT MUSE_Pico-ITX para mejorar el rendimiento del sistema
