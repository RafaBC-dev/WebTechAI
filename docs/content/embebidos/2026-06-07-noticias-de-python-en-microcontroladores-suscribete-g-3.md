El boletín de noticias de Python para Microcontroladores es una publicación semanal que recopila las últimas noticias sobre Python en hardware, incluyendo microcontroladores y ordenadores de placa única como Raspberry Pi. El boletín se enfoca en informar sobre CircuitPython, MicroPython y Python, y recopila información sobre software, eventos, proyectos y hardware de todo el mundo. Puedes suscribirte de forma gratuita y cancelar en cualquier momento.
