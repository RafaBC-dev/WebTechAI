## Introducción

Sanctuary Systems ha lanzado el Sentinel Core, una tarjeta de expansión para Raspberry Pi Compute Module 5 que agrega un slot PCIe x16 para conectar tarjetas gráficas y acelerar tareas de video y procesamiento de lenguaje natural.

## ¿Qué es?

El Sentinel Core es una tarjeta de expansión que se conecta a un Raspberry Pi Compute Module 5 y agrega funcionalidades como un slot PCIe x16, dos puertos HDMI, un puerto Gigabit Ethernet, dos puertos USB 3.0 y una zona de prototipado.

## ¿Cómo funciona?

La tarjeta de expansión se conecta a un Raspberry Pi Compute Module 5 a través de dos conectores de 100 pines B2B, lo que permite acceder a las funcionalidades del módulo. El slot PCIe x16 permite conectar tarjetas gráficas y acelerar tareas de video y procesamiento de lenguaje natural.

## ¿Por qué importa?

El Sentinel Core importa porque permite a los desarrolladores y usuarios avanzados aprovechar al máximo las capacidades del Raspberry Pi Compute Module 5, especialmente en aplicaciones que requieren procesamiento de video y lenguaje natural, como la inteligencia artificial y la automatización.
