# Olimex lanza una placa de automatización para Raspberry Pi Pico con alta tensión

**Fecha:** 2026-07-17
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** Olimex PICO-EVB automation board adds high-voltage terminals to Raspberry Pi Pico family

## Introducción

Olimex ha lanzado la placa PICO-EVB, diseñada para la automatización con alta tensión en proyectos con Raspberry Pi Pico y compatibles. Esta placa ofrece cuatro entradas de alta tensión y cuatro relés de 15V DC/220V AC.

## ¿Qué es?

La placa PICO-EVB es una placa de automatización diseñada para proyectos con Raspberry Pi Pico y compatibles. Permite la conexión de cuatro entradas de alta tensión de 3.3-30V DC o 110-220V AC y cuatro relés de 15V DC/220V AC. También cuenta con un conector UEXT para la expansión de señales I2C, UART y SPI.

## ¿Cómo funciona?

La placa PICO-EVB se conecta a un Raspberry Pi Pico compatible a través de dos headers de 20 pines. El Pico se encarga de leer los sensores y controlar los relés. La placa también cuenta con un conector USB-C para la alimentación y programación, y LEDs de estado para los relés y entradas de alta tensión.

## ¿Por qué importa?

La placa PICO-EVB es importante porque ofrece una solución de automatización con alta tensión para proyectos con Raspberry Pi Pico y compatibles. Esto permite la creación de proyectos más complejos y robustos en áreas como la automatización industrial, la iluminación y la seguridad.
