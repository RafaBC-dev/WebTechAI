# Revisa las pantallas HMI Open Nextion con ESP32-S3 y Arduino IDE

## Introducción

ITEAD ha enviado muestras de las pantallas HMI Open Nextion ONX3248G035 y ONX2432G028 para su revisión. Estas pantallas son parte de la nueva línea de productos Open Nextion, que se diferencia de la familia Nextion original en su uso de un microcontrolador ESP32-S3 para el procesamiento lógico, control de la pantalla y conectividad inalámbrica.

## ¿Qué es?

Las pantallas HMI Open Nextion son dispositivos de interfaz de usuario que combinan un microcontrolador ESP32-S3 con una pantalla táctil y conectividad inalámbrica. Estos dispositivos están diseñados para ser utilizados en aplicaciones de IoT y prototipado de paneles de control inteligentes.

## ¿Cómo funciona?

Las pantallas HMI Open Nextion funcionan utilizando el microcontrolador ESP32-S3, que se encarga de procesar la lógica de la aplicación, controlar la pantalla y establecer la conectividad inalámbrica. El dispositivo también incluye una pantalla táctil y una variedad de interfaces de entrada y salida, como sensores y actuadores.

## ¿Por qué importa?

Las pantallas HMI Open Nextion son importantes porque ofrecen una solución completa para el prototipado de aplicaciones de IoT y paneles de control inteligentes. También permiten a los desarrolladores y makers aprovechar las características del microcontrolador ESP32-S3, como la conectividad inalámbrica y la capacidad de procesamiento de alto rendimiento.

## Consejo técnico

Si deseas utilizar las pantallas HMI Open Nextion con Arduino IDE, asegúrate de instalar la biblioteca LVGL y configurar el microcontrolador ESP32-S3 según las instrucciones del fabricante.

```bash
Instalar la biblioteca LVGL en Arduino IDE: `Sketch > Bibliotecas > Agregar biblioteca > Buscar LVGL`
```
