# Pimoroni lanza una nueva gama de dispositivos de wearable para Raspberry Pi

**Fecha:** 2026-05-01
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** Raspberry Pi badgeware

---

## Introducción

Pimoroni, un fabricante conocido por sus dispositivos para Raspberry Pi, ha lanzado una nueva gama de dispositivos de wearable que ofrecen una experiencia de usuario más avanzada y funcionalidades adicionales. Estos dispositivos están diseñados para ser utilizados con Raspberry Pi y ofrecen una forma fácil de crear proyectos de wearable y dispositivos de IoT.

## ¿Qué es?

Los dispositivos de wearable de Pimoroni son dispositivos pequeños y portátiles que se pueden llevar en la mano o en la ropa. Están basados en la placa de circuito impreso RP2350 y ofrecen una variedad de funcionalidades, incluyendo pantallas, botones y conectividad USB-C. Los dispositivos están diseñados para ser fáciles de usar y programar, y se pueden utilizar para crear proyectos de wearable y dispositivos de IoT.

## ¿Cómo funciona?

Los dispositivos de wearable de Pimoroni funcionan conectándolos a un Raspberry Pi y programándolos utilizando MicroPython. Los dispositivos tienen una pantalla que muestra un menú de aplicaciones preinstaladas, y se pueden programar utilizando botones y conectividad USB-C. Los dispositivos también tienen una batería integrada que puede ser recargada utilizando un cable USB-C.

## ¿Por qué importa?

Estos dispositivos de wearable importan porque ofrecen una forma fácil de crear proyectos de wearable y dispositivos de IoT. Los dispositivos son fáciles de usar y programar, y se pueden utilizar para crear proyectos de todo tipo, desde dispositivos de seguimiento de salud hasta dispositivos de automatización del hogar.

## Consejo técnico

Si estás interesado en crear un proyecto de wearable utilizando los dispositivos de Pimoroni, considera utilizar la librería MicroPython para programar tus dispositivos. MicroPython es una versión de Python que se puede utilizar en dispositivos de wearable y ofrece una forma fácil de programar y personalizar tus dispositivos.

```bash
pip install micropython
```
