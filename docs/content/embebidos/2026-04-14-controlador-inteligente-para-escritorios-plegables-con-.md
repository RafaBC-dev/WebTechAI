# Controlador inteligente para escritorios plegables con integración con Home Assistant

**Fecha:** 2026-04-14
**Categoría:** embebidos
**Tags:** robotica, esp32, linux, diseño-3d
**Título original:** DeskUp Pro smart standing desk controller integrates with Home Assistant and Homey Pro Smart Home hubs

---

## Introducción

El proyecto DeskUp Pro permite automatizar el control de escritorios plegables inteligentes, integrándolos con sistemas de hogar como Home Assistant y Homey Pro. Esto permite a los usuarios programar recordatorios de salud, controlar el altura del escritorio y utilizar asistentes de voz como Google Home y Alexa.

## ¿Qué es?

El DeskUp Pro es un controlador inteligente diseñado para escritorios plegables que utilizan una conexión RJ11/RJ12 estándar. Se basa en un microcontrolador ESP32 y se conecta directamente al controlador del escritorio para controlar su altura y otros parámetros.

## ¿Cómo funciona?

El DeskUp Pro utiliza la conexión RJ11/RJ12 para comunicarse con el controlador del escritorio y controlar su altura. También se puede programar para enviar comandos de salud y recordatorios a los usuarios. Se puede configurar a través de una interfaz web y se expone a través de una API REST para integrarse con otros sistemas de hogar.

## ¿Por qué importa?

El DeskUp Pro importa porque permite a los usuarios automatizar y controlar sus escritorios plegables inteligentes de manera más sencilla y eficiente. Esto puede ayudar a reducir problemas de salud como la ciática causada por la postura incorrecta durante largos períodos de tiempo.

## Consejo técnico

Si deseas integrar tu escritorio plegable con Home Assistant, puedes utilizar el DeskUp Pro y configurarlo a través de la interfaz web. Primero, asegúrate de que tu escritorio plegable tenga una conexión RJ11/RJ12 y que el DeskUp Pro esté configurado correctamente.
