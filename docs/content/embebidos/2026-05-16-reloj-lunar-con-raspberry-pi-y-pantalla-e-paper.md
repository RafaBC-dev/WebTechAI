## Introducción

Un entusiasta ha creado un reloj lunar que muestra la fase actual del satélite y los horarios de las mareas en un puerto del Reino Unido. Este proyecto es un ejemplo de cómo la tecnología puede ser utilizada para crear dispositivos útiles y atractivos.

## ¿Qué es?

El reloj lunar es un dispositivo que utiliza un Raspberry Pi y una pantalla E-Paper para mostrar la fase actual del satélite, la fecha y los horarios de las mareas en el puerto de Whitstable, en el Reino Unido. El dispositivo tiene un diseño estilizado en estilo Steam-Punk y utiliza un Pi Pico W para conectarse a Internet.

## ¿Cómo funciona?

El reloj lunar utiliza la información de la fecha y la hora para calcular la fase actual del satélite y mostrarla en la pantalla E-Paper. También utiliza la información de las mareas para mostrar los horarios de las mareas en el puerto de Whitstable. El dispositivo utiliza el protocolo HTTP para conectarse a Internet y obtener la información necesaria.

## ¿Por qué importa?

Este proyecto es importante porque muestra cómo la tecnología puede ser utilizada para crear dispositivos útiles y atractivos. El reloj lunar puede ser utilizado por personas que quieren saber la fase actual del satélite o los horarios de las mareas en un puerto específico. Además, el proyecto demuestra la capacidad de los dispositivos Raspberry Pi para realizar tareas complejas y mostrar información en tiempo real.

## Consejo técnico

Si deseas crear un dispositivo similar, puedes utilizar la librería `epaper` para Raspberry Pi para controlar la pantalla E-Paper y la librería `requests` para conectarte a Internet y obtener la información necesaria. También puedes utilizar el Pi Pico W para conectarte a Internet y obtener la información necesaria.

```bash
pip install epaper requests
```
