## Introducción

Hamzah Abugosh ha creado un dispositivo portátil que combina múltiples sensores para medir la temperatura, humedad, presión, gases, luz UV, proximidad, color, distancia, sonido y señales inalámbricas. ¿Qué hace que este tricorder sea tan especial?

## ¿Qué es?

El ThriCorder v3.8 es un dispositivo de mano que utiliza el ESP32-S3 como cerebro y cuenta con una variedad de sensores para medir diferentes parámetros ambientales. Se inspira en el tricorder de Star Trek y busca ser una herramienta práctica para la exploración y el análisis del entorno.

## ¿Cómo funciona?

El ThriCorder utiliza el ESP32-S3 para procesar la información de los sensores, que incluyen un sensor de temperatura y humedad (BME688), un sensor de luz UV (LTR390), un sensor de proximidad y color (APDS9999), un sensor de distancia (VL53L4CX) y un sensor de sonido (PDM Microphone Breakout). La información se muestra en una pantalla de 1.8 pulgadas con una matriz de LEDs de 8x8 y se puede reproducir audio a través de un altavoz.

## ¿Por qué importa?

Este tricorder casero puede ser útil para científicos, ingenieros y entusiastas de la tecnología que buscan explorar y analizar su entorno de manera más detallada. También puede ser una herramienta educativa para aprender sobre la electrónica y la programación.

## Consejo técnico

Si deseas crear un dispositivo similar, considera utilizar la plataforma ESP32-S3 y la biblioteca de Adafruit para acceder a una variedad de sensores y componentes. También puedes utilizar la herramienta de diseño 3D para crear un caso personalizado para tu dispositivo.
