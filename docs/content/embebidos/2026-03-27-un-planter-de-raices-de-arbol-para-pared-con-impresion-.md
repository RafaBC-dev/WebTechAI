## Introducción

Adafruit celebra cada jueves a la comunidad de impresión 3D con proyectos innovadores. En esta ocasión, presentamos un planter de raíces de árbol para pared diseñado para ser impreso en 3D.

## ¿Qué es?

El planter de raíces de árbol para pared es un proyecto de impresión 3D que permite a los usuarios crear un planter para sus plantas en una pared. Se puede descargar el archivo de modelo en el sitio web MakerWorld.

## ¿Cómo funciona?

El planter se imprime en 3D utilizando un modelo digital que se puede descargar desde MakerWorld. El usuario puede personalizar el diseño y la forma del planter según sus preferencias.

## ¿Por qué importa?

Este proyecto es importante porque permite a los usuarios crear un planter para sus plantas en una pared de manera fácil y sencilla. También es un ejemplo de cómo la impresión 3D puede ser utilizada para crear objetos únicos y personalizados.

## Consejo técnico

Si deseas crear un planter similar, puedes descargar el archivo de modelo de MakerWorld y imprimirlo en 3D utilizando un equipo de impresión 3D. Recuerda personalizar el diseño según tus preferencias.
