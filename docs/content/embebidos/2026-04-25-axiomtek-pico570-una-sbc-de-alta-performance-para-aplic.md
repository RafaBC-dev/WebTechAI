# Axiomtek PICO570: una SBC de alta performance para aplicaciones de IA

**Fecha:** 2026-04-25
**Categoría:** embebidos
**Tags:** robotica, ia-local, embebidos
**Título original:** Axiomtek PICO570 – An Intel Core Ultra Meteor Lake Pico-ITX SBC with DDR5-5600 SO-DIMM memory, HDMI 2.1 up to 4Kp120

## Introducción

Axiomtek ha lanzado el PICO570, una SBC de alta performance que combina un procesador Intel Core Ultra Meteor Lake-U con una memoria DDR5-5600 y una interfaz HDMI 2.1. Esta SBC es ideal para aplicaciones de IA, automatización y sistemas de infraestructura inteligente.

## ¿Qué es?

El PICO570 es una SBC (Single-Board Computer) de alta performance que se basa en el procesador Intel Core Ultra Meteor Lake-U. Este procesador ofrece una potencia de cálculo de 15W y una arquitectura de 12 núcleos, lo que la hace ideal para aplicaciones de IA y automatización.

## ¿Cómo funciona?

La SBC cuenta con una memoria DDR5-5600 de hasta 64GB, una interfaz HDMI 2.1 que soporta resoluciones de hasta 4K a 120Hz, y una interfaz de red de 2.5GbE. También cuenta con una interfaz de serie RS-232/422/485 y dos puertos USB 3.2 Gen 2.

## ¿Por qué importa?

El PICO570 es importante porque ofrece una solución de alta performance para aplicaciones de IA y automatización. Puede ser utilizada en sistemas de infraestructura inteligente, controles de automatización, y aplicaciones de IA en tiempo real.

## Consejo técnico

Si deseas utilizar el PICO570 en una aplicación de IA, asegúrate de instalar la última versión del software de Intel OpenVINO, que ofrece una herramienta de desarrollo y depuración de modelos de IA.
