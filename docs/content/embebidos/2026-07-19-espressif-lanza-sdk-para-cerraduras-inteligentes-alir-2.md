# Espressif lanza SDK para cerraduras inteligentes Aliro basadas en ESP32

**Fecha:** 2026-07-19
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y linux
**Título original:** Espressif releases SDK for Aliro smart door locks based on ESP32-C or ESP32-H SoCs

---

## Introducción

Espressif Systems ha lanzado el SDK para Aliro, una tecnología de control de acceso digital que permite a los usuarios acceder a sus hogares de manera segura y conveniente. Esta tecnología se basa en microcontroladores ESP32-C y ESP32-H, y está diseñada para reducir la fragmentación en el mercado de las cerraduras inteligentes.

## ¿Qué es?

Aliro es un estándar de control de acceso digital que permite a los usuarios acceder a sus hogares de manera segura y conveniente. Se basa en tecnologías como NFC, Bluetooth LE y UWB, y está diseñado para ser compatible con dispositivos de diferentes fabricantes. El SDK de Espressif para Aliro permite a los desarrolladores crear aplicaciones que se integren con esta tecnología.

## ¿Cómo funciona?

El SDK de Espressif para Aliro se basa en microcontroladores ESP32-C y ESP32-H, que proporcionan conectividad WiFi y Thread. La tecnología Aliro utiliza NFC para autenticar a los usuarios y permitirles acceder a sus hogares. El SDK también incluye soporte para Bluetooth LE y UWB, que permiten a los usuarios acceder a sus hogares de manera más segura y conveniente.

## ¿Por qué importa?

La tecnología Aliro es importante porque permite a los usuarios acceder a sus hogares de manera segura y conveniente. También reduce la fragmentación en el mercado de las cerraduras inteligentes, lo que significa que los usuarios pueden elegir entre diferentes dispositivos y tecnologías sin tener que cambiar su forma de acceder a sus hogares.

## Consejo técnico

Si deseas crear una aplicación que se integre con la tecnología Aliro, puedes utilizar el SDK de Espressif y el microcontrolador ESP32-C o ESP32-H. También puedes utilizar la biblioteca de desarrollo de Espressif para crear aplicaciones que se integren con esta tecnología.

```bash
git clone https://github.com/espressif/esp-aliro.git
```
