# M5Stack StopWatch: un devkit compacto con AMOLED y audio para IoT y wearables

**Fecha:** 2026-05-22
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y linux
**Título original:** M5Stack StopWatch ESP32-S3 devkit offers 1.75-inch touch AMOLED, microphone, speaker, and GPIO expansion

---

## Introducción

La empresa M5Stack ha lanzado un nuevo devkit llamado StopWatch, que combina una pantalla AMOLED de 1,75 pulgadas con un micrófono, un altavoz y conexiones GPIO. Este dispositivo está diseñado para crear dispositivos IoT portátiles, etiquetas electrónicas y objetos conectados.

## ¿Qué es?

El M5Stack StopWatch es un devkit basado en el procesador ESP32-S3, que cuenta con una pantalla AMOLED táctil de 1,75 pulgadas, 16MB de memoria flash y 8MB de PSRAM. También incluye un micrófono, un altavoz de 1W, un sensor IMU de 6 ejes y una batería de 450mAh.

## ¿Cómo funciona?

El devkit utiliza el procesador ESP32-S3, que cuenta con un microcontrolador dual-core Tensilica LX7 con instrucciones vectoriales para acelerar la inteligencia artificial. La pantalla AMOLED se controla mediante un driver específico y se puede interactuar con ella mediante botones programables y un motor de vibración.

## ¿Por qué importa?

Este devkit es relevante para desarrolladores que buscan crear dispositivos IoT portátiles y objetos conectados con una pantalla táctil y audio. También puede ser utilizado para crear etiquetas electrónicas y objetos conectados con una batería de larga duración.

## Consejo técnico

Para empezar a trabajar con el M5Stack StopWatch, puedes utilizar la biblioteca M5Unified y M5GFX en el entorno de desarrollo Arduino IDE, o utilizar PlatformIO para crear proyectos más complejos.

```bash
sudo apt-get install platformio
```
