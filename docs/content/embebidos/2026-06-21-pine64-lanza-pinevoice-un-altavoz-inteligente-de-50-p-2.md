# Pine64 lanza PineVoice, un altavoz inteligente de $50 para Home Assistant

**Fecha:** 2026-06-21
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software y linux
**Título original:** Pine64 Pinevoice – A $50 RISC-V Smart Speaker for Home Assistant based on Bouffalo Lab BL606P MCU

---

## Introducción

Pine64 ha lanzado PineVoice, un altavoz inteligente de $50 basado en el microcontrolador RISC-V Bouffalo Lab BL606P. Este dispositivo está diseñado para trabajar con Home Assistant y ofrece varias características interesantes.

## ¿Qué es?

PineVoice es un altavoz inteligente que utiliza el microcontrolador RISC-V Bouffalo Lab BL606P, que cuenta con un procesador 64-bit T-Head C906 a 480 MHz, un procesador 32-bit T-Head E907 a 320 MHz, 544KB de RAM y 16MB de PSRAM. También cuenta con WiFi, Bluetooth y Zigbee, lo que lo hace ideal para integrarse con Home Assistant.

## ¿Cómo funciona?

PineVoice funciona conectando el dispositivo a la red y configurando Home Assistant para que se comunique con él. El dispositivo utiliza el protocolo ESPHome, que es el protocolo preferido para este tipo de dispositivos. El altavoz puede reproducir audio y también puede ser utilizado para la detección de voz.

## ¿Por qué importa?

PineVoice es importante porque ofrece una opción asequible para crear un sistema de voz inteligente en casa. También es una excelente opción para desarrolladores que quieren crear aplicaciones de voz inteligente utilizando Home Assistant.

## Consejo técnico

Si estás interesado en crear un sistema de voz inteligente con PineVoice, considera utilizar el protocolo ESPHome para configurar el dispositivo y Home Assistant para que se comunique con él.

```bash
esp-home config --board pinevoice
```
