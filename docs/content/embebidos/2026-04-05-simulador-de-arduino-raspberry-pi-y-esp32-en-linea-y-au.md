# Simulador de Arduino, Raspberry Pi y ESP32 en línea y autónomo

**Fecha:** 2026-04-05
**Categoría:** embebidos
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** Velxio is an open-source, self-hosted Arduino, Raspberry Pi, and ESP32 simulator

---

## Introducción

Velxio es un simulador de código abierto y autónomo para Arduino, ESP32 y Raspberry Pi que se ejecuta directamente en el navegador web. Puede arrastrar y soltar tableros, conectar componentes y módulos, escribir y ejecutar código en Arduino o Python, y acceder a la consola serial, todo sin hardware.

## ¿Qué es?

Velxio es un proyecto de código abierto que permite simular y desarrollar proyectos de electrónica y robótica sin necesidad de hardware. Ofrece una interfaz gráfica de usuario intuitiva y fácil de usar, permitiendo a los usuarios crear y ejecutar proyectos de manera rápida y eficiente.

## ¿Cómo funciona?

El simulador utiliza una combinación de emulación de hardware y software para simular la conducta de los componentes y módulos electrónicos. Esto permite a los usuarios probar y depurar sus proyectos sin necesidad de hardware físico. Velxio también ofrece soporte para múltiples arquitecturas de procesador, incluyendo AVR8, Arm Cortex-M0+, RISC-V y Xtensa.

## ¿Por qué importa?

Velxio es importante porque ofrece una forma rápida y eficiente de desarrollar y probar proyectos de electrónica y robótica sin necesidad de hardware físico. Esto puede ahorrar tiempo y recursos, y permitir a los usuarios crear proyectos más complejos y sofisticados.

## Consejo técnico

Si deseas empezar a utilizar Velxio, te recomendamos descargar el código fuente del proyecto y seguir las instrucciones de instalación en el archivo README. Esto te permitirá instalar el simulador en tu máquina local y empezar a crear proyectos de manera rápida y eficiente.

```bash
docker run -d -p 3080:80 ghcr.io/davidmonterocrespo24/velxio:master
```
