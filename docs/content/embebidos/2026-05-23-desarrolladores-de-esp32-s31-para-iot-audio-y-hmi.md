# Desarrolladores de ESP32-S31 para IoT, audio y HMI

**Fecha:** 2026-05-23
**Categoría:** embebidos
**Tags:** esp32, robotica, embebidos
**Título original:** ESP32-S31 development boards unveiled for IoT, Smart Audio, and HMI applications

## Introducción

La empresa ESP32 ha lanzado dos nuevos desarrolladores de ESP32-S31 para aplicaciones IoT, audio y HMI. Estos desarrolladores ofrecen una amplia gama de características, incluyendo Gigabit Ethernet, WiFi 6 y Bluetooth 5.4.

## ¿Qué es?

El ESP32-S31 es un microcontrolador inalámbrico con dos núcleos RISC-V, que ofrece una amplia gama de características, incluyendo Gigabit Ethernet, WiFi 6 y Bluetooth 5.4. Está diseñado para aplicaciones IoT, audio y HMI.

## ¿Cómo funciona?

El ESP32-S31 utiliza dos núcleos RISC-V, uno para realizar tareas de alto rendimiento y otro para realizar tareas de bajo consumo de energía. También cuenta con una memoria de 512 KB de SRAM y 16 MB de PSRAM.

## ¿Por qué importa?

El ESP32-S31 es importante porque ofrece una amplia gama de características para aplicaciones IoT, audio y HMI. También es compatible con una amplia gama de protocolos, incluyendo Zigbee, Thread y Matter.

## Consejo técnico

Si deseas desarrollar aplicaciones con el ESP32-S31, te recomendamos utilizar la documentación oficial de ESP32 para obtener más información sobre la configuración y el uso del desarrollador.
