# Cámara de germinación de semillas con impresión 3D

## Introducción

Un proyecto de impresión 3D busca mejorar la germinación de semillas creando un entorno controlado y eficiente. El objetivo es crear un sistema que permita a los usuarios cultivar semillas de manera fácil y efectiva.

## ¿Qué es?

El proyecto consiste en una cámara de germinación de semillas creada mediante impresión 3D. La cámara tiene dos compartimentos, uno para las semillas y otro para el agua, lo que permite controlar el nivel de humedad y evitar que el agua se acumule en el compartimento de las semillas.

## ¿Cómo funciona?

La cámara de germinación funciona mediante el principio de comunicación de vasos, que establece que el nivel de un líquido en un recipiente se iguala al nivel de un líquido en otro recipiente conectado a él. En este caso, el compartimento de agua se conecta al compartimento de las semillas a través de un pequeño orificio, lo que permite que el agua se evapore y se humedezca el compartimento de las semillas sin que el agua se acumule.

## ¿Por qué importa?

Este proyecto importa porque ofrece una solución innovadora y eficiente para la germinación de semillas. Al permitir a los usuarios controlar el nivel de humedad y evitar la acumulación de agua, se reduce el riesgo de enfermedades y se mejora la tasa de germinación.

## Consejo técnico

Si deseas crear tu propia cámara de germinación de semillas, puedes descargar los modelos de impresión 3D disponibles en el proyecto original y personalizarlos según tus necesidades. Recuerda que es importante ajustar el tamaño del orificio que conecta los compartimentos para asegurarte de que el agua se evapore de manera adecuada.
