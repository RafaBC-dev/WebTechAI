# Sensores de visión ultra-bajos en consumo para aplicaciones de IA y visión siempre encendidas

**Fecha:** 2026-04-30
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** STMicro VD65G4 and VD55G4 0.56MP global shutter image sensors enable ultra-low-power always-on event-driven vision

---

## Introducción

STMicroelectronics ha presentado dos sensores de visión ultra-bajos en consumo, el VD65G4 y el VD55G4, diseñados para aplicaciones de IA y visión siempre encendidas. Estos sensores son ideales para wearables, AR/VR y dispositivos de hogar inteligentes.

## ¿Qué es?

Los sensores VD65G4 y VD55G4 son sensores de visión de baja resolución (0,56 MP) con captura global y bajo consumo de energía. El VD65G4 es un sensor de color RGB, mientras que el VD55G4 es un sensor monocromo que captura luz visible y near-infrarroja.

## ¿Cómo funciona?

Estos sensores utilizan tecnologías como Back Side Illuminated (BSI), CDTI y 3D stacking para reducir su tamaño y consumo de energía. El sensor puede funcionar en diferentes modos, incluyendo un modo de auto-despertar que consume solo 1-2 mW.

## ¿Por qué importa?

Estos sensores son importantes porque pueden ser utilizados en aplicaciones de IA y visión siempre encendidas, como wearables, AR/VR y dispositivos de hogar inteligentes. También pueden ser utilizados en cámaras IP para reducir falsos alertas.

## Consejo técnico

Si deseas utilizar estos sensores en tu proyecto, asegúrate de utilizar la biblioteca de drivers de STMicroelectronics para Linux, que proporciona una interfaz fácil de usar para acceder a los sensores.

```bash
sudo apt-get install st-vd65g4-driver
```
