## Introducción

Adafruit presenta una caja para Raspberry Pi Zero 2W con pantalla e-Paper y UPS, ideal para proyectos de robótica y automatización. Esta caja es perfecta para crear proyectos portátiles con Raspberry Pi.

## ¿Qué es?

La caja es un proyecto de impresión 3D que incluye una pantalla e-Paper de 2.13 pulgadas y un módulo de UPS (Uninterruptible Power Supply) de 18650, diseñado para Raspberry Pi Zero 2W. La caja cuenta con un botón de encendido y diferentes estilos de tapa.

## ¿Cómo funciona?

La caja se imprime en 3D utilizando un modelo de archivo disponible en el sitio web de MakerWorld. La pantalla e-Paper se conecta a Raspberry Pi a través de una placa de expansión, mientras que el módulo de UPS proporciona alimentación a la placa cuando se pierde la conexión a la red eléctrica.

## ¿Por qué importa?

Esta caja es importante porque permite a los usuarios crear proyectos de robótica y automatización portátiles y fiables. La pantalla e-Paper y el módulo de UPS hacen que la caja sea ideal para aplicaciones que requieren una conexión constante a la red eléctrica.

## Consejo técnico

Para imprimir la caja en 3D, es recomendable utilizar un modelo de archivo de alta resolución y un material de impresión de alta calidad para garantizar la precisión y la durabilidad de la caja.
