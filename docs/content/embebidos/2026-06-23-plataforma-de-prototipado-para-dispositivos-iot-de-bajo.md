# Plataforma de prototipado para dispositivos IoT de bajo consumo

**Fecha:** 2026-06-23
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** Coin-cell powered Nordic nRF54L15 Tag prototyping platform supports Bluetooth Channel Sounding, Matter, Edge AI

---

## Introducción

Nordic Semiconductor ha lanzado la plataforma de prototipado nRF54L15 Tag, diseñada para acelerar el desarrollo de dispositivos IoT de ultra-bajo consumo, incluyendo rastreadores de activos compatibles con los ecosistemas Apple Find My y Google Find Hub.

## ¿Qué es?

La nRF54L15 Tag es una plataforma de prototipado de 33 mm de diámetro que utiliza una batería de 2,4 GHz y cuenta con dos antenas integradas para mejorar la fiabilidad de la medición de distancia con el sonido de canal de Bluetooth. Incorpora un procesador Arm Cortex-M33, un sensor de movimiento 6-axial y un sensor ambiental.

## ¿Cómo funciona?

La plataforma utiliza el procesador nRF54L15 de Nordic Semi, que cuenta con un CPU Arm Cortex-M33 de hasta 128 MHz y un coprocesador RISC-V para periféricos 'software-definidos'. La plataforma también incluye memoria SRAM de 256 KB y almacenamiento de 1,5 MB de memoria no volátil.

## ¿Por qué importa?

La nRF54L15 Tag es importante porque permite la creación de dispositivos IoT de ultra-bajo consumo que pueden ser utilizados en aplicaciones como rastreadores de activos, monitoreo ambiental y procesamiento de datos de Edge AI.

## Consejo técnico

Para aprovechar al máximo la plataforma nRF54L15 Tag, es recomendable utilizar la biblioteca de software Aliro para desarrollar aplicaciones compatibles con la norma de acceso control y seguridad de la Connectivity Standards Alliance.
