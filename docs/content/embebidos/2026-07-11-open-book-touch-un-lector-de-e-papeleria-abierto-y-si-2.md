# Open Book Touch: un lector de e-papelería abierto y sin DRM

## Introducción

Oddly Specific Objects ha lanzado una campaña de crowdfunding para el Open Book Touch, un lector de e-papelería con pantalla táctil y conexión WiFi. Este dispositivo abierto y sin DRM ofrece una experiencia de lectura única y personalizable.

## ¿Qué es?

El Open Book Touch es un lector de e-papelería basado en el SoC ESP32-S3 de Espressif Systems, con una pantalla de 4,26 pulgadas y 480 x 800 resolución. Tiene una memoria de 8 MB de PSRAM, una tarjeta microSD para almacenamiento y una batería de 800 mAh reemplazable. El dispositivo se puede programar con Arduino o CircuitPython y viene con un firmware preinstalado para una experiencia de lectura óptima.

## ¿Cómo funciona?

El Open Book Touch utiliza el firmware preinstalado basado en el framework ESP-IDF para optimizar el rendimiento y la eficiencia energética. El firmware soporta formatos de archivo EPUB y texto plano, con características como la hípernea, la justificación y las imágenes en línea. También ofrece soporte multilingüe con la fuente Unifont y una interfaz de usuario localizable en siete idiomas.

## ¿Por qué importa?

El Open Book Touch es importante porque ofrece una experiencia de lectura personalizable y sin DRM, lo que permite a los usuarios transferir sus libros a través de la interfaz web integrada. Además, el dispositivo es abierto y sin licencia, lo que permite a los desarrolladores personalizar y mejorar el firmware y la interfaz de usuario.

## Consejo técnico

Si deseas personalizar el firmware del Open Book Touch, puedes utilizar el framework ESP-IDF y la fuente Unifont para crear una interfaz de usuario personalizada. También puedes utilizar la interfaz web integrada para transferir tus libros y configurar el dispositivo según tus necesidades.

```bash
esp-idf
```
