# Amplificador de audio de alta calidad con ESP32 y Home Assistant

## Introducción

El desarrollador de Esparagus ha lanzado un nuevo proyecto: un amplificador de audio de alta calidad basado en ESP32 que soporta Home Assistant y Squeezelite. Este módulo compacto puede ser utilizado para crear sistemas de audio en todo el hogar, actualizar altavoces vintage con características de streaming modernas o alimentar configuraciones de altavoces personalizadas como subwoofers o sistemas de altavoces bi-amp.

## ¿Qué es?

El Esparagus Audio Brick es un módulo de amplificador de audio de alta calidad que se basa en el procesador dual-core ESP32 o ESP32-S3. Está diseñado para ser compacto y fácil de instalar, con una carcasa de plástico y un soporte de DIN-rail. Soporta Wi-Fi y Ethernet, y tiene una interfaz de audio I²S que permite conectar altavoces de alta calidad.

## ¿Cómo funciona?

El Esparagus Audio Brick utiliza un procesador ESP32 o ESP32-S3 para procesar la señal de audio y controlar el amplificador. El módulo incluye un DAC Texas Instruments TAS5825M que proporciona una salida de audio de alta calidad, y un DSP que permite realizar ajustes de igualización, compresión dinámica y control de volumen. El módulo también incluye una interfaz USB Type-C para cargar el firmware y una LED RGB para indicar el estado del módulo.

## ¿Por qué importa?

El Esparagus Audio Brick es importante porque proporciona una solución de alta calidad para crear sistemas de audio en todo el hogar. Puede ser utilizado para actualizar altavoces vintage con características de streaming modernas, o para alimentar configuraciones de altavoces personalizadas. Además, soporta Home Assistant y Squeezelite, lo que lo hace compatible con una amplia gama de sistemas de streaming de audio.

## Consejo técnico

Si estás interesado en crear un sistema de audio en todo el hogar, considera utilizar el Esparagus Audio Brick como base para tu proyecto. Puedes conectarlo a tu red local y utilizar Home Assistant para controlar y configurar el módulo.

```bash
Para instalar Home Assistant en tu Raspberry Pi, ejecuta el siguiente comando: `sudo apt-get update && sudo apt-get install homeassistant`
```
