# Modem 5G RedCap y 4G LTE compacto para IoT industrial

**Fecha:** 2026-05-18
**Categoría:** embebidos
**Tags:** robotica, embebidos, linux
**Título original:** Semtech FX86E – A Compact 5G RedCap and 4G LTE modem for industrial IoT applications

## Introducción

Semtech ha lanzado el modem FX86E, un dispositivo compacto que combina la conectividad 5G RedCap y 4G LTE para aplicaciones industriales de IoT. Este modem es ideal para cámaras de seguridad resistentes, vehículos pesados y dispositivos alimentados por baterías o energía solar.

## ¿Qué es?

El modem FX86E es un dispositivo de conectividad celular diseñado para aplicaciones industriales de IoT. Se basa en el módulo EM8695 M.2 de Semtech, que utiliza el procesador Snapdragon X35 5G RedCap. Ofrece una conectividad 5G RedCap y 4G LTE, con soporte para redes públicas y privadas.

## ¿Cómo funciona?

El modem FX86E utiliza el procesador Snapdragon X35 5G RedCap para proporcionar conectividad 5G RedCap y 4G LTE. El módulo EM8695 M.2 se conecta a un dispositivo industrial mediante una interfaz M.2. El modem soporta múltiples frecuencias y bandas, incluyendo 5G Sub-6 GHz y LTE Cat-4.

## ¿Por qué importa?

El modem FX86E es importante porque ofrece una conectividad segura y confiable para aplicaciones industriales de IoT. Puede ser utilizado en cámaras de seguridad resistentes, vehículos pesados y dispositivos alimentados por baterías o energía solar. Su diseño compacto y su bajo consumo de energía lo hacen ideal para aplicaciones de IoT que requieren una conectividad estable y segura.

## Consejo técnico

Si estás desarrollando un proyecto de IoT industrial, considera utilizar el modem FX86E para proporcionar conectividad 5G RedCap y 4G LTE a tus dispositivos. Puedes configurar el modem utilizando la interfaz web de OpenWrt y realizar actualizaciones de firmware a través de HTTPS.

```bash
AT+CGDCONT=1,"IP","apn.telnx.com"
```
