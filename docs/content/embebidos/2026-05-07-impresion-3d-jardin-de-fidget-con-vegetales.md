# Impresión 3D: Jardín de Fidget con Vegetales

**Fecha:** 2026-05-07
**Categoría:** embebidos

**Título original:** Fidget Garden (Carrot, Radish, Broccoli, Corn) #3DPrinting #3DThursday

## Introducción

La comunidad de impresión 3D en Adafruit ha creado un nuevo proyecto llamado Fidget Garden, donde se pueden imprimir modelos de vegetales como zanahorias, rábanos, brócoli y maíz. Este proyecto es una excelente oportunidad para los entusiastas de la impresión 3D y la robótica para crear objetos únicos y personalizados.

## ¿Qué es?

El Fidget Garden es un conjunto de modelos de impresión 3D que representan diferentes tipos de vegetales, como zanahorias, rábanos, brócoli y maíz. Estos modelos están diseñados para ser impresos en una impresora 3D y pueden ser personalizados para crear objetos únicos.

## ¿Cómo funciona?

Para imprimir los modelos del Fidget Garden, se pueden utilizar herramientas como Cura o Slic3r para preparar la impresión, y luego se pueden imprimir utilizando una impresora 3D. Los modelos pueden ser personalizados utilizando software de diseño asistido por computadora (CAD) como Blender o Tinkercad.

## ¿Por qué importa?

El Fidget Garden es importante porque ofrece una forma divertida y creativa de aprender sobre la impresión 3D y la robótica. También puede ser utilizado para crear objetos únicos y personalizados para decorar el hogar o regalar a amigos y familiares.

## Consejo técnico

Si deseas imprimir los modelos del Fidget Garden, asegúrate de utilizar un software de diseño CAD como Blender o Tinkercad para personalizar los modelos y agregar detalles adicionales.
