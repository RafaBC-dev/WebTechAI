# KiCad 10: software de diseño de PCB con mejoras significativas

**Fecha:** 2026-03-23
**Categoría:** diseño-3d
**Tags:** robotica, diseño-3d, herramientas
**Título original:** KiCad 10 release – Dark mode, graphical DRC rule editor, new file importers, and more

---

## Introducción

La versión 10 del software de diseño de PCB KiCad ha sido lanzada con varias mejoras importantes, incluyendo soporte para modo oscuro, editores gráficos de reglas de diseño y nuevos importadores de archivos. Esto significa que los diseñadores de PCB pueden trabajar de manera más eficiente y con herramientas más avanzadas.

## ¿Qué es?

KiCad es un software de código abierto para el diseño de PCB (Placas de Circuitos Impresos) que permite a los usuarios crear y editar diseños de PCB de manera gratuita. Es una herramienta popular entre los diseñadores de electrónica y los ingenieros de hardware.

## ¿Cómo funciona?

KiCad 10 ofrece una interfaz de usuario mejorada con soporte para modo oscuro, herramientas personalizables y una barra de herramientas configurable. También incluye un editor gráfico de reglas de diseño, que permite a los usuarios crear y editar reglas de diseño de manera visual. Además, se han agregado nuevos importadores de archivos para Allegro, PADS y gEDA/Lepton PCB.

## ¿Por qué importa?

KiCad 10 es importante porque ofrece una herramienta de diseño de PCB más avanzada y eficiente, lo que puede ahorrar tiempo y esfuerzo a los diseñadores de electrónica y los ingenieros de hardware. También es una herramienta gratuita y de código abierto, lo que la hace accesible a una amplia gama de usuarios.

## Consejo técnico

Si estás utilizando KiCad 10, considera crear un proyecto de diseño de PCB nuevo con el editor gráfico de reglas de diseño para aprovechar al máximo las nuevas funcionalidades.

```bash
sudo add-apt-repository ppa:kicad/kicad-10.0-releases
sudo apt install kicad
```
