# Convertir una impresora 3D antigua en una cortadora de vinilo a bajo costo

## Introducción

Una comunidad de impresión 3D ha encontrado una forma innovadora de dar nueva vida a sus impresoras 3D antiguas, convirtiéndolas en cortadoras de vinilo de bajo costo. Esto abre nuevas posibilidades para los entusiastas de la impresión 3D y la creación de arte.

## ¿Qué es?

La Polycut es un proyecto que permite convertir una impresora 3D en una cortadora de vinilo, utilizando un software que genera código G para controlar la impresora. Esto permite a los usuarios crear vinilos personalizados sin necesidad de una máquina especializada.

## ¿Cómo funciona?

El proyecto Polycut utiliza un software que toma un archivo SVG y lo convierte en código G para la impresora. Luego, se necesita ajustar la calibración de la Z para obtener resultados precisos. La impresora 3D se equipa con una hoja de corte especializada y un soporte para la hoja de corte, que se puede imprimir en 3D.

## ¿Por qué importa?

Esta solución es importante porque permite a los entusiastas de la impresión 3D y la creación de arte crear vinilos personalizados a bajo costo, sin necesidad de una máquina especializada. Esto abre nuevas posibilidades para la creación de arte y la personalización de productos.

## Consejo técnico

Si deseas convertir tu impresora 3D en una cortadora de vinilo, asegúrate de ajustar la calibración de la Z correctamente para obtener resultados precisos. Puedes utilizar el software Polycut para generar código G y configurar la impresora.

```bash
polycut
```
