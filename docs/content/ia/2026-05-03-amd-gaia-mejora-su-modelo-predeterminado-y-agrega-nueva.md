# AMD GAIA mejora su modelo predeterminado y agrega nuevas funcionalidades

**Fecha:** 2026-05-03
**Categoría:** ia
**Tags:** llms, ia-local, linux
**Título original:** AMD's GAIA Defaults To Better Model, Continued Improvements For Local AI

---

## Introducción

AMD ha lanzado una nueva versión de GAIA, su software de código abierto para Windows y Linux, que busca hacer que sea fácil crear agentes de IA local en tu ordenador con todos los procesos de IA realizados localmente en los procesadores, tarjetas gráficas y NPUs de AMD. Esta versión incluye mejoras significativas en el modelo predeterminado y la adición de nuevas funcionalidades.

## ¿Qué es?

GAIA es un software de código abierto desarrollado por AMD para crear agentes de IA local en ordenadores con procesadores, tarjetas gráficas y NPUs de AMD. Utiliza la Lemonade SDK y busca hacer que sea fácil crear y ejecutar agentes de IA en Linux y Windows.

## ¿Cómo funciona?

GAIA utiliza un modelo de lenguaje llamado Gemma 4 E4B, que es un modelo multimodal de ~4,5 billones de parámetros con una ventana de contexto de 128K. El modelo se utiliza para realizar tareas de procesamiento de lenguaje natural, como la generación de texto y la respuesta a preguntas. GAIA también incluye una interfaz de usuario y una suite de evaluación para probar y mejorar el rendimiento de los agentes de IA.

## ¿Por qué importa?

GAIA es importante porque permite a los usuarios crear agentes de IA local en sus ordenadores, lo que puede ser útil para tareas como la generación de texto, la respuesta a preguntas y la automatización de tareas. La adición de nuevas funcionalidades y mejoras en el modelo predeterminado hacen que GAIA sea una herramienta más potente y fácil de usar.

## Consejo técnico

Si deseas probar GAIA en tu ordenador con Linux, puedes instalarlo utilizando pip con el comando `pip install amd-gaia[ui] && gaia chat --ui`.

```bash
pip install amd-gaia[ui] && gaia chat --ui
```
