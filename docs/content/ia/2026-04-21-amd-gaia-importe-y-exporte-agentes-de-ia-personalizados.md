# AMD GAIA: importe y exporte agentes de IA personalizados con facilidad

**Fecha:** 2026-04-21
**Categoría:** ia
**Tags:** llms, ia-local, linux
**Título original:** AMD's GAIA Makes It Easier To Import/Export Custom AI Agents Across PCs

---

## Introducción

AMD ha lanzado una actualización de su plataforma GAIA, que permite a los usuarios importar y exportar agentes de IA personalizados entre sistemas de manera sencilla. Esto abre nuevas posibilidades para la creación de aplicaciones de IA local y la colaboración en proyectos de IA.

## ¿Qué es?

AMD GAIA es una plataforma de software que permite a los usuarios crear y ejecutar agentes de IA local en hardware AMD, incluyendo CPUs, GPUs y NPUs. Los agentes de IA son programas que pueden aprender y tomar decisiones basadas en datos y patrones. Con GAIA, los usuarios pueden crear y personalizar sus propios agentes de IA para realizar tareas específicas.

## ¿Cómo funciona?

La plataforma GAIA utiliza la biblioteca Lemonade SDK para crear y ejecutar agentes de IA. La actualización de GAIA 0.17.3 permite a los usuarios crear un instalador personalizado para sus agentes de IA, que se puede exportar y importar entre sistemas. Esto se logra mediante la utilización de formatos de archivo JSON y HMAC-SHA256 para la caché de RAG. Además, la plataforma ahora preserva las URLs de base compatibles con OpenAI para trabajar con back-ends de inferencia alternativos.

## ¿Por qué importa?

La capacidad de importar y exportar agentes de IA personalizados entre sistemas es fundamental para la creación de aplicaciones de IA local y la colaboración en proyectos de IA. Esto permite a los usuarios compartir y reutilizar agentes de IA, lo que puede acelerar el desarrollo de aplicaciones de IA y mejorar su calidad.

## Consejo técnico

Si deseas aprovechar al máximo la plataforma GAIA, te recomendamos utilizar la biblioteca Lemonade SDK para crear agentes de IA personalizados. Puedes descargar la biblioteca desde GitHub y seguir los tutoriales y documentación disponibles para aprender a utilizarla.

```bash
npm install lemonade-sdk
```
