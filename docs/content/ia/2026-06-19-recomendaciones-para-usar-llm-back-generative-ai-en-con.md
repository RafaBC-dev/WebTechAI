# Recomendaciones para usar LLM-back generative AI en contribuciones de FOSS

**Fecha:** 2026-06-19

**Título original:** The Software Freedom Conservancy's LLM-backed generative AI recommendations

## Introducción

La Software Freedom Conservancy ha lanzado recomendaciones para usar sistemas de inteligencia artificial generativa respaldados por LLM en contribuciones de software libre. Estas recomendaciones buscan ayudar a los desarrolladores de software libre a navegar por las complejas decisiones que se presentan al utilizar estas tecnologías.

## ¿Qué es?

Las LLM-back generative AI son sistemas de inteligencia artificial que utilizan modelos de lenguaje largo (LLM) para generar texto, imágenes o otros contenidos. Estos sistemas pueden ser útiles en contribuciones de software libre, pero también plantean desafíos en términos de privacidad, seguridad y propiedad intelectual.

## ¿Cómo funciona?

Las recomendaciones de la Software Freedom Conservancy se basan en un estudio exhaustivo de la forma en que los desarrolladores de software libre están utilizando LLM-back generative AI en sus contribuciones. Los voluntarios de la organización han creado un conjunto de prácticas recomendadas para minimizar el daño causado por el uso de sistemas propietarios.

## ¿Por qué importa?

Estas recomendaciones son importantes porque ayudan a los desarrolladores de software libre a navegar por las complejas decisiones que se presentan al utilizar LLM-back generative AI. Al seguir estas recomendaciones, los desarrolladores pueden minimizar el daño causado por el uso de sistemas propietarios y proteger su libertad y privacidad.

## Consejo técnico

Si estás utilizando LLM-back generative AI en tus contribuciones de software libre, considera utilizar herramientas como la biblioteca de Python `transformers` para trabajar con modelos de lenguaje largo. Esto te permitirá crear modelos personalizados y controlar mejor el proceso de generación de contenido.

```bash
pip install transformers
```
