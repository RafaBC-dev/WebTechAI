# Modelos de lenguaje visual entrenan a robots para leer emociones humanas

**Fecha:** 2026-06-18

## Introducción

Los robots podrían trabajar junto a humanos, pero para hacerlo necesitan leer nuestras emociones. Un estudio reciente ha entrenado a robots para leer emociones humanas mediante modelos de lenguaje visual.

## ¿Qué es?

Los modelos de lenguaje visual (VLM) son algoritmos que pueden leer imágenes y texto. En este estudio, se utilizaron VLM para entrenar a robots a leer emociones humanas en videos. Los robots aprendieron a reconocer expresiones faciales y contextos de interacción para determinar la emoción de una persona.

## ¿Cómo funciona?

Los investigadores entrenaron un VLM utilizando el software Gemini 2.5 y lo evaluaron con 40 voluntarios. Los voluntarios describieron las emociones de las personas en videos de robots entregando objetos a humanos. El VLM aprendió a reconocer expresiones faciales y contextos de interacción, como el ritmo cardíaco o el lenguaje corporal, para determinar la emoción de la persona.

## ¿Por qué importa?

Este estudio es importante porque muestra que los robots pueden aprender a leer emociones humanas de manera efectiva. Esto podría mejorar la interacción entre humanos y robots en entornos de trabajo o en la vida cotidiana.

## Consejo técnico

Si deseas entrenar a un robot para leer emociones humanas, puedes utilizar el software Gemini 2.5 y un modelo de lenguaje visual como el utilizado en este estudio.

```bash
python -m pip install gemini
```
