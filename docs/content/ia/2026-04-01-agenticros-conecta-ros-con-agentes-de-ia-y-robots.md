# AgenticROS: Conecta ROS con agentes de IA y robots

**Fecha:** 2026-04-01

## Introducción

El proyecto AgenticROS permite a los desarrolladores de robótica conectar agentes de IA con robots utilizando el framework ROS2. Esto abre nuevas posibilidades para la robótica autónoma y la automatización.

## ¿Qué es?

AgenticROS es un proyecto de código abierto que agrega soporte a ROS2 a múltiples agentes de IA, incluyendo OpenClaw, Claude Code, Claude Desktop/Dispatch y Google Gemini. Permite a los desarrolladores conectar estos agentes con robots y otros dispositivos para crear sistemas de robótica autónoma.

## ¿Cómo funciona?

AgenticROS ofrece cuatro modos de despliegue para conectar agentes de IA con robots, dependiendo de la ubicación del agente en relación con el robot. Los modos incluyen ejecución local, red local, nube remota y uso de Zenoh. El proyecto también incluye una arquitectura de plugins llamada Skills, que permite a los desarrolladores crear y compartir herramientas personalizadas para sus sistemas de robótica.

## ¿Por qué importa?

AgenticROS importa porque permite a los desarrolladores crear sistemas de robótica autónoma más complejos y eficientes. También ofrece una plataforma para compartir y reutilizar herramientas y plugins, lo que puede acelerar el desarrollo de la robótica y la automatización.

## Consejo técnico

Si estás desarrollando un sistema de robótica autónoma, considera utilizar AgenticROS para conectar tus agentes de IA con tus robots. Puedes empezar instalando el proyecto y explorando la documentación para aprender a utilizar la arquitectura de plugins Skills y los diferentes modos de despliegue.

```bash
npm install agenticros
```
