# Nueva placa base eATX con procesador AMD EPYC Embedded 8004 para aplicaciones de IA en borde

**Fecha:** 2026-04-08
**Categoría:** ia
**Tags:** ia-local, robotica, embebidos
**Título original:** AMD EPYC Embedded 8004 eATX motherboard supports up to 576GB DDR5 memory for Edge AI applications

---

## Introducción

IBASE Technology ha lanzado la placa base MBB1002, una solución de alta gama para aplicaciones de inteligencia artificial (IA) en borde que utiliza el procesador AMD EPYC Embedded 8004 y puede alcanzar hasta 576 GB de memoria DDR5.

## ¿Qué es?

La placa base MBB1002 es una solución de alta gama diseñada para aplicaciones de IA en borde, como la automatización industrial, la transporte y la infraestructura de cómputo en borde. Está equipada con el procesador AMD EPYC Embedded 8004, que ofrece hasta 64 núcleos y 128 hilos de ejecución, y puede alcanzar velocidades de memoria de hasta 4800 MHz.

## ¿Cómo funciona?

La placa base MBB1002 cuenta con cinco slots de expansión PCIe Gen5 x16 para integrar diferentes GPUs y aceleradores de IA, así como dos puertos de red 10 GbE, almacenamiento NVMe y SATA, y puertos USB 3.2 con control de poder periférico (PDPC). También cuenta con un temporizador de watchdog, un monitor de hardware y cuatro conectores de ventilador de 4 pines.

## ¿Por qué importa?

Esta placa base es importante porque ofrece una solución de alta gama para aplicaciones de IA en borde, lo que permite a los desarrolladores crear soluciones más eficientes y escalables para diferentes industrias. También es interesante porque utiliza el procesador AMD EPYC Embedded 8004, que ofrece una gran cantidad de recursos de procesamiento y memoria.

## Consejo técnico

Si estás interesado en utilizar la placa base MBB1002, asegúrate de instalar el sistema operativo Ubuntu 22.04, Windows Server 2022 o Windows Server 2025, ya que estos son los sistemas operativos soportados por la placa. También es importante mencionar que la placa no incluye drivers, por lo que debes asegurarte de que el sistema operativo tenga los drivers necesarios para funcionar correctamente.

```bash
sudo apt-get install ubuntu-drivers-common
```
