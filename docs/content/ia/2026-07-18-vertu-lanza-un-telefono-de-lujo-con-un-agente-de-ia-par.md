# Vertu lanza un teléfono de lujo con un agente de IA para ejecutivos

**Fecha:** 2026-07-18
**Categoría:** ia
**Tags:** ia-local, agentes, linux
**Título original:** Vertu wants executives to pay $6,880 for an AI agent — here’s how it actually performs

---

## Introducción

Vertu, fabricante de teléfonos de lujo, ha lanzado el Alphafold, un teléfono plegable que se dirige a ejecutivos y empresarios de alto nivel. Este teléfono no se centra en especificaciones técnicas, sino en proporcionar un asistente de IA personalizado para automatizar tareas y mejorar la productividad.

## ¿Qué es?

El Alphafold es un teléfono plegable de lujo que incorpora un agente de IA llamado Hermes Agent, desarrollado sobre la base del proyecto de código abierto Hermes. Este agente puede analizar archivos, automatizar tareas en aplicaciones, recordar conversaciones y transferir solicitudes a un concierge humano cuando sea necesario.

## ¿Cómo funciona?

El Hermes Agent se basa en el proyecto de código abierto Hermes y se ejecuta en el teléfono Alphafold. Puede realizar flujos de trabajo complejos para automatizar tareas y mejorar la productividad del usuario. El agente puede interactuar con múltiples aplicaciones y recordar conversaciones para proporcionar una experiencia de usuario personalizada.

## ¿Por qué importa?

El Alphafold y su agente de IA pueden ser beneficiosos para ejecutivos y empresarios que necesitan automatizar tareas y mejorar su productividad. El agente puede ayudar a reducir el tiempo dedicado a tareas repetitivas y mejorar la eficiencia en la gestión de documentos, análisis de spreadsheets y planificación de viajes.

## Consejo técnico

Si deseas implementar un agente de IA similar en tu teléfono o dispositivo, puedes investigar el proyecto Hermes y aprender a desarrollar tus propias aplicaciones de IA utilizando herramientas como Python y librerías de IA como TensorFlow o PyTorch.

```bash
pip install hermes
```
