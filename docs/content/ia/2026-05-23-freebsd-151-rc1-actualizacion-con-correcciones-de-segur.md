# FreeBSD 15.1-RC1: Actualización con correcciones de seguridad gracias a IA

**Fecha:** 2026-05-23
**Categoría:** ia
**Tags:** linux, codigo, herramientas
**Título original:** FreeBSD 15.1-RC1 Released: Fixes With Now Seeing More AI-Discovered Security Issues

## Introducción

La versión de pruebas de FreeBSD 15.1-RC1 ha sido lanzada con una serie de correcciones de seguridad, gracias a la colaboración de herramientas de descubrimiento de vulnerabilidades impulsadas por inteligencia artificial. Esto se suma a la reciente afluencia de vulnerabilidades de seguridad en Linux.

## ¿Qué es?

FreeBSD es un sistema operativo de código abierto que se enfoca en la estabilidad y la seguridad. Es una alternativa popular a Linux y se utiliza en una variedad de aplicaciones, desde servidores hasta dispositivos embebidos.

## ¿Cómo funciona?

FreeBSD 15.1-RC1 incluye correcciones de seguridad para varios problemas, incluyendo una vulnerabilidad de uso después de liberación del kernel, una falta de validación en ptrace y una vulnerabilidad de ejecución de código remoto en el instalador de FreeBSD. Estas correcciones se han identificado gracias a la colaboración de herramientas de descubrimiento de vulnerabilidades impulsadas por inteligencia artificial, como Calif.io y AISLE Research Team.

## ¿Por qué importa?

Estas correcciones de seguridad son importantes porque protegen a los usuarios de FreeBSD de posibles ataques maliciosos. La colaboración de herramientas de descubrimiento de vulnerabilidades impulsadas por inteligencia artificial ayuda a identificar y corregir problemas de seguridad antes de que puedan ser explotados.

## Consejo técnico

Si estás utilizando FreeBSD, te recomendamos actualizar a la versión 15.1-RC1 para aprovechar las correcciones de seguridad incluidas. Puedes hacerlo ejecutando el comando `freebsd-update -r 15.1-RC1 upgrade`.

```bash
freebsd-update -r 15.1-RC1 upgrade
```
