## Resumen

La cámara instantánea es una tecnología analógica que permite obtener fotografías químicas en segundos. BoxArt ha creado una cámara instantánea con un Raspberry Pi y una impresora, ofreciendo una alternativa más asequible a las cámaras instantáneas comerciales. Esta cámara utiliza un Pi Zero y una cámara para imprimir imágenes en papel en grises.

## Puntos clave

- Cámara instantánea con tecnología Raspberry Pi
- Uso de una impresora para imprimir imágenes
- Alternativa asequible a cámaras comerciales

## Consejo técnico

Puedes utilizar un Pi Zero y una cámara para crear una cámara instantánea similar.
