## ¿Qué es?

ChatGPT es un modelo de lenguaje inteligente desarrollado por OpenAI que permite a los usuarios interactuar con él de manera natural. El modelo utiliza técnicas de aprendizaje automático para generar respuestas a las preguntas y solicitudes de los usuarios. El nuevo plan Pro de $100/mes se centra en el uso diario de la herramienta de código Codex, que permite a los usuarios escribir código de manera más eficiente y rápida.

## ¿Cómo funciona?

El plan Pro de $100/mes ofrece 5 veces más capacidad de código que el plan Plus de $20/mes. El modelo Codex utiliza técnicas de procesamiento de lenguaje natural para analizar y generar código de manera automática. El usuario puede interactuar con el modelo a través de una interfaz de usuario amigable, lo que permite escribir código de manera más eficiente y rápida.
