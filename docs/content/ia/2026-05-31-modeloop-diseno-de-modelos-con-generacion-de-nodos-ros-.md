# Modeloop: diseño de modelos con generación de nodos ROS 2

**Fecha:** 2026-05-31
**Categoría:** ia
**Tags:** robotica, ros2, linux
**Título original:** Modeloop - model-based design with ROS 2 node generation

## Introducción

Modeloop es un entorno en la nube que permite diseñar modelos de manera gráfica y generar código para robots y sistemas embebidos. Esta herramienta es especialmente útil para desarrolladores de robótica que trabajan con ROS 2.

## ¿Qué es?

Modeloop es un entorno de diseño de modelos en la nube que permite conectar bloques en un diagrama, simular dinámicas y generar código. Se basa en la tecnología de modelado de Simulink, pero con un editor que se ejecuta en el navegador y un formato de modelo que es texto plano y compatible con Git.

## ¿Cómo funciona?

El modelo se crea en el navegador, donde se pueden conectar bloques y simular dinámicas. Luego, se puede generar código para ROS 2 utilizando la herramienta de toolbox. La toolbox también permite importar mensajes y servicios de ROS, crear nodos de publicación y suscripción y configurar QoS.

## ¿Por qué importa?

Modeloop es importante porque permite a los desarrolladores de robótica diseñar modelos de manera gráfica y generar código para ROS 2 de manera eficiente. Esto puede ayudar a acelerar el desarrollo de proyectos de robótica y mejorar la calidad del código.

## Consejo técnico

Si estás trabajando con ROS 2 y quieres aprovechar las funcionalidades de Modeloop, puedes unirte a la lista de espera y obtener acceso a la herramienta de toolbox para generar código y importar mensajes y servicios de ROS.

```bash
ros2 run modeloop_toolbox
```
