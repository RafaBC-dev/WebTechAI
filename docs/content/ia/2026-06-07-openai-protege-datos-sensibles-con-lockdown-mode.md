# OpenAI protege datos sensibles con Lockdown Mode

**Fecha:** 2026-06-07
**Categoría:** ia
**Tags:** ia-local, linux, codigo
**Título original:** OpenAI unveils Lockdown Mode to protect sensitive data from prompt injection attacks

---

## Introducción

OpenAI ha anunciado Lockdown Mode, una característica que busca proteger datos sensibles de ataques de inyección de instrucciones en chatbots. Este nuevo modo reduce el riesgo de compartir información confidencial.

## ¿Qué es?

Lockdown Mode es una función de seguridad que deshabilita la navegación en vivo en Internet, la recuperación y visualización de imágenes desde la web, la investigación profunda y el modo de agente en ChatGPT. Su objetivo es reducir la probabilidad de compartir datos sensibles.

## ¿Cómo funciona?

Lockdown Mode funciona desactivando ciertas funciones de ChatGPT, como la navegación en vivo y la recuperación de imágenes, para minimizar el riesgo de inyección de instrucciones maliciosas. Aunque esto no elimina completamente el riesgo, reduce la probabilidad de compartir datos sensibles.

## ¿Por qué importa?

Lockdown Mode es importante para personas y organizaciones que manejan datos sensibles y buscan una mayor protección contra riesgos de extracción de datos relacionados con la inyección de instrucciones. Esto puede ser especialmente relevante en entornos de trabajo o en situaciones donde la privacidad es crítica.

## Consejo técnico

Si manejas datos sensibles y deseas habilitar Lockdown Mode en ChatGPT, asegúrate de revisar los requisitos de acceso y configuración de tu cuenta de ChatGPT Business o personal. Puedes verificar si estás elegible para habilitar esta característica en la configuración de tu cuenta.
