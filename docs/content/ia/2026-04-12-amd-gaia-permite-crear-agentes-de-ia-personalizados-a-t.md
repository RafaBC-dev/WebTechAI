# AMD GAIA permite crear agentes de IA personalizados a través de chat

**Fecha:** 2026-04-12
**Categoría:** ia
**Tags:** ia-local, agentes, python
**Título original:** AMD's GAIA Now Allows Building Custom AI Agents Via Chat, Becomes "True Desktop App"

---

## Introducción

AMD ha lanzado una actualización de su proyecto GAIA, que ahora permite a los usuarios crear agentes de IA personalizados a través de un chat. Esto se considera un paso importante hacia la creación de aplicaciones de IA más accesibles y personalizadas.

## ¿Qué es?

GAIA es un proyecto de AMD que se centra en la creación de agentes de IA local, es decir, que se ejecutan en el propio dispositivo y no en la nube. Esto permite a los usuarios crear aplicaciones de IA que sean más seguras, privadas y eficientes en términos de recursos.

## ¿Cómo funciona?

La actualización de GAIA 0.17.2 permite a los usuarios crear agentes de IA personalizados a través de un chat con el Builder Agent. Este proceso se realiza de manera interactiva, donde el usuario puede proporcionar información y configurar las características del agente. Una vez creado, el agente se puede ejecutar en diferentes plataformas, incluyendo Windows, Linux y macOS.

## ¿Por qué importa?

La capacidad de crear agentes de IA personalizados a través de un chat es importante porque permite a los usuarios crear aplicaciones de IA que se adapten a sus necesidades específicas. Esto puede ser especialmente útil en campos como la robótica, la automatización y la inteligencia artificial en general.

## Consejo técnico

Si deseas crear un agente de IA personalizado con GAIA, asegúrate de descargar la versión 0.17.2 desde GitHub y seguir las instrucciones del Builder Agent para crear y configurar tu agente.

```bash
Para crear un agente de IA con GAIA, puedes utilizar el comando `gaia-builder` en la terminal. Esto te permitirá interactuar con el Builder Agent y crear tu agente de manera personalizada.
```
