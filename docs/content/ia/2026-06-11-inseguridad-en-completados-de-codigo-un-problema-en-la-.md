# Inseguridad en completados de código: un problema en la IA

## Introducción

El desarrollador de seguridad de la Fundación de Python, Seth Larson, ha descubierto que el plugin de completado de código en PyCharm sugiere código que podría llevar a vulnerabilidades graves. ¿Es esto un problema de seguridad o no?

## ¿Qué es?

El plugin de completado de código en PyCharm utiliza un módulo de aprendizaje profundo local para ofrecer sugerencias de código. Sin embargo, este plugin sugiere código que podría llevar a vulnerabilidades graves, lo que plantea dudas sobre si esto es un problema de seguridad

## ¿Cómo funciona?

El plugin de completado de código en PyCharm utiliza un módulo de aprendizaje profundo local para analizar el código y ofrecer sugerencias. Sin embargo, este módulo no siempre es capaz de detectar vulnerabilidades y puede sugerir código que podría llevar a problemas de seguridad

## ¿Por qué importa?

Este problema es importante porque puede afectar a los desarrolladores que utilizan el plugin de completado de código en PyCharm. Si el plugin sugiere código que podría llevar a vulnerabilidades graves, esto puede comprometer la seguridad de los proyectos de código abierto y cerrado

## Consejo técnico

Si utilizas el plugin de completado de código en PyCharm, es importante revisar cuidadosamente las sugerencias de código y asegurarte de que no sugiere código que podría llevar a vulnerabilidades graves. Puedes hacer esto revisando la documentación del plugin y asegurándote de que estás utilizando la versión más reciente
