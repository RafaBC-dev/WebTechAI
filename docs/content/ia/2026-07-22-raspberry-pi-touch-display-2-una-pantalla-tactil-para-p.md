## Introducción

Raspberry Pi ha lanzado una nueva pantalla táctil de 10,1 pulgadas para proyectos interactivos, como tablets, sistemas de entretenimiento y tableros de información. Esta pantalla es compatible con Raspberry Pi 5 y Compute Modules conectados a una placa de carrier con un conector MIPI.

## ¿Qué es?

El Raspberry Pi Touch Display 2 es una pantalla táctil de 10,1 pulgadas con una resolución de 1200 x 1920 píxeles y un ángulo de visión de 85 grados. Es una pantalla multi-táctil con soporte para hasta 10 dedos y tiene una respuesta de hasta 35 ms. Está diseñada para proyectos interactivos y es compatible con Raspberry Pi 5 y Compute Modules.

## ¿Cómo funciona?

La pantalla se conecta a Raspberry Pi 5 a través de un cable de alimentación para el conector GPIO y un cable de datos para el conector DSI. La instalación es sencilla y solo requiere dos conexiones. La pantalla también viene con un cable de alimentación, un cable de datos FPC y ocho tornillos M2.5.

## ¿Por qué importa?

Esta pantalla es importante porque permite a los desarrolladores crear proyectos interactivos con Raspberry Pi, como tablets, sistemas de entretenimiento y tableros de información. También es compatible con Compute Modules, lo que la hace más versátil.

## Consejo técnico

Si deseas utilizar esta pantalla, asegúrate de conectar los cables en el orden correcto para evitar dañar la pantalla. También es importante utilizar un cable de alimentación adecuado para evitar sobrecargar la pantalla.

```bash
sudo apt-get install raspbian
```
