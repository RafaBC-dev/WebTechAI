## Introducción

La empresa X Square Robot busca crear robots inteligentes capaces de aprender y adaptarse a diferentes situaciones. Para lograr esto, han desarrollado un stack de inteligencia artificial que integra percepción, planificación y control.

## ¿Qué es?

El stack de inteligencia artificial de X Square Robot es un conjunto de componentes que trabajan juntos para permitir a los robots aprender y adaptarse a diferentes situaciones. Incluye un modelo del mundo físico, un modelo de acción y un sistema de aprendizaje basado en interacciones.

## ¿Cómo funciona?

El stack se basa en tres principios clave: la unidad básica de datos es una interacción, no un trayectoria; el preentrenamiento debe producir capacidad usable, no solo una inicialización para la fine-tuning; y el comportamiento debe modelarse en torno a eventos físicos, no a slices de tiempo fijos. El sistema utiliza un modelo del mundo físico y un modelo de acción que se complementan y son independientes, pero comparten un código base.

## ¿Por qué importa?

Este stack de inteligencia artificial es importante porque podría permitir a los robots aprender y adaptarse a diferentes situaciones de manera más efectiva. Esto podría tener aplicaciones en la automatización, la robótica y la inteligencia artificial en general.

## Consejo técnico

python -m x_square_robot.umi --help
