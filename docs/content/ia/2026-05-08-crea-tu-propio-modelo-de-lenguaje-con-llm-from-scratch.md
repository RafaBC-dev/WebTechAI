## Introducción

¿Quieres aprender sobre modelos de lenguaje de manera práctica? El proyecto LLM From Scratch te permite crear tu propio modelo de lenguaje desde cero, utilizando Python y bibliotecas comunes como numpy y torch.

## ¿Qué es?

LLM From Scratch es un taller práctico que te guía a través de la creación de un modelo de lenguaje básico. Comienza con la tokenización, pasa a la transformación, el bucle de entrenamiento, la generación de texto y finalmente, integra el modelo con un poeta AI. Inspirado en nanoGPT, este proyecto se enfoca en entrenar el modelo en solo una hora en una computadora común.

## ¿Cómo funciona?

El taller se divide en seis partes: la tokenización, la transformación, el bucle de entrenamiento, la generación de texto, la integración del modelo con un poeta AI y una sección de referencias con artículos interesantes. Puedes entrenar el modelo en una computadora con Mac, Linux o Windows, utilizando Python y bibliotecas como numpy y torch, y aprovechar el acelerador de GPU de Apple o NVIDIA si lo tienes.

## ¿Por qué importa?

Crear un modelo de lenguaje desde cero te permite entender mejor cómo funcionan los modelos de lenguaje y cómo pueden ser entrenados para realizar tareas específicas. Esto puede ser útil para desarrolladores de IA y científicos de datos que buscan mejorar sus habilidades en este campo.

## Consejo técnico

Si deseas profundizar en la creación de modelos de lenguaje, prueba a entrenar tu propio modelo utilizando el proyecto LLM From Scratch y luego intégralo con un poeta AI para crear contenido interesante.
