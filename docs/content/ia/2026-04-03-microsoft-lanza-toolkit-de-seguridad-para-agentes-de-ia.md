# Microsoft lanza toolkit de seguridad para agentes de IA

**Fecha:** 2026-04-03
**Categoría:** ia
**Tags:** ia-local, agentes, linux
**Título original:** Microsoft's Newest Open-Source Project: Runtime Security For AI Agents

---

## Introducción

Microsoft ha anunciado un nuevo proyecto de código abierto llamado Agent Governance Toolkit, diseñado para proporcionar seguridad y gobernanza para agentes de IA autónomos. Este toolkit se enfoca en abordar los riesgos de seguridad identificados por OWASP en 2025. La seguridad de los agentes de IA es un tema cada vez más importante en la industria tecnológica.

## ¿Qué es?

El Agent Governance Toolkit es un conjunto de herramientas de código abierto que se enfoca en proporcionar seguridad y gobernanza para agentes de IA autónomos. Está disponible para Python, Rust, TypeScript, Go y .NET. El toolkit incluye varios componentes, como Agent OS, Agent Mesh, Agent Runtime, Agent SRE, Agent Compliance, Agent Marketplace y Agent Lightning.

## ¿Cómo funciona?

El Agent Governance Toolkit funciona interceptando las acciones de los agentes de IA antes de su ejecución, utilizando una política de seguridad. El toolkit también proporciona una red de seguridad para la comunicación entre agentes, así como mecanismos para la ejecución dinámica y la verificación de cumplimiento. Además, incluye una tienda de plugins para la gestión de la vida útil de los plugins y un mecanismo de aprendizaje por refuerzo para la gobernanza de la formación de los agentes.

## ¿Por qué importa?

Este toolkit es importante porque aborda los riesgos de seguridad identificados por OWASP en 2025, como el secuestro de objetivos, el abuso de identidad, la explotación de la confianza entre humanos y agentes, y la creación de agentes maliciosos. La seguridad de los agentes de IA es crucial para evitar daños a las personas y a la sociedad en general.

## Consejo técnico

Para implementar el Agent Governance Toolkit, puedes comenzar por instalar el código en GitHub y configurar la política de seguridad según tus necesidades. Luego, puedes agregar plugins para mejorar la seguridad y la gobernanza de tus agentes de IA.

```bash
git clone https://github.com/microsoft/AgentGovernanceToolkit.git
```
