# AMD presenta GAIA Agent UI para aplicaciones de IA local privadas

**Fecha:** 2026-03-29
**Categoría:** ia
**Tags:** llms, ia-local, linux
**Título original:** AMD Introduces GAIA Agent UI For Privacy-First Web App For Local AI Agents

---

## Introducción

AMD ha lanzado una nueva versión de su framework de agentes de IA local, GAIA, con una interfaz de usuario privada y segura para ejecutar tareas de IA en hardware Ryzen AI sin necesidad de conexión a la nube.

## ¿Qué es?

GAIA es un framework de agentes de IA local que permite a los usuarios ejecutar tareas de IA en su hardware Ryzen AI sin necesidad de conexión a la nube. La versión 0.17 de GAIA introduce una interfaz de usuario privada y segura llamada Agent UI, que permite a los usuarios interactuar con los agentes de IA de manera segura y privada.

## ¿Cómo funciona?

La Agent UI de GAIA se ejecuta en el cliente y utiliza una arquitectura de capas para separar la interfaz de usuario de la lógica de negocio. La interfaz de usuario está escrita en React/TypeScript y utiliza un shell de Electron para proporcionar una experiencia de usuario nativa. Los agentes de IA se ejecutan en el servidor y se comunican con la interfaz de usuario a través de una API REST.

## ¿Por qué importa?

GAIA Agent UI es importante porque permite a los usuarios ejecutar tareas de IA de manera segura y privada en su hardware Ryzen AI. Esto es especialmente útil para profesionales de la tecnología que necesitan ejecutar tareas de IA en entornos de trabajo seguros y privados.

## Consejo técnico

Si deseas probar GAIA Agent UI, puedes descargar la versión 0.17 de GAIA desde el repositorio de GitHub de AMD y seguir las instrucciones de instalación para configurar tu entorno de desarrollo.

```bash
npm install @gaia/agent-ui
```
