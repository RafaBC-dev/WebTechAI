# Banana Pi BPI-OM7: una cámara 3D AI con procesamiento de imágenes en tiempo real

**Fecha:** 2026-05-04
**Categoría:** ia
**Tags:** robotica, ia-local, linux
**Título original:** Banana Pi BPI-OM7 AI 3D camera pairs BPI-M7 RK3588 SBC with ORBBEC Gemini 2 depth camera

---

## Introducción

La empresa Banana Pi ha lanzado una cámara 3D AI llamada BPI-OM7, que combina una placa de desarrollo SBC con una cámara de profundidad ORBBEC Gemini 2. Esta cámara es ideal para aplicaciones de visión 3D, robótica y percepción espacial.

## ¿Qué es?

La BPI-OM7 es una cámara 3D AI que integra una placa de desarrollo SBC con una cámara de profundidad ORBBEC Gemini 2. Esta cámara utiliza un procesador Rockchip RK3588 octa-core y un acelerador de inteligencia artificial NPU de 6 TOPS para procesar imágenes en tiempo real.

## ¿Cómo funciona?

La BPI-OM7 funciona conectando la placa de desarrollo SBC con la cámara de profundidad ORBBEC Gemini 2. El procesador Rockchip RK3588 procesa las imágenes capturadas por la cámara y el acelerador de inteligencia artificial NPU realiza el procesamiento de visión 3D en tiempo real.

## ¿Por qué importa?

La BPI-OM7 es importante porque ofrece una solución de visión 3D AI en tiempo real para aplicaciones como robótica, automatización y percepción espacial. También es compatible con Ubuntu 24.04 y ofrece una API para desarrolladores.

## Consejo técnico

Si deseas aprovechar al máximo la BPI-OM7, instala el SDK de Orbbec y utiliza la herramienta RKNPU/RKNN para desarrollar aplicaciones de visión 3D AI en tiempo real.

```bash
sudo apt-get install orbbec-sdk rknpu/rknn
```
