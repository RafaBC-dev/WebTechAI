# DeepInfra se une a Hugging Face como proveedor de inferencia

## Introducción

DeepInfra, una plataforma de inferencia de inteligencia artificial serverless, se ha unido a Hugging Face como proveedor de inferencia. Esto permite a los desarrolladores integrar una variedad de modelos de IA en sus aplicaciones con facilidad.

## ¿Qué es?

DeepInfra es una plataforma de inferencia de IA que ofrece un precio competitivo por token, con un catálogo de más de 100 modelos. Soporta una amplia gama de tipos de modelos, desde modelos de lenguaje a modelos de imagen y video.

## ¿Cómo funciona?

DeepInfra se integra en la plataforma de Hugging Face, permitiendo a los desarrolladores acceder a modelos de IA populares como DeepSeek V4 y Kimi-K2.6. Los desarrolladores pueden configurar sus propias claves de API y ordenar los proveedores de inferencia según sus preferencias.

## ¿Por qué importa?

La unión de DeepInfra con Hugging Face permite a los desarrolladores integrar modelos de IA de manera más fácil y eficiente en sus aplicaciones. Esto puede ser especialmente útil para proyectos que requieren una gran cantidad de procesamiento de lenguaje natural.

## Consejo técnico

Si estás utilizando Hugging Face, considera configurar tus propias claves de API en DeepInfra para aprovechar al máximo sus características de inferencia serverless.

```bash
Para configurar tus claves de API en DeepInfra, dirígete a la página de configuración de tu cuenta en Hugging Face y sigue las instrucciones para establecer tus claves de API en DeepInfra.
```
