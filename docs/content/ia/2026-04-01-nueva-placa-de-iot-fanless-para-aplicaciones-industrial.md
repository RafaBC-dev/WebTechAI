# Nueva placa de IoT fanless para aplicaciones industriales

## Introducción

Forlinx ha lanzado la FCU1501, una placa de IoT fanless diseñada para aplicaciones industriales que requieren alta confiabilidad y resistencia a temperaturas extremas.

## ¿Qué es?

La FCU1501 es una placa de IoT fanless basada en el procesador Rockchip RK3506J, con un tri-core Cortex-A7 y un solo core Cortex-M0. Ofrece conectividad dual Ethernet, Wi-Fi y Bluetooth 5.0, así como interfaces RS485, RS232 y CAN FD.

## ¿Cómo funciona?

La placa utiliza el procesador Rockchip RK3506J para proporcionar una alta velocidad de procesamiento y una baja consumo de energía. La conectividad dual Ethernet permite la comunicación con dispositivos en red, mientras que la Wi-Fi y el Bluetooth 5.0 permiten la conexión a redes inalámbricas. Las interfaces RS485, RS232 y CAN FD permiten la comunicación con dispositivos periféricos y sensores.

## ¿Por qué importa?

La FCU1501 es importante para aplicaciones industriales que requieren alta confiabilidad y resistencia a temperaturas extremas. Puede ser utilizada en aplicaciones como automatización, transporte ferroviario y fabricación inteligente.

## Consejo técnico

Si estás desarrollando una aplicación industrial que requiere alta confiabilidad y resistencia a temperaturas extremas, considera utilizar la FCU1501 como plataforma de base. Puedes conectar dispositivos periféricos y sensores a través de las interfaces RS485, RS232 y CAN FD.
