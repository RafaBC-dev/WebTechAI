## Introducción

La empresa X Square Robot busca crear robots capaces de realizar tareas generales, pero para lograrlo, necesita desarrollar una receta equivalente a la que han dado los modelos de lenguaje grandes. Esto implica construir un stack de base que incluya un modelo del mundo, un modelo de acción y un modelo de interacción.

## ¿Qué es?

X Square Robot es una empresa china de inteligencia artificial incorporada que busca crear robots capaces de realizar tareas generales. Para lograrlo, están desarrollando un stack de base que incluya un modelo del mundo, un modelo de acción y un modelo de interacción. El stack se basa en tres principios clave: que la unidad básica de datos del robot sea una interacción, no una trayectoria; que el entrenamiento pre sea capaz de producir una capacidad útil, no solo una inicialización para ajustar; y que el comportamiento se modele en torno a eventos físicos, no a slices fijos de tiempo.

## ¿Cómo funciona?

El stack de base de X Square Robot se compone de tres componentes principales: el modelo del mundo, el modelo de acción y el modelo de interacción. El modelo del mundo se encarga de predecir cambios en el mundo físico, mientras que el modelo de acción se encarga de generar comportamientos ejecutables del robot. El modelo de interacción se encarga de capturar las interacciones entre el robot y su entorno. Estos componentes se conectan a través de un pequeño conjunto de principios que garantizan que el stack sea coherente y funcione de manera efectiva.

## ¿Por qué importa?

El stack de base de X Square Robot tiene importantes implicaciones para la robótica general. Si se logra desarrollar un stack que pueda ser utilizado en robots de diferentes tamaños y tipos, se podría crear una plataforma común para la creación de robots capaces de realizar tareas generales. Esto podría revolucionar la forma en que se diseñan y se utilizan los robots en la vida diaria.
