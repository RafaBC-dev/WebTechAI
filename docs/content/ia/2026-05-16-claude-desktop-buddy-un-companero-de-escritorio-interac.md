# Claude Desktop Buddy: un compañero de escritorio interactivo para dispositivos ESP32-S3

**Fecha:** 2026-05-16
**Categoría:** ia
**Tags:** ia-local, esp32, codigo, herramientas
**Título original:** Anthropic’s open-source Claude Desktop Buddy turns ESP32-S3 devices into interactive AI desk companions

---

## Introducción

Anthropic ha lanzado una interfaz de hardware abierta para dispositivos ESP32-S3, permitiendo crear compañeros de escritorio interactivos que conecten directamente con la aplicación Claude sobre Bluetooth Low Energy (BLE).

## ¿Qué es?

Claude Desktop Buddy es un proyecto de código abierto que convierte dispositivos ESP32-S3 en compañeros de escritorio interactivos que se conectan directamente con la aplicación Claude sobre Bluetooth Low Energy (BLE). Esto permite a los usuarios recibir promociones y aprobar o denegar acciones directamente en el dispositivo físico, sin necesidad de volver a la pantalla de escritorio.

## ¿Cómo funciona?

El proyecto utiliza la interfaz de hardware abierta de Claude, que permite a los dispositivos ESP32-S3 conectarse a la aplicación Claude sobre BLE. El dispositivo muestra actualizaciones en tiempo real de la actividad del agente de inteligencia artificial y permite a los usuarios responder a solicitudes de permiso directamente utilizando botones físicos.

## ¿Por qué importa?

Claude Desktop Buddy resuelve un problema clave de aprobaciones de usuario frecuentes que otros sistemas de inteligencia artificial similares no abordan. Al utilizar un dispositivo físico, los usuarios pueden recibir promociones y aprobar o denegar acciones de manera rápida y privada, sin necesidad de volver a la pantalla de escritorio.

## Consejo técnico

Si deseas crear tu propio compañero de escritorio interactivo, puedes utilizar el código abierto de Claude Desktop Buddy y la interfaz de hardware abierta de Claude para conectar tu dispositivo ESP32-S3 a la aplicación Claude sobre BLE.

```bash
npm install claudedesktopbuddy
```
