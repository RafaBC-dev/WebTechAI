## Introducción

El CEO de Y Combinator, Garry Tan, ha compartido su configuración de Claude Code, un conjunto de habilidades para agentes de IA que ha generado gran emoción y debate en la comunidad de tecnología. Tan ha admitido que está experimentando 'psicosis cibernética' debido a su entusiasmo por la tecnología.

## ¿Qué es?

Claude Code es un conjunto de habilidades para agentes de IA que permiten a los desarrolladores crear aplicaciones personalizadas y automatizar tareas. Se basa en la plataforma de IA Claude, que ofrece una API para interactuar con agentes de IA. Las habilidades de Claude Code se almacenan en archivos 'skill.md' que contienen instrucciones para el agente de IA.

## ¿Cómo funciona?

Para crear una aplicación con Claude Code, los desarrolladores deben crear habilidades personalizadas que se almacenan en archivos 'skill.md'. Estos archivos contienen instrucciones para el agente de IA sobre cómo interactuar con el usuario y realizar tareas. Los desarrolladores pueden combinar estas habilidades para crear aplicaciones complejas y automatizar tareas.

## ¿Por qué importa?

La configuración de Claude Code de Garry Tan es importante porque muestra cómo la tecnología de IA puede ser utilizada para crear aplicaciones personalizadas y automatizar tareas. Esto puede tener un impacto significativo en la productividad y la eficiencia en diversas industrias y sectores.

## Consejo técnico

Si deseas crear tus propias habilidades para Claude Code, puedes empezar por investigar la documentación oficial de la plataforma y crear un archivo 'skill.md' que contenga instrucciones para el agente de IA.
