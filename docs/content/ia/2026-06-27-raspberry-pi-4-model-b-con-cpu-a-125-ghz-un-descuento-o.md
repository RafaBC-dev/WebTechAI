## Introducción

Cytron ha lanzado una versión especial del Raspberry Pi 4 Model B con una velocidad de CPU reducida a 1.25 GHz, pero ¿qué significa esto para los usuarios?

## ¿Qué es?

El Raspberry Pi 4 Model B es una placa de desarrollo de código abierto que se utiliza para proyectos de robótica, IA y aprendizaje automático. Esta versión especial tiene 4GB o 8GB de RAM y una velocidad de CPU reducida a 1.25 GHz.

## ¿Cómo funciona?

La placa utiliza el procesador Broadcom BCM2711C0 con un núcleo Cortex-A72 a 1.25 GHz y un GPU VideoCore VI. También cuenta con 2x micro HDMI, 2x USB 3.0, 2x USB 2.0, Ethernet y WiFi 5.

## ¿Por qué importa?

Esta versión especial del Raspberry Pi 4 Model B puede ser interesante para aquellos que buscan una placa de desarrollo asequible con características similares a la versión estándar, pero con una velocidad de CPU reducida.

## Consejo técnico

Si estás considerando comprar esta versión especial, asegúrate de verificar la compatibilidad con tus proyectos actuales y de investigar cómo afecta la velocidad de CPU reducida a tus necesidades.
