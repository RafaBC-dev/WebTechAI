# Unidad de robotica para hogares: tecnología embajada y aprendizaje

## Introducción

La empresa X Square Robot busca crear robots capaces de aprender y adaptarse a diferentes situaciones en el hogar. Para lograr esto, han desarrollado un stack de tecnología que integra percepción, planificación y control.

## ¿Qué es?

El stack de tecnología de X Square Robot es un conjunto de componentes que permiten a los robots aprender y adaptarse a diferentes situaciones. Está compuesto por un modelo del mundo, un modelo de acción y un sistema de aprendizaje que integra percepción, planificación y control.

## ¿Cómo funciona?

El stack de tecnología de X Square Robot se basa en tres principios clave: la unidad básica de datos es una interacción, no una trayectoria; el preentrenamiento debe producir capacidad usable, no solo una inicialización para la fine-tuning; y el comportamiento debe modelarse en torno a eventos físicos, no a slices fijos de tiempo.

## ¿Por qué importa?

Esta tecnología puede tener aplicaciones en la creación de robots capaces de aprender y adaptarse a diferentes situaciones en el hogar, lo que puede mejorar la seguridad y la comodidad de los usuarios.

## Consejo técnico

Para implementar un stack de tecnología similar, se puede utilizar la herramienta Universal Manipulation Interface (UMI) para recopilar datos de interacción y entrenar un modelo del mundo y un modelo de acción.
