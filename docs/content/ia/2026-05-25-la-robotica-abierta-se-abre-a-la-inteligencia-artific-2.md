# La robótica abierta se abre a la inteligencia artificial

## Introducción

La comunidad de robótica ha estado trabajando en software abiertos desde la década de 1990, pero ahora se está aplicando la inteligencia artificial para hacer que los robots piensen y tomen decisiones. Esto podría cambiar la forma en que se construyen los robots y reducir el tiempo de desarrollo.

## ¿Qué es?

El software abiertos para robótica se refiere a la creación de herramientas y frameworks que pueden ser utilizados y modificados por cualquier persona. El Robot Operating System (ROS) es un ejemplo de esto, que se lanzó en 2007 y se ha convertido en el estándar de facto para la robótica. ROS es un framework que se ejecuta sobre Linux y proporciona herramientas para controlar y programar robots.

## ¿Cómo funciona?

ROS se compone de una serie de herramientas y frameworks que trabajan juntos para permitir que los robots interactúen con su entorno. Incluye herramientas para controlar el movimiento del robot, generar mapas y planificar rutas, entre otras. Los desarrolladores pueden utilizar ROS para crear aplicaciones de robótica sin tener que crear la infraestructura básica desde cero.

## ¿Por qué importa?

La aplicación de la inteligencia artificial a la robótica abierta podría reducir el tiempo de desarrollo de los robots y hacer que sean más capaces de realizar tareas complejas. Esto podría tener un impacto significativo en industrias como la manufactura, la logística y la atención médica, donde los robots pueden ser utilizados para realizar tareas repetitivas y peligrosas.

## Consejo técnico

Si deseas empezar a trabajar con ROS, te recomendamos que comiences por instalar el entorno de desarrollo de ROS y familiarizarte con las herramientas y frameworks que ofrece. Puedes empezar por el tutorial oficial de ROS y luego explorar las diferentes herramientas y aplicaciones que están disponibles.

```bash
sudo apt-get install ros-kinetic-desktop-full
```
