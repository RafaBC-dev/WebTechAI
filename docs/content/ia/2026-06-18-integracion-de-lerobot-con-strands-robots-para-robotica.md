# Integración de LeRobot con Strands Robots para robótica de código abierto

## Introducción

La empresa Amazon ha lanzado una integración innovadora entre LeRobot y Strands Robots, permitiendo a los desarrolladores crear robots autónomos de código abierto con una sola herramienta. Esta integración revoluciona la forma en que se crean y se despliegan robots, facilitando la creación de agentes que pueden aprender y adaptarse a nuevas tareas.

## ¿Qué es?

Strands Robots es un SDK de código abierto de Amazon que expone las abstracciones de robot, simulación y el stack de LeRobot como herramientas de agente que se pueden componer en un solo agente. La integración con LeRobot permite a los desarrolladores crear robots autónomos que pueden aprender y adaptarse a nuevas tareas.

## ¿Cómo funciona?

La integración funciona mediante la creación de un agente que se compone de herramientas de LeRobot, como la grabación de nuevas demostraciones, la simulación y la implementación en hardware. El agente puede aprender y adaptarse a nuevas tareas mediante la utilización de datasets y políticas que se pueden intercambiar de manera sencilla.

## ¿Por qué importa?

Esta integración es importante porque permite a los desarrolladores crear robots autónomos de código abierto que pueden aprender y adaptarse a nuevas tareas. Esto abre nuevas posibilidades para la automatización y la robótica en diversas industrias.

## Consejo técnico

Si deseas aprovechar esta integración, puedes comenzar creando un agente con las herramientas de LeRobot y Strands Robots. Puedes utilizar el ejemplo de código proporcionado en GitHub para empezar a desarrollar tus propios robots autónomos.

```bash
git clone https://github.com/StrandsRobots/examples/lerobot/hub_to_hardware.py
```
