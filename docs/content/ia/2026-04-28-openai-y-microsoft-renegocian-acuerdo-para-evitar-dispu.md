# OpenAI y Microsoft renegocian acuerdo para evitar disputas legales

**Fecha:** 2026-04-28

## Introducción

OpenAI y Microsoft han anunciado un nuevo acuerdo que resuelve disputas legales sobre el uso de tecnología de inteligencia artificial. Este acuerdo beneficia a ambas partes y permite a OpenAI expandir su presencia en diferentes proveedores de nube.

## ¿Qué es?

OpenAI es una empresa que desarrolla tecnologías de inteligencia artificial, incluyendo modelos de lenguaje y agentes de inteligencia artificial. El acuerdo con Microsoft permite a OpenAI utilizar la plataforma de nube de Microsoft para desarrollar y desplegar sus productos.

## ¿Cómo funciona?

El nuevo acuerdo establece que Microsoft tendrá un acceso no exclusivo a la tecnología de OpenAI hasta 2032. Esto significa que OpenAI podrá utilizar diferentes proveedores de nube para desplegar sus productos, incluyendo Amazon. El acuerdo también establece que OpenAI seguirá siendo un cliente importante de la plataforma de nube de Microsoft.

## ¿Por qué importa?

Este acuerdo es importante porque resuelve disputas legales entre OpenAI y Microsoft y permite a OpenAI expandir su presencia en diferentes proveedores de nube. Esto significa que los usuarios de OpenAI podrán acceder a sus productos en diferentes plataformas de nube, lo que aumenta la flexibilidad y la escalabilidad.

## Consejo técnico

Si estás utilizando tecnología de inteligencia artificial, considera utilizar la plataforma de nube de Microsoft para desarrollar y desplegar tus productos. Esto te permitirá aprovechar la experiencia y la infraestructura de Microsoft para desarrollar soluciones escalables y confiables.
