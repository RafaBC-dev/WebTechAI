## Introducción

La Raspberry Pi 5 de 1GB ofrece una opción más asequible para usuarios que necesitan procesamiento adicional y/o características como un conector PCIe. En este artículo, exploramos proyectos que aprovechan al máximo el rendimiento de la Raspberry Pi 5 de 1GB sin requerir una gran cantidad de RAM.

## ¿Qué es?

La Raspberry Pi 5 de 1GB es una variante de la Raspberry Pi 5 que utiliza el mismo procesador BCM2712 que otras versiones, pero con solo 1GB de RAM. A pesar de esto, ofrece una opción más asequible para usuarios que necesitan procesamiento adicional y/o características como un conector PCIe.

## ¿Cómo funciona?

La Raspberry Pi 5 de 1GB puede conectarse a un SSD NVMe mediante un conector M.2, lo que proporciona almacenamiento adicional y permite arrancar el sistema desde el SSD en lugar de la tarjeta microSD estándar. Esto acelera el rendimiento general con velocidades de lectura y escritura rápidas.

## ¿Por qué importa?

La Raspberry Pi 5 de 1GB es ideal para crear un centro de medios (para streamear películas, series y música) y/o un NAS (almacenamiento conectado a la red) con velocidades de lectura y escritura rápidas. También puede emular muchos ordenadores clásicos y consolas.

## Consejo técnico

Para crear un centro de medios, utiliza una distribución de Kodi como LibreELEC o OSMC, y para configurar un NAS, sigue el tutorial de configuración de Samba en la documentación oficial.

```bash
sudo apt-get install libreelec
```
