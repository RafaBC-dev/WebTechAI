## Introducción

¿Te has encontrado con la frustración de que el límite de tokens se agote en modelos como Claude o ChatGPT? Ahora puedes evitar la espera instalando un modelo LLM en tu propia computadora.

## ¿Qué es?

Un modelo LLM (Large Language Model) es un modelo de inteligencia artificial que puede procesar y generar texto de manera autónoma. Se utiliza en aplicaciones como asistentes virtuales y chatbots.

## ¿Cómo funciona?

Para instalar un modelo LLM en tu computadora, debes evaluar tu hardware para asegurarte de que tenga la capacidad necesaria. Luego, debes elegir un modelo compatible y descargar la interfaz de modelo LM Studio. Finalmente, debes configurar los ajustes de contexto, GPU y hilos de CPU para obtener el mejor rendimiento.

## ¿Por qué importa?

Instalar un modelo LLM en tu computadora te permite procesar y generar texto de manera autónoma, sin depender de servicios en la nube que pueden tener límites de tokens. Esto es especialmente útil para proyectos que requieren procesamiento de texto en masa.

## Consejo técnico

Utiliza la herramienta canirun.ai para evaluar tu hardware y elegir un modelo LLM compatible con tu computadora. También puedes agregar plugins de herramientas como Beledarian para permitir que el modelo interactúe con archivos locales y la web.

```bash
canirun.ai
```
