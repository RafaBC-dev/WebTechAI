## Introducción

NVIDIA ha lanzado Cosmos 3, un modelo de razonamiento físico abierto que combina generación de mundos, razonamiento físico y generación de acciones en un solo modelo. Esto revoluciona la forma en que se desarrollan sistemas de inteligencia artificial física.

## ¿Qué es?

Cosmos 3 es un modelo de razonamiento físico que utiliza una arquitectura de mezcla de transformadores (MoT) para combinar generación de mundos, razonamiento físico y generación de acciones en un solo modelo. Esto permite a los desarrolladores crear sistemas de inteligencia artificial física capaces de entender y interactuar con el mundo real.

## ¿Cómo funciona?

Cosmos 3 utiliza una arquitectura de MoT para combinar diferentes componentes, incluyendo generación de mundos, razonamiento físico y generación de acciones. Esto permite a los desarrolladores crear modelos que pueden razonar y generar diferentes modalidades en una sola pasada de entrada.

## ¿Por qué importa?

Cosmos 3 es importante porque permite a los desarrolladores crear sistemas de inteligencia artificial física capaces de entender y interactuar con el mundo real. Esto tiene aplicaciones en robótica, vehículos autónomos y espacios inteligentes, entre otros.

## Consejo técnico

Para aprovechar al máximo Cosmos 3, los desarrolladores pueden utilizar los scripts de entrenamiento post-entrenamiento disponibles en GitHub para entrenar el modelo en sus propios datos. Esto les permitirá crear modelos personalizados para sus necesidades específicas.

```bash
git clone https://github.com/NVIDIA/Cosmos-3.git
```
