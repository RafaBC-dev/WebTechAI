# Un modelo de lenguaje AI alimentado por un manubrio de mano

**Fecha:** 2026-06-11
**Categoría:** ia
**Tags:** llms, ia-local, python, linux
**Título original:** AI The Truly Environmentally Friendly Way

## Introducción

Un equipo de desarrolladores ha creado un modelo de lenguaje AI que se alimenta de energía humana, utilizando un manubrio de mano para cargar un Raspberry Pi 5. Este proyecto busca reducir el impacto ambiental de los modelos de lenguaje AI.

## ¿Qué es?

El proyecto, llamado Squeezlabs, utiliza un Raspberry Pi 5 como base para un modelo de lenguaje AI llamado llama.cpp. El modelo se alimenta de energía humana a través de un manubrio de mano que carga el dispositivo.

## ¿Cómo funciona?

El manubrio de mano carga un capacitor que ayuda a estabilizar la energía y prevenir brownouts. El modelo de lenguaje AI utiliza una gran cantidad de procesamiento, lo que requiere una cantidad significativa de energía. El equipo ha utilizado una tecnología de capacitor para ayudar a manejar la demanda de energía.

## ¿Por qué importa?

Este proyecto es importante porque busca reducir el impacto ambiental de los modelos de lenguaje AI. Los modelos de lenguaje AI requieren una gran cantidad de energía para funcionar, lo que puede tener un impacto negativo en el medio ambiente. El uso de un manubrio de mano como fuente de energía es una forma innovadora de reducir este impacto.

## Consejo técnico

Si deseas crear un modelo de lenguaje AI similar, considera utilizar un Raspberry Pi 5 y el modelo llama.cpp. También es importante investigar sobre la tecnología de capacitor y cómo puede ser utilizada para manejar la demanda de energía en proyectos de IA.

```bash
sudo apt-get install python3-pip && pip3 install llama-cpp
```
