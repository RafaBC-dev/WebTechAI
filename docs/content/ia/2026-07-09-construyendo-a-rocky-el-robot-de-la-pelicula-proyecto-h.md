# Construyendo a Rocky, el robot de la película 'Proyecto Hail Mary'

**Fecha:** 2026-07-09
**Categoría:** ia
**Tags:** robotica, ia-local, raspberry-pi
**Título original:** AI Rocky from ‘Project Hail Mary’

## Introducción

Un ingeniero ha creado un robot llamado Rocky, inspirado en la película 'Proyecto Hail Mary', utilizando un Raspberry Pi y tecnologías de IA. Este proyecto es un ejemplo de cómo la robótica y la IA pueden ser utilizadas para crear sistemas interactivos y útiles.

## ¿Qué es?

Rocky es un robot que utiliza un Raspberry Pi 5, un PCA9685 servo driver HAT, un micrófono y un altavoz para interactuar con los usuarios. El robot puede reconocer la voz y responder a las preguntas y órdenes, utilizando la tecnología de IA para aprender y mejorar su comportamiento.

## ¿Cómo funciona?

El robot Rocky utiliza la biblioteca Claude Code para reconocer la voz y generar respuestas en tiempo real. El sistema también utiliza servos para mover las piernas del robot y crear movimientos realistas. El ingeniero utilizó un software de diseño 3D para crear el cuerpo del robot y lo imprimió en 3D utilizando un material multicolor.

## ¿Por qué importa?

Este proyecto es importante porque demuestra la capacidad de la robótica y la IA para crear sistemas interactivos y útiles. El robot Rocky puede ser utilizado en aplicaciones como la asistencia a personas con discapacidad o la educación en robótica y programación.

## Consejo técnico

Si deseas crear un robot similar a Rocky, puedes utilizar la biblioteca Claude Code y un Raspberry Pi para reconocer la voz y generar respuestas en tiempo real. También puedes utilizar un software de diseño 3D para crear el cuerpo del robot y lo imprimir en 3D utilizando un material multicolor.

```bash
sudo apt-get install python3-pip && pip3 install clauded
```
