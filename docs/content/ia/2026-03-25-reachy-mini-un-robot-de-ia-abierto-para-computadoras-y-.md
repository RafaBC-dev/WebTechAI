# Reachy Mini: un robot de IA abierto para computadoras y Raspberry Pi

**Fecha:** 2026-03-25
**Categoría:** ia
**Tags:** robotica, ia-local, python
**Título original:** Hugging Face’s Reachy Mini is an open-source AI robot for your computer or Raspberry Pi CM4

## Introducción

Hugging Face ha lanzado Reachy Mini, un robot de IA abierto que se puede conectar a computadoras y Raspberry Pi. Con una cámara, micrófonos y un altavoz, puede interactuar con el mundo físico y ejecutar aplicaciones de IA.

## ¿Qué es?

Reachy Mini es un robot de IA abierto que se puede conectar a computadoras y Raspberry Pi. Tiene una cámara, cuatro micrófonos y un altavoz de 5W, y puede mover su cabeza y cuerpo gracias a nueve motores servo. Está diseñado para ejecutar aplicaciones de IA que interactúen con el mundo físico.

## ¿Cómo funciona?

Reachy Mini utiliza una arquitectura basada en Python, con soporte para JavaScript y aplicaciones web. Puede integrarse con modelos de lenguaje de máquina (LLMs) para crear aplicaciones y publicarlas en Hugging Face. El SDK de Reachy Mini se encuentra en GitHub y ofrece varias muestras de aplicaciones, como una aplicación de conversación y una aplicación de seguimiento de manos.

## ¿Por qué importa?

Reachy Mini es importante porque permite a los desarrolladores crear aplicaciones de IA que interactúen con el mundo físico de manera sencilla. Puede ser utilizado para crear aplicaciones de visión computacional, reconocimiento de voz y seguimiento de objetos.

## Consejo técnico

Si deseas empezar a trabajar con Reachy Mini, te recomendamos descargar el SDK de GitHub y explorar las muestras de aplicaciones disponibles. Puedes empezar con la aplicación de conversación para hablar con Reachy Mini de manera natural.
