# El software abierto ayuda a los robots a pensar y tomar decisiones

## Introducción

La comunidad de robótica ha estado trabajando en proyectos de código abierto desde la década de 1990, pero ahora se está aplicando a la inteligencia artificial. Esto podría acelerar el desarrollo de robots capaces de razonar y tomar decisiones.

## ¿Qué es?

El software abierto en robótica se refiere a la creación y compartición de código fuente para que los desarrolladores puedan construir y personalizar robots de manera flexible. El Robot Operating System (ROS) es un marco de software que sitúa en la cima de Linux y maneja fundamentos robóticos como la comunicación de datos, la interacción con hardware y la planificación de rutas.

## ¿Cómo funciona?

ROS se compone de herramientas y modelos que se pueden utilizar para desarrollar robots. Los desarrolladores pueden utilizar ROS para crear robots que puedan razonar, decidir y actuar de manera autónoma. El software abierto permite a los desarrolladores contribuir y mejorar el código, lo que acelera el desarrollo de robots más avanzados.

## ¿Por qué importa?

La aplicación de software abierto en robótica podría reducir el tiempo y el esfuerzo necesarios para desarrollar robots capaces de razonar y tomar decisiones. Esto podría tener un impacto significativo en la industria de la robótica y la inteligencia artificial.

## Consejo técnico

Si deseas empezar a trabajar con ROS, puedes descargar el software de código abierto y comenzar a explorar sus herramientas y modelos. Puedes empezar con proyectos simples y avanzar a proyectos más complejos a medida que te sientas más cómodo con el software.

```bash
sudo apt-get install ros-<distribución>-desktop
```
