# Google lanza una aplicación de dictado con IA que funciona offline

**Fecha:** 2026-04-07
**Categoría:** ia
**Tags:** ia-local, robotica, linux
**Título original:** Google quietly launched an AI dictation app that works offline

---

## Introducción

Google ha lanzado una aplicación de dictado llamada Google AI Edge Eloquent que utiliza modelos de reconocimiento de voz automatizado (ASR) para convertir la voz en texto. La aplicación es gratuita y puede funcionar sin conexión a Internet.

## ¿Qué es?

Google AI Edge Eloquent es una aplicación de dictado que utiliza modelos de ASR basados en Gemma para convertir la voz en texto. La aplicación puede funcionar offline y ofrece varias características, como la eliminación de palabras de relleno y la opción de transformar el texto en diferentes formatos.

## ¿Cómo funciona?

La aplicación utiliza modelos de ASR basados en Gemma para reconocer la voz y convertirla en texto. Los modelos se descargan en la aplicación y se utilizan para procesar la voz en tiempo real. La aplicación también ofrece la opción de utilizar procesamiento local o en la nube para limpiar el texto.

## ¿Por qué importa?

Esta aplicación es importante porque ofrece una forma de dictado que funciona offline y es más precisa que otras aplicaciones de dictado. La aplicación también ofrece varias características útiles, como la eliminación de palabras de relleno y la opción de transformar el texto en diferentes formatos.

## Consejo técnico

Si deseas probar la aplicación, asegúrate de descargar los modelos de ASR basados en Gemma y de configurar la aplicación para utilizar procesamiento local. Esto te permitirá utilizar la aplicación sin conexión a Internet y obtener resultados más precisos.

```bash
python -m pip install gemma
```
