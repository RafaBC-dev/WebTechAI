# M5Stack Core2 se convierte en un macro pad para Codex Micro con firmware abierto

**Fecha:** 2026-07-19
**Categoría:** ia
**Tags:** ia-local, microcontroladores, software-y-linux
**Título original:** M5Stack Core2 gets open-source firmware to reproduce OpenAI’s Codex Micro features

---

## Introducción

IAMLIUBO ha creado un firmware para el M5Stack Core2 que le permite a los desarrolladores reproducir las características del Codex Micro, un macro pad de $230 para la plataforma de agentes de código AI de OpenAI.

## ¿Qué es?

El Codex Micro es un macro pad con un encuadre de CNC, un encoder rotativo y un joystick, pero es caro y solo se envía a los EE. UU. IAMLIUBO ha creado un firmware para el M5Stack Core2 que reproduce las mismas funciones, utilizando la pantalla táctil de 2 pulgadas del dispositivo ESP32.

## ¿Cómo funciona?

El firmware presenta el Core2 como un dispositivo HID BLE compatible con la información de Codex Micro de Work Louder, y maneja el estado de la tarea, el estado de la batería, la reasignación de comandos y la integración de 'habla' a través de la aplicación de escritorio de ChatGPT.

## ¿Por qué importa?

Este proyecto es importante porque permite a los desarrolladores crear un macro pad similar al Codex Micro a un costo mucho más bajo, utilizando el M5Stack Core2 y el firmware abierto de IAMLIUBO.

## Consejo técnico

Si deseas crear un macro pad similar al Codex Micro, puedes utilizar el M5Stack Core2 y el firmware abierto de IAMLIUBO, y personalizarlo según tus necesidades utilizando la aplicación de escritorio de ChatGPT.
