# Cámara de seguridad con armas defensivas para proteger tu hogar

## Introducción

Una cámara de seguridad personalizada con armas defensivas y visión artificial para proteger tu hogar de intrusos. Un proyecto innovador que combina tecnología y seguridad.

## ¿Qué es?

La cámara de seguridad es un proyecto personalizado que utiliza una cámara web montada en un soporte de giro y panórmico, controlado por un Raspberry Pi con software personalizado. La cámara puede operar en modo autónomo o ser controlada manualmente desde una tableta Android.

## ¿Cómo funciona?

La cámara utiliza un sistema de visión artificial para detectar y seguir objetos, y puede activar una Nerf machine gun con láser y luces de spotlight para disuadir a los intrusos. El sistema también incluye un altavoz para comunicarse con el usuario y un sistema de voz para controlarlo.

## ¿Por qué importa?

Este proyecto importa porque ofrece una solución innovadora para la seguridad en el hogar, utilizando tecnología de visión artificial y armas defensivas para disuadir a los intrusos. También puede ser utilizado para proteger áreas específicas del hogar, como la cocina o el garaje.

## Consejo técnico

Si deseas crear un sistema de seguridad similar, considera utilizar el software de visión artificial OpenCV en tu proyecto y configura el sistema de voz para que pueda ser controlado con comandos de voz.

```bash
sudo apt-get install opencv-python
```
