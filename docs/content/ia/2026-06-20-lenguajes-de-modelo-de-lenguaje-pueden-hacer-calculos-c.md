# Lenguajes de modelo de lenguaje: ¿pueden hacer cálculos correctamente?

**Fecha:** 2026-06-20
**Categoría:** ia
**Tags:** llms, ia-local, herramientas
**Título original:** How LLMs can be Assisted to do Arithmetic Correctly

---

## Introducción

Los lenguajes de modelo de lenguaje (LLM) son capaces de realizar tareas complejas, pero ¿qué sucede cuando se les pide que hagan cálculos simples? Un proyecto reciente busca mejorar la capacidad de los LLM para realizar aritmética.

## ¿Qué es?

El proyecto Rune es un mecanismo consciente de compilación JIT para lenguajes de modelo de lenguaje que busca mejorar la capacidad de los LLM para realizar aritmética. Aunque es estadísticamente imposible que un LLM pueda realizar cálculos aritméticos de manera correcta, el proyecto Rune intenta interferir en el proceso de inferencia del modelo para mejorar los resultados.

## ¿Cómo funciona?

El proyecto Rune utiliza una técnica de compilación JIT para identificar los parámetros de una operación aritmética y luego interferir en el proceso de inferencia del modelo para introducir el resultado correcto. Esto se hace monitorizando el estado interno del modelo y ajustando los parámetros para mejorar los resultados.

## ¿Por qué importa?

Aunque el proyecto Rune no ha tenido éxito en mejorar significativamente la capacidad de los LLM para realizar aritmética, es importante porque muestra la complejidad de la tarea y la necesidad de mejorar la capacidad de los LLM para realizar tareas numéricas.

## Consejo técnico

Si estás trabajando con lenguajes de modelo de lenguaje y necesitas mejorar la capacidad de realizar aritmética, considera utilizar herramientas como Rune o investigar en técnicas de compilación JIT para mejorar los resultados.
