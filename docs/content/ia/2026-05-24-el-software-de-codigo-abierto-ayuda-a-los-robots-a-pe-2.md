## Introducción

La comunidad de robótica ha estado trabajando en proyectos de código abierto desde la década de 1990, pero ahora se está aplicando este enfoque a la inteligencia artificial. Esto podría acelerar el desarrollo de robots más inteligentes.

## ¿Qué es?

El software de código abierto para robótica se refiere a herramientas y frameworks que permiten a los desarrolladores crear robots más inteligentes y autónomos. Esto incluye frameworks como ROS (Robot Operating System) que proporciona una plataforma para desarrollar aplicaciones de robótica.

## ¿Cómo funciona?

ROS es un framework que se ejecuta sobre Linux y proporciona herramientas para gestionar la comunicación entre componentes, interactuar con hardware, crear mapas y planificar rutas. También ofrece herramientas de desarrollo como registro de datos y visualización.

## ¿Por qué importa?

La aplicación de la inteligencia artificial en robótica podría permitir a los robots tomar decisiones más complejas y actuar de manera más autónoma. Esto podría tener aplicaciones en áreas como la medicina, la logística y la seguridad.

## Consejo técnico

Si estás interesado en desarrollar aplicaciones de robótica con ROS, puedes empezar explorando las herramientas y frameworks disponibles en el sitio web de Open Robotics. Puedes descargar ROS y comenzar a desarrollar aplicaciones de robótica utilizando la documentación oficial.

```bash
sudo apt-get install ros-kinetic-desktop-full
```
