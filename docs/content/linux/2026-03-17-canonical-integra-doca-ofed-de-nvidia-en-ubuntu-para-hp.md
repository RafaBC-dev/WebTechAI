# Canonical integra DOCA-OFED de NVIDIA en Ubuntu para HPC y IA

**Fecha:** 2026-03-17
**Categoría:** linux
**Tags:** ia-local, hpc, linux
**Título original:** Canonical Plans To Integrate NVIDIA DOCA-OFED Into The Ubuntu Archive

---

## Introducción

Canonical, el proveedor de Ubuntu, ha anunciado que integrará el framework de software DOCA-OFED de NVIDIA en la distribución Linux para mejorar la experiencia de despliegue en entornos de alto rendimiento y IA.

## ¿Qué es?

DOCA-OFED es un framework de software que proporciona un alto rendimiento de red para aplicaciones de alto rendimiento y IA. Es una versión simplificada del framework de software DOCA de NVIDIA, que se enfoca en la gestión de redes y la comunicación entre nodos en entornos de alto rendimiento.

## ¿Cómo funciona?

DOCA-OFED se integra con el kernel de Ubuntu y proporciona una capa de abstracción para la gestión de redes y la comunicación entre nodos. Utiliza la tecnología RDMA (acceso directo a memoria remota) para permitir la transferencia de datos de manera eficiente y segura.

## ¿Por qué importa?

La integración de DOCA-OFED en Ubuntu permitirá a los usuarios de alto rendimiento y IA aprovechar al máximo la potencia de sus equipos y mejorar la eficiencia de sus aplicaciones. También reducirá la complejidad de la gestión de redes y la comunicación entre nodos.

## Consejo técnico

Si estás trabajando con Ubuntu y necesitas mejorar la experiencia de despliegue en entornos de alto rendimiento y IA, considera utilizar el framework de software DOCA-OFED. Puedes descargar la versión más reciente de DOCA-OFED desde el repositorio de Ubuntu y seguir las instrucciones de instalación para configurarlo en tu sistema.

```bash
sudo apt-get install doca-ofed
```
