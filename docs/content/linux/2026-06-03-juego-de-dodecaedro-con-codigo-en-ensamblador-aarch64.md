## Introducción

Un entusiasta de la robótica ha creado un juego de consola portátil que utiliza código en ensamblador AArch64 y se ejecuta en un Raspberry Pi 3 sin sistema operativo. El juego se llama GameDodecahedron y se juega en un dodecaedro de plástico con botones y una tarjeta de memoria SD.

## ¿Qué es?

El GameDodecahedron es un proyecto de robótica que consiste en un juego de consola portátil que se ejecuta en un Raspberry Pi 3 sin sistema operativo. El juego utiliza código en ensamblador AArch64 y se juega en un dodecaedro de plástico con botones y una tarjeta de memoria SD. El objetivo del proyecto es demostrar que es posible crear un juego de consola sin el uso de un sistema operativo.

## ¿Cómo funciona?

El juego se ejecuta en un Raspberry Pi 3 sin sistema operativo, utilizando código en ensamblador AArch64. El código se carga en una tarjeta de memoria SD que se conecta al dodecaedro de plástico, que contiene botones y una pantalla LCD. El juego utiliza una Schmitt trigger para debounce los botones y una GPIO para leer los pulsos de los botones.

## ¿Por qué importa?

Este proyecto es importante porque demuestra que es posible crear un juego de consola sin el uso de un sistema operativo. Esto puede ser útil para proyectos de robótica que requieren una respuesta rápida y eficiente. Además, el uso de código en ensamblador AArch64 permite una mayor eficiencia en el uso de recursos del procesador.

## Consejo técnico

Si deseas crear un juego de consola sin sistema operativo, puedes utilizar el código en ensamblador AArch64 del GameDodecahedron como referencia. Puedes descargar el código desde el repositorio de GitHub y adaptarlo a tu propio proyecto.

git clone https://github.com/Inkbox/GameDodecahedron.git
