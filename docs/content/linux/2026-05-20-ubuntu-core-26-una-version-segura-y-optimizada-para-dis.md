# Ubuntu Core 26: una versión segura y optimizada para dispositivos IoT y embebidos

**Fecha:** 2026-05-20
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Ubuntu Core 26 targets IoT devices and embedded systems, offers up to 15 years of security maintenance

---

## Introducción

Canonical ha lanzado Ubuntu Core 26, una versión de Ubuntu diseñada específicamente para dispositivos IoT y sistemas embebidos. Esta versión ofrece una seguridad extendida de hasta 15 años y una serie de características innovadoras que mejoran la experiencia de usuario y la seguridad en general.

## ¿Qué es?

Ubuntu Core 26 es una versión minimalista e inmutable del sistema operativo Ubuntu, diseñada para funcionar en dispositivos IoT y sistemas embebidos. Se basa en Ubuntu 26.04 LTS y ofrece una serie de características innovadoras como actualizaciones OTA optimizadas y parches de kernel en vivo.

## ¿Cómo funciona?

Ubuntu Core 26 utiliza un sistema de construcción llamado Chisel, que permite crear versiones precisas del sistema operativo con un tamaño reducido. Cada componente del sistema es un contenedorizado snap, similar a las versiones anteriores de Ubuntu Core. Esto permite una actualización más rápida y segura del sistema.

## ¿Por qué importa?

Ubuntu Core 26 es importante porque ofrece una seguridad extendida de hasta 15 años, lo que es crucial para dispositivos IoT y sistemas embebidos que pueden estar en funcionamiento durante décadas. Además, esta versión ofrece una serie de características innovadoras que mejoran la experiencia de usuario y la seguridad en general.

## Consejo técnico

Si estás trabajando con dispositivos IoT o sistemas embebidos, considera utilizar Ubuntu Core 26 para aprovechar sus características innovadoras y seguridad extendida. Puedes comenzar por crear un contenedorizado snap utilizando la herramienta Snapcraft.

```bash
snap create my-snap
```
