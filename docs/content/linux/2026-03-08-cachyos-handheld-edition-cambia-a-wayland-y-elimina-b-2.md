# CachyOS Handheld Edition cambia a Wayland y elimina Bcachefs

**Fecha:** 2026-03-08
**Categoría:** linux
**Tags:** Linux, Steam Deck, Wayland, CachyOS, Arch Linux
**Título original:** CachyOS Handheld Edition Switches To Wayland, CachyOS Installer Drops Bcachefs

---

## Introducción

La versión de marzo de 2026 de CachyOS Handheld Edition está disponible, con mejoras significativas en el instalador y la experiencia de usuario. Esta actualización es especialmente importante para propietarios de dispositivos de juego portátiles como la Steam Deck y Legion Go. La versión de marzo de 2026 de CachyOS trae consigo cambios importantes en la experiencia de usuario y la funcionalidad del sistema operativo.

## ¿Qué es?

CachyOS es una distribución de Linux basada en Arch Linux, diseñada para dispositivos de juego portátiles como la Steam Deck y Legion Go. Se enfoca en brindar una experiencia de usuario óptima y una funcionalidad completa para los jugadores. La versión Handheld Edition es una edición especial de CachyOS diseñada específicamente para estos dispositivos de juego portátiles.

## ¿Cómo funciona?

La versión de marzo de 2026 de CachyOS Handheld Edition utiliza Wayland en lugar de X11 como gestor de ventana, lo que proporciona una experiencia de usuario más fluida y eficiente. Además, el instalador de CachyOS ahora puede mostrar previsualizaciones animadas de GIF y WebP, y admite JPEG-XL para imágenes más pequeñas. El sistema operativo también ha sido actualizado para utilizar el nuevo Plasma Login Manager en lugar de SDDM.

## ¿Por qué importa?

Estos cambios son importantes porque mejoran la experiencia de usuario y la funcionalidad de CachyOS Handheld Edition. La transición a Wayland proporciona una experiencia más fluida y eficiente, mientras que las mejoras en el instalador hacen que sea más fácil instalar y configurar el sistema operativo. Además, la eliminación de Bcachefs simplifica la configuración del sistema de archivos.

## Consejo técnico

Si estás considerando instalar CachyOS Handheld Edition en tu dispositivo de juego portátil, asegúrate de leer las instrucciones de instalación cuidadosamente y de configurar el sistema de archivos correctamente para aprovechar al máximo la experiencia de usuario y la funcionalidad de este sistema operativo.

```bash
sudo pacman -S cachyos-handheld-edition
```
