# Intel XPU Manager agrega soporte para Arc Pro B65 y B70

**Fecha:** 2026-06-03
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Intel XPU Manager Adds Support For Arc Pro B65 + Arc Pro B70

---

## Introducción

Intel ha lanzado actualizaciones para su herramienta XPU Manager y el driver de NPU Linux. La herramienta XPU Manager ahora soporta la gestión y monitoreo de GPUs de datos centrales Intel, incluyendo las recién lanzadas Arc Pro B65 y B70.

## ¿Qué es?

Intel XPU Manager es una herramienta de código abierto para gestionar y monitorear GPUs de datos centrales Intel. Permite la gestión y monitoreo de flotas de sistemas o clústeres en el centro de datos a través de interfaces de línea de comandos o APIs RESTful. Funciona en sistemas Windows y Linux.

## ¿Cómo funciona?

La herramienta XPU Manager utiliza una arquitectura de código abierto para interactuar con las GPUs de datos centrales Intel. Algunas de las características incluyen la detección y configuración de comandos SMI, así como la reportación de números de serie OEM cuando estén disponibles.

## ¿Por qué importa?

La actualización de XPU Manager es importante porque permite a los usuarios gestionar y monitorear sus GPUs de datos centrales Intel de manera más eficiente. Esto es especialmente relevante para empresas que utilizan flotas de sistemas o clústeres en el centro de datos.

## Consejo técnico

Si estás utilizando GPUs de datos centrales Intel, te recomiendo verificar la compatibilidad de tu sistema con la última versión de XPU Manager y actualizar si es necesario. Puedes hacerlo consultando la documentación de GitHub.

```bash
Para verificar la versión actual de XPU Manager, puedes utilizar el comando `git pull` en tu repositorio local.
```
