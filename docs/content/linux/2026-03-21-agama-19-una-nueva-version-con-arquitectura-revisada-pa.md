## Introducción

La versión 19 del instalador Agama para openSUSE y SUSE ha sido lanzada, con cambios significativos en la arquitectura y la interfaz web. Esto permitirá a los desarrolladores crear una API más simple y consistente para controlar el instalador.

## ¿Qué es?

Agama es un instalador para openSUSE y SUSE que permite a los usuarios instalar y configurar sus sistemas de manera fácil y segura. Es una herramienta importante para la comunidad de usuarios de Linux, ya que facilita la instalación y configuración de los sistemas.

## ¿Cómo funciona?

La versión 19 de Agama introduce una arquitectura revisada que permite a los desarrolladores crear una API más simple y consistente para controlar el instalador. Esto se logra mediante la creación de una capa de abstracción entre la interfaz web y las herramientas de línea de comandos, lo que permite a los desarrolladores crear una experiencia de usuario más uniforme y fácil de usar.

## ¿Por qué importa?

Esta versión de Agama es importante porque permite a los desarrolladores crear una API más simple y consistente para controlar el instalador. Esto facilitará la creación de herramientas y aplicaciones que se integren con Agama, lo que a su vez mejorará la experiencia de usuario para los usuarios de openSUSE y SUSE.
