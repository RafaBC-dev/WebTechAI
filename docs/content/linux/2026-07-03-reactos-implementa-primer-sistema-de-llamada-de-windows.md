# ReactOS implementa primer sistema de llamada de Windows NT6

**Fecha:** 2026-07-03
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** ReactOS Implements First Windows NT6 System Call In Step Toward Vista Compatibility

---

## Introducción

El proyecto ReactOS, que busca ser una versión abierta de Windows, ha alcanzado un nuevo hito en su camino hacia la compatibilidad con Windows Vista y versiones posteriores. Después de años de trabajo, ReactOS ha implementado por primera vez un sistema de llamada de Windows NT6.

## ¿Qué es?

ReactOS es un proyecto de software que busca crear una versión abierta de Windows, compatible con los mismos drivers y software que la versión comercial. Esto significa que los usuarios pueden ejecutar aplicaciones y juegos que están diseñados para Windows en una plataforma abierta y gratuita.

## ¿Cómo funciona?

ReactOS utiliza un enfoque modular para implementar los sistemas de llamada de Windows NT6. El proyecto ha implementado el sistema de llamada NtGetCurrentProcessorNumberEx, que permite determinar el número del procesador lógico en el que se está ejecutando el código. Esto es solo el primer paso hacia la compatibilidad completa con Windows Vista y versiones posteriores.

## ¿Por qué importa?

La implementación de sistemas de llamada de Windows NT6 es importante porque permite a ReactOS ejecutar aplicaciones y juegos que están diseñados para Windows, lo que amplía la compatibilidad de la plataforma. Esto también significa que los desarrolladores pueden crear software que sea compatible con ReactOS y Windows al mismo tiempo.
