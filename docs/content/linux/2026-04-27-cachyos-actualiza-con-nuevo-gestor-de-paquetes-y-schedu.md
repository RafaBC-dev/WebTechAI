# CachyOS Actualiza con Nuevo Gestor de Paquetes y Scheduler para NVMe

**Fecha:** 2026-04-27
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** CachyOS Introduces New Default GUI Package Manager, Kyber For NVMe I/O Scheduler

---

## Introducción

La distribución Arch Linux basada CachyOS ha lanzado una actualización con importantes mejoras, incluyendo un nuevo gestor de paquetes gráfico y un scheduler para NVMe que ofrece mejoras en la respuesta y privacidad.

## ¿Qué es?

CachyOS es una distribución de Linux basada en Arch Linux que ofrece una experiencia de usuario rápida y segura. La distribución ha actualizado su gestor de paquetes gráfico a Shelly, un reimaginado moderno del gestor de paquetes de Arch Linux.

## ¿Cómo funciona?

Shelly es un gestor de paquetes gráfico que permite a los usuarios instalar y gestionar paquetes de software de manera fácil y sencilla. Además, la distribución ha implementado el scheduler Kyber para NVMe, que ofrece mejoras en la respuesta y privacidad gracias a la gestión de DNS-over-HTTPS.

## ¿Por qué importa?

Esta actualización es importante porque ofrece mejoras en la experiencia de usuario, incluyendo una respuesta más rápida y una mayor privacidad. El scheduler Kyber para NVMe también ofrece mejoras en la gestión de recursos del sistema.

## Consejo técnico

Si estás utilizando CachyOS, te recomendamos actualizar a la versión más reciente para aprovechar las mejoras en el gestor de paquetes y el scheduler para NVMe. Puedes hacerlo ejecutando el comando `sudo pacman -Syu` y luego reiniciando el sistema.

```bash
sudo pacman -Syu
```
