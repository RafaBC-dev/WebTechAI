# piBrick PocketCM5: un ordenador portátil Linux basado en Raspberry Pi CM5

**Fecha:** 2026-06-11

**Título original:** piBrick PocketCM5 – An open-source handheld Linux computer kit for Raspberry Pi CM5

## Introducción

El ingeniero indonesio Ahmad Amarullah ha diseñado un ordenador portátil Linux llamado piBrick PocketCM5, basado en el Raspberry Pi CM5. Este dispositivo es perfecto para desarrolladores, makers y administradores de sistemas que buscan una herramienta para experimentar y desarrollar aplicaciones en un entorno de escritorio.

## ¿Qué es?

El piBrick PocketCM5 es un ordenador portátil Linux que se basa en el Raspberry Pi CM5 y cuenta con una pantalla táctil AMOLED de 3,92 pulgadas, un teclado QWERTY físico y varias opciones de expansión. Es ideal para tareas como la experimentación general, el desarrollo de aplicaciones embebidas y el acceso remoto.

## ¿Cómo funciona?

El piBrick PocketCM5 utiliza el Raspberry Pi CM5 como núcleo y cuenta con una variedad de componentes, incluyendo un MCU Raspberry Pi RP2040 para la entrada de teclado y trackpad, un slot de tarjeta SD, un conector M.2 NVMe para SSDs y una salida HDMI. También cuenta con una batería de 5000mAh y un conector USB-C para carga y alimentación.

## ¿Por qué importa?

El piBrick PocketCM5 es importante porque ofrece una plataforma de desarrollo y experimentación en un entorno de escritorio, lo que permite a los desarrolladores y makers crear aplicaciones y proyectos de manera más eficiente y efectiva. También es una herramienta valiosa para administradores de sistemas que necesitan acceder a sus sistemas de manera remota.

Si estás interesado en crear un proyecto con el piBrick PocketCM5, te recomiendo investigar sobre la configuración del sistema operativo y la instalación de herramientas de desarrollo como Python y Git. También es recomendable explorar las opciones de expansión y personalización del dispositivo.
