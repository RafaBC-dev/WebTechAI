## Introducción

Si estás comenzando en el mundo de la programación, Python es una excelente opción. Es fácil de leer y escribir, y hay muchos recursos disponibles para aprender. En este artículo, te mostraremos cómo crear un juego de preguntas sencillas con Python utilizando la plataforma Raspberry Pi.

## ¿Qué es?

Un juego de preguntas es un programa que te hace preguntas y te proporciona una respuesta según tu respuesta. En este caso, creamos un juego de preguntas que te hace preguntar un año y te dice si es correcto o no. El juego utiliza la biblioteca de Python llamada 'input' para leer la respuesta del usuario y la función 'print' para mostrar la respuesta.

## ¿Cómo funciona?

Para crear el juego, utilizamos la plataforma Thonny, que viene incluida en la distribución de Raspberry Pi OS. En Thonny, escribimos un programa que utiliza la función 'input' para leer la respuesta del usuario y la función 'print' para mostrar la respuesta. El programa también utiliza variables para almacenar la respuesta del usuario y la respuesta correcta.

## ¿Por qué importa?

Este juego de preguntas es importante porque te permite practicar tus habilidades en programación y aprender a utilizar la biblioteca de Python llamada 'input'. También te permite crear juegos y aplicaciones sencillas utilizando la plataforma Raspberry Pi.

## Consejo técnico

Si quieres crear un juego de preguntas más complejo, puedes utilizar la función 'if' para comprobar si la respuesta del usuario es correcta o no. Por ejemplo, puedes crear un juego que te hace preguntar un número y te dice si es par o impar.
