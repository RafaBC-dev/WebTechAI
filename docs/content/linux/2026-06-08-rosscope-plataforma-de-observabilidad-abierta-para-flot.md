# ROSscope: plataforma de observabilidad abierta para flotas de ROS 2

**Fecha:** 2026-06-08

**Tags:** robotica, linux, codigo
**Título original:** ROSscope Open source observability platform for ROS 2 fleets (v0.1.0)

## Introducción

La comunidad de ROS ha recibido un regalo con ROSscope, una plataforma de observabilidad abierta que permite monitorear en tiempo real los sistemas ROS 2. Con ROSscope, los desarrolladores pueden detectar anormalidades y mejorar la eficiencia de sus sistemas.

## ¿Qué es?

ROSscope es una plataforma de observabilidad abierta diseñada específicamente para flotas de ROS 2. Permite monitorear en tiempo real todos los temas, detectar anormalidades automáticamente y mapear las relaciones de publicador y suscriptor en tiempo real.

## ¿Cómo funciona?

ROSscope utiliza una combinación de Prometheus y Grafana para proporcionar una visión completa de los sistemas ROS 2. Detecta anormalidades automáticamente utilizando estadísticas de z-score y mantiene un registro de las relaciones de publicador y suscriptor en tiempo real.

## ¿Por qué importa?

ROSscope es importante porque permite a los desarrolladores de ROS 2 detectar anormalidades y mejorar la eficiencia de sus sistemas. También proporciona una herramienta útil para la gestión de alertas y la resolución de problemas.

## Consejo técnico

Si estás trabajando con ROS 2, prueba ROSscope para mejorar la observabilidad de tus sistemas. Puedes instalarlo en un solo comando utilizando `curl -sSL https://raw.githubusercontent.com/jawad-glitch/ROSscope/main/install.sh | bash`.

```bash
curl -sSL https://raw.githubusercontent.com/jawad-glitch/ROSscope/main/install.sh | bash
```
