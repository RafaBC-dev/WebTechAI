# NVIDIA prepara soporte para 'Blackwell-Next' en Linux 7.2

**Fecha:** 2026-06-21
**Categoría:** linux
**Tags:** linux, nvidia, gpu
**Título original:** Linux 7.2 Begins Making Preparations For NVIDIA "Blackwell-Next"

---

## Introducción

NVIDIA ha comenzado a preparar el soporte para su próxima generación de GPUs, denominada 'Blackwell-Next', en la versión 7.2 del kernel de Linux. Esto significa que los usuarios de Linux podrán disfrutar de una mayor compatibilidad con las futuras GPUs de NVIDIA.

## ¿Qué es?

Blackwell-Next es la próxima generación de GPUs de NVIDIA, que se espera que sea una mejora significativa sobre la actual generación de GPUs Blackwell. La tecnología CXL (Compute Express Link) es una característica importante de Blackwell-Next, que permite una mayor velocidad y eficiencia en la transferencia de datos entre la GPU y el sistema.

## ¿Cómo funciona?

La actualización del driver NVGrace-GPU VFIO para Linux 7.2 incluye cambios en la gestión de la tecnología CXL para preparar el soporte para Blackwell-Next. Esto implica que los desarrolladores de NVIDIA están trabajando en la implementación de la tecnología CXL en el kernel de Linux para garantizar una mayor compatibilidad con las futuras GPUs de la empresa.

## ¿Por qué importa?

El soporte para Blackwell-Next en Linux 7.2 es importante porque permitirá a los usuarios de Linux disfrutar de una mayor compatibilidad con las futuras GPUs de NVIDIA, lo que puede mejorar la experiencia de usuario y la eficiencia del sistema.

## Consejo técnico

Si deseas disfrutar de la compatibilidad con Blackwell-Next en Linux 7.2, asegúrate de actualizar tu kernel de Linux a la versión 7.2 y verificar que tu sistema esté configurado correctamente para utilizar la tecnología CXL.

```bash
sudo apt-get update && sudo apt-get install linux-image-7.2
```
