# Banana Pi lanza BPI-VP10: un módulo de control de servos a bajo costo

**Fecha:** 2026-06-30
**Categoría:** linux
**Tags:** robotica, microcontroladores, embebidos
**Título original:** Banana Pi BPI-VP10 – A low-cost servo control board based on Fortior FU7512L dual-core RISC-V MCU

---

## Introducción

Banana Pi ha lanzado el BPI-VP10, un módulo de control de servos a bajo costo basado en el microcontrolador Fortior FU7512L. Este módulo está diseñado para aplicaciones de control industrial que requieren servos de 100W/200W. Con el BPI-VP10, los desarrolladores pueden experimentar con el microcontrolador FU7512L y diseñar proyectos o productos alrededor de él.

## ¿Qué es?

El BPI-VP10 es un módulo de control de servos que utiliza el microcontrolador Fortior FU7512L y el pre-amplificador FD6288Q. Este módulo soporta diferentes modos de control, incluyendo pulso/dirección, analógico y RS485, y es compatible con codificadores absolutos (protocolos Tamagawa/BiSS-C), codificadores incrementales y codificadores de campo magnético.

## ¿Cómo funciona?

El BPI-VP10 utiliza el microcontrolador FU7512L, que cuenta con un núcleo de 32 bits RISC-V a 48 MHz y un motor de motor ME2 a 48 MHz con módulo de control de velocidad y módulo de córdico. El módulo también incluye un pre-amplificador FD6288Q con tres drivers de puente de media onda independientes. El módulo soporta la conexión de codificadores absolutos, codificadores incrementales y codificadores de campo magnético.

## ¿Por qué importa?

El BPI-VP10 es importante porque ofrece una plataforma a bajo costo para que los desarrolladores experimenten con el microcontrolador FU7512L y diseñen proyectos o productos alrededor de él. Esto puede ser especialmente útil para aplicaciones de control industrial que requieren servos de 100W/200W.

## Consejo técnico

Si estás interesado en experimentar con el BPI-VP10, puedes descargar el firmware fuente en C y los binarios que se basan en la herramienta FTM32ForIDE de Fortior Tech. También puedes utilizar la herramienta de software de Fortior Tech Servo Studio 2 para programar y configurar el módulo.

```bash
git clone https://github.com/banana-pi/bpi-vp10-firmware.git
```
