# Aspiradora robótica abierta y sin nube: una alternativa a las rentas

**Fecha:** 2026-07-22
**Categoría:** linux
**Tags:** robotica, linux, diseño-3d
**Título original:** Open Source Vacuum Avoids Cloud

---

## Introducción

La tendencia de la tecnología a convertirse en una suscripción ha llevado a un cambio en la comunidad abierta, donde se están creando reemplazos para estos servicios. Ahora, esto se está aplicando también en hardware.

## ¿Qué es?

OOMWOO es una aspiradora robótica completamente abierta, desde el hardware hasta el software, que no requiere servicios en la nube. Utiliza un Raspberry Pi con ROS 2 y un sistema de lidar 2D para mapear y navegar.

## ¿Cómo funciona?

La aspiradora utiliza un Raspberry Pi como computadora central, que ejecuta ROS 2 y se integra con herramientas como Home Assistant. El sistema de lidar 2D permite mapear y navegar los espacios, y la impresión 3D se utiliza para crear la mayoría de los componentes.

## ¿Por qué importa?

Esta aspiradora robótica abierta es importante porque ofrece una alternativa a las rentas y permite a los usuarios controlar su propia tecnología. También es una oportunidad para que los desarrolladores contribuyan y mejorar el proyecto.

## Consejo técnico

Si deseas contribuir al proyecto, puedes empezar adaptando los archivos de impresión 3D para utilizar partes off-the-shelf. Puedes encontrar los archivos en la página de GitHub del proyecto.

```bash
git clone https://github.com/OOMWOO/oomwoo.git
```
