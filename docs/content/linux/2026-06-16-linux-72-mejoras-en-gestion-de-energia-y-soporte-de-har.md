# Linux 7.2: mejoras en gestión de energía y soporte de hardware

**Fecha:** 2026-06-16
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Linux 7.2 Power Management Adds New Hardware Support While Dropping AMD Elan

---

## Introducción

La versión 7.2 del kernel de Linux ha sido actualizada con mejoras significativas en la gestión de energía y soporte de hardware. Estas actualizaciones incluyen soporte para nuevos dispositivos y la eliminación de soporte para hardware obsoleto.

## ¿Qué es?

Linux es un sistema operativo de código abierto que se utiliza en una amplia variedad de dispositivos, desde servidores hasta dispositivos embebidos. El kernel de Linux es el núcleo del sistema operativo y se encarga de gestionar los recursos del sistema y proporcionar servicios básicos.

## ¿Cómo funciona?

El kernel de Linux utiliza una arquitectura modular para gestionar los recursos del sistema. Las actualizaciones de la versión 7.2 incluyen la adición de soporte para nuevos dispositivos, como el Qualcomm Shikra / QCM2390, y la eliminación de soporte para hardware obsoleto, como el AMD Elan SC4. También se han agregado mejoras en la gestión de energía y la compresión de datos.

## ¿Por qué importa?

Estas actualizaciones son importantes porque permiten a los desarrolladores crear aplicaciones y sistemas más eficientes y escalables. También permiten a los usuarios de dispositivos embebidos aprovechar al máximo el potencial de sus dispositivos.

## Consejo técnico

Si estás desarrollando un dispositivo embebido con Linux, considera utilizar el kernel de Linux 7.2 para aprovechar las mejoras en la gestión de energía y soporte de hardware.

```bash
Para verificar la versión del kernel de Linux en tu dispositivo, puedes utilizar el comando `uname -r`.
```
