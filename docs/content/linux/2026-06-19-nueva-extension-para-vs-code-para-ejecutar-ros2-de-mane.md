# Nueva extensión para VS Code para ejecutar ROS2 de manera rápida

**Fecha:** 2026-06-19
**Categoría:** linux
**Tags:** robotica, codigo, linux
**Título original:** A VS Code extension for quickly running ROS2 launch files, Python nodes, and C++ nodes

---

## Introducción

La comunidad de desarrolladores de ROS2 ha recibido una noticia importante con la creación de una nueva extensión para VS Code que permite ejecutar launch files, nodos Python y C++ de manera rápida y sencilla.

## ¿Qué es?

La extensión llamada ROS2 Quick Runner es una herramienta desarrollada para facilitar el trabajo con ROS2 en Visual Studio Code (VS Code). Permite ejecutar launch files, nodos Python y C++ de manera rápida y sencilla, eliminando la necesidad de escribir comandos en la terminal.

## ¿Cómo funciona?

La extensión funciona mediante el uso de comandos contextuales en VS Code. Al hacer clic derecho en un archivo .launch.py, .py o .cpp, se puede seleccionar la opción 'ros2 launch' o 'ros2 run' para ejecutar el archivo correspondiente. La extensión también permite obtener el nombre del workspace y construir el workspace de ROS2 mediante el comando 'colcon build'.

## ¿Por qué importa?

Esta extensión es importante porque simplifica el trabajo con ROS2 en VS Code, lo que facilita la creación de proyectos de robótica y automatización. También reduce el tiempo de desarrollo y mejora la productividad de los desarrolladores.

## Consejo técnico

Si estás trabajando con ROS2 en VS Code, instala la extensión ROS2 Quick Runner y aprovecha sus funciones para ejecutar launch files y nodos de manera rápida y sencilla.

```bash
ros2-quick-runner.ros2launch
```
