## Introducción

Python es el lenguaje de programación más popular del mundo, según el Índice TIOBE. Su simplicidad y facilidad de lectura lo convierten en el ideal para iniciarse en la programación. En este artículo, aprenderás a crear un juego de preguntas y respuestas sencillo utilizando Python.

## ¿Qué es?

El juego de preguntas y respuestas es un proyecto de programación que utiliza Python para crear un entorno interactivo donde el usuario puede responder preguntas y recibir respuestas. Se utiliza la biblioteca estándar de Python para leer la entrada del usuario y mostrar la respuesta correcta.

## ¿Cómo funciona?

El juego utiliza la función `input()` para leer la entrada del usuario y la función `print()` para mostrar la respuesta correcta. El usuario puede responder preguntas y recibir respuestas hasta que decida salir del juego. El código se escribe en el entorno de desarrollo Thonny, que viene incluido en la distribución de Raspberry Pi OS.

## ¿Por qué importa?

Este juego de preguntas y respuestas es importante porque te permite practicar la programación en Python y aprender a crear interfaces interactivas. También te permite crear juegos y aplicaciones personalizadas para Raspberry Pi.

## Consejo técnico

Para crear más preguntas y respuestas, puedes utilizar la función `input()` para leer la entrada del usuario y la función `print()` para mostrar la respuesta correcta. Puedes agregar más preguntas y respuestas utilizando el mismo código y modificando la lógica del juego.

```bash
input() y print()
```
