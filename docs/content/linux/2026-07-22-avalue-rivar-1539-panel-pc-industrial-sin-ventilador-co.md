# Avalue RIVAR-1539: panel PC industrial sin ventilador con Intel N150

**Fecha:** 2026-07-22
**Categoría:** linux
**Tags:** robotica, linux, embebidos
**Título original:** Avalue RIVAR-1539 is a 15.6-inch fanless industrial panel PC powered by an Intel N150 Twin Lake processor

---

## Introducción

Avalue Technology ha presentado el RIVAR-1539, un panel PC industrial sin ventilador de 15,6 pulgadas con procesador Intel N150. Este sistema está diseñado para aplicaciones de edge computing y HMI, como kioscos de autoorden, contadores inteligentes y sistemas de señalización digital.

## ¿Qué es?

El RIVAR-1539 es un panel PC industrial sin ventilador que se basa en el procesador Intel N150 quad-core Twin Lake. Este procesador tiene una velocidad de 800 MHz y puede alcanzar hasta 3,6 GHz en modo turbo, con un consumo de energía de 6W. El panel PC cuenta con una pantalla Full HD de 15,6 pulgadas, con una resolución de 1920 x 1080 píxeles y un ángulo de visión de 89°.

## ¿Cómo funciona?

El RIVAR-1539 cuenta con una pantalla Full HD de 15,6 pulgadas y un procesador Intel N150 quad-core Twin Lake. El sistema también cuenta con 8GB de memoria LPDDR5 y 128GB de almacenamiento eMMC. El panel PC tiene una conexión HDMI y cuatro puertos USB 3.0, así como dos puertos de red Gigabit Ethernet. El sistema también cuenta con una conexión Wi-Fi 5 y Bluetooth 5.0 mediante una ranura de expansión.

## ¿Por qué importa?

El RIVAR-1539 es importante porque ofrece una solución de edge computing y HMI para aplicaciones industriales y comerciales. El sistema puede ser utilizado en kioscos de autoorden, contadores inteligentes y sistemas de señalización digital, entre otros. El RIVAR-1539 también cuenta con una conectividad Wi-Fi 5 y Bluetooth 5.0, lo que lo hace ideal para aplicaciones que requieren conectividad inalámbrica.

## Consejo técnico

Si deseas utilizar el RIVAR-1539 con Linux, puedes configurar el sistema para que use el kernel Linux 5.10 o superior. También puedes utilizar la herramienta de configuración U-Boot para personalizar el sistema.

```bash
sudo apt-get update && sudo apt-get install -y linux-image-5.10.0-13-generic
```
