# Triple LED Matrix Bonnet para Raspberry Pi: controla hasta 3 pantallas

**Fecha:** 2026-06-03
**Categoría:** linux
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** Triple LED Matrix Bonnet for Raspberry Pi – The Raspberry Pi Official Magazine, Issue 166 @rpimagazine

---

## Introducción

La Raspberry Pi Official Magazine ha lanzado un nuevo bonnet para Raspberry Pi que permite controlar hasta tres pantallas LED en paralelo. Este dispositivo es ideal para proyectos de visualización y animación.

## ¿Qué es?

El Triple LED Matrix Bonnet es un dispositivo que se conecta a la GPIO del Raspberry Pi y permite controlar hasta tres pantallas LED en paralelo. Estas pantallas pueden tener tamaños de 16 × 32, 32 × 32, 32 × 64 o 64 × 64. El dispositivo no tiene salida de poder, por lo que es necesario conectar un suministro de poder externo de 5V y 10A o más.

## ¿Cómo funciona?

Una vez conectado el bonnet al Raspberry Pi, se puede elegir entre diferentes software para controlar las pantallas. Para Raspberry Pi 4 y anteriores, se recomienda utilizar el driver rpi-rgb-led-matrix de Henner Zeller, que tiene bindings en Python y ejemplos de código. Para Raspberry Pi 5, se puede utilizar la biblioteca Piomatter de Adafruit, que aprovecha la funcionalidad de PIO.

## ¿Por qué importa?

Este dispositivo es importante porque permite a los desarrolladores crear proyectos de visualización y animación más complejos y sofisticados. Puede ser utilizado en proyectos de arte digital, educación, entretenimiento y más.

## Consejo técnico

Si deseas utilizar este bonnet, asegúrate de conectar un suministro de poder externo adecuado y de utilizar el software recomendado por Adafruit.

```bash
rpimag.co/rpirgbledmatrix (driver rpi-rgb-led-matrix)
```
