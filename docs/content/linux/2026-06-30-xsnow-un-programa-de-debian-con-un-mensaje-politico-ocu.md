# Xsnow: un programa de Debian con un mensaje político oculto

## Introducción

La aplicación xsnow, que genera un efecto de nieve en pantallas X11, ha sido encontrada con un mensaje político oculto en su código. El mantenimiento del programa ha incluido una protesta política en forma de un 'huevo de Pascua' que se activa cuando el idioma del programa se establece en ruso.

## ¿Qué es?

Xsnow es una aplicación que genera un efecto de nieve en pantallas X11, permitiendo a los usuarios disfrutar de una animación agradable. Fue creado para proporcionar una experiencia visual atractiva en entornos de escritorio X11.

## ¿Cómo funciona?

La aplicación xsnow funciona generando un efecto de nieve en la pantalla, utilizando la biblioteca X11 para crear la animación. El código incluye un 'huevo de Pascua' que se activa cuando el idioma del programa se establece en ruso, mostrando un mensaje político oculto.

## ¿Por qué importa?

El caso de xsnow importa porque muestra cómo un programa de código abierto puede incluir mensajes políticos ocultos, lo que puede ser un problema para algunos usuarios. También destaca la importancia de revisar cuidadosamente el código de las aplicaciones que se utilizan.

## Consejo técnico

Si deseas evitar que un programa incluya mensajes políticos ocultos, puedes revisar el código fuente antes de instalarlo. Puedes utilizar herramientas como 'git' para ver las modificaciones recientes en el código y detectar posibles mensajes ocultos.

```bash
git log -p
```
