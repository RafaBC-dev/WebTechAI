# Ubuntu MATE sigue adelante a pesar de la ausencia de Ubuntu MATE 26.04

**Fecha:** 2026-06-09
**Categoría:** linux
**Tags:** linux, ubuntu, mate
**Título original:** Ubuntu MATE Is Continuing Despite No Ubuntu MATE 26.04 Release

---

## Introducción

La comunidad de Ubuntu MATE ha estado en estado de alerta después de que Martin Wimpress dejara su cargo como líder del proyecto y no se lanzara la versión 26.04. Sin embargo, la noticia es que Ubuntu MATE seguirá adelante con un nuevo equipo en el timón.

## ¿Qué es?

Ubuntu MATE es una distribución de Linux basada en Ubuntu que utiliza el escritorio MATE, un clon de GNOME2. Esta distribución está diseñada para ofrecer una experiencia de usuario sencilla y fácil de usar.

## ¿Cómo funciona?

Ubuntu MATE se puede instalar desde un ISO o actualizar desde una versión anterior de Ubuntu. Los usuarios pueden instalar el escritorio MATE utilizando el comando `sudo apt install ubuntu-mate-desktop` después de instalar Ubuntu 26.04.

## ¿Por qué importa?

La noticia es importante porque asegura que Ubuntu MATE seguirá adelante con un nuevo equipo en el timón, lo que significa que los usuarios pueden seguir utilizando esta distribución sin preocupaciones. Además, el nuevo equipo está trabajando para cubrir los items faltantes y puede lanzar una versión 26.10 en octubre de 2026.

## Consejo técnico

Para instalar Ubuntu MATE en una máquina virtual, puedes utilizar VirtualBox y descargar el ISO de Ubuntu 26.04. Luego, puedes instalar el escritorio MATE utilizando el comando `sudo apt install ubuntu-mate-desktop`.

```bash
sudo apt install ubuntu-mate-desktop
```
