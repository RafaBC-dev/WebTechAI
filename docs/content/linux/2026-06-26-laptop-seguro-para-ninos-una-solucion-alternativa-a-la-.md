# Laptop seguro para niños: una solución alternativa a la verificación de edad

**Fecha:** 2026-06-26
**Categoría:** linux
**Tags:** linux, ia-local, robotica
**Título original:** This Kid-Friendly Laptop Build is The Antidote to Age Verification

---

## Introducción

La verificación de edad en línea ha sido un tema de debate reciente, pero ¿qué hay detrás de esta idea? Un ingeniero ha creado un laptop seguro para niños que puede ser una solución alternativa.

## ¿Qué es?

El proyecto consiste en un laptop antiguo con Kubuntu, una distribución de Linux, que se ha configurado para ser seguro para niños. Se ha eliminado el acceso a la aplicación de navegación y se ha agregado un atajo para acceder al terminal si es necesario.

## ¿Cómo funciona?

El laptop se ha configurado con una selección cuidadosa de aplicaciones educativas y de entretenimiento, y se ha utilizado Vivaldi's app mode para acceder a juegos de navegador y herramientas en la nube. El acceso a la aplicación de navegación aún no está disponible para los niños, ya que el proyecto está dirigido a niños de 7 años.

## ¿Por qué importa?

Este proyecto importa porque muestra que es posible crear un entorno seguro para niños sin necesidad de leyes de verificación de edad. También destaca la importancia de la educación técnica para que las personas puedan tomar decisiones informadas sobre su seguridad en línea.

## Consejo técnico

Si deseas crear un entorno seguro para tus hijos en tu propio laptop, puedes utilizar la distribución de Linux Kubuntu y configurarla para eliminar el acceso a la aplicación de navegación. También puedes utilizar Vivaldi's app mode para acceder a juegos de navegador y herramientas en la nube.

```bash
sudo apt-get install kubuntu-desktop
```
