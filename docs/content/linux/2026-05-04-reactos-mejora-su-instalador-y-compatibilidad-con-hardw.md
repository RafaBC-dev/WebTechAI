# ReactOS mejora su instalador y compatibilidad con hardware

**Fecha:** 2026-05-04

**Tags:** linux, codigo, herramientas
**Título original:** ReactOS Gets Unified Installer Image and a New Storage Stack

## Introducción

La comunidad de ReactOS ha logrado dos avances importantes en la estabilidad y compatibilidad de su sistema operativo. Ahora es posible instalar y probar ReactOS de manera más sencilla y con una mayor compatibilidad con hardware.

## ¿Qué es?

ReactOS es un sistema operativo gratuito y de código abierto que busca ser compatible con Windows. Aunque no busca reemplazar a Windows como sistema operativo de escritorio, busca ofrecer una alternativa más segura y personalizable.

## ¿Cómo funciona?

El nuevo instalador de ReactOS combina las imágenes de arranque y de CD en vivo en una sola imagen, lo que facilita la instalación y prueba del sistema. Además, se ha implementado un nuevo stack de almacenamiento PnP-aware para ATA y AHCI, lo que mejora la compatibilidad con hardware.

## ¿Por qué importa?

Estos avances mejoran la experiencia de instalación y uso de ReactOS, acercándola a la de Windows y muchas distribuciones de Linux. Si tienes problemas con ReactOS en hardware real, ahora es el momento de darle otra oportunidad y proporcionar retroalimentación a los desarrolladores.

## Consejo técnico

Si deseas probar ReactOS en tu hardware, asegúrate de descargar la última versión del instalador unificado y sigue las instrucciones para configurar el nuevo stack de almacenamiento.

```bash
reactos-usb.img
```
