# Linux 7.1-rc1: nuevo driver NTFS y mejoras para seguridad y rendimiento

**Fecha:** 2026-04-27
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Linux 7.1-rc1 Released With New NTFS Driver, FRED By Default & Much More

---

## Introducción

La versión de prueba Linux 7.1-rc1 ha sido lanzada, con importantes mejoras en seguridad, rendimiento y compatibilidad. Este lanzamiento es el primer paso hacia la versión estable de Linux 7.1, que se espera que esté disponible a mediados de junio.

## ¿Qué es?

Linux es un sistema operativo de código abierto que se utiliza en una amplia variedad de dispositivos, desde servidores hasta dispositivos móviles. La versión 7.1-rc1 es una versión de prueba de la próxima versión estable de Linux, que incluye mejoras en seguridad, rendimiento y compatibilidad.

## ¿Cómo funciona?

La versión 7.1-rc1 incluye un nuevo driver NTFS que pretende ser la opción de driver de NTFS de código abierto más superior. También se han agregado soporte para 12 nuevos SoCs, drivers para la gama Lenovo Legion Go, soporte para Intel QAT Zstd y más. Además, se han realizado mejoras en la seguridad y el rendimiento, incluyendo el soporte para AMD Zen 6 y la implementación de Intel Linear Address Space Separation (LASS).

## ¿Por qué importa?

Estas mejoras son importantes porque mejoran la seguridad y el rendimiento de Linux, lo que significa que los usuarios pueden disfrutar de una experiencia más segura y eficiente. Además, el soporte para nuevos SoCs y dispositivos significa que Linux puede ser utilizado en una mayor variedad de dispositivos.

## Consejo técnico

Si estás utilizando Linux, es recomendable actualizar a la versión 7.1-rc1 para aprovechar las mejoras en seguridad y rendimiento. Puedes hacerlo ejecutando el comando `sudo apt-get update && sudo apt-get upgrade` en tu terminal.

```bash
sudo apt-get update && sudo apt-get upgrade
```
