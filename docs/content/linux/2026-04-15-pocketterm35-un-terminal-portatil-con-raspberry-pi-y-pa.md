# PocketTerm35: un terminal portátil con Raspberry Pi y pantalla táctil

**Fecha:** 2026-04-15
**Categoría:** linux
**Tags:** robotica, embebidos, linux
**Título original:** PocketTerm35 – A Raspberry Pi 4/5-based handheld Linux terminal with 3.5-inch touch display and built-in keyboard

## Introducción

Waveshare ha lanzado el PocketTerm35, un terminal portátil que combina la potencia de un Raspberry Pi con una pantalla táctil y un teclado QWERTY. Este dispositivo es ideal para juegos de retro, pero también puede ser utilizado como terminal y controlador HMI portátil.

## ¿Qué es?

El PocketTerm35 es un dispositivo portátil que se basa en un Raspberry Pi 4 o 5, con una pantalla táctil de 3,5 pulgadas y un teclado QWERTY integrado. Incluye puertos de Gigabit Ethernet, USB 3.0 y 2.0, y un altavoz estéreo.

## ¿Cómo funciona?

El dispositivo utiliza el sistema operativo Raspberry Pi OS y cuenta con una arquitectura que permite la conexión de un teclado y un mouse externos. La pantalla táctil permite interactuar con el sistema operativo y los aplicaciones, mientras que el teclado QWERTY facilita la entrada de texto.

## ¿Por qué importa?

El PocketTerm35 es importante porque ofrece una solución portátil y flexible para juegos de retro, así como para aplicaciones de terminal y controlador HMI. Su diseño compacto y ligero lo hace ideal para llevar en un bolso o mochila.

## Consejo técnico

Si deseas utilizar el PocketTerm35 como terminal portátil, asegúrate de configurar correctamente el archivo config.txt para habilitar la conexión de teclado y mouse externos.

```bash
dtoverlay=waveshare-35dpi-4b
```
