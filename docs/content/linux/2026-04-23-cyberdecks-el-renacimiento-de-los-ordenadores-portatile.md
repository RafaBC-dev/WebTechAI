## Introducción

La comunidad de makers está reviviendo la idea de los cyberdecks, ordenadores portátiles que combinan tecnología moderna con diseños futuristas. Y la Raspberry Pi es el corazón de muchos de estos proyectos.

## ¿Qué es?

Un cyberdeck es un ordenador portátil que combina una Raspberry Pi con componentes adicionales para crear un dispositivo que puede ser utilizado en entornos de ciencia ficción o en proyectos de hacking y seguridad informática.

## ¿Cómo funciona?

Los cyberdecks suelen utilizar una Raspberry Pi como procesador central, que se conecta a una pantalla, teclado y otros componentes para crear un dispositivo portátil. Algunos proyectos también incluyen componentes adicionales como sensores, cámaras o dispositivos de almacenamiento.

## ¿Por qué importa?

Los cyberdecks importan porque permiten a los makers y desarrolladores experimentar con tecnologías nuevas y creativas, y también pueden ser utilizados para proyectos de hacking y seguridad informática.

## Consejo técnico

Si deseas crear tu propio cyberdeck, considera utilizar una Raspberry Pi 4 como procesador central y conectarlo a una pantalla y teclado para crear un dispositivo portátil. También puedes utilizar herramientas como Manjaro Linux y ytfzf para crear un entorno de desarrollo y ejecución.

```bash
sudo apt-get install manjaro-lxde
```
