# Linux en Android: computación potente a bajo costo

**Fecha:** 2026-05-27
**Categoría:** linux
**Tags:** linux, android, robotica
**Título original:** Linux on Android Provides Inexpensive, Powerful Computing

## Introducción

En algunas partes del mundo, los operadores de telefonía venden teléfonos a precios muy bajos, siempre y cuando estén vinculados a su red. Esto puede ser una oportunidad para los hackers, ya que estos teléfonos pueden ejecutar un entorno de Linux, convirtiéndolos en una fuente de hardware de computación potente a bajo costo.

## ¿Qué es?

Linux on Android es una técnica que permite ejecutar un entorno de Linux en un teléfono Android, aprovechando su hardware para realizar tareas de computación intensiva. Esto se logra utilizando una terminal emuladora como Termux, que proporciona un entorno de Linux dentro de Android.

## ¿Cómo funciona?

Para implementar Linux on Android, se instala Termux en el teléfono y se configuran varias opciones de configuración para mejorar la experiencia de Linux. Luego, se instalan una extensión de X11 y una aplicación que expone una API para Termux. Finalmente, se instala un entorno de escritorio como XFCE y se escribe un script para iniciar automáticamente.

## ¿Por qué importa?

Linux on Android ofrece una forma económica y potente de computación, lo que puede ser útil para tareas como la edición de video, la programación y la automatización. Además, permite aprovechar el hardware de los teléfonos Android, que de otro modo serían desechados.

## Consejo técnico

Si deseas aprovechar el hardware de tu teléfono Android, instala Termux y configura las opciones de configuración para mejorar la experiencia de Linux. Luego, instala XFCE y escribe un script para iniciar automáticamente el entorno de escritorio.

```bash
sudo apt-get update
```
