# Forlinx lanza módulo de sistema Rockchip RK3572 con Linux 6.12

**Fecha:** 2026-06-04
**Categoría:** linux
**Tags:** robotica, microcontroladores, software y linux
**Título original:** Forlinx launches Rockchip RK3572 system-on-module (SoM) and development board with Linux 6.12 BSP

---

## Introducción

Forlinx ha lanzado el módulo de sistema Rockchip RK3572, un procesador de alta gama diseñado para aplicaciones de HMI (Interfaz de Usuario Humano) que aprovechan la inteligencia artificial en la orilla del borde. Este módulo está basado en el procesador Rockchip RK3572, que cuenta con un octa-core Cortex-A73/A53 y un NPU de 4 TOPS.

## ¿Qué es?

El módulo de sistema Rockchip RK3572 es un componente de hardware que se utiliza en aplicaciones de HMI que requieren procesamiento de imágenes y audio en tiempo real. Está diseñado para ser utilizado en dispositivos de consumo, control industrial, cómputo en la orilla del borde, seguridad inteligente y terminales de vehículos.

## ¿Cómo funciona?

El módulo de sistema Rockchip RK3572 cuenta con un procesador octa-core Cortex-A73/A53, un NPU de 4 TOPS y un GPU Mali-G310V2 MC1. También cuenta con conectores para memoria, almacenamiento y dispositivos periféricos como cámaras, sensores y pantallas. El módulo está diseñado para funcionar con sistemas operativos como Linux 6.12, Android 16 y Debian 13.

## ¿Por qué importa?

El módulo de sistema Rockchip RK3572 es importante porque permite a los desarrolladores crear dispositivos de HMI que pueden procesar imágenes y audio en tiempo real, lo que es esencial en aplicaciones como la seguridad inteligente, el control industrial y los terminales de vehículos.

## Consejo técnico

Si estás desarrollando un dispositivo de HMI que requiere procesamiento de imágenes y audio en tiempo real, considera utilizar el módulo de sistema Rockchip RK3572 y el SDK de Rockchip para aprovechar la potencia del NPU y el GPU.

```bash
Para instalar el SDK de Rockchip, ejecuta el comando `sudo apt-get install rockchip-sdk` en tu sistema Linux.
```
