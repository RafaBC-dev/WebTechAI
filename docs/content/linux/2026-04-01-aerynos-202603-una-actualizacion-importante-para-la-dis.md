## Introducción

La distribución Linux AerynOS ha lanzado su versión 2026.03, que incluye mejoras significativas en el entorno de escritorio y la gestión de paquetes. Esta actualización es importante para los usuarios de AerynOS y para la comunidad Linux en general.

## ¿Qué es?

AerynOS es una distribución Linux desde cero, originalmente conocida como Serpent OS. Se enfoca en ofrecer una experiencia de usuario moderna y segura, con un enfoque en la simplicidad y la facilidad de uso.

## ¿Cómo funciona?

AerynOS utiliza el gestor de paquetes Moss, que permite a los usuarios buscar y instalar paquetes de manera sencilla. La distribución también incluye el entorno de escritorio GNOME 50, que ofrece una experiencia de usuario intuitiva y personalizable.

## ¿Por qué importa?

Esta actualización es importante porque incluye mejoras significativas en el entorno de escritorio y la gestión de paquetes. Los usuarios de AerynOS pueden disfrutar de una experiencia de usuario más segura y fácil de usar, con acceso a una amplia variedad de paquetes y herramientas.

## Consejo técnico

Si estás utilizando AerynOS, te recomendamos actualizar a la versión 2026.03 y explorar las nuevas características y mejoras del gestor de paquetes Moss. Puedes buscar paquetes específicos utilizando el comando `moss search <nombre_del_paquete>`.

```bash
moss search <nombre_del_paquete>
```
