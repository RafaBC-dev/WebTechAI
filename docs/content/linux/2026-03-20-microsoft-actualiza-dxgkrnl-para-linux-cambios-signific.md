# Microsoft actualiza DXGKRNL para Linux: cambios significativos después de cuatro años

## Introducción

Microsoft ha lanzado una nueva versión de la driver DXGKRNL para Linux, que incluye cambios significativos después de cuatro años de inactividad. Esta actualización se enfoca en mejorar la compatibilidad con el entorno de desarrollo de Windows Subsystem for Linux (WSL2).

## ¿Qué es?

DXGKRNL es una driver de kernel desarrollada por Microsoft para exponer la compatibilidad de DirectX en Linux. Esta driver se enfoca en permitir la ejecución de aplicaciones de Windows en un entorno de Linux, específicamente en el contexto del WSL2.

## ¿Cómo funciona?

La driver DXGKRNL se integra en el kernel de Linux para proporcionar acceso a la API de DirectX. Esto permite a las aplicaciones de Windows utilizar la gráfica y la aceleración de DirectX en un entorno de Linux. La actualización incluye soporte para adaptadores de cómputo solo, sincronización de memoria y fijaciones de bug.

## ¿Por qué importa?

Esta actualización es importante porque brinda soporte adicional para el WSL2, lo que permite a los desarrolladores ejecutar aplicaciones de Windows en un entorno de Linux de manera más eficiente. También incluye mejoras en la compatibilidad y la estabilidad del sistema.

## Consejo técnico
