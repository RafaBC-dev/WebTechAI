## Introducción

Python es el lenguaje de programación más popular del mundo, según el Índice TIOBE. Es fácil de aprender y utilizar, lo que lo hace ideal para principiantes. En este artículo, aprenderás a crear un juego de preguntas y respuestas sencillo utilizando Python y el entorno de desarrollo Thonny.

## ¿Qué es?

El juego de preguntas y respuestas es un proyecto simple que utiliza Python para pedir al usuario que responda a una pregunta y luego verificar si la respuesta es correcta. El juego se puede extender para incluir más preguntas y respuestas.

## ¿Cómo funciona?

El juego utiliza la función `input()` de Python para pedir al usuario que responda a una pregunta. La respuesta se almacena en una variable y luego se compara con la respuesta correcta utilizando un operador de comparación. Si la respuesta es correcta, el juego imprime un mensaje de éxito, si no, imprime un mensaje de error.

## ¿Por qué importa?

Este juego es importante porque te permite practicar la programación y aprender a utilizar la función `input()` de Python. También te enseña a crear un juego sencillo que puede ser extensible para incluir más preguntas y respuestas.

## Consejo técnico

Para crear más preguntas y respuestas, puedes utilizar un archivo de texto para almacenar las preguntas y respuestas. Luego, puedes leer el archivo utilizando la función `open()` de Python y agregar las preguntas y respuestas al juego.

```bash
print('Hello', input('¿Cuál es tu nombre?'))
```
