## Introducción

La empresa MSI ha lanzado una placa base AMD Ryzen AM5 para Linux y open-source, diseñada para entusiastas de la comunidad de código abierto. Esta placa base es competitiva en precio y ofrece una experiencia de firmware completamente open-source.

## ¿Qué es?

La placa base MSI PRO B850-P WIFI es un proyecto de la empresa 3mdeb, que busca portar el firmware open-source Coreboot y el código de inicialización de CPU openSIL a una placa base de escritorio. Esto permitirá a los usuarios tener un control total sobre su firmware y evitar depender de software cerrado.

## ¿Cómo funciona?

La placa base se basa en la arquitectura AMD Ryzen AM5 y utiliza el firmware open-source Coreboot, que se puede personalizar y modificar. El código de inicialización de CPU openSIL se utiliza para inicializar el procesador de manera segura y eficiente. La empresa 3mdeb está trabajando en portar el código de Coreboot y openSIL a la placa base, lo que permitirá a los usuarios tener una experiencia de firmware completamente open-source.

## ¿Por qué importa?

Esta placa base es importante porque ofrece una experiencia de firmware completamente open-source, lo que permite a los usuarios tener un control total sobre su sistema. Esto es especialmente relevante para los entusiastas de la comunidad de código abierto, que buscan evitar depender de software cerrado y tener una mayor flexibilidad en la personalización de su sistema.
