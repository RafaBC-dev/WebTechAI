# Crea tus gafas de NeoPixel con un efecto de escáner de estilo cyberpunk

## Introducción

¿Quieres crear un proyecto de moda con un toque de ciencia ficción? Con las gafas de NeoPixel, puedes crear un efecto de escáner de estilo cyberpunk que hará que te sientas como un personaje de una película de acción. En este artículo, te mostraré cómo hacerlo.

## ¿Qué es?

Las gafas de NeoPixel son un proyecto de moda que utiliza una cinta de LED NeoPixel para crear un efecto de escáner de estilo cyberpunk. Se basa en el concepto de escáner de Glen Larson, que se utilizó en la serie de televisión Knight Rider y Battlestar Galactica. Ahora, puedes personalizar el color y el efecto de la cinta de LED para crear un diseño único.

## ¿Cómo funciona?

Para crear las gafas de NeoPixel, necesitarás un Trinket, una cinta de LED NeoPixel, un soldador y algunos materiales de conexión. La cinta de LED NeoPixel se conecta al Trinket, que es un microcontrolador pequeño que puede controlar la cinta de LED. Puedes programar el Trinket para que cree diferentes efectos de escáner utilizando el software de Adafruit.

## ¿Por qué importa?

Las gafas de NeoPixel son importantes porque permiten a los entusiastas de la tecnología crear un proyecto de moda con un toque de ciencia ficción. Puedes personalizar el color y el efecto de la cinta de LED para crear un diseño único que refleje tu personalidad. Además, este proyecto es una excelente manera de aprender sobre la electrónica y la programación.

## Consejo técnico

Si estás interesado en crear tus propias gafas de NeoPixel, asegúrate de elegir una cinta de LED NeoPixel que sea compatible con el Trinket. También es importante seguir las instrucciones del software de Adafruit para programar el Trinket correctamente.
