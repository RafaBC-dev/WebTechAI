## Introducción

La empresa Beelink ha lanzado un nuevo mini PC llamado EQi 304, que cuenta con un procesador Intel Core 3 304 y 16 GB de memoria RAM. En este artículo, se analiza la instalación y configuración de Ubuntu 26.04 en este dispositivo.

## ¿Qué es?

El Beelink EQi 304 es un mini PC que se basa en el procesador Intel Core 3 304, un chip de cinco núcleos que ofrece una gran potencia de cálculo. Este dispositivo también cuenta con 16 GB de memoria RAM y una tarjeta gráfica Intel Wildcat Lake.

## ¿Cómo funciona?

La instalación de Ubuntu 26.04 en el Beelink EQi 304 se realizó mediante la inserción de una tarjeta NVMe de 256 GB en un socket PCIe x2. El sistema operativo se instaló en la tarjeta NVMe, mientras que la partición de Windows 11 se mantuvo intacta en la memoria UFS interna.

## ¿Por qué importa?

La instalación de Ubuntu 26.04 en el Beelink EQi 304 es importante porque ofrece una alternativa a Windows 11 para los usuarios que buscan una experiencia de usuario más ligera y personalizable. Además, la gran potencia de cálculo del procesador Intel Core 3 304 hace que este dispositivo sea ideal para tareas que requieren una gran cantidad de recursos.

## Consejo técnico

Si deseas instalar Ubuntu 26.04 en tu Beelink EQi 304, asegúrate de tener una tarjeta NVMe de al menos 256 GB para instalar el sistema operativo. También es importante verificar que la partición de Windows 11 no se dañe durante el proceso de instalación.

```bash
sudo inxi -Fc0
```
