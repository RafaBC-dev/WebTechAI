# Controladores Razer y Betop serán compatibles con Linux 7.0

**Fecha:** 2026-04-05
**Categoría:** linux
**Tags:** linux, robotica, codigo
**Título original:** Razer Wolverine V3 Pro & Betop KP50 Controllers To Be Supported By Linux 7.0

---

## Introducción

La próxima versión de Linux, Linux 7.0, traerá soporte para dos nuevos controladores de juego: Razer Wolverine V3 Pro y Betop KP50. Esto permitirá a los usuarios de Linux disfrutar de una experiencia de juego más completa y sin problemas de compatibilidad.

## ¿Qué es?

Los controladores Razer Wolverine V3 Pro y Betop KP50 son dispositivos de entrada para juegos que permiten a los jugadores interactuar con los juegos de manera más precisa y cómoda. El Razer Wolverine V3 Pro utiliza el protocolo Xbox 360 y es compatible con modos de conexión tanto cableados como inalámbricos.

## ¿Cómo funciona?

El soporte para estos controladores se ha agregado a la driver XPad, que es responsable de manejar la entrada de dispositivos de juego en Linux. Para que los controladores funcionen, se necesitan nuevos IDs y algunas configuraciones específicas en el driver XPad.

## ¿Por qué importa?

El soporte para estos controladores es importante porque permite a los usuarios de Linux disfrutar de una experiencia de juego más completa y sin problemas de compatibilidad. Esto es especialmente relevante para los jugadores que utilizan Linux como sistema operativo principal.
