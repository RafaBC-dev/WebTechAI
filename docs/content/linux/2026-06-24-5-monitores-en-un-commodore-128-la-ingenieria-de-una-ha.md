# 5 monitores en un Commodore 128: la ingeniería de una hazaña retro

**Fecha:** 2026-06-24

## Introducción

El ingeniero de retroinformática The 8-Bit Guy ha logrado conectar 5 monitores a un Commodore 128 sin modificar el ordenador en sí, utilizando una combinación de hardware y software creativos.

## ¿Qué es?

El proyecto consiste en conectar 5 monitores a un Commodore 128 utilizando una tarjeta de expansión diseñada por Joe Burks, que permite dividir el señal de color en cuatro monitores y utilizar la salida del VIC-II para el quinto monitor.

## ¿Cómo funciona?

La tarjeta de expansión utiliza la señal de RGBI del Commodore 128 para dividir el color en cuatro monitores, mientras que el quinto monitor se conecta a la salida del VIC-II. El software utilizado es una implementación de BASIC que permite crear imágenes de múltiples colores para cada monitor.

## ¿Por qué importa?

Este proyecto es importante porque muestra la creatividad y la ingeniería que se pueden aplicar a la retroinformática, y también porque demuestra la capacidad de conectar múltiples monitores a un ordenador de la época.

## Consejo técnico

Si deseas intentar conectar múltiples monitores a tu Commodore 128, puedes utilizar la tarjeta de expansión diseñada por Joe Burks y encontrar la implementación de BASIC en GitHub.

```bash
git clone https://github.com/joeburks/rgbimon
```
