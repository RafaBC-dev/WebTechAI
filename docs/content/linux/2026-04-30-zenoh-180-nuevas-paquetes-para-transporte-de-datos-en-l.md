# Zenoh 1.8.0: Nuevas paquetes para transporte de datos en Linux

## Introducción

La Fundación OSR ha lanzado paquetes preestablecidos de Zenoh 1.8.0, una herramienta para transporte de datos en tiempo real. Estos paquetes están disponibles en el repositorio de prerelease de packages.osrfoundation.org y serán promovidos a la versión estable después de una prueba exhaustiva.

## ¿Qué es?

Zenoh es una herramienta de transporte de datos en tiempo real que permite la comunicación entre dispositivos en redes de área amplia. Utiliza un protocolo de comunicación eficiente y escalable para transmitir datos de manera segura y confiable.

## ¿Cómo funciona?

Zenoh utiliza un modelo de publicación y suscripción para permitir que los dispositivos publiquen y suscriban a canales de datos. Los paquetes de Zenoh 1.8.0 están disponibles para Linux y pueden ser utilizados en proyectos de robótica y automatización industrial.

## ¿Por qué importa?

Zenoh 1.8.0 es relevante para desarrolladores de software que necesitan implementar transporte de datos en tiempo real en sus proyectos. La herramienta es escalable y segura, lo que la hace ideal para aplicaciones de robótica y automatización industrial.

## Consejo técnico

Si estás trabajando con Gazebo y necesitas implementar transporte de datos en tiempo real, considera utilizar los paquetes de Zenoh 1.8.0. Puedes encontrar más información sobre la herramienta y cómo utilizarla en el repositorio de prerelease de packages.osrfoundation.org.

```bash
sudo apt-get install libzenohc1.8.0
```
