# ESP32-E22: módulo Wi-Fi 6E y Bluetooth 5.4 con certificación Wi-Fi CERTIFIED

**Fecha:** 2026-06-14
**Categoría:** linux
**Tags:** esp32, robotica, linux

## Introducción

Espressif ha obtenido la certificación Wi-Fi CERTIFIED para su módulo ESP32-E22, que cuenta con Wi-Fi 6E y Bluetooth 5.4. Además, ha lanzado drivers de Linux para el chip.

## ¿Qué es?

El ESP32-E22 es un módulo de Wi-Fi 6E y Bluetooth 5.4 que cuenta con un procesador dual-core RISC-V de hasta 500 MHz, 1 MB de RAM y 41 pines GPIO. Está diseñado para sistemas de comunicaciones inalámbricas con Wi-Fi 6E y Bluetooth 5.4 a través de interfaces PCIe 2.0 o SDIO.

## ¿Cómo funciona?

El módulo ESP32-E22 funciona en modo estación con 2x2 MIMO, hasta 40 MHz de ancho de canal en 2,4 GHz, 160 MHz en 5 GHz y 6 GHz. El chip ha sido probado en Linux, pero también se espera que funcione en Windows. Los drivers de Linux están disponibles en GitHub y permiten la conexión a Wi-Fi y Bluetooth.

## ¿Por qué importa?

La certificación Wi-Fi CERTIFIED garantiza que el módulo ESP32-E22 cumple con los estándares de Wi-Fi y Bluetooth. Esto es importante para sistemas de comunicaciones inalámbricas que requieren una conexión estable y segura. Además, los drivers de Linux permiten a los desarrolladores crear aplicaciones y proyectos que utilicen el módulo ESP32-E22.

## Consejo técnico

Si estás desarrollando un proyecto que requiere Wi-Fi 6E y Bluetooth 5.4, considera utilizar el módulo ESP32-E22 y los drivers de Linux disponibles en GitHub. Esto te permitirá crear una conexión estable y segura para tu proyecto.

```bash
git clone https://github.com/espressif/esp32-e22-linux-driver.git
```
