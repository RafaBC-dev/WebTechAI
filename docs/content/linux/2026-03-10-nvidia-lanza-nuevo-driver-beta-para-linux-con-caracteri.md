# NVIDIA lanza nuevo driver beta para Linux con características Vulkan

**Fecha:** 2026-03-10

**Tags:** Vulkan, Linux, NVIDIA, driver beta

## Introducción

NVIDIA ha lanzado un nuevo driver beta para Linux que incluye características Vulkan importantes. Este driver es parte del programa de beta de desarrolladores de Vulkan y ofrece mejoras significativas en rendimiento y funcionalidad. La noticia es relevante para desarrolladores de aplicaciones que utilizan Vulkan en Linux.

## ¿Qué es?

El driver beta 595.44.02 de NVIDIA es una versión experimental del driver para Linux que incluye características Vulkan nuevas. Vulkan es una API de gráficos que permite a los desarrolladores crear aplicaciones gráficas de alta calidad en Linux. El driver beta es una versión en desarrollo que se utiliza para probar y mejorar las características Vulkan antes de que sean lanzadas en versiones estables.

## ¿Cómo funciona?

El driver beta 595.44.02 de NVIDIA incluye varias características nuevas, como el soporte para descriptorHeapCaptureReplay, que permite a los desarrolladores capturar y reproducir secuencias de captura de pantalla de aplicaciones Vulkan. También incluye soporte para compresión de imágenes en formatos multi-planares YCbCr y exportación de DMA-BUF con modificadores de formato DRM. Estas características mejoran la funcionalidad y el rendimiento de las aplicaciones Vulkan en Linux.

## ¿Por qué importa?

La noticia es importante para desarrolladores de aplicaciones que utilizan Vulkan en Linux, ya que ofrece mejoras significativas en rendimiento y funcionalidad. Las características nuevas incluidas en el driver beta permiten a los desarrolladores crear aplicaciones más complejas y eficientes. Además, el soporte para compresión de imágenes y exportación de DMA-BUF mejora la compatibilidad con hardware y software existentes.
