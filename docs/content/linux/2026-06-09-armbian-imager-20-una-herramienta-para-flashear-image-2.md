# Armbian Imager 2.0: una herramienta para flashear imágenes en más de 300 tableros

**Fecha:** 2026-06-09
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Armbian Imager 2.0 release supports over 300 boards from 64 SBC vendors, custom user profiles

---

## Introducción

La comunidad de Armbian ha lanzado la versión 2.0 de su herramienta de flasheo, Armbian Imager, que permite fácilmente instalar imágenes pre-construidas de Ubuntu o Debian en más de 338 tableros de 64 fabricantes diferentes. Esta herramienta es similar a la Raspberry Pi Imager 2.0, pero cubre un ecosistema más amplio de ordenadores de sobremesa

## ¿Qué es?

Armbian Imager 2.0 es una herramienta de línea de comandos y GUI que permite flashear imágenes pre-construidas de Ubuntu o Debian en más de 338 tableros de 64 fabricantes diferentes. Esta herramienta es gratuita y disponible para Linux, Windows y macOS

## ¿Cómo funciona?

Para usar Armbian Imager 2.0, el usuario debe seleccionar el fabricante del tablero, luego el modelo del tablero, y finalmente la imagen de sistema que desea instalar. La herramienta automáticamente descarga y descomprime la imagen y la escribe en la tarjeta SD. La herramienta también permite crear perfiles de usuario personalizados para cada tablero

## ¿Por qué importa?

Armbian Imager 2.0 es importante porque permite a los usuarios instalar sistemas operativos en una variedad de tableros de manera fácil y rápida. Esto es especialmente útil para desarrolladores de software y hardware que necesitan probar sus aplicaciones en diferentes plataformas

## Consejo técnico

Si estás usando Armbian Imager 2.0, asegúrate de seleccionar la imagen de sistema correcta para tu tablero y de crear un perfil de usuario personalizado para cada tablero. Esto te permitirá personalizar la configuración de cada tablero según tus necesidades
