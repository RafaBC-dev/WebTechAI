# Soporte adicional para Razer en Linux con OpenRazer 3.12.1

**Fecha:** 2026-04-04
**Categoría:** linux
**Tags:** linux, robotica, herramientas
**Título original:** OpenRazer 3.12.1 Enables Two More Razer Devices Under Linux

---

## Introducción

La comunidad de Linux ha recibido un regalo con la actualización de OpenRazer 3.12.1, que agrega soporte para dos dispositivos Razer más. Esto significa que los usuarios pueden disfrutar de una experiencia más completa con sus productos Razer en Linux.

## ¿Qué es?

OpenRazer es un conjunto de drivers open-source para hardware Razer en Linux. Su objetivo es proporcionar una experiencia más fluida y completa para los usuarios de Linux que utilizan productos Razer. Con la actualización de OpenRazer 3.12.1, se han agregado dos nuevos dispositivos compatibles.

## ¿Cómo funciona?

OpenRazer utiliza una arquitectura modular para proporcionar soporte a diferentes dispositivos Razer. La actualización de OpenRazer 3.12.1 incluye cambios en el código fuente para agregar soporte a la Razer Basilisk Mobile y la Lian Li O11 Dynamic - Razer Edition. Estos cambios permiten a los usuarios utilizar estos dispositivos en Linux de manera nativa.

## ¿Por qué importa?

El soporte adicional para Razer en Linux es importante porque permite a los usuarios disfrutar de una experiencia más completa con sus productos. Esto incluye la capacidad de personalizar la iluminación, controlar el dispositivo y acceder a características avanzadas. Además, la comunidad de Linux puede contribuir a mejorar y expandir el soporte para Razer.

## Consejo técnico

Si deseas agregar soporte a tu Razer Basilisk Mobile en Linux, asegúrate de instalar la versión más reciente de OpenRazer y configurar el dispositivo según las instrucciones proporcionadas en la documentación de OpenRazer.

```bash
sudo apt-get install openrazer
```
