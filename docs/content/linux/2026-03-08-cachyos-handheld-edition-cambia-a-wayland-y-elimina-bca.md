# CachyOS Handheld Edition cambia a Wayland y elimina Bcachefs

**Fecha:** 2026-03-08
**Categoría:** linux
**Tags:** Linux, CachyOS, Wayland, Bcachefs, Steam Deck
**Título original:** CachyOS Handheld Edition Switches To Wayland, CachyOS Installer Drops Bcachefs

---

## Introducción

La versión de marzo de 2026 de CachyOS Handheld Edition, una distribución de Linux para dispositivos de juego portátiles como el Steam Deck, ha sido actualizada con varias mejoras y cambios importantes. Entre ellos, el cambio a Wayland como gestor de ventana en lugar de X11, lo que debería mejorar la experiencia de usuario. Además, se han eliminado algunas dependencias y se han agregado nuevas características para facilitar la instalación y configuración del sistema.

## ¿Qué es?

CachyOS es una distribución de Linux basada en Arch Linux diseñada para dispositivos de juego portátiles como el Steam Deck y el Legion Go. Ofrece una experiencia de usuario personalizada y optimizada para juegos y aplicaciones de entretenimiento. La versión Handheld Edition es específicamente diseñada para estos dispositivos y ofrece características y mejoras para mejorar la experiencia de juego y la productividad.

## ¿Cómo funciona?

La versión de marzo de 2026 de CachyOS Handheld Edition utiliza Wayland como gestor de ventana en lugar de X11, lo que debería mejorar la estabilidad y la eficiencia del sistema. Además, se ha eliminado Bcachefs, un sistema de archivos que dependía de un módulo de kernel personalizado. En su lugar, se utiliza un sistema de archivos más tradicional. La instalación también ofrece una nueva interfaz de usuario con animaciones y soporte para imágenes en formato JPEG-XL.

## ¿Por qué importa?

Estos cambios y mejoras son importantes porque mejoran la experiencia de usuario y la estabilidad del sistema. La transición a Wayland debería proporcionar una experiencia más fluida y eficiente, mientras que la eliminación de Bcachefs simplifica la instalación y configuración del sistema. Además, la nueva interfaz de usuario ofrece una experiencia más atractiva y fácil de usar.

## Consejo técnico

Si estás utilizando CachyOS Handheld Edition, te recomendamos actualizar a la versión de marzo de 2026 para aprovechar las mejoras y cambios mencionados. También es recomendable explorar la nueva interfaz de usuario y configurar el sistema para aprovechar al máximo sus características.

```bash
sudo pacman -Syyu && sudo pacman -S cachyos-handheld-edition
```
