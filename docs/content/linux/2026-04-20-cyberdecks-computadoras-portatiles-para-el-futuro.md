# Cyberdecks: computadoras portátiles para el futuro

## Introducción

La comunidad de makers ha visto un auge en la creación de cyberdecks, computadoras portátiles inspiradas en la ciencia ficción. Estos dispositivos están siendo construidos con Raspberry Pi y otros componentes, ofreciendo una forma innovadora de interactuar con la tecnología.

## ¿Qué es?

Un cyberdeck es una computadora portátil que combina una pantalla, un teclado y un sistema operativo en un solo dispositivo. Pueden ser construidos con componentes reutilizados y personalizados, ofreciendo una forma única de experimentar con la tecnología.

## ¿Cómo funciona?

Los cyberdecks suelen ser construidos con Raspberry Pi como sistema operativo, lo que les permite ejecutar aplicaciones y juegos en un entorno de código abierto. Pueden ser conectados a una pantalla, un teclado y otros dispositivos para ofrecer una experiencia de usuario completa.

## ¿Por qué importa?

Los cyberdecks importan porque ofrecen una forma innovadora de interactuar con la tecnología y pueden ser utilizados para crear experiencias de usuario únicas y personalizadas. También pueden ser utilizados para aprender sobre programación y electrónica.

## Consejo técnico

Si deseas construir tu propio cyberdeck, considera utilizar una Raspberry Pi como sistema operativo y conectarlo a una pantalla y un teclado. Puedes utilizar herramientas como Manjaro Linux y ytfzf para crear una experiencia de usuario personalizada.

```bash
sudo apt-get install manjaro-linux ytfzf
```
