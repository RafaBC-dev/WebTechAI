# Calculadora 3D impresionada con Raspberry Pi Pico

## Introducción

La Raspberry Pi Official Magazine ha lanzado un nuevo número con un artículo destacado: una calculadora 3D impresionada con tecnología mecánica y un Raspberry Pi Pico como cerebro.

## ¿Qué es?

La calculadora, llamada Flapulator, es un dispositivo físico que utiliza un display mecánico 3D impresionado y un Raspberry Pi Pico como procesador. El dispositivo cuenta con teclas mecánicas Gateron KS-33 y un botón dedicado para el sistema operativo Raspberry Pi.

## ¿Cómo funciona?

El Flapulator utiliza un Raspberry Pi Pico como procesador, que se encarga de realizar las operaciones matemáticas y mostrar los resultados en el display mecánico. Las teclas mecánicas Gateron KS-33 permiten al usuario ingresar números y realizar cálculos de manera táctil. El dispositivo también cuenta con funciones trigonométricas que solo funcionan en radianes.

## ¿Por qué importa?

La Flapulator es importante porque ofrece una experiencia de cálculo física y táctil que no se puede replicar con dispositivos digitales. El dispositivo también demuestra la capacidad del Raspberry Pi Pico para ser utilizado en proyectos de hardware y electrónica.

## Consejo técnico

Si deseas crear un proyecto similar, puedes utilizar la biblioteca de software de Raspberry Pi para programar el Raspberry Pi Pico y crear un dispositivo personalizado. Puedes empezar investigando sobre la biblioteca de software de Raspberry Pi y sus recursos para desarrolladores.
