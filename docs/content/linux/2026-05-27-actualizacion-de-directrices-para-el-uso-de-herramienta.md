## Introducción

La comunidad de desarrolladores de Python ha actualizado sus directrices para el uso de herramientas de inteligencia artificial al contribuir al código fuente de CPython. Esta actualización es importante para todos los contribuyentes, ya sean experimentados o nuevos.

## ¿Qué es?

CPython es la implementación de referencia de la lengua de programación Python. Es un proyecto de código abierto que ofrece una plataforma para desarrollar aplicaciones de todo tipo. Las herramientas de inteligencia artificial son tecnologías que permiten a los sistemas tomar decisiones y realizar tareas de manera autónoma.

## ¿Cómo funciona?

Las herramientas de IA se integran en el código de CPython para mejorar la eficiencia y la precisión de las tareas de programación. Sin embargo, es importante recordar que los desarrolladores son responsables de lo que contribuyen, incluyendo el uso de herramientas de IA.

## ¿Por qué importa?

Esta actualización de directrices es importante porque refuerza la responsabilidad de los desarrolladores al utilizar herramientas de IA. Esto significa que deben ser conscientes de cómo funcionan y cómo pueden afectar el código que contribuyen.

## Consejo técnico

Si estás contribuyendo al código de CPython, asegúrate de revisar las directrices actualizadas sobre el uso de herramientas de IA. Puedes encontrar más información en el Python Developer's Guide.
