# Agama 19: Una Nueva Arquitectura para el Instalador de openSUSE y SUSE

## Introducción

La versión 19 del instalador Agama para openSUSE y SUSE ha sido lanzada, con cambios significativos en su diseño arquitectónico y organización de la interfaz web. Estos cambios permiten una mayor simplicidad y consistencia en la programación del instalador.

## ¿Qué es?

Agama es un instalador para sistemas operativos basados en Linux, específicamente diseñado para openSUSE y SUSE. Su objetivo es proporcionar una experiencia de instalación sencilla y personalizable.

## ¿Cómo funciona?

La versión 19 de Agama se basa en una arquitectura renovada, que define un API (Interfaz de Programación de Aplicaciones) para controlar el core del instalador. Esto permite una mayor consistencia y simplicidad en la programación de la interfaz web, herramientas de línea de comandos y la instalación no interactiva.

## ¿Por qué importa?

Esta versión de Agama es importante porque proporciona una base sólida para futuras mejoras y desarrollo. También permite a los usuarios y desarrolladores aprovechar al máximo las características de openSUSE y SUSE.

## Consejo técnico

Si deseas aprovechar las nuevas características de Agama 19, puedes comenzar investigando sobre la documentación oficial de la API de Agama y cómo se puede utilizar para personalizar la experiencia de instalación.
