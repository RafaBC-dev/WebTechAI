# Calculadora 3D impresora con Raspberry Pi Pico

## Introducción

Una calculadora física con pantalla mecánica 3D impresora, alimentada por un Raspberry Pi Pico, ha sido creada por un científico informático. Esta innovadora calculadora, llamada Flapulator, combina la precisión de un cálculo digital con la experiencia táctil de un dispositivo físico.

## ¿Qué es?

El Flapulator es una calculadora que utiliza una pantalla mecánica 3D impresora, alimentada por un Raspberry Pi Pico, y cuenta con teclas Gateron KS-33 de bajo perfil. Esta calculadora está diseñada para realizar cálculos rápidos y simples, y su pantalla mecánica proporciona una experiencia visual única.

## ¿Cómo funciona?

El Flapulator utiliza un Raspberry Pi Pico como procesador, que se conecta a la pantalla mecánica 3D impresora y a las teclas Gateron KS-33. El dispositivo utiliza un software específico para realizar cálculos y mostrar los resultados en la pantalla mecánica.

## ¿Por qué importa?

El Flapulator es importante porque ofrece una experiencia de cálculo única que combina la precisión de un cálculo digital con la experiencia táctil de un dispositivo físico. Esto puede ser especialmente útil para aquellos que necesitan realizar cálculos rápidos y simples en entornos de trabajo o estudio.

## Consejo técnico

Si deseas crear una calculadora similar, puedes utilizar el software de Raspberry Pi Pico y la biblioteca de teclas Gateron KS-33 para crear un dispositivo personalizado. Recuerda que debes diseñar cuidadosamente la pantalla mecánica y las teclas para asegurarte de que funcionen correctamente.

```bash
sudo apt-get install raspbian-pico
```
