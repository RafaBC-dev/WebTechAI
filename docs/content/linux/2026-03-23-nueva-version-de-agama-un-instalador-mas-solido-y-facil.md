# Nueva versión de Agama: un instalador más sólido y fácil de usar

**Fecha:** 2026-03-23
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Agama 19 released

## Introducción

La versión 19 de Agama, un instalador para openSUSE y SUSE, ha sido lanzada con cambios significativos en su diseño arquitectónico y la organización de la interfaz web. Esto permitirá una mayor flexibilidad y personalización en la instalación de sistemas operativos.

## ¿Qué es?

Agama es un instalador de sistemas operativos diseñado para openSUSE y SUSE. Su objetivo es proporcionar una experiencia de instalación sencilla y personalizable, permitiendo a los usuarios configurar sus sistemas de manera flexible.

## ¿Cómo funciona?

La versión 19 de Agama introduce un nuevo diseño arquitectónico que permite controlar el core del instalador a través de una interfaz de programación de aplicaciones (API) simple y consistente. Esto permite que la interfaz web, las herramientas de línea de comandos y la instalación automática se construyan sobre esta API genérica.

## ¿Por qué importa?

Esta versión de Agama es importante porque resuelve problemas de compatibilidad y flexibilidad en la instalación de sistemas operativos. Ahora, los usuarios pueden personalizar su instalación de manera más fácil y sencilla, lo que facilita la configuración de sus sistemas.

## Consejo técnico

Si deseas aprovechar al máximo la nueva versión de Agama, revisa la documentación oficial para aprender a utilizar la API genérica y personalizar tu instalación de acuerdo con tus necesidades específicas.
