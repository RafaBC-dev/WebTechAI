## Introducción

El lenguaje de programación Python sigue siendo el más popular del mundo, según el Índice TIOBE. Para aprender a programar, nada mejor que crear algo con tus propias manos. En este artículo, aprenderás a construir un juego de preguntas y respuestas sencillo utilizando Python y el entorno de desarrollo Thonny.

## ¿Qué es?

Un juego de preguntas y respuestas es un programa que te pide preguntas y te da respuestas basadas en tus ingresos. En este caso, se trata de un juego de adivinanza que intenta adivinar el año en el que se escribió un determinado evento histórico.

## ¿Cómo funciona?

El juego utiliza la función input() de Python para pedir preguntas al usuario y almacenar sus respuestas en variables. Luego, utiliza la función print() para mostrar las respuestas en la consola. El juego se puede extender fácilmente para agregar más preguntas y respuestas.

## ¿Por qué importa?

Este juego es importante porque te enseña a crear programas sencillos utilizando Python y el entorno de desarrollo Thonny. También te enseña a utilizar variables y funciones para almacenar y procesar datos.

## Consejo técnico
