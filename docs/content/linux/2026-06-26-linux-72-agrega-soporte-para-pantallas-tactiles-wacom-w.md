# Linux 7.2 agrega soporte para pantallas táctiles Wacom W9000

**Fecha:** 2026-06-26
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Linux 7.2 Adds New Driver For Wacom W9000 Pen-Enabled Touchscreens

---

## Introducción

La versión 7.2 de Linux ha agregado un nuevo driver para pantallas táctiles Wacom W9000, lo que permitirá a los usuarios aprovechar al máximo estas pantallas en sus dispositivos Linux. Esta actualización es especialmente relevante para aquellos que utilizan pantallas táctiles en sus proyectos de desarrollo de hardware o software.

## ¿Qué es?

El driver Wacom W9000 es una actualización para el sistema de entrada de Linux que permite la compatibilidad con pantallas táctiles Wacom W9000. Estas pantallas son capaces de reconocer gestos y escritura con lápices, lo que las hace ideales para aplicaciones de diseño gráfico, educación y entretenimiento.

## ¿Cómo funciona?

El driver Wacom W9000 se conecta a la pantalla táctil a través de I2C y utiliza un lápiz pasivo Wacom para reconocer los gestos y escritura. El driver está diseñado para ser compatible con varias pantallas táctiles Wacom, incluyendo el W9002 y el W9007A, que se utilizan en dispositivos como la Microsoft Surface Pro.

## ¿Por qué importa?

La adición de soporte para pantallas táctiles Wacom W9000 en Linux 7.2 es importante porque permite a los desarrolladores aprovechar al máximo estas pantallas en sus proyectos. Esto puede ser especialmente útil para aquellos que trabajan en proyectos de desarrollo de hardware o software que requieren la interacción con pantallas táctiles.

## Consejo técnico

Si deseas probar el driver Wacom W9000 en tu dispositivo Linux, asegúrate de verificar que tu pantalla táctil esté compatible y que tengas un lápiz pasivo Wacom. Puedes verificar la compatibilidad y obtener más información sobre el driver en el pull request de Linux 7.2.

```bash
git pull --no-edit https://git.kernel.org/pub/scm/linux/kernel/git/torvalds/linux.git refs/heads/linux-7.2
```
