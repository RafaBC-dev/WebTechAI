# Toradex Zinnia: un gateway IoT industrial con conectividad y simplificación de software

**Fecha:** 2026-05-06
**Categoría:** linux
**Tags:** robotica, ia-local, linux
**Título original:** Toradex Zinnia Linux IoT Gateway offers dual GbE, WiFi 5, 4G LTE, I/Os, and simplify software deployment

---

## Introducción

Toradex ha lanzado el Zinnia, un gateway IoT industrial que ofrece conectividad dual GbE, WiFi 5, 4G LTE y simplificación de software para aplicaciones de edge AI y automatización industrial.

## ¿Qué es?

El Toradex Zinnia es un gateway IoT industrial basado en el sistema en módulo Verdin de la empresa, que se puede configurar con diferentes SoC de NXP o Texas Instruments. Ofrece conectividad dual GbE, WiFi 5, Bluetooth 5 y conectividad 4G LTE o 5G opcional. También cuenta con puertos USB, conectores I/O y una amplia gama de voltaje de alimentación.

## ¿Cómo funciona?

El Zinnia se basa en el SoC NXP i.MX 8M Plus, que cuenta con un procesador quad-core Cortex-A53 a 1.6/1.8 GHz y un core de tiempo real Cortex-M7 a 800 MHz. También cuenta con un acelerador de inteligencia artificial de 2.3 TOPS y un procesador de vídeo que puede codificar y decodificar vídeo H.264 y H.265 a 1080p60.

## ¿Por qué importa?

El Toradex Zinnia importa porque ofrece una solución de gateway IoT industrial que simplifica la implementación de aplicaciones de edge AI y automatización industrial. También cuenta con conectividad dual GbE y WiFi 5, lo que lo hace ideal para aplicaciones que requieren alta velocidad y conectividad.

## Consejo técnico

Si estás implementando un proyecto de edge AI o automatización industrial, considera utilizar el Toradex Zinnia como gateway IoT industrial para simplificar la implementación y mejorar la conectividad.
