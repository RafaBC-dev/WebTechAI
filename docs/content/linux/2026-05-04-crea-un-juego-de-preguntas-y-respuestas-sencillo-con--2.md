## Introducción

El lenguaje de programación Python sigue siendo el más popular del mundo, según el índice TIOBE. Una de las razones es que sus programas son fáciles de leer y entender, incluso para principiantes. En este artículo, aprenderás a crear un juego de preguntas y respuestas sencillo utilizando Python y el entorno de desarrollo Thonny en Raspberry Pi.

## ¿Qué es?

Un juego de preguntas y respuestas es un programa que muestra una pregunta al usuario y espera a que responda. En este caso, se utilizará Python para crear un juego de preguntas y respuestas sencillo que pregunte al usuario sobre un año específico.

## ¿Cómo funciona?

El juego utiliza la función `input()` para pedir al usuario que responda una pregunta. La respuesta se almacena en una variable llamada `guess`. Luego, se utiliza la función `print()` para mostrar la respuesta correcta al usuario. El juego también utiliza la función `int()` para convertir la respuesta del usuario a un número entero.

## ¿Por qué importa?

Este juego es importante porque te permite practicar la programación en Python y aprender a crear programas sencillos. También te enseña a utilizar las funciones `input()` y `print()` para interactuar con el usuario.

## Consejo técnico

Si quieres crear un juego de preguntas y respuestas más complejo, puedes agregar más preguntas y respuestas a tu programa utilizando la función `input()` y la variable `guess`. También puedes utilizar la función `random()` para generar preguntas y respuestas aleatorias.
