## Introducción

Waveshare ha lanzado una nueva placa de expansión para Raspberry Pi que ofrece una amplia gama de características, incluyendo conectividad 4G/5G, Ethernet dual, y más. Esta placa es ideal para proyectos de IoT y automatización.

## ¿Qué es?

La placa de expansión CM5-ETH-RS485-4G-BASE es una tarjeta de expansión diseñada para trabajar con el módulo de computadora Raspberry Pi CM5. Ofrece una variedad de características, incluyendo conectividad 4G/5G, Ethernet dual, y más.

## ¿Cómo funciona?

La placa de expansión utiliza el módulo de computadora Raspberry Pi CM5 como procesador central. La conectividad 4G/5G se proporciona mediante un módulo M.2 B-Key y una tarjeta SIM. La Ethernet dual se proporciona mediante dos puertos RJ45, uno de los cuales es de 2,5 GbE.

## ¿Por qué importa?

Esta placa de expansión es importante porque ofrece una amplia gama de características que pueden ser utilizadas en proyectos de IoT y automatización. La conectividad 4G/5G permite la comunicación remota y la supervisión en tiempo real, mientras que la Ethernet dual proporciona una conexión estable y rápida a la red.

## Consejo técnico

Si deseas utilizar esta placa de expansión con conectividad 4G/5G, asegúrate de utilizar un módulo M.2 B-Key compatible y una tarjeta SIM de buena calidad.
