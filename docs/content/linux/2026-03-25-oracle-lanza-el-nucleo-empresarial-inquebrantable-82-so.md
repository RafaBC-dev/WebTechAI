# Oracle lanza el núcleo empresarial inquebrantable 8.2 sobre Linux 6.12 LTS

**Fecha:** 2026-03-25
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Oracle Releases Unbreakable Enterprise Kernel 8.2 - Still On Linux 6.12 LTS Base

---

## Introducción

Oracle ha lanzado el Unbreakable Enterprise Kernel 8.2 (UEK 8.2) para su distribución Oracle Linux. Este núcleo empresarial es una alternativa al kernel RHEL-clone y ofrece características adicionales y hardening. El UEK 8.2 sigue utilizando una base de Linux 6.12 LTS, pero se espera que se actualice a la rama Linux 6.18 LTS en la versión UEK9.

## ¿Qué es?

El Unbreakable Enterprise Kernel (UEK) es el núcleo Linux principal de Oracle Linux, diseñado para ofrecer seguridad y estabilidad en entornos empresariales. El UEK 8.2 es una actualización del núcleo empresarial inquebrantable de Oracle, que incluye características adicionales y mejoras en seguridad y rendimiento.

## ¿Cómo funciona?

El UEK 8.2 se construye sobre una base de Linux 6.12 LTS y aprovecha las últimas mejoras de la rama upstream. Incorpora parches de Intel Trust Domain Extensions (TDX) para mejorar la computación confidencial, así como actualizaciones de drivers para mejorar el rendimiento y la estabilidad. El núcleo también incluye mejoras en la protección de memoria y diagnóstico.

## ¿Por qué importa?

El UEK 8.2 es importante porque ofrece una alternativa segura y estable para los entornos empresariales de Oracle Linux. La actualización de drivers y la mejora en la computación confidencial permiten a los usuarios aprovechar al máximo sus sistemas y proteger sus datos. Además, el núcleo empresarial inquebrantable de Oracle sigue siendo una opción viable para aquellos que buscan una distribución Linux segura y confiable.

## Consejo técnico

Para aprovechar al máximo el UEK 8.2, los usuarios deben actualizar sus sistemas a la versión más reciente de Oracle Linux y verificar que sus drivers estén actualizados. También es recomendable verificar la documentación de Oracle para obtener más información sobre las características y mejoras del núcleo empresarial inquebrantable.
