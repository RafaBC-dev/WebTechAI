## Introducción

La programación es un mundo fascinante que puede ser accesible a todos, especialmente con lenguajes como Python que ofrecen una sintaxis clara y fácil de leer. En este artículo, aprenderás a crear un juego de trivia sencillo utilizando Python y el entorno de desarrollo Thonny en Raspberry Pi.

## ¿Qué es?

Un juego de trivia es un tipo de programa que presenta preguntas y respuestas a los usuarios, con la finalidad de evaluar su conocimiento sobre un tema específico. En este caso, se trata de un juego de trivia que pregunta sobre la fecha de un evento histórico.

## ¿Cómo funciona?

El juego utiliza la función `input()` de Python para solicitar al usuario que ingrese su respuesta, y la función `print()` para mostrar la pregunta y la respuesta correcta. El programa también utiliza variables para almacenar la respuesta del usuario y la pregunta correcta.

## ¿Por qué importa?

Este juego de trivia es importante porque permite a los usuarios practicar su conocimiento sobre un tema específico de manera divertida y interactiva. Además, es un ejemplo práctico de cómo utilizar Python para crear programas sencillos y útiles.

## Consejo técnico

Si deseas crear un juego de trivia más complejo, puedes utilizar la función `random()` de Python para generar preguntas y respuestas aleatorias. Por ejemplo, puedes crear un juego que pregunte sobre diferentes temas y utilice una base de datos para almacenar las preguntas y respuestas.

```bash
import random
```
