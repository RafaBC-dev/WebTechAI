## ¿Qué es?

Un juego de preguntas y respuestas es un programa que te pide respuestas a preguntas y te muestra el resultado. En este caso, creamos un juego de preguntas y respuestas con Python que te permite crear un juego de adivinanza de años.

## ¿Cómo funciona?

Para crear el juego, utilizamos el entorno de desarrollo Thonny, que viene incluido en la distribución de Raspberry Pi OS. El juego utiliza la función input() para pedir respuestas al usuario y la función print() para mostrar el resultado. También utilizamos variables para almacenar la respuesta del usuario y realizar comparaciones.

## ¿Por qué importa?

Este juego de preguntas y respuestas es importante porque te permite practicar la programación y aprender a crear programas interactivos. También te muestra cómo utilizar variables y funciones en Python.
