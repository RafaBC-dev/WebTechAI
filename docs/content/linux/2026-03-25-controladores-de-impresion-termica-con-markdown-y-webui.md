## Introducción

¿Tienes un controlador de impresión térmica que se ha quedado sin uso? ThermalMarky es aquí para ayudarte a darle una segunda vida con soporte básico para Markdown y una interfaz web intuitiva.

## ¿Qué es?

ThermalMarky es un proyecto que permite controlar impresoras térmicas mediante un lenguaje de marcado llamado Markdown. Esto significa que puedes crear documentos y textos con formato utilizando un lenguaje sencillo y fácil de leer. Además, ThermalMarky incluye una interfaz web básica para editar y visualizar tus documentos.

## ¿Cómo funciona?

ThermalMarky soporta Markdown básico, incluyendo encabezados, texto en negrita y subrayado, y listas. También permite la creación de formatos personalizados con etiquetas como [align=center] para alinear texto y [hr] para crear líneas horizontales. La interfaz web incluye un editor con atajos para facilitar la edición de documentos.

## ¿Por qué importa?

ThermalMarky es importante porque ofrece una forma sencilla de controlar impresoras térmicas y crear documentos con formato. Esto puede ser útil para proyectos de automatización, impresión de etiquetas y etiquetas, y otros escenarios donde se requiere una impresión de alta calidad.

## Consejo técnico

Si deseas aprovechar al máximo ThermalMarky, prueba a utilizar la función de CLI para imprimir documentos directamente desde la terminal o mediante un flujo de contenido. Esto te permitirá integrar ThermalMarky en tus scripts y aplicaciones.

```bash
thermalmarky <archivo>
```
