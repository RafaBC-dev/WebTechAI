# Opera GX ya está disponible en Flatpak y Snap para Linux

**Fecha:** 2026-04-24
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Opera GX Browser Gets Flatpak'ed & Snap'ed On Linux

## Introducción

La popular aplicación de navegación Opera GX ha llegado a Linux en formatos de paquete Flatpak y Snap, ofreciendo una opción más para los usuarios de este sistema operativo.

## ¿Qué es?

Opera GX es una versión de la aplicación de navegación Opera diseñada específicamente para gamers, con características como un tema oscuro y un control de recursos para minimizar la sobrecarga del sistema.

## ¿Cómo funciona?

La versión de Flatpak de Opera GX está disponible en Flathub, mientras que la versión de Snap se puede encontrar en el Snap Store. Ambas versiones ofrecen una forma de instalar y ejecutar la aplicación de manera sandboxed, lo que significa que se ejecutan en un entorno aislado del sistema.

## ¿Por qué importa?

La disponibilidad de Opera GX en Flatpak y Snap es importante para los usuarios de Linux que buscan una aplicación de navegación que se adapte a sus necesidades de gaming, ya que ofrece una forma más fácil de instalar y ejecutar la aplicación en su sistema operativo.

## Consejo técnico

Si deseas instalar Opera GX en tu sistema Linux, puedes buscar la versión de Flatpak en Flathub o la versión de Snap en el Snap Store, y seguir las instrucciones para instalar y ejecutar la aplicación.

```bash
flatpak install flathub com.opera.gx
```
