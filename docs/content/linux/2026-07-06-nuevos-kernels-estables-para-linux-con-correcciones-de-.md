# Nuevos kernels estables para Linux con correcciones de seguridad

**Fecha:** 2026-07-06
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Seven stable kernels for Saturday including two security fixes

---

## Introducción

Greg Kroah-Hartman ha anunciado la liberación de siete kernels estables para Linux, incluyendo correcciones importantes para vulnerabilidades de seguridad. Estos kernels son una actualización esencial para garantizar la seguridad de los sistemas Linux.

## ¿Qué es?

Los kernels estables son versiones de código de Linux que han sido probadas y aprobadas para su uso en producción. Son versiones estables y bien probadas de los kernels de Linux, que se utilizan en la mayoría de los sistemas Linux.

## ¿Cómo funciona?

Los kernels estables se construyen a partir de la versión principal de Linux y se prueban exhaustivamente para asegurarse de que no contengan errores ni vulnerabilidades. Los desarrolladores de Linux revisan y corren los errores encontrados en la versión anterior, lo que garantiza que los kernels estables sean seguros y estables.

## ¿Por qué importa?

Estos kernels estables son importantes porque incluyen correcciones para vulnerabilidades de seguridad que podrían permitir a un atacante escapar de un contenedor y obtener acceso root. Es fundamental actualizar a estos kernels para garantizar la seguridad de los sistemas Linux.

## Consejo técnico

Para actualizar a los kernels estables, puedes utilizar el comando `sudo apt-get update && sudo apt-get upgrade` en sistemas Ubuntu o Debian, o `sudo yum update` en sistemas CentOS o RHEL. Asegúrate de que tu sistema esté actualizado antes de proceder.

```bash
sudo apt-get update && sudo apt-get upgrade
```
