Esta tarjeta es compatible con todos los módulos de computadora Raspberry Pi Compute Module 5, lo que la hace una opción atractiva para desarrolladores y fabricantes de hardware.

## ¿Qué es?

La tarjeta de expansión Raspberry Pi CM5 es una placa de circuito impreso que se conecta a la computadora Raspberry Pi Compute Module 5 para agregar características adicionales. Ofrece conectividad de red dual Ethernet, cuatro puertos RS485, conectividad 4G LTE/5G, y mucho más.

## ¿Cómo funciona?

La tarjeta de expansión Raspberry Pi CM5 se conecta a la computadora Raspberry Pi Compute Module 5 mediante una interfaz de comunicación. La tarjeta ofrece una amplia gama de características, incluyendo conectividad de red dual Ethernet, cuatro puertos RS485, conectividad 4G LTE/5G, y una salida HDMI 4K. También cuenta con una ranura M.2 para conectar una tarjeta de estado sólido o un módulo de aceleración de inteligencia artificial.
