# Dongle USB para DECT NR+: una herramienta para desarrolladores de IoT

**Fecha:** 2026-07-09
**Categoría:** linux
**Tags:** robotica, embebidos, linux
**Título original:** Norik Systems introduces nRF9151-based USB Dongle for DECT NR+ deployments

---

## Introducción

Norik Systems ha lanzado un dongle USB basado en el nRF9151 de Nordic Semiconductor, diseñado para trabajar como nodo hijo o como gateway USB en redes DECT NR+. Esta tecnología es ideal para aplicaciones IoT de largo alcance y baja potencia.

## ¿Qué es?

El DECT NR+ es un estándar no celular de 5G para aplicaciones de largo alcance y baja potencia, que funciona a 1,9 GHz y soporta hasta un millón de nodos a través de topologías de red en malla, estrella y punto a punto. El dongle USB de Norik Systems permite a los desarrolladores agregar fácilmente DECT NR+ a gateways o nodos hijos.

## ¿Cómo funciona?

El dongle USB se conecta directamente a un puerto USB y se muestra como un dispositivo serial y debugger. Viene pre-flasheado con firmware de consola que permite a los desarrolladores descubrir redes, unirse o crear redes privadas, realizar pruebas de enlace y evaluar el rendimiento de la radio a través de una terminal serial. También se puede reprogramar con la aplicación personalizada utilizando ejemplos de código abierto incluidos en la SDK de Nordic nRF Connect.

## ¿Por qué importa?

Esta herramienta es importante porque permite a los desarrolladores crear aplicaciones IoT de largo alcance y baja potencia de manera más fácil y eficiente. El DECT NR+ es una tecnología ideal para aplicaciones como sensores de temperatura, sensores de humedad, etc.

## Consejo técnico

Si deseas probar el dongle USB de Norik Systems, asegúrate de descargar la SDK de Nordic nRF Connect y seguir las instrucciones de configuración y programación incluidas en la documentación.

```bash
sudo apt-get install nrf-connect-sdk
```
