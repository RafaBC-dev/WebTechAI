# Llega pkgcli, una interfaz de línea de comandos moderna y amigable

**Fecha:** 2026-06-14

## Introducción

Después de casi dos décadas, se ha lanzado una nueva herramienta para gestionar paquetes en Linux, diseñada para ser más fácil de usar y con una experiencia de scripting mejorada. pkgcli es el nombre de esta herramienta, que se ha desarrollado como parte del proyecto PackageKit.

## ¿Qué es?

pkgcli es una interfaz de línea de comandos moderna y amigable para gestionar paquetes en Linux. Se ha diseñado para ser fácil de usar, con comandos y salida de información claros, y ofrece una experiencia de scripting mejorada a través de líneas de JSON. Esta herramienta se basa en la capa de abstracción de PackageKit, que ha sido utilizada durante años en Linux.

## ¿Cómo funciona?

pkgcli se ha desarrollado desde cero para ofrecer una experiencia de usuario mejorada. Se ha diseñado para ser fácil de usar, con comandos y salida de información claros. También ofrece una experiencia de scripting mejorada a través de líneas de JSON, lo que permite a los usuarios automatizar tareas y crear scripts más eficientes. La herramienta se basa en la capa de abstracción de PackageKit, que se encarga de gestionar los paquetes en Linux.

## ¿Por qué importa?

pkgcli es importante porque ofrece una experiencia de usuario mejorada para gestionar paquetes en Linux. Esto significa que los usuarios pueden instalar, actualizar y eliminar paquetes de manera más fácil y rápida. Además, la herramienta ofrece una experiencia de scripting mejorada, lo que permite a los usuarios automatizar tareas y crear scripts más eficientes.

## Consejo técnico

Si deseas empezar a usar pkgcli, puedes probar instalando la última versión de PackageKit en tu distribución de Linux. Luego, puedes ejecutar el comando `pkgcli --help` para obtener una lista de comandos disponibles y aprender a usar la herramienta.

```bash
pkgcli --help
```
