# Wine 11.12: Escalado fraccionario y mejoras en Wayland

**Fecha:** 2026-06-30

## Introducción

La versión 11.12 de Wine ha sido lanzada con soporte para escalado fraccionario en Wayland, lo que permite una experiencia de escalado de interfaz de usuario más agradable en pantallas de alta resolución.

## ¿Qué es?

Wine es un proyecto de software que permite ejecutar aplicaciones de Windows en sistemas operativos Linux y macOS. Se basa en una capa de compatibilidad que traduce llamadas de sistema de Windows a llamadas de sistema de Linux.

## ¿Cómo funciona?

La versión 11.12 de Wine incluye una implementación de escalado fraccionario en Wayland, que permite a los usuarios establecer escalados diferentes para cada pantalla. Además, se han agregado mejoras en la gestión de monitores y se han corregido varios bugs.

## ¿Por qué importa?

La versión 11.12 de Wine es importante porque ofrece una experiencia de escalado de interfaz de usuario más agradable en pantallas de alta resolución, lo que es especialmente útil para usuarios que utilizan pantallas de alta densidad.

## Consejo técnico

Si deseas aprovechar las mejoras de escalado fraccionario en Wayland, asegúrate de actualizar tu versión de Wine a la 11.12 y configura el escalado de pantalla adecuadamente en tu sistema.

```bash
winecfg
```
