# Un clásico digital se renueva con tecnología moderna

**Categoría:** linux
**Tags:** robotica, embebidos, linux
**Título original:** Digital Organizer Given Modern Upgrade

## Introducción

Un dispositivo de la década de 1990 ha sido transformado en un dispositivo más capaz gracias a la integración de un Raspberry Pi Zero y una pantalla LCD de alta resolución.

## ¿Qué es?

Un digital organizer es un dispositivo portátil que permite almacenar y gestionar información personal, como números de teléfono y citas. El Casio Business Organizer Scheduling System SF-5580 es un ejemplo de estos dispositivos, que se popularizaron en la década de 1990.

## ¿Cómo funciona?

El dispositivo original ha sido reemplazado por un Raspberry Pi Zero, que se conecta a una pantalla LCD de 480 x 800 píxeles y a un teclado de 82 teclas. El Raspberry Pi Pico se encarga de leer los datos del teclado y enviarlos al Raspberry Pi Zero. El dispositivo está alimentado por una batería de 6000 mAh.

## ¿Por qué importa?

Este proyecto importa porque muestra cómo es posible dar nueva vida a dispositivos antiguos con tecnología moderna. También destaca la capacidad de los dispositivos embebidos para ser utilizados en aplicaciones creativas y personalizadas.

## Consejo técnico

Si deseas crear un proyecto similar, puedes utilizar el software de Raspberry Pi para personalizar la interfaz de usuario y agregar funcionalidades a tu dispositivo. Puedes comenzar descargando el software de Raspberry Pi y siguiendo las instrucciones de configuración.

```bash
sudo apt-get install raspbian
```
