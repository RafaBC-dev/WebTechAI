# CircuitPython agrega soporte a audio USB para proyectos de Morse

## Introducción

Adafruit ha lanzado una guía para utilizar la biblioteca TinyUSB en CircuitPython para crear un teclado de Morse que envíe señales de audio a un ordenador o equipo digital de audio.

## ¿Qué es?

La biblioteca TinyUSB es una herramienta fundamental en CircuitPython que permite a los proyectos utilizar interfaces USB como almacenamiento, CDC, HID, MIDI y más. La versión 10.3.0-alpha.3 de CircuitPython agrega soporte para tipos de audio USB, lo que permite a los proyectos generar tonos y enviarlos a un ordenador o equipo digital de audio.

## ¿Cómo funciona?

La guía muestra cómo conectar un teclado básico de Morse a un proyecto de CircuitPython y utilizar la función de generación para convertir cadenas en señales de audio. La biblioteca TinyUSB se utiliza para establecer la conexión USB y enviar las señales de audio al ordenador o equipo digital de audio.

## ¿Por qué importa?

Este proyecto importa porque permite a los desarrolladores de proyectos de robótica y electrónica crear interfaces de usuario interactivas y enviar señales de audio en tiempo real. También es una herramienta útil para educar sobre el Morse y la programación en CircuitPython.

## Consejo técnico

Si deseas crear un proyecto de Morse con CircuitPython, asegúrate de utilizar la versión 10.3.0-alpha.3 o superior de la biblioteca TinyUSB y sigue la guía de Adafruit para conectar tu teclado de Morse y enviar señales de audio a tu ordenador o equipo digital de audio.

```bash
pip install circuitpython-tinyusb
```
