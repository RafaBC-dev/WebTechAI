## Introducción

Después de analizar el hardware y probar el mini PC con Windows 11 Pro, ahora exploramos su rendimiento con Ubuntu 25.10. ¿Cómo se comporta el AMD Ryzen 5 7530U en Linux?

## ¿Qué es?

El GEEKOM A5 Pro 2026 es un mini PC equipado con un procesador AMD Ryzen 5 7530U, 16 GB de memoria RAM y un disco duro de 1 TB. Se puede instalar Ubuntu 25.10 en dual boot con Windows 11 Pro

## ¿Cómo funciona?

La instalación de Ubuntu 25.10 se realizó sin problemas, y el sistema se configuró con GNOME 46, compositor Wayland y kernel Linux 6.17. El procesador Ryzen 5 7530U ofrece 12 hilos de ejecución y gráficos Radeon integrados

## ¿Por qué importa?

Este mini PC con Ubuntu 25.10 es relevante para aquellos que buscan una opción de escritorio ligera y eficiente, y también para desarrolladores que desean probar sus aplicaciones en un entorno Linux

## Consejo técnico

Si deseas instalar Ubuntu 25.10 en tu mini PC, asegúrate de reducir el tamaño de la partición de Windows antes de proceder con la instalación, y asegúrate de que el BIOS esté configurado para priorizar el arranque de Linux

Para obtener información sobre el sistema, puedes ejecutar comandos como `cat /etc/lsb-release` para ver la versión de Ubuntu, o `uname -a` para obtener detalles sobre el kernel
