# Ubuntu Core 26: seguridad y eficiencia para dispositivos IoT y embebidos

**Fecha:** 2026-05-19
**Categoría:** linux
**Tags:** linux, codigo, herramientas
**Título original:** Ubuntu Core 26 targets IoT devices and embedded systems, offers up to 15 years of security maintenance

---

## Introducción

Canonical ha lanzado Ubuntu Core 26, una versión de Linux diseñada específicamente para dispositivos IoT y sistemas embebidos. Esta versión ofrece una seguridad y eficiencia mejoradas, lo que la convierte en una opción atractiva para empresas que buscan cumplir con los requisitos de seguridad de la Unión Europea.

## ¿Qué es?

Ubuntu Core 26 es una versión de Linux minimal y inmutable que se enfoca en la seguridad y la eficiencia. Se basa en Ubuntu 26.04 LTS y ofrece una serie de características innovadoras, como la actualización OTA (over-the-air) optimizada y la protección de hardware para misiones críticas.

## ¿Cómo funciona?

Ubuntu Core 26 utiliza un sistema de construcción llamado Chisel, que permite crear versiones precisas de Linux con un tamaño de archivo reducido. Cada componente es un contenedorizado snap, lo que facilita la actualización y la seguridad. Además, la versión utiliza la tecnología de parcheo de kernel en vivo, lo que permite actualizar el kernel sin reiniciar el sistema.

## ¿Por qué importa?

Ubuntu Core 26 es importante porque ofrece una seguridad y eficiencia mejoradas para dispositivos IoT y sistemas embebidos. Esto es especialmente relevante para empresas que buscan cumplir con los requisitos de seguridad de la Unión Europea, como el CRA (Cyber Resilience Act).

## Consejo técnico

Si deseas aprovechar al máximo las características de Ubuntu Core 26, utiliza la herramienta Chisel para crear versiones precisas de Linux y optimizar las actualizaciones OTA.

```bash
chisel
```
