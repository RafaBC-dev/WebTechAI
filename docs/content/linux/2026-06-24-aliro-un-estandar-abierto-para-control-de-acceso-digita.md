# Aliro: un estándar abierto para control de acceso digital

**Fecha:** 2026-06-24
**Categoría:** linux
**Tags:** robotica, microcontroladores, software y linux
**Título original:** Aliro is a vendor-agnostic standard for digital access control working over NFC, Bluetooth LE, or UWB

---

## Introducción

La Connectivity Standards Alliance ha lanzado Aliro, un estándar abierto para control de acceso digital que funciona sobre NFC, Bluetooth LE o UWB. Este estándar busca abordar la fragmentación del mercado y permitir la interoperabilidad entre diferentes fabricantes.

## ¿Qué es?

Aliro es un estándar abierto para control de acceso digital que utiliza tecnologías como NFC, Bluetooth LE y UWB para permitir la autenticación y el acceso a edificios, vehículos o equipos. Es un estándar diseñado para ser independiente de los fabricantes y permitir la interoperabilidad entre diferentes sistemas.

## ¿Cómo funciona?

Aliro utiliza un marco de trabajo que se basa en criptografía asimétrica para garantizar la seguridad y la confianza en las interacciones entre dispositivos de usuario y lectores. El estándar también incluye la integración con carteras digitales de Apple, Google, Samsung y otros proveedores. Los transportes utilizados por Aliro son NFC para acceso cercano, Bluetooth LE para comunicación a larga distancia iniciada por el usuario y Bluetooth LE con UWB para autenticación sin tocar.

## ¿Por qué importa?

Aliro es importante porque busca abordar la fragmentación del mercado de control de acceso digital y permitir la interoperabilidad entre diferentes fabricantes. Esto significa que los usuarios pueden utilizar cualquier aplicación compatible con Aliro para acceder a edificios o equipos, sin importar el fabricante del sistema de control de acceso.

## Consejo técnico

Si estás interesado en implementar Aliro en tus sistemas de control de acceso, puedes descargar la especificación de Aliro 1.0 desde el sitio web de la Connectivity Standards Alliance y comenzar a trabajar en la integración con tus sistemas existentes.
