# Cyberdecks: los ordenadores portátiles de ciencia ficción se hacen realidad

**Fecha:** 2026-04-18
**Categoría:** linux
**Tags:** robotica, microcontroladores, software-y-linux
**Título original:** In celebration of cyberdecks

---

## Introducción

La comunidad de makers ha visto un aumento en el número de proyectos de cyberdecks, ordenadores portátiles inspirados en la ciencia ficción. Entre ellos, destacan los que utilizan la Raspberry Pi como plataforma de base.

## ¿Qué es?

Un cyberdeck es un ordenador portátil que se inspira en la ciencia ficción, especialmente en la serie de películas y series de televisión de cyberpunk. Pueden ser construidos utilizando diferentes componentes, incluyendo la Raspberry Pi.

## ¿Cómo funciona?

Los cyberdecks suelen ser construidos utilizando una Raspberry Pi como plataforma de base, que se combina con otros componentes como pantallas, teclados y baterías. Algunos ejemplos destacan la utilización de seis pantallas para una experiencia de usuario más inmersiva.

## ¿Por qué importa?

Los cyberdecks importan porque muestran la creatividad y la innovación de la comunidad de makers. Además, pueden ser utilizados para crear experiencias de usuario inmersivas y nuevas formas de interactuar con la tecnología.

## Consejo técnico

Si deseas construir tu propio cyberdeck, considera utilizar la Raspberry Pi como plataforma de base y explorar la posibilidad de utilizar múltiples pantallas para una experiencia de usuario más inmersiva.

```bash
sudo apt-get install raspbian
```
