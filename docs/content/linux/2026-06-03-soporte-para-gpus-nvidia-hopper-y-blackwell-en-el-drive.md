# Soporte para GPUs NVIDIA Hopper y Blackwell en el driver Nova se acerca

**Fecha:** 2026-06-03

## Introducción

La comunidad de desarrollo de Linux ha logrado un avance significativo en el soporte para las GPUs NVIDIA Hopper y Blackwell en el driver Nova, un proyecto de código abierto que busca ofrecer una alternativa a los drivers propietarios de NVIDIA.

## ¿Qué es?

El driver Nova es un proyecto de código abierto que busca ofrecer un controlador de gráficos para las GPUs NVIDIA de manera gratuita y abierta. Se está desarrollando en Rust y busca ofrecer una alternativa a los drivers propietarios de NVIDIA.

## ¿Cómo funciona?

El driver Nova utiliza el código de la GPU System Processor (GSP) para proporcionar soporte a las GPUs Hopper y Blackwell. El código se está desarrollando en Rust y se está integrando con la infraestructura de control de gráficos de Linux.

## ¿Por qué importa?

El soporte para las GPUs Hopper y Blackwell en el driver Nova es importante porque permite a los desarrolladores de Linux aprovechar al máximo el potencial de estas GPUs, lo que puede llevar a mejoras en el rendimiento y la eficiencia de los sistemas basados en Linux.
