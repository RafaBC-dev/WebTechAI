## Introducción

La exploración del océano es crucial para la investigación marina, pero las condiciones extremas y el alto costo hacen que sea peligroso y difícil para los humanos. Los vehículos autónomos pueden manejar el trabajo, pero hardware asequible y software accesible son esenciales para abrir este área de conservación.

## ¿Qué es?

La empresa MDM Team está desarrollando robots marinos autónomos impulsados por Raspberry Pi Compute Module 4 para la exploración del océano, el monitoreo ambiental y la inspección de infraestructura. Estos vehículos incluyen barcos de superficie autónomos (ASVs) y vehículos submarinos autónomos (AUVs) que pueden realizar tareas como sondeos hidrográficos o inspecciones de infraestructura sin la necesidad de un operador humano a bordo.

## ¿Cómo funciona?

Cada uno de los vehículos desarrollados por MDM Team, Cariddi y Stok, cuenta con un sistema dual construido alrededor de Raspberry Pi Compute Module 4. Un tablero de expansión personalizado conecta el módulo a la restante hardware del vehículo, integrando sistemas de navegación como GPS y conectando sensores utilizados para navegación y medición. Raspberry Pi Compute Modules son el sistema nervioso central de cada robot, coordinando sensores, sistemas de propulsión y procesamiento de datos de manera simultánea.

## ¿Por qué importa?

La importancia de este proyecto radica en que puede ayudar a resolver problemas de exploración del océano, monitoreo ambiental y inspección de infraestructura de manera más eficiente y segura, lo que puede tener un impacto significativo en la conservación del medio ambiente y la seguridad de la infraestructura.
