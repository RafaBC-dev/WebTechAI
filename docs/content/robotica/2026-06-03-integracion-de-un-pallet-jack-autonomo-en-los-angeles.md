# Integración de un pallet jack autónomo en Los Ángeles

**Fecha:** 2026-06-03
**Categoría:** robotica
**Tags:** robotica, ia-local, linux
**Título original:** Robotics integrator — autonomous pallet jack bring-up (1–2 week contract, on-site Los Angeles)

---

## Introducción

Una empresa está buscando un integrador de robótica para llevar a cabo el hardware de un pallet jack autónomo en Los Ángeles. El objetivo es convertir un camión de mano estándar en un vehículo autónomo que pueda moverse por su cuenta en un área restringida.

## ¿Qué es?

El proyecto consiste en integrar un sistema de autonomía en un camión de mano existente, utilizando un NVIDIA Jetson Orin Nano y software de ROS 2. El objetivo es crear un sistema que pueda moverse por su cuenta y realizar tareas de forma autónoma.

## ¿Cómo funciona?

El sistema utiliza un NVIDIA Jetson Orin Nano y software de ROS 2 para controlar el movimiento del camión de mano. El sistema también incluye sensores y un actuator de dirección para permitir que el camión se mueva de forma autónoma. El proceso de integración incluye la configuración del hardware, la programación del software y la pruebas del sistema.

## ¿Por qué importa?

Este proyecto es importante porque puede revolucionar la forma en que se manejan los pallets en los almacenes y fábricas. Un sistema autónomo puede mejorar la eficiencia y la seguridad del proceso de manejo de pallets, reduciendo el riesgo de accidentes y mejorando la productividad.

## Consejo técnico

Si estás interesado en desarrollar un sistema de autonomía para vehículos, es importante familiarizarte con la tecnología de ROS 2 y el uso de sensores y actuator para controlar el movimiento del vehículo.

```bash
ros2 launch <nombre_del_launch>
```
