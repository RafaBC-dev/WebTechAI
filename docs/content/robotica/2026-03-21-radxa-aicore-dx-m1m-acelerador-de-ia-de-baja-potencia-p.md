# Radxa AICore DX-M1M: acelerador de IA de baja potencia para robots y dispositivos IoT

**Fecha:** 2026-03-21
**Categoría:** robotica
**Tags:** ia-local, robotica, embebidos
**Título original:** Radxa AICore DX-M1M M.2 2242 low-power AI module delivers 25 TOPS of edge AI performance for just 3W of power

## Introducción

Radxa ha lanzado el AICore DX-M1M, un módulo de aceleración de IA de baja potencia que ofrece hasta 25 TOPS de rendimiento en un consumo de solo 3W. Este módulo está diseñado para aplicaciones industriales y de IoT, como robots armados, vehículos autónomos y dispositivos de inteligencia artificial.

## ¿Qué es?

El AICore DX-M1M es un módulo de aceleración de IA compacto y de baja potencia que utiliza el procesador de unidades de procesamiento neural (NPU) DeepX DX-M1M. Ofrece hasta 25 TOPS de rendimiento en INT8 y consume solo 3W de potencia.

## ¿Cómo funciona?

El AICore DX-M1M utiliza el procesador NPU DeepX DX-M1M para acelerar la ejecución de modelos de IA. El módulo utiliza una interfaz PCIe Gen3 x2 y es compatible con sistemas x86 y Arm, incluyendo la Raspberry Pi 5 y los SBCs de Radxa. El módulo también incluye 1GB de memoria LPDDR4X y 1Gbit de almacenamiento QSPI NAND/NOR.

## ¿Por qué importa?

El AICore DX-M1M es importante porque ofrece una solución de aceleración de IA de baja potencia para aplicaciones industriales y de IoT. Esto permite a los desarrolladores crear sistemas más eficientes y escalables que puedan realizar tareas de IA complejas sin aumentar el consumo de energía.

## Consejo técnico

Si deseas utilizar el AICore DX-M1M con tu proyecto de IA, asegúrate de utilizar la SDK de DeepX DXNN, que incluye herramientas para compilar y optimizar modelos de IA para el procesador NPU. También puedes utilizar el DX-All Suite, un paquete de instalación simplificado que incluye el compilador y el runtime de DXNN.

```bash
deepx-dxnn-sdk install
```
