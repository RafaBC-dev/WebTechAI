# Connext Robotics Toolkit para ROS Lyrical Luth ahora disponible

**Fecha:** 2026-05-27
**Categoría:** robotica
**Tags:** robotica, linux, codigo
**Título original:** Connext Robotics Toolkit for ROS Lyrical Luth is now available

---

## Introducción

La comunidad de ROS 2 acaba de recibir una actualización importante en el Connext Robotics Toolkit, que facilita la integración de sistemas de robótica con native DDS aplicaciones. Esto abre nuevas posibilidades para la producción y el desarrollo de sistemas de robótica escalables y seguros.

## ¿Qué es?

El Connext Robotics Toolkit es una herramienta que proporciona una instalación simple y única de Lyrical y Connext DDS RMW en Ubuntu Linux, compatible con plataformas como Raspberry Pi y NVIDIA Orin. Se ofrece de forma gratuita para uso prototipo, investigación, no comercial y académico.

## ¿Cómo funciona?

El toolkit utiliza Connext 7.7 LTS para mejorar la interoperabilidad entre ROS y DDS, con nuevas características como la aliasación de temas y tipos que facilitan la integración de sistemas de robótica con aplicaciones native DDS. Esto se logra mediante la creación de un mapeo automático entre nombres de ROS y DDS.

## ¿Por qué importa?

La actualización del Connext Robotics Toolkit es importante porque permite a los usuarios de ROS 2 pasar de la prototipificación a la producción, con sistemas más escalables, seguros y eficientes. Esto abre nuevas posibilidades para el desarrollo de sistemas de robótica complejos y confiables.

## Consejo técnico

Si deseas aprovechar al máximo la actualización del Connext Robotics Toolkit, prueba la característica de aliasación de temas y tipos para ver cómo facilita la integración de tus sistemas de robótica con aplicaciones native DDS.

```bash
apt install connext-robotics-toolkit
```
