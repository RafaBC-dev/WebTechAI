# Robots recuperan chips de circuitos para reducir residuos electrónicos

**Fecha:** 2026-05-20

## Introducción

La cantidad de residuos electrónicos se está volviendo un problema cada vez más grave. La Unión Europea y California están implementando nuevas regulaciones para reducir la cantidad de residuos electrónicos que se exportan o se destruyen. Una startup llamada Tuurny está desarrollando un sistema robótico para recuperar chips de circuitos antes de que se destruyan.

## ¿Qué es?

Tuurny es una startup que está desarrollando un sistema robótico llamado Nantul para identificar y extraer chips de memoria de circuitos antes de que se destruyan. El objetivo es recuperar chips que pueden ser reutilizados en sistemas legados donde es difícil encontrar componentes de repuesto.

## ¿Cómo funciona?

El sistema Nantul utiliza robots para identificar y extraer chips de memoria de circuitos. Luego, los chips se clasifican por modelo o material y se envían a laboratorios de prueba para determinar si pueden ser reutilizados como chips nuevos o se envían a refinerías y fundiciones para procesarlos.

## ¿Por qué importa?

La cantidad de residuos electrónicos se está volviendo un problema cada vez más grave. La Unión Europea y California están implementando nuevas regulaciones para reducir la cantidad de residuos electrónicos que se exportan o se destruyen. La tecnología de Tuurny puede ayudar a recuperar metales valiosos y reducir la cantidad de residuos electrónicos que se destruyen.
