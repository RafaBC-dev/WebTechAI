## Introducción

La comunidad de ROS (Robot Operating System) ha lanzado nuevas funcionalidades y herramientas para mejorar la experiencia de desarrollo de robótica. Entre ellas, se destacan la capacidad de visualizar datos en 3D con RVizSplat y la implementación de navegación sin GPS para drones.

## ¿Qué es?

ROS (Robot Operating System) es un sistema operativo para robótica que permite a los desarrolladores crear aplicaciones y herramientas para controlar y programar robots. ROS proporciona una plataforma para la integración de hardware y software, facilitando la creación de sistemas de robótica complejos.

## ¿Cómo funciona?

La nueva versión de ROS incluye la capacidad de visualizar datos en 3D con RVizSplat, que utiliza técnicas de splatting para mostrar la información de manera clara y precisa. También se ha implementado la navegación sin GPS para drones, utilizando la técnica TERCOM+ESKF, que permite a los drones navegar en áreas sin acceso a señales de GPS.

## ¿Por qué importa?

Estas nuevas funcionalidades y herramientas son importantes porque permiten a los desarrolladores crear aplicaciones de robótica más avanzadas y precisas. La capacidad de visualizar datos en 3D con RVizSplat facilita la comprensión y el análisis de la información, mientras que la navegación sin GPS para drones abre nuevas posibilidades para la exploración y el trabajo en áreas sin acceso a señales de GPS.

## Consejo técnico

Si estás trabajando con ROS, prueba RVizSplat para visualizar tus datos en 3D y mejorar la comprensión de tu aplicación. También puedes explorar la implementación de navegación sin GPS para drones y ver cómo puedes adaptarla a tus necesidades.

```bash
ros2 run rviz_splat
```
