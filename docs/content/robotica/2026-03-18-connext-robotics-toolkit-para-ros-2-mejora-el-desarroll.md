# Connext Robotics Toolkit para ROS 2: mejora el desarrollo de sistemas robóticos

## Introducción

La comunidad de ROS 2 ha recibido una noticia importante: RTI ha lanzado el Connext Robotics Toolkit, una herramienta que facilita el desarrollo de sistemas robóticos utilizando ROS 2 y Connext. Esto es especialmente relevante para startups y equipos de desarrollo que buscan escalar desde prototipos a producción.

## ¿Qué es?

El Connext Robotics Toolkit es una herramienta que proporciona soporte para ROS 2 y rmw_connextdds, permitiendo a los usuarios aprovechar las características de Connext y DDS para mejorar su experiencia de desarrollo. Incluye herramientas como network tuning y debugging, sistema observabilidad y soporte para diversas redes, desde memoria compartida hasta WAN.

## ¿Cómo funciona?

El Connext Robotics Toolkit se basa en la implementación de DDS de RTI, que ya está disponible para uso no comercial. La herramienta proporciona una plataforma completa para la comunicación entre nodos ROS 2, incluyendo herramientas de depuración y observabilidad del sistema. También incluye soporte para redes diversas y herramientas de tuning de red.

## ¿Por qué importa?

El Connext Robotics Toolkit es importante porque permite a los desarrolladores de sistemas robóticos aprovechar las características de Connext y DDS para mejorar su experiencia de desarrollo. Esto es especialmente relevante para startups y equipos de desarrollo que buscan escalar desde prototipos a producción. Además, el toolkit proporciona soporte para redes diversas y herramientas de depuración y observabilidad del sistema.

## Consejo técnico

Si estás explorando formas de utilizar ROS en sistemas comerciales, considera utilizar el Connext Robotics Toolkit para aprovechar las características de Connext y DDS. Puedes encontrar más información y instrucciones de instalación en la página de RTI.

```bash
rti connext robotics toolkit
```
