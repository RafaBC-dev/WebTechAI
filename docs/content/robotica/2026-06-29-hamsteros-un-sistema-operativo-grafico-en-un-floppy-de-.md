# HamsterOS: un sistema operativo gráfico en un floppy de 1,44 MB

**Fecha:** 2026-06-29
**Categoría:** robotica
**Tags:** robotica, linux, embebidos
**Título original:** HamsterOS Crams Complete Graphical Desktop onto 1.44 MB Floppy

## Introducción

El proyecto HamsterOS busca revivir la emoción de trabajar con hardware vintage, ofreciendo un sistema operativo gráfico completo en un floppy de 1,44 MB. Este proyecto es relevante para entusiastas de la retrocomputación que buscan experimentar con tecnologías antiguas.

## ¿Qué es?

HamsterOS es un sistema operativo gráfico multitarea de 32 bits que se ejecuta en hardware de la época 386 y 486. Se diseñó para funcionar en un floppy de 1,44 MB, pero también puede instalarse en un disco duro. Incluye una suite de aplicaciones nativas y soporte para DOS.

## ¿Cómo funciona?

HamsterOS utiliza una arquitectura de multitarea para permitir la ejecución de múltiples aplicaciones simultáneamente. El sistema también incluye un contador de fallas de arranque CMOS, que fuerza el sistema a entrar en un modo seguro VGA después de tres intentos fallidos de arranque.

## ¿Por qué importa?

HamsterOS importa porque ofrece una forma de experimentar con tecnologías antiguas de una manera más fácil y accesible. El sistema también incluye herramientas para facilitar la interfaz con hardware vintage, como la interfaz gráfica HamsterWeazle para el dispositivo Greaseweazle.

## Consejo técnico

Si deseas experimentar con HamsterOS, asegúrate de descargar y utilizar el dispositivo Greaseweazle para leer y escribir en discos de 1,44 MB. La interfaz gráfica HamsterWeazle puede ayudarte a configurar y utilizar el dispositivo de manera más fácil.
