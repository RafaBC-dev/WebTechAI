# ROS 2 Jazzy ahora disponible en Homebrew para macOS

**Fecha:** 2026-06-28
**Categoría:** robotica
**Tags:** robotica, linux, codigo
**Título original:** 🍺 ros2-jazzy is now available on Homebrew!

## Introducción

La comunidad de robótica ha obtenido un gran avance con la disponibilidad de ROS 2 Jazzy en Homebrew para macOS, lo que permite una experiencia nativa y sin necesidad de contenedores o máquinas virtuales.

## ¿Qué es?

ROS 2 Jazzy es una distribución de Robot Operating System (ROS) 2, una plataforma de software de código abierto para desarrollar sistemas de robótica. Esta distribución proporciona una suite completa de herramientas y librerías para la creación de sistemas de robótica, incluyendo simulación, control y navegación autónoma.

## ¿Cómo funciona?

Para instalar ROS 2 Jazzy en macOS, se puede utilizar el comando `brew tap idesign0/ros2` y luego `brew install ros2-jazzy`. Esto instalará la distribución completa con todas las herramientas y librerías necesarias. La distribución se mantiene en sincronización con la versión upstream, lo que significa que se actualizará automáticamente con las últimas mejoras y correcciones.

## ¿Por qué importa?

La disponibilidad de ROS 2 Jazzy en Homebrew para macOS es importante porque permite a los desarrolladores de robótica trabajar en un entorno nativo y sin necesidad de contenedores o máquinas virtuales. Esto facilita la creación de sistemas de robótica y permite una mayor colaboración y compartición de conocimientos entre la comunidad.

## Consejo técnico

Si deseas probar las tutoriales de MoveIt 2 o TurtleBot4 en macOS, puedes utilizar los enlaces proporcionados en el artículo para acceder a los tutoriales patcheados para la distribución Jazzy.

```bash
brew tap idesign0/ros2 && brew install ros2-jazzy
```
