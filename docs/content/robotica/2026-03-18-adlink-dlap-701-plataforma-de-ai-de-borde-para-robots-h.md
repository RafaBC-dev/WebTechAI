# ADLINK DLAP-701: plataforma de AI de borde para robots humanoides

## Introducción

ADLINK ha lanzado la plataforma de AI de borde DLAP-701, diseñada para robots humanoides, robots móviles autónomos y sistemas de visión. Esta plataforma se basa en la tecnología NVIDIA Jetson T5000/T4000 y ofrece una amplia gama de características y conectividad.

## ¿Qué es?

La DLAP-701 es una plataforma de AI de borde que se basa en la tecnología NVIDIA Jetson T5000/T4000. Ofrece una gran cantidad de memoria (hasta 128 GB) y una amplia gama de características, incluyendo conectividad Ethernet, USB, HDMI y slots M.2 para Wi-Fi 6, 5G y almacenamiento NVMe.

## ¿Cómo funciona?

La DLAP-701 se basa en la tecnología NVIDIA Jetson T5000/T4000, que proporciona una gran cantidad de procesamiento y memoria para aplicaciones de AI. La plataforma ofrece una amplia gama de características y conectividad, incluyendo conectividad Ethernet, USB, HDMI y slots M.2 para Wi-Fi 6, 5G y almacenamiento NVMe. La DLAP-701 también incluye un módulo de seguridad TPM 2.0 y un sensor de temperatura interno.

## ¿Por qué importa?

La DLAP-701 es importante porque ofrece una plataforma de AI de borde que puede ser utilizada en una variedad de aplicaciones, incluyendo robots humanoides, robots móviles autónomos y sistemas de visión. La plataforma ofrece una gran cantidad de memoria y características, lo que la hace ideal para aplicaciones que requieren un gran procesamiento y conectividad.

## Consejo técnico

Si deseas utilizar la DLAP-701 con Ubuntu, asegúrate de instalar la versión más reciente del JetPack SDK con CUDA acceleration, TensorRT y drivers para la plataforma. Esto te permitirá aprovechar al máximo las características de la plataforma.

```bash
sudo apt-get install jetpack-sdk
```
