# Sistema de desarrollo de robótica abierto con procesador Intel Core Ultra X7

**Fecha:** 2026-04-16
**Categoría:** robotica
**Tags:** robotica, ia-local, linux
**Título original:** Intel Core Ultra X7 358H-powered open robotics development system targets humanoid robots and autonomous vehicles

---

## Introducción

AAEON y Intel presentan un sistema de desarrollo de robótica abierto que utiliza el procesador Intel Core Ultra X7, diseñado para la creación de robots humanoides y vehículos autónomos con inteligencia artificial.

## ¿Qué es?

El AAEON CEXD-INTRBL es un sistema de desarrollo de robótica abierto que utiliza el procesador Intel Core Ultra X7, un procesador de 16 núcleos con una velocidad de hasta 4,8 GHz y una capacidad de procesamiento de 180 TOPS. Este sistema está diseñado para la creación de robots humanoides y vehículos autónomos con inteligencia artificial.

## ¿Cómo funciona?

El sistema utiliza el procesador Intel Core Ultra X7 para ejecutar aplicaciones de inteligencia artificial y procesar datos de sensores. También cuenta con una GPU de 12 núcleos, un acelerador de IA y una memoria de hasta 64 GB. El sistema admite hasta 8 cámaras GMSL2 y tiene conectores para sensores de LiDAR, infrarrojos y profundidad.

## ¿Por qué importa?

Este sistema de desarrollo de robótica abierto es importante porque permite a los desarrolladores crear robots humanoides y vehículos autónomos con inteligencia artificial de manera más eficiente y efectiva. También ofrece una plataforma para la creación de aplicaciones de IA y procesamiento de datos de sensores.

## Consejo técnico

Si deseas utilizar el sistema AAEON CEXD-INTRBL, asegúrate de instalar el sistema operativo Ubuntu 24.04 o posterior, ya que es la versión oficialmente soportada por AAEON y Intel.
