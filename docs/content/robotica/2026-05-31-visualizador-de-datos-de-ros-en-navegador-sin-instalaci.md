## Introducción

Un desarrollador ha creado una herramienta para visualizar datos de ROS en el navegador sin necesidad de instalación previa. Esto puede ser útil para desarrolladores de robótica que necesitan analizar datos de sus proyectos sin tener que configurar un entorno de desarrollo completo.

## ¿Qué es?

BAGEL es un visualizador de datos de ROS que permite a los desarrolladores visualizar, escrutar y analizar datos de ROS 1 y ROS 2 en el navegador sin necesidad de instalación previa. Puede trabajar con archivos de datos de ROS en formato .bag, .mcap y .db3.

## ¿Cómo funciona?

BAGEL utiliza una arquitectura cliente-servidor para procesar los datos de ROS en el navegador. Utiliza web workers y decodificadores de chunks para manejar grandes cantidades de datos de robótica. También utiliza la biblioteca Three.js para visualizar datos de 3D.

## ¿Por qué importa?

BAGEL es importante porque permite a los desarrolladores de robótica analizar datos de sus proyectos de manera rápida y sencilla sin necesidad de configurar un entorno de desarrollo completo. También puede ser útil para compartir resultados con colaboradores que no tienen experiencia en ROS.
