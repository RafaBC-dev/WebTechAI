# ROS 2: Un camino de aprendizaje para desarrolladores de robótica

**Fecha:** 2026-07-22
**Categoría:** robotica
**Tags:** robotica, linux, codigo
**Título original:** ROS 2 Learning Path Architecture v0.1 — 14 programs benchmarked, zero competency models: a skill tree and gap map for discussion

## Introducción

La comunidad de ROS 2 ha estado buscando una estructura de aprendizaje coherente para desarrolladores de robótica. Después de evaluar 14 programas de aprendizaje, se ha encontrado que ninguna de ellas define un modelo de competencia o progreso en rama. Por lo tanto, se ha propuesto una arquitectura de aprendizaje para ROS 2, diseñada para ser mantenida y contribuida por la comunidad.

## ¿Qué es?

ROS 2 es una plataforma de software de código abierto para desarrollar aplicaciones de robótica. La arquitectura de aprendizaje propuesta es un esfuerzo para crear un camino de aprendizaje estructurado y coherente para desarrolladores de robótica que utilizan ROS 2.

## ¿Cómo funciona?

La arquitectura de aprendizaje propuesta consta de un árbol de habilidades y una tabla de brechas. El árbol de habilidades muestra las habilidades y conocimientos que un desarrollador de robótica debe adquirir en diferentes niveles de competencia. La tabla de brechas identifica las áreas en las que la comunidad necesita mejorar o agregar contenido.

## ¿Por qué importa?

Esta arquitectura de aprendizaje es importante porque proporciona una estructura coherente para desarrolladores de robótica que buscan aprender y mejorar sus habilidades. También permite a la comunidad contribuir y mejorar el contenido, lo que puede ayudar a crear una comunidad más unida y comprometida.

## Consejo técnico

Si estás interesado en aprender ROS 2, te recomendamos comenzar con el árbol de habilidades y la tabla de brechas propuestos. Puedes encontrar el contenido en la página de GitHub de la arquitectura de aprendizaje. También puedes contribuir al contenido y mejorar la arquitectura.
