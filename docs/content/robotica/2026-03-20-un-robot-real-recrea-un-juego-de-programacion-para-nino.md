# Un robot real recrea un juego de programación para niños

**Fecha:** 2026-03-20
**Categoría:** robotica
**Tags:** robotica, python, linux
**Título original:** Real Robot Makes Debut in Programming Game

---

## Introducción

Un ingeniero ha creado un robot que reproduce un juego de programación para niños, utilizando una aplicación de Python y una interfaz de usuario personalizada. Este proyecto busca fomentar la experimentación y el descubrimiento de la programación en niños.

## ¿Qué es?

El proyecto consiste en un robot que reproduce el juego de programación 'Robot Turtles', utilizando una aplicación de Python y una interfaz de usuario personalizada. El robot puede ser controlado mediante teclado o mediante tarjetas NFC que se colocan en un lector NFC. El juego se puede jugar en tiempo real o con movimientos programados.

## ¿Cómo funciona?

La aplicación de Python se ejecuta en un ordenador y se conecta con el robot a través de una interfaz de usuario personalizada. El robot utiliza una tarjeta NFC para leer las tarjetas del juego y realizar las acciones correspondientes. La aplicación utiliza la biblioteca PyGame para crear una interfaz de usuario gráfica.

## ¿Por qué importa?

Este proyecto importa porque busca fomentar la experimentación y el descubrimiento de la programación en niños. También muestra cómo se pueden utilizar tecnologías como la programación y la robótica para recrear juegos y actividades educativas.

## Consejo técnico

Si deseas crear un proyecto similar, puedes utilizar la biblioteca PyGame para crear una interfaz de usuario gráfica y la biblioteca NFC para leer tarjetas NFC. Puedes también utilizar la aplicación de Python para controlar el robot y reproducir el juego.

```bash
python -m pygame --init
```
