# Digit aprende a bailar gracias a un modelo de IA genérico

## Introducción

El equipo de IA de una empresa de robótica ha desarrollado un modelo de aprendizaje automático que permite a un robot llamado Digit aprender a bailar en cuestión de horas. Este avance es un paso importante hacia la creación de inteligencia general para el mundo físico.

## ¿Qué es?

El modelo de IA genérico, llamado GEN-1, es una red neuronal que puede aprender a realizar tareas físicas simples de manera autónoma. Se basa en datos de movimiento y se entrena mediante un proceso de aprendizaje por refuerzo.

## ¿Cómo funciona?

El GEN-1 se entrena mediante la combinación de datos de movimiento obtenidos mediante captura de movimiento, animación y teleoperación. Luego, se aplica un proceso de aprendizaje por refuerzo en un entorno simulado para aprender a realizar las tareas de manera eficiente.

## ¿Por qué importa?

Este avance es importante porque permite a los robots aprender a realizar tareas complejas de manera autónoma, lo que podría revolucionar la industria de la robótica y permitir la creación de robots que puedan ayudar a las personas en una variedad de tareas.
