# Un robot inchworm construye casas con bloques LEGO

**Fecha:** 2026-05-09
**Categoría:** robotica
**Tags:** robotica, diseño-3d, linux
**Título original:** Could Your Next House be Built from Giant Lego By an Inchworm Robot?

## Introducción

Un equipo de investigadores de MIT ha desarrollado un robot que puede construir estructuras con bloques LEGO gigantes. Este robot inchworm podría revolucionar la forma en que se construyen casas en el futuro.

## ¿Qué es?

El robot inchworm es un dispositivo automatizado que se mueve de manera similar a un gusano, utilizando actuadores en ambos extremos para agarrar y colocar bloques LEGO gigantes. Estos bloques se llaman 'voxels' y son estructuras de espacio de marco diseñadas para unirse fácilmente.

## ¿Cómo funciona?

El robot inchworm funciona como sigue: se ancla a una parte de la estructura existente, agarra un bloque voxel y se ancla a él, repitiendo este proceso para construir la estructura. El robot tiene 5 grados de libertad y no tiene base tradicional, lo que le permite moverse de manera flexible.

## ¿Por qué importa?

Este proyecto importa porque podría reducir la energía incorporada en la construcción de casas, ya que los bloques voxel pueden ser fabricados con materiales más sostenibles que el hormigón. Además, la construcción con bloques voxel podría ser más rápida y eficiente que la construcción tradicional.

## Consejo técnico

Si deseas experimentar con la construcción de estructuras con bloques voxel, puedes descargar el código abierto del robot inchworm y comenzar a diseñar tus propias estructuras.

```bash
git clone https://github.com/mit-robotics/milabot
```
