# Nueva ROS 2 node para la cámara HD de PlayStation 5

## Introducción

La comunidad de ROS 2 ha recibido una noticia interesante: se ha implementado una nueva node para la cámara HD de PlayStation 5, una cámara estéreo con alta resolución y un precio asequible. Esto puede ser una excelente opción para aplicaciones de visión SLAM.

## ¿Qué es?

La cámara HD de PlayStation 5 es una cámara estéreo que cuenta con dos cámaras monoculares con resolución de 1080p y una velocidad de captura de hasta 60 FPS. Se trata de una opción asequible (60 USD) que puede ser utilizada en aplicaciones de visión SLAM.

## ¿Cómo funciona?

La nueva node ROS 2 permite a los desarrolladores utilizar la cámara HD de PlayStation 5 en sus proyectos de visión SLAM. La node se ha implementado utilizando la plataforma ROS 2, lo que permite una integración fácil con otros componentes de la plataforma.

## ¿Por qué importa?

La cámara HD de PlayStation 5 es una opción asequible y de alta calidad para aplicaciones de visión SLAM. Esto puede ser especialmente útil para desarrolladores que buscan una solución económica para sus proyectos.

## Consejo técnico

Si deseas utilizar la cámara HD de PlayStation 5 en tu proyecto de visión SLAM, asegúrate de instalar la última versión de ROS 2 y de seguir las instrucciones de instalación de la node en la documentación oficial.

```bash
ros2 run ps5_hd_camera
```
