## Introducción

Un desarrollador de robots busca consejos para llegar a los usuarios de ROS 2, una herramienta clave para automatizar el desarrollo de robots. ¿Cómo se construyó esta herramienta y qué problemas resuelve?

## ¿Qué es?

La herramienta se llama Jointly y se utiliza para automatizar el desarrollo de robots utilizando ROS 2. Con Jointly, los desarrolladores pueden crear un entorno de trabajo de robot verificado y listo para usar en minutos, en lugar de horas o días.

## ¿Cómo funciona?

Jointly utiliza un conjunto de herramientas y técnicas para automatizar la conversión de asamblajes CAD, la configuración de articulaciones, la generación de descripciones de robot y la depuración del primer setup. Esto se logra mediante un proceso de automatización que utiliza ROS 2 como base.

## ¿Por qué importa?

La importancia de Jointly radica en que resuelve un problema común en el desarrollo de robots: la repetitividad y la falta de eficiencia en la creación de un entorno de trabajo de robot. Con Jointly, los desarrolladores pueden ahorrar tiempo y esfuerzo, lo que les permite enfocarse en la creación de robots más avanzados y complejos.

## Consejo técnico

Si estás desarrollando herramientas de automatización para ROS 2, considera utilizar la biblioteca de herramientas de ROS 2 para simplificar el proceso de automatización y mejorar la eficiencia.

```bash
ros2 launch jointly.launch.py
```
