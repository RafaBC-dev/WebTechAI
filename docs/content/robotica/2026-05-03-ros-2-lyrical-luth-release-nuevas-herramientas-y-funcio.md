# ROS 2 Lyrical Luth Release: Nuevas Herramientas y Funcionalidades

**Fecha:** 2026-05-03
**Categoría:** robotica
**Tags:** robotica, linux, codigo
**Título original:** ROS News for the Week of April 27th, 2026

## Introducción

La comunidad de ROS (Robot Operating System) ha lanzado una nueva versión de ROS 2, llamada Lyrical Luth, que incluye una serie de herramientas y funcionalidades innovadoras para mejorar la experiencia de desarrollo de robótica. Esta versión es el resultado de la colaboración de la comunidad y ofrece una plataforma más estable y segura para crear sistemas de robótica complejos.

## ¿Qué es?

ROS 2 es una plataforma de software de código abierto para la creación de sistemas de robótica. Es una extensión de la plataforma original ROS, diseñada para ser más escalable y segura. La versión Lyrical Luth incluye herramientas como RVizSplat, que permite visualizar datos de 3D en tiempo real, y mesh_tools, que agrega herramientas para trabajar con meshes 3D.

## ¿Cómo funciona?

La versión Lyrical Luth de ROS 2 se basa en una arquitectura modular que permite a los desarrolladores crear y compartir módulos de software de manera fácil y segura. La plataforma utiliza un sistema de paquetes llamado colcon, que permite a los desarrolladores crear y gestionar paquetes de software de manera eficiente. RVizSplat, una de las herramientas incluidas en esta versión, utiliza una técnica de visualización en tiempo real llamada Gaussian Splatting para mostrar datos de 3D de manera clara y precisa.

## ¿Por qué importa?

La versión Lyrical Luth de ROS 2 es importante porque ofrece una plataforma más estable y segura para crear sistemas de robótica complejos. La inclusión de herramientas como RVizSplat y mesh_tools permite a los desarrolladores crear visualizaciones de datos de 3D en tiempo real y trabajar con meshes 3D de manera más eficiente. Esto puede ser especialmente útil en aplicaciones como la navegación de UAVs sin GPS.

## Consejo técnico

Si estás trabajando con ROS 2, te recomiendo probar RVizSplat para visualizar tus datos de 3D en tiempo real. Puedes instalar RVizSplat utilizando el comando `sudo apt-get install ros-<distro>-rviz-splat` (reemplaza `<distro>` con la versión de ROS 2 que estés utilizando).

```bash
sudo apt-get install ros-<distro>-rviz-splat
```
