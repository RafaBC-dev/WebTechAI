# OpenUSD: Estándar para la interoperabilidad de activos de simulación en robótica

**Fecha:** 2026-06-17
**Categoría:** robotica
**Tags:** robotica, linux, codigo
**Título original:** Draft REP 158: OpenUSD Conventions for Simulation Asset Interoperability in Open Source Robotics

---

## Introducción

La comunidad de ROS está trabajando en un nuevo estándar para la interoperabilidad de activos de simulación en robótica. Esto permitirá a los desarrolladores crear activos de simulación una vez y utilizarlos en diferentes plataformas de simulación y visualización.

## ¿Qué es?

OpenUSD (Universal Scene Description) es un estándar para la descripción de escenas 3D. El objetivo de este proyecto es establecer un estándar para la interoperabilidad de activos de simulación en robótica, permitiendo a los desarrolladores crear activos de simulación una vez y utilizarlos en diferentes plataformas de simulación y visualización. Esto incluye la creación de un conjunto de reglas y convenciones para la representación de interfaces ROS en USD, la definición de reglas para conversión de formatos y la creación de registros comunitarios para extensiones y activos compatibles.

## ¿Cómo funciona?

El proyecto OpenUSD establece un conjunto de reglas y convenciones para la representación de interfaces ROS en USD. Esto incluye la definición de esquemas declarativos para ROS Interface, como RosContextAPI, Topics, Services, Actions y Frames. Además, se establecen reglas para la conversión de formatos y se crean registros comunitarios para extensiones y activos compatibles.

## ¿Por qué importa?

Este proyecto es importante porque permitirá a los desarrolladores crear activos de simulación una vez y utilizarlos en diferentes plataformas de simulación y visualización. Esto reducirá la complejidad y el costo de desarrollo, y permitirá a los desarrolladores enfocarse en la creación de soluciones innovadoras en robótica.

## Consejo técnico

Si deseas contribuir a este proyecto, puedes empezar revisando el repositorio de OpenUSD y proporcionando retroalimentación sobre la propuesta. También puedes unirte a la comunidad de ROS y participar en la discusión sobre este proyecto.
