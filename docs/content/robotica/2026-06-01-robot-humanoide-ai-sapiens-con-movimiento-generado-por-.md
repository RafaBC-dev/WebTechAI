# Robot humanoide AI Sapiens con movimiento generado por texto

**Fecha:** 2026-06-01

## Introducción

La empresa ROBOTIS ha presentado su plataforma de robot humanoide AI Sapiens, que utiliza la tecnología NVIDIA Kimodo para generar movimientos directamente a partir de texto. Este avance permite a los desarrolladores crear movimientos complejos para sus robots de manera sencilla y eficiente.

## ¿Qué es?

AI Sapiens es una plataforma de robot humanoide de código abierto que utiliza la tecnología NVIDIA Kimodo para generar movimientos directamente a partir de texto. Esta plataforma se basa en la arquitectura ROS 2 y utiliza actuadores DYNAMIXEL-Q (QDD) para proporcionar control preciso y estabilidad.

## ¿Cómo funciona?

La plataforma AI Sapiens utiliza la tecnología NVIDIA Kimodo para generar movimientos directamente a partir de texto. La información de texto se procesa utilizando aprendizaje por refuerzo (RL) para controlar los movimientos del robot. La plataforma también utiliza la arquitectura ROS 2 para proporcionar una capa de abstracción y facilitar la integración con otros sistemas.

## ¿Por qué importa?

Este avance es importante porque permite a los desarrolladores crear movimientos complejos para sus robots de manera sencilla y eficiente. La plataforma AI Sapiens también proporciona una herramienta útil para la investigación y el desarrollo de robots humanoides.
