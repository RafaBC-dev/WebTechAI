## Introducción

El robot humanoide Honor Lightning ha batido el récord mundial de media maratón con un tiempo de 50 minutos y 26 segundos. ¿Qué tecnología o técnica ha permitido lograr este récord impresionante?

## ¿Qué es?

El robot humanoide Honor Lightning es un robot diseñado para correr maratones con eficiencia. Consiste en un sistema de piernas que se mueven alternativamente para propulsar el cuerpo del robot. El robot utiliza motores eléctricos para producir torque y moverse.

## ¿Cómo funciona?

El robot utiliza motores eléctricos con un sistema de engranajes para amplificar el torque y reducir la velocidad. El engranaje óptimo permite minimizar el consumo de energía y reducir la pérdida de calor. El robot utiliza un modelo físico para estimar el consumo de energía y la producción de calor en función del engranaje utilizado.

## ¿Por qué importa?

La tecnología utilizada en el robot humanoide Honor Lightning puede ser aplicada en la creación de robots humanoides más eficientes y capaces de realizar tareas complejas. Esto puede tener implicaciones en la robótica y la automatización.
