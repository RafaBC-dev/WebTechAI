## Introducción

El robot humanoid 'Honor Lightning' ha roto el récord mundial de maratón con un tiempo de 50 minutos y 26 segundos. Pero ¿qué secreto ha permitido a este robot superar a los humanos y a otros robots?

## ¿Qué es?

El Honor Lightning es un robot humanoide diseñado para correr maratones. Utiliza una combinación de motores eléctricos y un sistema de transmisión de engranajes para lograr su objetivo. El robot ha sido diseñado para minimizar la pérdida de energía y maximizar la eficiencia.

## ¿Cómo funciona?

El robot utiliza un sistema de transmisión de engranajes para amplificar la potencia de los motores eléctricos. La potencia se minimiza en un punto óptimo de relación de engranaje, lo que permite al robot correr de manera eficiente. El robot también utiliza un modelo de física para estimar la potencia consumida durante la carrera.

## ¿Por qué importa?

La creación del Honor Lightning tiene implicaciones importantes para la robótica y la automatización. El robot ha demostrado que es posible crear sistemas de movimiento eficientes y precisos, lo que puede tener aplicaciones en áreas como la medicina, la logística y la exploración espacial.
