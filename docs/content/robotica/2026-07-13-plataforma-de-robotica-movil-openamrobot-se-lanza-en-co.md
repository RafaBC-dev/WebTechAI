# Plataforma de robótica móvil OpenAMRobot se lanza en código abierto

**Fecha:** 2026-07-13
**Categoría:** robotica
**Tags:** robotica, ia-local, linux
**Título original:** OpenAMRobot v0.0.1 - Open-source ROS 2 mobile robot platform

---

## Introducción

La comunidad de robótica acaba de recibir un regalo importante: OpenAMRobot, una plataforma de robótica móvil de código abierto que permite a los desarrolladores crear robots autónomos y experimentar con Inteligencia Artificial. Esta plataforma es el resultado de un esfuerzo colaborativo y está diseñada para ser utilizada en la investigación, la educación y la prototipificación rápida.

## ¿Qué es?

OpenAMRobot es una plataforma de robótica móvil que se basa en ROS 2 Jazzy y está diseñada para la investigación, la educación y la prototipificación rápida. Incluye hardware, software y firmware para crear robots autónomos que pueden navegar y interactuar con su entorno.

## ¿Cómo funciona?

La plataforma se compone de un robot móvil con ruedas diferenciales, un firmware basado en Teensy 4.0 con micro-ROS, un simulador de Gazebo Harmonic y una interfaz de usuario web. El robot puede navegar utilizando la biblioteca Nav2 y puede detectar objetos con LiDAR SLAM y AprilTag. Además, puede transmitir video y telemetría en tiempo real.

## ¿Por qué importa?

OpenAMRobot es importante porque proporciona a los desarrolladores una plataforma de código abierto para crear robots autónomos y experimentar con Inteligencia Artificial. Esto puede ayudar a acelerar la investigación y el desarrollo en el campo de la robótica y la IA.

## Consejo técnico

Si deseas empezar a trabajar con OpenAMRobot, te recomendamos que comiences por configurar el simulador de Gazebo Harmonic y que explores la biblioteca Nav2 para aprender a hacer que tu robot navegue de manera autónoma.

```bash
ros2 run nav2_map_server
```
