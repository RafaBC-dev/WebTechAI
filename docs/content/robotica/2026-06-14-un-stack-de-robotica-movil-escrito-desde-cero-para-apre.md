# Un stack de robótica móvil escrito desde cero para aprender

## Introducción

Un desarrollador ha lanzado un proyecto de código abierto para crear un stack de robótica móvil desde cero, con el objetivo de entender mejor la tecnología detrás de la robótica

## ¿Qué es?

El proyecto, llamado taorobot, es un stack de robótica móvil escrito en C++ y basado en ROS 2, que incluye algoritmos de localización, planificación y control. Está diseñado para ser una herramienta de aprendizaje para desarrolladores que buscan entender la tecnología detrás de la robótica

## ¿Cómo funciona?

El stack taorobot está compuesto por varios módulos, cada uno de los cuales implementa un algoritmo específico, como la localización mediante particulas, el mapeo de ocupación y la planificación de trayectorias. Los módulos están diseñados para ser lo más independientes posible y se comunican entre sí a través de ROS 2

## ¿Por qué importa?

El proyecto taorobot importa porque ofrece una visión completa de la tecnología detrás de la robótica móvil, desde la localización hasta el control. Los desarrolladores pueden aprender de este proyecto y aplicar sus conocimientos en proyectos reales de robótica

## Consejo técnico

Si deseas aprender más sobre la implementación de algoritmos de localización y planificación en ROS 2, puedes empezar por revisar el código del proyecto taorobot y entender cómo se comunican los módulos entre sí

```bash
catkin build
```
