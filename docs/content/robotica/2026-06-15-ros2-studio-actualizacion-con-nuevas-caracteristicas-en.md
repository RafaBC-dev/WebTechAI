# ROS2 Studio: Actualización con nuevas características en el grabador de bag

## Introducción

La herramienta ROS2 Studio ha recibido una actualización con mejoras en su grabador de bag, permitiendo a los usuarios grabar y almacenar datos de manera más eficiente. Esta actualización es relevante para desarrolladores de sistemas embebidos y robótica que utilizan ROS2.

## ¿Qué es?

ROS2 Studio es una herramienta de monitoreo y gestión de ROS2, una plataforma de software de código abierto para la robótica y los sistemas embebidos. Permite a los usuarios monitorear el rendimiento de sus sistemas, grabar y reproducir datos de bag, y realizar otras tareas de gestión.

## ¿Cómo funciona?

El grabador de bag de ROS2 Studio permite a los usuarios seleccionar el formato de almacenamiento, configurar la duración de la grabación y dividir los archivos en chunks. Esto se puede hacer directamente desde la interfaz gráfica del usuario (GUI) sin necesidad de utilizar comandos en la terminal.

## ¿Por qué importa?

Esta actualización es importante para desarrolladores de sistemas embebidos y robótica que utilizan ROS2, ya que les permite grabar y almacenar datos de manera más eficiente. Esto puede ser especialmente útil para la depuración y el análisis de datos en sistemas complejos.

## Consejo técnico

Si estás utilizando ROS2 Studio, prueba la nueva característica de grabación de bag con formato sqlite3 o mcap. Puedes seleccionar el formato directamente desde la GUI y configurar la duración de la grabación y la división de los archivos.
