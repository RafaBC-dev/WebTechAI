## Herramientas destacadas esta semana

Esta semana, se destaca la creciente importancia de los **subagentes** como patrón de ingeniería para manejar tareas complejas y superar las limitaciones de contexto de los LLMs. La implementación de subagentes en **OpenAI Codex** (con "explorer", "worker" y "default" predefinidos, y la posibilidad de agentes personalizados) es un ejemplo de esta tendencia, siendo muy similar a la ya existente en **Claude Code**.

Además, se ha revelado cómo **VS Code** utiliza la IA en su propio flujo de desarrollo, incluyendo el **modo agente de GitHub Copilot**, para tareas como pruebas automatizadas y revisión de código, lo que subraya la madurez y utilidad de esta funcionalidad en entornos profesionales.

## ¿Cuál elegir según tu caso?

-   **Si eres principiante**:
    *   **GitHub Copilot**: Si ya usas VSCode o JetBrains, es una extensión familiar y potente, con un plan educativo gratuito.
    *   **Windsurf (Codeium)**: Ofrece una capa gratuita generosa y se integra como editor, facilitando el inicio con IA.
    *   **Cursor**: Un fork de VSCode con IA integrada, tiene una capa gratuita para empezar a explorar.
-   **Si eres developer profesional**:
    *   **Claude Code**: Para proyectos complejos, refactoring y debugging a gran escala, operando sobre la base de código completa.
    *   **Cursor**: Un IDE completo con capacidades de IA avanzadas, ideal para cambios masivos y soporte multi-archivo.
    *   **GitHub Copilot**: Si ya estás en el ecosistema Microsoft/GitHub, su modo agente es muy útil para integrar la IA en tu flujo de trabajo existente.
    *   **Aider**: Para quienes prefieren la terminal y un control granular, con la flexibilidad de usar diferentes LLMs.
-   **Si trabajas en equipo**:
    *   **Windsurf (Codeium)**: Diseñado para equipos, con una capa gratuita generosa que puede ser útil para presupuestos limitados.
    *   **GitHub Copilot**: Su integración con GitHub y su uso extendido lo hacen una opción sólida para equipos que ya usan la plataforma.
    *   **Cursor**: Como IDE profesional, ofrece las características necesarias para un desarrollo colaborativo asistido por IA.
-   **Si quieres privacidad / modelos locales**:
    *   **Continue**: Extensión open source que permite conectar con LLMs locales como Ollama, ofreciendo control total sobre tus datos y modelos.
    *   **Aider**: Al ser open source y compatible con Ollama, te da la flexibilidad de ejecutar modelos localmente y mantener la privacidad.
