# WiFi y Bluetooth LE se pueden usar simultáneamente en Arduino con NINA-W102

**Fecha:** 2026-03-08
**Categoría:** embebidos
**Dificultad:** Principiante
**Tags:** Arduino, ESP32, WiFi, Bluetooth LE
**Título original:** WiFi and Bluetooth LE can now be used simultaneously on Arduino boards with NINA-W102 (ESP32) module

---

## Resumen

Se ha resuelto un problema de años en los boards Arduino que utilizan el módulo ESP32, permitiendo ahora el uso simultáneo de WiFi y Bluetooth LE. Esto se logra gracias a una nueva versión del firmware y de las bibliotecas WiFi y BLE. Los usuarios de Arduino Nano RP2040 Connect, Arduino MKR WiFi 1010 y Arduino Nano 33 IoT pueden actualizar sus boards y disfrutar de esta mejora.

## Puntos clave

- Se ha resuelto un problema de años en los boards Arduino que utilizan el módulo ESP32
- Se permite el uso simultáneo de WiFi y Bluetooth LE
- Se requiere actualizar el firmware y las bibliotecas WiFi y BLE
